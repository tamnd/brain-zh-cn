辅音突变是辛达语语法的定义特征。 当你遇到一个以与其字典形式不同的辅音开头的单词时，这就是一个突变。 这不是错误，而是语法。 

托尔金写道：“辛达林语的变体或‘变异’是故意设计的，以使其在语音起源和语法使用上与威尔士语类似；但在语音起源或语法使用上却不相同。”

 ## 为什么存在突变

 在古辛达林语中，大多数单词以元音结尾（如昆雅语）。 当这些元音随着时间的推移而消失时，它们后面的辅音已经被周围的元音“削弱”了——而这种削弱被保留为语法特征。 一个单词的软突变本质上是它之前丢失的元音的化石回声。 

这与威尔士语发生突变的原因完全相同——托尔金故意植入的平行进化。 

## 5 种突变类型

 | 突变| 频率| 主要触发因素 |
 |----------|------------------------|-----------------|
 | **软（轻柔）** | 很常见| 名词后的形容词、直接宾语、许多介词、冠词 *i* |
 | **鼻** | 常见 | 复数冠词 *in*，介词 *an* (to/for) |
 | **混合** | 中等| 定语介词+单数名词 |
 | **停止** | 罕见/有争议| 跟随 *d* 或 *t* |
 | **液体** | 理论| 跟随 *l* 或 *r* |

 ---

 ## 软突变（Lenition）

 最重要的突变。 如果你只学一门，就学这一门。 

从历史上看，软突变源于元音之间的辅音。 清音停顿变成了浊音； 浊音停顿变成了上进心。 

### 软突变表

 | 声母| 软突变后| 国际音标更改 | 笔记|
 |--------------------|---------------------|------------|--------|
 | **p** | **b** | [p] → [b] | 清音 → 浊音 |
 | **t** | **d** | [t] → [d] | 清音 → 浊音 |
 | **c** | **克** | [k] → [g] | 清音 → 浊音 |
 | **b** | **v** | [b] → [v] | 浊停→浊思念|
 | **d** | **DH** | [d] → [ð] | 浊停→浊思念|
 | **克** | **（消失）** | [g] → ∅ | 完全删除|
 | **米** | **v** | [m] → [v] | 鼻音 → 浊思念 |
 | **h** | **ch** | [h] → [x] | 软化为软颚摩擦音 |
 | **s** | **h** | [s] → [h] | 精神力减弱|
 | **lh** | **l** | [ɬ] → [l] | 清音 → 浊音 |
 | **右** | **r** | [r̥] → [r] | 清音 → 浊音 |

 在软突变下**不突变**的字母：**n、l、r、f、v、dh、th**（已经是“软”）

 ### 工作示例：软突变

 | 基本形式| 软突变 | 背景 | 翻译 |
 |------------|-------------|---------|------------|
 | *加拉德*（树）| *'阿拉德* | *我'阿拉德* | 树（文章后 g 消失）|
 | *perian*（霍比特人）| *伯里安* | *我是贝里安* | 霍比特人 |
 | *卡兰*（日）| *加兰* | *我加兰* | 那天|
 | *贝莱格*（很棒）| *维莱格* | *加拉德·韦莱格* | 一棵伟大的树（形容词，在名词之后）|
 | *dûn*（西）| *杜恩* | *我是* | 西方|
 | *提力斯*（手表）| *迪里斯* | *塞宁·迪里斯* | 我看着（我看到了 watch-mut。） |
 | *明登*（塔）| *文登* | *我文登* | 塔 (m→v) |
 | *sûl*（风）| *呼尔* | *我呼* | 风 (s→h) |
 | *号角*（追逐）| *喇叭* | （h词：一些争议） | |

 ### 软突变的触发因素

 **1. 名词后面的形容词**（最常见）：
 - * adan *（男人）+ * beleg *（伟大）= * adan veleg *（一个伟大的人） - * beleg * → * veleg *

 **2. 动词后的直接宾语**：
 - *cenin galadh* （我看到一棵树） — 直接宾语 *galadh* 接受变法：*cenin 'aladh*
 - 在实践中，这一点存在争议； 许多文本显示了基本形式

 **3. 在定冠词 *i***（单数）之后：
 - *i 'aladh*（树）、*i berian*（霍比特人）、*i galan*（白天）

 **4. 介词之后**：ab、adel、am、dad、di、nu、trî、ú-
 - *ab galadh* → *ab 'aladh* （在树后面）
 - *ú-beleg* (不太好) — 前缀 ú- 触发 lenition

 **5. 在*na***（将/到）之后：
 - *na veleg*（很棒）

 **6。 在所有格和某些助词之后**

 ## 鼻腔突变

由前面的 *-n* 引起。 结果类似于爱尔兰的日食。 

### 鼻腔突变表

 | 声母| 鼻部突变后 | 笔记|
 |--------------------|----------------------|--------|
 | **p** | **b**（或**ph**）| 争议; b 在新辛达林语中很常见 |
 | **t** | **d** | |
 | **c** | **克** | |
 | **b** | **米** | |
 | **d** | **n** | |
 | **克** | **ng** [ŋg] | **ng** [ŋg] | |
 | **米** | **米** | 不变|
 | **s** | **h** | |
 | **妈** | **f**（不变）| |

 元音开头的单词：*in* 或 *an* 的前面的 *n* 可能会出现，使得突变在前面的单词上可见，而不是在后面的单词上：*in + edain* = *i Nedain*（但这有争议）。 

### 鼻突变的触发因素

 **1. 复数定冠词*in***（在辅音之前）：
 - *edain*（男性，复数）→ *i Nedain*（男性）——冠词*in*触发以下名词的鼻音突变

 **2. 介词 *an***（至、为、旨在）：
 - *an pengolodh degil*（给老师的一支钢笔） — *pengolodh* 来自 *tengolodh*； *degil*来自*tegil*
 - *t* → *d*（鼻突变）； *p* → 鼻音... = *an bengolodh* （实际上：p→b 位于鼻音下方 = *an bengolodh*）

 ### 示例：鼻腔突变

 *Annon a phengolodh degil* — “我给老师一支笔”
 - *pengolodh*（老师）：*an* → 鼻音突变后：p→b = *bengolodh*
 - *tegil*（笔）：直接宾语→软突变：t→d = *degil*
 - 完整：*annon an bengolodh degil*

 ## 混合突变

 出现在带有单数名词的定介词（介词+冠词融合）之后。 结果就像鼻突变和软突变的混合。 

### 混合突变表

 | 声母| 混合突变后|
 |--------------------|----------------------|
 | **p** | **b** |
 | **t** | **d** |
 | **c** | **克** |
 | **b** | **米** |
 | **d** | **n** |
 | **克** | **ng** |
 | **米** | **米** |
 | **s** | **h** |
 | **妈** | **v** |
 | **lh** | **l** |
 | **右** | **r** |

 ### 混合突变的触发因素

 单数名词前介词的限定版本 (介词 + *i* = "the")：
 - *o* (来自) + *i* = *o'n* 或类似→以下名词的混合突变
 - *tren*（通过）= *trî* + *i* → 混合突变

 示例：*o'n dûn*（来自西方）- *dûn* 混合突变后保持 *dûn*（d 在混合与鼻音中没有变化......这因分析而异）

 ---

 ## 停止突变

 出现在前面的 *d* 或 *t* 之后（作为前缀或介词）。 没有得到充分证实并且有些争议。 

### 停止突变表

 | 声母| 停止突变后|
 |--------------------|---------------------|
 | **p** | **b** |
 | **t** | **d** |
 | **c** | **克** |
 | **b** | **b** → **mb** （前缀可见）|
 | **d** | **d** → **nd** |
 | **克** | **g** → **ng** |
 | **米** | 仍然**米** |

 David Salo 将终止突变视为五种突变之一； 托尔金的证明有限。 

---

 ## 液体突变

 理论上是由前面的 *l* 或 *r* 引起的。 最弱证明的突变——可能存在，也可能不存在。 大多数新辛达语作家在写作时都会忽略它，因为它很少出现。 

如果存在：
 - **p** → **b**； **t** → **d**; **c** → **g** （类似于软突变）

 ## 常见错误

 1. **忘记 *g* 在软突变下消失** - *i galadh* 是错误的； 应该是 *i 'aladh* （撇号标记被删除 g）

 2. **混淆 b** 的软突变和鼻音突变 — *b* 的软突变 = **v**； *b* = **m** 鼻腔突变。 这些是不同的！ *i veleg*（伟大的，形容词用作名词 - b→v 软）与 *in Mellyn*（朋友们，鼻音来自 *Bellyn*？ - b→m 鼻音）

3. **将突变应用于一切**——突变是由特定的语法上下文触发的； 并非每个相邻的单词都会发生变异

 4. **忘记辅音声母与元音声母** — 突变仅影响*辅音*声母单词； 元音开头的词不会变异（但前面的助词可能会改变形式）
