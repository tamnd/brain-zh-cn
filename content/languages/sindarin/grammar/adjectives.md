辛达林语形容词具有三个不同于英语的关键属性：

 1. 它们出现在所修饰的名词**之后**（而不是之前）
 2. 他们在名词后进行**软突变**（放宽）
 3. 它们有**复数形式**（通过 i-affection 与复数名词一致）

 ## 位置：名词后形容词

 | 英语 | 辛达林语 | 笔记|
 |--------|----------|--------|
 | 一个伟大的人| *阿丹·维莱格* | *beleg*（伟大）→ *veleg* 在名词之后 |
 | 绿叶| *盖伦小姐* | *卡伦*（绿色）→ *盖伦* |
 | 一座白色的塔| *明登·蒂伦* | *thîren*（白色/亮色）如下 |
 | 美丽的精灵| *埃德尔徒劳* | *bain*（美丽）→ *虚荣* |

 ## Lenition：形容词的软突变

 当形容词跟在名词后面时，它的首辅音会发生**软突变**。 

| 形容词| 基本形式| 名词后 (Lenited) |
 |----------|----------|----------------------|
 | *贝莱格*（很棒）| 贝莱格| **v**腿 |
 | *贝恩*（美丽）| 贝恩| **v**ain |
 | *卡伦*（绿色）| 卡伦| **g**艾伦 |
 | *dûr*（深色）| 杜尔 | **dh**ûr | **dh**ûr |
 | *gorn*（石头；形容词，意义“石头”）| 戈尔恩 | **(∅)**orn（g 消失）|
 | *maen*（熟练）| 梅恩 | **v**aen |
 | *早上*（深色、黑色）| 早上| **v**orn |
 | *帕兰*（远方和远方）| 帕兰| **b**艾伦 |
 | *先生*（流动；形容词）| 先生| **h**îr |
 | *tîr*（观看）| tîr | **d**îr |

 ### 为什么形容词被限制

 历史原因：古辛达林语中的名词通常以元音结尾。 这些名词词尾元音后面的形容词处于“元音间”位置，导致我们在突变中看到辅音弱化（弱化）。 当名词的最后一个元音丢失时，突变仍然作为语法标记。

 ## 复数形容词

 当名词是复数时，形容词也必须是复数形式。 复数形容词由与名词相同的 i-affection 元音变化形成：

 | 调整。 （单数）| 调整。 （复数）| 元音变化 |
 |-----------------|----------------------------|--------------|
 | *贝莱格*（很棒）| *贝利格* | e→i |
 | *贝恩*（美丽）| *贝因* | ai → ei |
 | *卡伦*（绿色）| *塞林* | a→e, e→i |
 | *早上*（黑暗）| *迈恩* | o→y |
 | *dûr*（深色）| *杜尔* | 长 û→ui |
 | *maen*（熟练）| *分钟* | ae→î |

 ### 复数形容词示例

 | 名词短语（单数）| 名词短语（复数）| 翻译 |
 |------------------------|------------------------|--------------|
 | *阿丹·维莱格* | *埃丹·维利格* | 伟人/伟人|
 | *埃德尔徒劳* | *艾德希尔静脉* | 美丽的精灵/美丽的精灵|
 | *阿蒙·盖伦* | *埃明·杰林* | 绿色山丘/绿色山丘|

 在一个完整的名词短语中：名词是复数（带有i-affection），形容词也是复数形式，并且形容词仍然是宽松的。

 ## 谓语形容词：“To Be”

 辛达林语在简单语句中表达“X is Y”（系动词句子），没有可见动词：

 - *Beleg aran* — “国王是伟大的”（字面意思是“伟大的国王”）
 - *Bain i edhel* — “精灵很美丽”

 对于过去和将来系动词，使用动词 *na-* 或 *nar-*（“to be”）：

 - *Naun belegui* — “我很棒”（重建）
 - *Nartha beleg* — “他会很棒的”

 这是一个积极重建的领域——不同学者的形式有所不同。

 ## 比较级和最高级

 辛达林语没有经过充分验证的比较级/最高级系统。 新辛达林使用：

 - **比较级**（“more X”）：介词 *na* 或 *och* + 形容词，或复合形式
 - *beleg nach*（大于）——推测
 - 可以使用结构：*[adj] na [名词]*（X 是与/比名词相比的形容词）

 - **最高级**（“最 X”）：前缀 *er-* 或 *ar-*（最、最好）
 - *arveleg* = 最伟大的（推测的新辛达语）

 ## 副词

 辛达林语副词的构成：
 1. 来自没有变化的形容词（在副词位置作表语）
 2. 使用前缀 *al-*（not）表示否定：*alveleg*（notgreat）
 3. 一些独立副词：*sí*（现在，这里），*palan*（远和宽），*nef*（在这一边），*ab*（之后）

 副词通常位于它所修饰的动词之后：
 - *Tôl Aragorn vorn* — “Aragorn 黑暗/秘密地到来” — *vorn*（黑暗）作为副词（轻描淡写，因为它跟在动词后面 — 有争议）
