Tengwar 是托尔金辛达林语的主要文字。 有两种 Sindarin Tengwar 模式； 本页详细介绍了两者。 对于休闲学习者，请先掌握**标准模式**。 **贝烈瑞安德模式**（用于杜林之门）更漂亮，更容易阅读，但不太常见。 

## 辛达语 Tengwar 与昆雅语有何不同

 托尔金用昆雅语编写了大部分 Tengwar 样本，因此许多在线资源描述了昆雅语 Tengwar 规则。 Sindarin Tengwar 的工作方式有所不同：

 | 特色 | 昆雅腾瓦 | 辛达林标准模式 |
 |--------------------|----------------|------------------------|
 | **元音位置** | Tehtar 位于*前面*辅音之上 | Tehtar 上面*后面*辅音 |
 | **双辅音** | 写一次，下面有栏 | 相同 |
 | **尾元音** | 位于前一个辅音之上 | 高于无声运营商（*电信*）|
 | **模式类型** | Ómatehtar（元音 = 标记）| Ómatehtar（同类型）|
 | **词尾短元音** | 带有 tehta 的短载体 | 带有 tehta 的短载体 |

 这意味着，如果您看到带有辅音簇的辛达林语单词，则元音标记位于单词中元音**后面**的辅音上方，而不是位于元音之前**的辅音上方。

 ### 额外的辛达林腾格瓦

 辛达林标准模式使用的一些字母与昆雅语对应的字母不同：
 - *hwesta sindarinwa*（[x] 的特殊字母）是辛达语特有的 Tengwa，用于 *ch* 声音 [x] — 与昆雅语 *harma* 不同，后者的声音分布略有不同
 - *Anto*（在许多分析中通常为 *nd*）用于辛达林语 *dh* [ð]

 ## 标准模式下的元音符号 (*Tehtar*)

 | 元音| *特塔* | 安置|
 |--------|---------|------------|
 | 一个 | 三点（上）| 以上辅音 |
 | 电子| 急性中风| 以上辅音 |
 | 我| 单点 | 以上辅音 |
 | 哦| 右卷曲| 以上辅音 |
 | 你| 左卷曲| 以上辅音 |
 | y | 特别标记| 以上辅音 |

 **关键规则**：辛达林语中的*Tehtar* 位于**以下辅音上方**。 

如果元音位于**单词末尾**（后面没有辅音）：在**短无声载体**（*telco*）上方写下*tehta*。 

### 长元音

 长元音（标有锐音符：á、é、í、ó、ú）由以下符号表示：
 - 写下 *tehta*，并在其所在字母下方添加 ** 横线，或者
 - 使用 **长载体** (*anna*) 代替短载体

## 贝尔兰的模式：完整模式

 在贝烈瑞安德调式中，元音是**完整字母**（不是变音符号）。 这使得文本更容易阅读，但需要学习额外的字母分配。 

### 贝烈瑞安德模式的特殊规则

 1. **元音** 使用特定的 *tengwar* 作为完整字母：
 - *a* → Anna（特定的腾瓦）
 - *e* → 雁塔（变体）
 - *i* → 另一种形式
 - *o* → *óre* 变体
 - *u* → *úre*

 2. 该模式下，字母*óre*代表*n*，*vala*代表*m*（与标准模式不同）

 3. **鼻化辅音** (nd, mb, ng) 用基本字母+上面的横线书写

 4. **双元音** *ae* 和 *oe* 有时被写成两个单独的 tengwar

 ### 都林之门铭文

 贝尔兰模式最著名的例子。 铭文如下：```
Ennyn Durin Aran Moria: pedo mellon a minno.
Im Narvi hain echant: Celebrimbor o Eregion teithan i thiw hin.
```翻译：“莫瑞亚领主都林之门。说吧，朋友，然后进入。
 我，纳尔维，创造了它们。 埃瑞吉安的凯勒布林博绘制了这些标志。”

 这是托尔金以贝烈瑞安德腾瓦语体写成的最完整的辛达语示例。

 ## 初学者常见错误

 1. **使用昆雅语元音位置** — 将 tehtar 放在*前一个*辅音上方，而不是*后一个*辅音上方
 2. **使用错误的模式** — 标准模式和贝尔兰的模式有不同的字母分配
 3. **忘记词末元音的无声载体**
 4. **使用昆雅语分配 *anto* 和 *ampa*** — 在辛达林标准模式下，它们分别代表 *dh* 和 *v*，而不是昆雅语声音
