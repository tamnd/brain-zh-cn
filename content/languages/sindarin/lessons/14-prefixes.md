## 为什么前缀很重要

 辛达林语的很大一部分词汇是通过**复合和前缀**构建的。 与严重依赖后缀派生的语言不同，辛达林语前置其形态：含义经常添加到基本单词的开头，以创建具有专门或扩展含义的新单词。 

这对学习者有实际影响：
 1. 知道前缀可以让你解码不熟悉的名字和你从未见过的单词
 2. 托尔金中的许多著名名字 — *Anduin*、*Mirkwood* (Sindarin *Taur-nu-Fuin*)、*Fornost*、*Dúnedain* — 一旦您知道前缀，就可以轻松分析
 3. 新辛达语造词遵循相同的前缀模式，因此您可以高效地构建新词汇

 关键原则：**前缀修饰基本词的含义**； 基本词的初始辅音可能会也可能不会突变，具体取决于语音环境。 许多化合物在接缝处保持基本形状不变； 有些表现出软突变。

 ## 方向前缀

 这些前缀表示方向、位置或空间方向。 

| 前缀 | 意义| 关键例子| 分析|
 |--------|---------|-------------|---------|
 | *上午-* | 向上，之上，向上| *Amon*（山），*amdir*（希望，字面意思是“向上看的方向”）| *am-* + 方向名词 |
 | *爸爸-* | 向下，向下| *Dadbenn*（下坡，下坡）| *爸爸-* + *佩恩*（斜坡）|
 | *nan-* / *nand-* | 下，山谷| *Nan Elmoth*（埃尔莫斯山谷）、*Nanduhirion*（昏暗溪流山谷）| *nan-* 作为山谷前缀 |
 | *为-* / *为-* | 北 | *福诺斯特*（北部堡垒）、*福罗德*（北部）、*福罗切尔*（北部冰湾）| *for-* 在辅音之前，*forn-* 变体 |
 | *哈拉德-* | 南| *Harad*（南方），*Haradwaith*（南方民间，南部地区）| |
 | *dún-* | 西 | *Dúnedain*（西方人），*Dúnhirion*（西河之地）| *dún-* + 基础 |
 | *amrûn-* / *rhûn-* | 东| *Amrûn*（东方），*Rhûn*（东方）— *rhûn* 作为独立 | |
 | *帕兰-* | 远方，远方，深远| *palan-díriel*（远眺）、*palan-tîr*（远视）| *palan-* + 动词 |
 | *nef-* | 在 | 的这一边 *nef aear*（在海的这一边）| 用作前缀和介词 |
 | *三-* | 通过 | *trîwath*（理论），定向使用 | |

 ### 详细示例：指南针前缀

 **北（*为-*）：**
 - *Fornost* = *for-* + *nost*（堡垒/家园）→ 北部堡垒； 阿尔诺的首府
 - *Forod* = 北作为独立名词（可能 *for-* + *od* “远离”）
 - *Forochel* = 北+ *ochel*（冷？）→ 北方的冰湾

 **西 (*dún-*):**
 - *Dúnedain* = *dún-* + *edain*（男人）→ 西方男人； 努门诺尔人后裔
 - *Annûn* = *an-* + *nûn*（日落方向）→ 西； *Ennyn Durin Aran Moria：pedo mellon a minno。 我是纳尔维·海恩·埃坎特。 Celebrimbor o Eregion teithant i thiw hin.* — 莫瑞亚的西门在其他文献中是 *Ennyn Annûn*

 **南（*哈拉德-*）：**
 - *Harad* = 南方（简单词干）
 - *Haradwaith* = 南方 + *gwaith*（人民、民间）→ 南方人民； 哈拉德人
 - *Umbar* = 可能与阿杜奈克语中的南方有关，但在辛达语中的命名不同

 **东 (*rhûn-*):**
 - *Rhûn* = 东方（独立）
 - *Amrûn* = *am-* + *rûn* = 向东，东方方向

 ## 否定前缀

 这些前缀否定或私自修改基本词。 

| 前缀| 意义| 突变触发| 关键例子|
 |--------|---------|--------------------|-------------|
 | *ú-* | 没有，不，不| **软突变** | *ú-chebin*（我没有保留），*Úmarth*（厄运）|
 | *al-* / *ail-* | 不，没有（替代）| 软突变| 辛达林语的使用有限； *alfirin*（不朽：不朽）|
 | *u-* / *û-* | 邪恶，坏（坏的强化词）| — | *Ungoliant*（大蜘蛛）|

 ### 主要否定前缀：*ú-*

 *ú-* 是有效的辛达林语私有前缀。 它附加在名词、形容词和动词上，意思是“没有、缺乏、不-”：

| 基地| 前缀形式 | 意义|
 |------|--------------|---------|
 | *chebin*（我保留 - *hebin* 的软突变）| *ú-chebin* | 我不保留|
 | *amarth*（命运，厄运）| *Ú玛斯* | 厄运，厄运 (ú- + amarth) |
 | *加拉德*（树）| *ú-'aladh* | 无树 (ú- + Galadh → g 滴) |
 | *埃斯特*（希望）| *ú-埃斯特尔* | 绝望，没有希望|
 | *maer*（好）| *ú-maer* → *ú-vaer* | 不好，有病 (m→v) |

 已证实：*ú-chebin estel anim* =“我对自己不抱任何希望”——摘自托尔金在《中土世界人民》中的注释。 

### *al-* / *ail-*：形态学替代

 *al-* 以几种形式出现：
 - *alfirin*（不朽）= *al-* + *firin*（凡人）→“非凡人”——托尔金使用的花名
 - 该前缀与昆雅语 *il-* （否定前缀）同源
 - 经过验证的辛达林语效率低于 *ú-*

 ## 强化和增强前缀

 这些前缀放大或扩展了基数的含义。 

| 前缀| 意义| 关键例子|
 |--------|---------|-------------|
 | *一个-* | 长，伟大； 最（最高级）| *Anduin*（长河）、*Annûn*（西-远方向）、*andave*（伟大、长）|
 | *ar-* | 高贵、皇家、高贵| *阿拉贡*（高贵的国王？）、*阿尔温*（高贵的少女）、*阿尔诺*（国王的土地）|
 | *焦油-* | 高远、崇高、高贵| *Tarcil*（高人，努门诺尔人），*Tartir*（高级观察者）|
 | *更-* | 深色, 黑色 | *魔多*（黑土地）、*莫里亚*（黑坑/裂缝）、*魔古*（黑色魔法）|
 | *加尔-* | 光、闪耀、光辉| *Galadriel*（gal + riel = 光芒四射的花环少女），*Galadh*（树 - 向上生长/发光的东西）|
 | *格林-* / *格洛-* | 黄金，荣耀| *Glorfindel*（金色头发 — *glor-* + *findel*）、*Glorhíril*（理论）|
 | *马尔-* | 黄金（黄金）| *Malen*（黄色）、*Mallorn*（金树 — mal + lorn）、*Malvorn* |
 | *名人-* | 银| *Celeborn*（银树）、*Celebrant*（银道 — Silverlode 河）、*Celebrimbor*（银拳头）|

 ### 详细分析：*an-*（长，伟大）

 *an-* 效率极高。 它出现在：
 - *Anduin* = *an-* + *duin*（河）→ 长河/大河
 - *Annûn* = *an-* + *nûn*（向下，向西）→ 西方（太阳长期向下的方向）
 - *Anfauglith* = *an-* + *fauglith*（喘气的尘埃）→ 令人窒息的灰烬平原（毁灭后的阿德加伦）
 - *Anfalas* = *an-* + *falas*（海岸）→ 长海岸（沿海地区）
 - *andave* = *an-* + *dave* (时间，发生) → 大大，很多次
 - *Andrath* = *an-* + *rath*（道路，路线）→ 漫长的道路

 作为**最高级前缀**（新辛达林惯例）：*an-* + *beleg* → *ambeleg*（最大；注意*an-*中前面的鼻音*n*的鼻音突变b→m）。 

### 详细分析：*celeb-*（银）

 *celeb-* 是托尔金命名中最明显的前缀之一，出现在：
 - *Celeborn* = *celeb-* + *orn*（树）→ 银树； 白树祖先； 也是加拉德瑞尔丈夫的名字
 - *Celebrant* = *celeb-* + *rant*（路线，河床）→ Silverlode； 它的西语翻译是“Silverlode”
 - *Celebrimbor* = *celeb-* + *rimbor* （拳头，来自 *rimb* “拳头”）→ 银拳头； 诺尔多林铁匠
 - *Celebdil* = *celeb-* + *dil*（点，尖峰）→ Silvertine； 莫里亚三峰之一

 ###详细分析：*mor-*（深色、黑色）

 *mor-* 是广泛出现在地名中的词根形式前缀：
 - *Mordor* = *mor-* + *dor*（土地）→ 黑土地； 索伦的领域
 - *Moria* = *mor-* + *ia*（虚空、深坑/裂缝）→ 黑色裂缝； 德瓦罗德尔夫
 - *Morgoth* = *mor-* + *goth*（可怕的敌人）→ 黑暗敌人（昆雅语形式；辛达语是平行的）
 - *Morgul* = *mor-* + *gûl*（巫术，幽灵魔法）→ 黑色巫术； 如*米那斯魔窟*
 - *Morannon* = *mor-* + *annon*（大门）→ 魔多黑门

 ## *go-* / *gwa-* 前缀：团结

 这个前缀值得特别关注，因为它在最著名的辛达林短语之一中扮演着重要角色。

| 变体 | 用途 | 示例 |
 |--------|--------|---------|
 | *去-* | 辅音前 | *govad-*（见面、去 + pad）|
 | *格瓦-* | 元音前 | *Gwaur*（污染，gwa + ? 根）|
 | *格瓦-* | 在化合物中| *Gwaith*（人员、公司 — gwa + ？）|

 ***govad-* 和 *Mae govannen* 的词源：**
 - *govad-* = *go-* + *pâd-*（行走、步骤）→“一起行走”= 见面
 - 过去分词是*govannen*（相遇，一起相遇）
 - *Mae govannen!* = “很高兴见到！” — *mae*（井）+ *govannen*（*govad-* 的过去分词）
 - *Mae* 与 *maer* 同根（好）
 - 这种问候语在《指环王》中得到了证明——字面意思是“走得很好！”

 其他 *go-* / *gwa-* 编队：
 - *Gwaith-i-Mírdain* = *gwaith*（公司，民间）+ *i*（the）+ *mírdain*（珠宝史密斯）→ 珠宝史密斯公司
 - *goheno nin* =“原谅我” — *go-* + *heno*（赦免/释放）+ *nin*（我）
 - *gohenam* —“我们宽恕”（互为：宽恕中的“我们一起行走”）

 ## 静态和结构前缀

 | 前缀| 意义| 关键例子|
 |--------|---------|-------------|
 | *去-/gwa-* | 一起，共同 | 见上文 |
 | *阿特拉-/塔尔-* | 横跨，横跨，超过| *athrad*（福特：交叉点；*athra-* + *râd* 道路/道路）|
 | *二-* | 之下，之下（静态）| *Dínen*（沉默 —“沉默之下”）、*Díriel*（凝视 — *di-* + *riel*？）|
 | *zh-* | 的，重新（增强器）| *Ennorath*（中土世界：*en-* + *norath* 土地）、*ennyn*（门：*annon* 的复数形式）|
 | *埃雷布-* | 孤立的，孤独的| *Erebor*（孤独山）、*Eregion*（冬青之地）|
 | *ost-* | 堡垒、城市| *奥斯吉利亚*（*ost* + *吉利亚*“星星”→ 星星堡垒），*奥斯提里安*（塔城）|

 ### 详细：*en-*（中/中）

 *En-* 在复合词中显示为所有格“of the”，并作为方格“中间”前缀：
 - *Ennorath* = *en-*（中间）+ *norath*（复数土地）→ 中部土地； 中土世界（陆地）
 - *Ennyn Durin* = *ennyn*（门，门 — *annon* 的复数形式，带有 *en-* 前缀？） → 都林之门
 - *Ered Engrin* = 铁山 — *engrin* 来自 *en-* + *grin* （铁，复数）？ 

### 详细：*athra-*（交叉）

 - *Athrad* = *athra-* + *râd*（路径、方式）→ 浅滩（十字路口）； 在*Sarn Athrad*（石头福特）证明
 - *Athelas* = *athra-* + *lass*（叶）→ 跨叶？ 治疗药草； 该前缀可能表明其具有深远的治疗作用

 ## 构建新单词：新辛达语前缀

 使用已建立的前缀模式的合法新辛达语构词的五个示例：

1. **“北方人”** = *for-* + *dîr*（男人）→ *Fordîr*（北方人）； 注意：*Dúnedain* 是“西方人”的既定术语，因此“北方人”遵循相同的 *direction + edain/dîr* 模式

 2. **“远见”** = *palan-* + *tîr*（手表）→ *Palantír* — 这已得到实际证明！ Palantíri（视明石）=“远视石”

 3. **“银手”** = *celeb-* + *cam*（手）→ *Celebcam* — 与 *Celebrimbor*（银拳）类比； 托尔金本人用*Celebrindal*（银脚）来代替伊缀尔

 4. **“Black-gate”** = *mor-* + *annon*（门）→ *Morannon* — 这被证明是魔多黑门的名称

 5. **“Long-road”** = *an-* + *rath*（道路，路线）→ *Andrath* — 被证明是一个地名（卡多兰语中的长狭道）

 这些例子表明前缀系统不是任意的——它遵循一致的规则，托尔金本人也反复使用这些模式，使它们对于学习者和新辛达林作曲家来说是可靠的。
