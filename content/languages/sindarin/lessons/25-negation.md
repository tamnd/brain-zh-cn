## 简介

 每种语言都需要有表达“不”、“不”和“不要”的方式。 辛达林语有一套丰富多样的否定策略，每种策略都用于不同的语法环境。 至关重要的是，其中一些否定形式在托尔金的文本中得到了**直接证明**——这意味着我们有实际的、已出版的辛达语否定句可供研究。 

本课程涵盖辛达林语的所有六种主要否定策略：

 1. **ú-** — 否定动词并形成否定复合词的前缀
 2. **avo** — 否定祈使助词（“don't！”）
 3. **baw** — 强烈禁止（“不！绝对不！”）
 4. **法律** — 简单的“不”答复
 5. **pen** — 复合词中的宾格“没有，缺乏”
 6. **ú-boe** — 否定的必然性

 ## 2. 否定前缀 *ú-*

 最通用的辛达林语否定是前缀 **ú-**。 它直接附加到动词（或名词/形容词）上以否定它。 

### *ú-* 触发软突变

 当 *ú-* 前缀于以可变辅音开头的单词时，它会触发该辅音上的**软突变**：

 | 原创| 在 *ú-* | 之后 突变规则|
 |----------|------------|--------------|
 | *提里奥* | *ú-硫磷* | t → th |
 | *别狗* | *ú-vedog* | b → v |
 | *加拉德* | *ú-'aladh* | g→'（零）|
 | *达斡尔族* | *ú-Dhaur* | D → Dh |
 | *希伯来语-* | *ú-cheb-*（见下文）| h → ch |

 > **重要警告：** *ú-* 的突变行为在辛达语学者中存在争议。 一些消息来源表明 *ú-* 会导致明显的“负突变”，而不是标准的软突变。 来自托尔金证实的 *ú-chebin*（见下文）的证据支持 *ú-* 影响后面的辅音，但确切的系统需要仔细阅读每个证实的形式。 

### 已证明：*ú-chebin estel anim*

 所有辛达林语中最著名的经证实的否定来自 **Gilraen 的 *linnod***（预言对句），可在《国王归来》的附录 A 中找到：

 > **Onen i-Estel Edain，ú-chebin estel anim**

 **全文及翻译：**
 - *Onen* =“我给予”——过去 1sg 的 *anna-*（给予）； 见第 22 课
 - *i-Estel* =“希望” — *i*（冠词）+ *Estel*（希望、信任），带有特殊的连字符冠词形式
 - *Edain* =“致杜内丹人”——*Adan* 的与格复数形式（三个家族的人）； 软突变 *A-* → 不变，但意思是“给男人”
 - *ú-chebin* = "我没有保留" — *ú-*（否定）+ *chebin*（我保留了 *heb-* 的过去现在形式）； 注意 *h* → *ch* 在 *ú-* 之后
 - *estel* = "hope" — 普通名词（小写），与名称 *Estel* 相反
 - *anim* = “为了我自己” — *an* (为了，为了) + *im* (我自己，反射 1sg)

 **翻译：** “我给了杜内丹人希望，但我自己没有保留任何希望。”

 **语法注释：**
 - *chebin* 是一种**过去现在**（完美式）形式：“我已经保留”——它表达了由过去的行为导致的状态。 这是一种罕见但重要的辛达语动词形式。 
- *Estel*（名字、礼物）与 *estel*（普通名词）的对比令人心酸——吉尔蕾恩放弃了自己的希望（化身为她的儿子阿拉贡，其昵称是 *Estel*），一无所有。 
- *ú-chebin*: *ú-* + *chebin* — *heb-* 的 *h*（保留）在 *ú-* 的影响下变成了 *ch*。 这种否定下的特定突变 (*h → ch*) 是一个显着特征。 

这句“linnod”是托尔金所有作品中最感人的句子之一。 记住它。 

### 更多 *ú-* 示例

 **否定动词：**

| 积极| 负面| 翻译 |
 |---------|---------|-------------|
 | *tiron*（我看）| *u-硫隆* | “我不看”|
 | *cenin*（我明白了）| *乌-白诗南* | “我没看到”|
 | *林农*（我唱歌）| *ú-林农* | “我不会唱歌”|
 | *托伦*（我来了）| *ú-托伦* | “我没来”|
 | *伊斯顿*（我知道）| *ú-伊斯顿* | “我不知道” |
 | *garnen*（我做的）| *ú-'arnen* | “我没有做”（g→'零）|

 **否定名词（私有）：**

 *ú-* 可以在名词前面形成否定复合词，意思是“un-X”或“ill-X”：

 | 复合| 意义| 笔记|
 |--------|---------|--------|
 | *乌玛斯* | “厄运，厄运”| *ú-* + *amarth*（命运）|
 | *乌加斯* | “恶行、侵犯、罪恶”| *ú-* + *garth*（契约？）|
 | *乌拉特* | “邪恶行为”| *ú-* + *rath*（当然，行动）|
 | *乌本* | “没有人”| *ú-* + *笔*（人）|

 **否定形容词：**

 | 复合| 意义|
 |---------|---------|
 | *乌瓦尔* | “不好，不好” (*ú-* + *maer* → *vaer*) |
 | *乌加伦* | “不绿，枯萎” (*ú-* + *calen* → *galen*) |

 ## 3. 否定命令 *avo*

 当您想用辛达林语说 **“不要做 X！”** 时，请勿使用 *ú-* + 命令式。 相反，您可以使用专用的否定命令助词 **avo**。 

### *Avo* + 软突变

 *Avo* 后跟 **动词的不定式/祈使词干**，具有 **软突变**：

 | 积极的命令| 否定命令| 翻译 |
 |--------------------|--------------------|-------------|
 | *蒂罗！*（观看！）| *Avo dhíro！* | “别看！” (t → dh?) |
 | *Garo！*（做/拥有它！）| *阿沃加罗！* | “别这么做！” |
 | *贝多！*（说话！）| *Avo vedo！* | “别说话！” (b → v) |
 | *恋童癖！*（说！） | *万岁！* | “别说！” (p→b)|

 > ***avo* 的证明：** 托尔金在他的语言笔记中证明 *avo* 是否定命令助词。 它触发的确切突变一直存在争议，但 *avo* 之后的软突变是共识。 

**句子示例：**

 - *Avo garo!* =“别这么做！” ——最常被引用的证明形式
 - *Avo vedo i thiw!* = “不要说手势！” （*bedo* = 说话；*b→v*；*thiw* = *têw* 的复数，符号）
 - *Avo menno!* = “别走！” （*menno* = go；初始 *m* 在软突变下不会突变 — *m* 没有软突变）
 - *Avo tiriad i varad!* =“不要看塔！” （在 *avo* 之后使用动词名词 *tiriad*）

 ### *Avo* 与命令式变形

 一些学者重构 *avo* 与动词的命令形式 (2sg) 结合，而另一些学者则认为它采用不定式/动名词形式。 最安全的方法：使用 *avo* + 动词词干以及任何适用的软突变。

 ## 4. 强烈禁止 *baw*

 **Baw** 是一个强调的“不！” 或“不要！” 表示强烈反对、厌恶或绝对禁止。 它比*avo*更强烈、更情绪化。 

### 证明

 *Baw* 在托尔金的语言笔记中被证明是辛达语感叹词/禁止词。 它似乎源自一个词根，意思是“拒绝、禁止”。 

### 用法

 *鲍*可以使用：
 1. **Alone** 作为感叹词：*Baw!* = “不！绝对不！别这样！”
 2. **动词前**表示强烈的否定命令：*Baw cerio!* = "Don’t run! (emphatic)"
 3. **回应**被拒绝的建议：*Baw, ú-boe* =“不，没有必要”

 ### *Baw* vs. *Avo* vs. *Ú-*

 | 表格 | 实力| 背景 |
 |------|----------|---------|
 | *ú-* | 中性| 陈述：“我不/他不” |
 | *阿沃* | 温和的命令| “请不要/我请求你不要” |
 | *咩* | 强烈的命令/感叹| “绝对不要！不！” |

 比较英语：“I don't see”(*ú-*) 与“Don't Look!” (*avo*) vs.“停下！不！” （*咩*）。

 ## 5. 简单否定答复*法则*

 **法律**（有时也*lav*）用作简单的**“否”**来回答是/否问题。 它并不否定特定的动词； 它对整个问题的回答是否定的。

| 情况| 辛达林语 | 英语 |
 |------------|---------|---------|
 | 问：*Ma cenig i aran？*（你看到国王了吗？）| *法律* | “不（我不）” |
 | 问：*Ma tolen？*（我可以来吗？） | *法律，ú-boe* | “不，没有必要”|
 | 强烈拒绝| *哇！* | “不！（强调）” |

 **证明：** *法律*在托尔金的语言笔记中被证明为辛达语否定答复粒子。 

### 正面回复：*Mae*

 为了完整起见：辛达林语中肯定的回答“是”是 **mae** （在某些来源中也是 *ôl*，但 *mae* 更常被引用）：
 - *Mae* =“是的，这很好/确实”
 - 不要与 *Mae govannen* 中的 *mae*（好）相混淆（很高兴！）

 ## 6. 私有前缀 *pen*

 **pen** 元素的功能是作为一个私有语：“没有、缺乏、没有 X”。 它主要用于**复合词**和复合名称。 

### *笔* + 软突变

 *Pen* 触发以下元素的软突变（就像大多数前缀一样）：

 | 复合| 意义| 分析|
 |---------|---------|---------|
 | *彭尼诺* | “没有一岁，尚未成年”| *笔* + *尼诺*？ = 没有年份 |
 | *Pen-i-bhûth* | “没有誓言”（假设）| *笔* + *i* + *pûth* → *bhûth* |
 | *压抑*？ | “没有牙齿”？ | ᴺS。 |

 更清楚地证明了相关形式**pín**（小，小——也许“缺乏尺寸”）和*Pinnath Gelin*（绿色山脊——*pinnath*=山脊，*pen*复合物？）等名称。 

### *Pen* 作为代词/名词

 另外，*pen* 在辛达林语中也意味着 **“一个，一个人”**（不定代词）：
 - *Pen tiritha* =“有人会观看”/“观看的人”
 - 这与 *úben*（无人）相反 — *ú-* + *pen*

 > **注意：** *Pen* 的意思是“人/一个”，而 *pen* 的意思是“没有”，尽管其起源相关，但它们是不同的用法。

 ## 7. 否定的必然性：*ú-boe*

 正如第 21 课中所述，*boe* 的意思是“这是必要的”。 它的否定 **ú-boe** 意思是“没有必要”——并且起到标准 *ú-* 否定的作用：

 | 辛达林语 | 英语 |
 |----------|---------|
 | *Boe ammen tiro* | “我们必须观察” |
 | *Ú-boe ammen tiro* | “我们不需要看/我们没有必要看”|
 | *Ú-boe i aran* | “国王不需要（这样做）” |

 **证明：** *Boe* 已得到证明； *ú-boe* 是使用相同 *ú-* 前缀的直接否定。 

---

 ## 8. 复合名词和地名中的 *Ú-*

 否定前缀 *ú-* 出现在托尔金给出的几个名字和单词中，为我们提供了其使用的额外证据：

 | 词| 分析| 意义|
 |------|---------|---------|
 | *Ú玛斯* | *ú-* + *amarth*（命运）| “厄运，厄运”|
 | *乌戈立安特* | *ú-* + *ngoliant*（蜘蛛网？）| 大蜘蛛； *ú-* 赋予邪恶的含义 |
 | *乌鲁伊* | *ú-* + *rui*？ 或者只是*ûr*（热）+*ui*？ | “八月”（炎热的月份） — *rui* 形式不确定 |

 **名称*Ungoliant*分析：**虽然传统上分析为昆雅语/Valarin，但*un-*元素可能与辛达林语*ú-*相同的负根相关。 原负根 *√UGU/UMU* 是昆雅语 *um-* 和辛达林语 *ú-* 的基础。

 ## 9. 复杂句子中的否定

 否定以重要的方式与复杂的句子结构（第 24 课）相互作用：

 ### 否定从句

 - *Tolen, an ú-thiron* = “我来，因为我不看”（因果从句否定）
 - *Ú-iston ma tolen* =“我不知道我是否会来”（主句否定）
 - *I aran i ú-thirant* =“没有观看的国王”（关系从句否定）

 ### 双重否定

 辛达林语不使用双重否定来表达肯定（与某些自然语言不同）。 *Ú-* 每个否定元素使用一次：
 - *Ú-thiron, ú-linnon* =“我不看，我不唱歌”（两个单独的否定）

 ## 11.练习

 **练习 1：** 使用 *ú-* 否定这些动词（在需要时应用软突变）：

 1.*menon*（我去）
 2. *ceritham*（我们会做）
 3.*tolon*（我来了）
 4. *bêd*（他说）

 **练习 2：** 使用 *avo* 将这些积极命令转换为消极命令：

 1. *蒂罗！*（观看！）
 2.*琳娜！*（唱歌！）
 3. *美诺！*（走！）
 4. *Pedo！*（说话！）

 **练习 3：** 为每个上下文选择正确的否定形式（*ú-*、*avo* 或 *baw*）：

 1. 礼貌地告诉朋友不要跑步：___
 2. 声称你没有看到国王：___
 3.大喊“绝对不！” 惊恐地：___
 4. 说“没必要”去：___

 **练习 4：** 从记忆中翻译 Gilraen 的 *linnod*（或重建它）：
 - “我给了杜内丹人希望，但我自己没有保留任何希望。”

 **练习 5：** 否定这些复合名词：
 1. *amarth*（命运）→ 厄运
 2. *maer*（好）→ 坏/邪恶

 ### 答案要点

 1.
 - *ú-venon* （*m* → *v* 在软突变下？实际上：*m* 在标准软突变表中不 lenite；所以 *ú-menon*）
 - *ú-cheritham* （软突变下的 *c* → *ch*？软突变：*c* → *g*，而不是 *ch*。所以：*ú-geritham*）
 - *ú-tholon* (*t* → *th*: 正确的软突变)
 - *ú-vêd* (*b* → *v*: 正确的软突变)

 > 答案注释：软突变：p→b、t→d、c→g、b→v、d→dh、g→'（零）、m→v、s→h。 所以： *menon* → *ú-menon* (m→v: *ú-venon*); *ceritham* → *ú-geritham* (c→g); *托隆* → *ú-多隆* (t→d); *bêd* → *ú-vêd* (b→v)。 

2.
 - *Avo dhíro!* （或 *Avo dhiro!*） — t→dh？ 软突变：t→d，所以*Avo diro！*
 - *Avo linna!* — l 不会变异
 - *Avo veno!* — m→v
 - *Avo bedo!* — p→b

 3.
 1. *Avo*（礼貌祈使否定）
 2. *ú-*（陈述否定）
 3. *Baw!*（强烈禁止/感叹词）
 4. *ú-boe*（否定必然性）

 4. *Onen i-Estel Edain，ú-chebin estel anim。*
 （“我给了杜内丹人希望，但我自己没有保留任何希望。”）

 5.
 - *úmarth* = 厄运 (*ú-* + *amarth*)
 - *úvaer* = 不好（*ú-* + *maer* → *vaer* 软突变后）
