## 两种伟大的精灵语言

 托尔金设计了两种主要的精灵语言——辛达林语（灰精灵语）和昆雅语（高精灵语）——就像历史语言学家对待真实语系一样。 它们之间的联系就像拉丁语与威尔士语的关系一样：都是印欧语系，但由于长期分歧，彼此无法理解，具有不同的语音系统、不同的语法结构和截然不同的美学。 

对于任何认真学习托尔金精灵语的学生来说，理解两种语言并清楚地区分它们是最重要的技能之一。 托尔金最著名的许多“精灵语”短语都是昆雅语，而不是辛达语——混淆它们就像混淆拉丁语和意大利语一样。 

---

 ## 历史关系

 ### 家谱

 这两种语言都源自**原始昆迪亚**（精灵的第一种语言）和**通用埃尔达语**（前往维林诺的埃尔达语言的共同祖先）。 

分歧点：
 - **Teleri**（那些最后出现的人，其中一些作为辛达人留在中土）通过与中土、矮人的接触以及他们自身数千年的进化发展了辛达语
 - **凡雅人和诺多**（最早到达维林诺并留下来的人）保持他们的语言更加保守，在维拉的影响下发展了昆雅语

 到了第三纪元：
 - **昆雅语** = 相当于拉丁语或古典希腊语 — 仪式性的、古老的、有声望的、用于传说和仪式； 日常使用中“死”的语言，但为学者所熟知
 - **辛达林语** = 相当于生活白话 — 类似威尔士语，日常使用，中土精灵和杜内丹人的通用语言

 当诺多族回到第一纪元的中土世界时，发现辛达人已经在那里讲辛达语了。 出于尊重（也考虑到弑亲事件周围的政治局势），他们采用辛达林语作为日常用语。 得知亲族屠杀事件后，多瑞亚斯国王辛葛禁止了昆雅语。 到了第三纪元，就连瑞文戴尔和洛斯罗林的诺多族也每天都说辛达语。

 ## 识别：区分辛达语和昆雅语

 | 特色 | 辛达林语 | 昆雅语 |

 | 有 TH [θ]、DH [ð]、LH [ɬ]、RH [r̥] | 是的——很常见 | 罕见到不存在 |
 | 有 QU [kw]、NY [nj]、LY [lj]、TY [tj] | 罕见| 共同点—特点|
 | 有元音 Y [y]（前圆角）| 是的——独特| 没有 |
 | 有双元音 AE [aɛ] | 是的 — 常见 | 否（使用 AI 代替）|
 | 单词经常以辅音结尾 | 是的 | 很少——大多数单词以元音结尾 |
 | 定冠词 (the) | *i* (sg), *in* (pl) | *i* (sg), *in* (pl) | 无 — 昆雅语没有文章 |
 | 突变系统| 是的 — 5 种类型，主要特点 | 最小——无系统突变|
 | 词序| VSO（默认）| SOV（默认）|
 | 形容词位置| 名词 | 后 名词前 |
 | 案件结局 | 否 - 使用介词 | 是的 — 5 个以上案例 |
 | 复数形式 | 内部元音变化（i-情感）| 外部后缀（-r、-i、-ë）|
 | 属格结构| *en* + 软突变 | 大小写后缀 *-o* 或 *-va* |

 ### 快速识别测试

 如果是一个短语或单词：
 - **以元音结尾** → 可能是昆雅语（但并非总是如此）
 - **包含 DH 或 LH** → 几乎肯定是辛达林
 - **包含 NY 或 TY** → 可能是昆雅语
 - **Y 作为单词中间的元音** → 可能是辛达林语
 - **有系统突变（初始辅音变化）** → 辛达林语
 - **名词后有格结尾** → 昆雅语

 ### *kw→p* 转变——辛达林语最戏剧性的声音变化

 辛达林语与昆雅语的最显着特征之一是辛达林语对普通精灵语 *kw-*（和 *tw-、pw-*）的处理。 昆雅语保留 *qu-* [kw]，辛达林语则移动 *kw→p*。 这就是为什么：
 - CE *kwet-*（说话）→ Q *quet-*，S *ped-*
 - CE *kwenê*（精灵，会说话的人）→ Q *Quendi*、S *Pent* 或 *Penni*？ 

这种转变也是托尔金写道昆雅语对于某些辛达语人来说“难以发音”的原因——不同的语音习惯。

 ## 语法比较

 ### 主要结构差异

| 特色 | 辛达林语 | 昆雅语 |
 |---------|----------|--------|
 | **复数** | 内部元音变化（*i-affection*）：*aran→erain* | 外部后缀：*elen→eleni* 或 *eldar→eldari* |
 | **案例系统** | None — 对所有关系使用介词 | 5+ 格：主格、宾格、属格、与格、与格、方格 |
 | **词序** | VSO（句子中动词在前）| SOV（动词在句子中的最后）|
 | **明确的文章** | *i* (sg)、*in* (pl) — 触发突变 | 无 — 没有昆雅语文章 |
 | **动词一致** | 动词人称后缀| 人称后缀+一些代词主语|
 | **突变系统** | 5种类型：柔软、鼻音、混合、液体、停止| 最小/不存在——没有系统的首辅音突变|
 | **形容词位置** | 名词后 (*aran veleg* = king Great) | 在名词之前 (*alta aran* = 伟大的国王) |
 | **属格** | 介词 *en* + 软突变 | 大小写后缀 *-o*（短所有格）或 *-va*（形容词所有格）|
 | **系动词“成为”** | *na-*，不规则； 经常被省略| *ná*，更常规 |
 | **动词时态** | 现在时、过去时（牙科后缀 *-ant-*）、一些过去被动语 | 更精细的时态系统 |

 ### 文章：关键诊断

 定冠词 (*i*, *in*) 的存在是辛达林语的直接标记。 昆雅语没有文章。 所以：
 - *i aran* = 国王（辛达语）
 - *aran* = 国王或国王（昆雅语 — 无冠词）

 由于昆雅语缺少冠词，而辛达林语的冠词会引发突变，因此几乎任何带有明确名词的辛达林语句子都会在名词上显示突变的首辅音。 这是一个实用的快速识别测试。

 ## 何时使用哪种语言

 ### 文化和历史背景

 **使用辛达林语用于：**
 - 中土世界（第三纪元）中的任何日常演讲
 - 向瑞文戴尔、洛斯罗林、幽暗密林、灰港的精灵致辞
 - 与杜内丹人（北方游骑兵，高血统的刚铎人）交谈
 - 与居住在中土世界的任何精灵进行实际交流
 - 讲辛达林语的精灵所起的个人名字

 **使用昆雅语用于：**
 - 正式仪式、宣誓和崇高仪式
 - 向维拉讲话或祈祷
 - 精灵传说诗歌（如 *Namárië* 挽歌）
 - 努门诺尔人的记录和铭文
 - 古老的精灵文物的名称
 - 托尔金本人使用“高等精灵”语域的任何上下文

 **与中世纪欧洲的类比：**
 受过教育的中世纪欧洲人既懂得拉丁语（学习、教会和仪式的语言），也懂得他们自己的方言（法语、英语、德语）。 他们不会使用拉丁语来购买面包，但会在弥撒或法律文件中使用它。 同样，受过教育的精灵或杜内丹人懂得辛达林语（日常）和昆雅语（仪式），但不会在早餐时使用昆雅语。

 ## 错误同源词和混淆点

 以下词语或情况让许多学习者感到困惑：

| 陷阱| 说明|
 |------|-------------|
 | *le* = 两者中的“你”| S. 和 Q. 都使用 *le* 表示“you/thee”，但句法作用和形式不同； 在昆雅语中，*le* 是一种礼貌/尊重的形式； 在辛达林语中，它是标准的第二人称代词 |
 | *ar* 名称与昆雅语 *ar* | 在昆雅语中，*ar* =“和”； 辛达林语 *ar-* 中的名称 = "贵族/皇家前缀" (*Aragorn, Arwen*); 但辛达林语“and” = *a*（不是 *ar*）|
 | *aran* = 两者皆为国王 | 保留相同的单词之一； 上下文（短语其余部分的语言）决定您使用哪种语言 |
 | *梅内尔* = 两者中的天空| 另一种保存形式； 两种语言都使用 *menel* 来表示天空/苍穹 |
 | *lómë* = 夜晚（仅 Q.）| 昆雅语中“夜晚”的意思； 辛达林语使用*fuin*（黑暗）或*dû*（黄昏）； 切勿将 *lómë* 用作辛达林语 |
 | *Namárië* = 昆雅语 | 加拉德瑞尔在离开洛斯罗瑞恩时所唱的著名告别挽歌完全是昆雅语，而不是辛达林语； 辛达语告别 = *纳瓦尔* |
 | *Elen síla...* = 昆雅语 | 佛罗多的问候语（“一颗星星在我们相遇的时刻闪耀”）是奎尼亚语（QUENYA），是比尔博/甘道夫教给他的； 这不是辛达林语 |
 | *Elbereth...* = 辛达语 | 星之赞歌（由夏尔和瑞文戴尔的精灵演唱）是辛达林语； 这是辛达林语的一个常见使用示例 |

 ### 关于 *Namárië* 最重要的三件事

 1. 它是昆雅语，而不是辛达林语——其中的每个词都是昆雅语。 
2. 这是托尔金所证实的最长的任何语言的连续精灵语文本。 
3. *Namárië* 本身的意思是“一切安好”（*namba-* 一切顺利 + *-rë* 可选）——即“再见”即“愿与你同在”。 

辛达林语的告别词是 *Navaer* — *na-* (be) + *vaer*（*maer* 的良好软突变）——语义内容相同，但发音完全不同，因为数千年的分歧。

 ## 阅读托尔金的双语段落

 托尔金经常在同一场景中混合使用辛达语和昆雅语。 识别哪种语言是哪种语言至关重要：

 | 文字| 语言 | 关键标记|
 |------|----------|------------|
 | *Elbereth Gilthoniel / silivren penna míriel...* | 辛达林语 | *A*（呼格助词）、*penna*（向下倾斜）、*míriel*（*-iel* 阴性后缀）|
 | *Namárië/哎！ laurië lantar lassi súrinen...* | 昆雅语 | 平滑的元音重线，*-inen* 方位后缀，*lantar*（fall，复数）|
 | *Elen síla lúmenn' omentielvo* | 昆雅语 | *-elvo* 双属格后缀； *lúmenn'* = *lúmenna*（按时间，代词）|
 | 戒指铭文：*Ash nazg durbatulûk...* | 黑色言论| 根本不是精灵语； 刺耳的、反辅音的、陌生的结构|
 | *Cuio i Pheriannath anann！* | 辛达林语 | *i Pheriannath* = 带有文章 + 突变的霍比特人 |
 | *佩多·梅隆和米诺人* | 辛达林语 | 冠词 *a* =“and”（辛达林语，而不是昆雅语 *ar*）； *minno* 中 *m* 的突变 |

 ### 诺多族的精灵双语

 第一纪元返回中土的诺多族的母语是昆雅语，但后来采用了辛达语。 到了第三纪元，埃尔隆德、加拉德瑞尔和格洛芬德尔都可以说双语了。 当他们使用昆雅语时，总是出于特定的原因：仪式、诗歌、传说或情感宣扬。 

因此，在远征队离开洛斯罗瑞恩时，加拉德瑞尔选择用昆雅语演唱“Namárië”是有意义的：她用记忆和不朽之地的语言来歌唱，因为她就像一个见过维林诺并知道自己会回到那里的人一样说话。 辛达语的告别*纳瓦尔*不足以弥补那一刻的沉重。

 ## 练习：识别语言

 将以下各项分类为辛达林语 (S.)、昆雅语 (Q.)、黑人语言 (BS.) 或其他/未知 (?)：

1. *Namárië!*（加拉德瑞尔的告别歌曲开场）
 2.*达罗！ Lasto beth nîn!*（洛斯罗林指挥部边界）
 3. *Ash nazg durbatulûk*（戒指铭文）
 4. *Elbereth Gilthoniel*（山姆的哭声和精灵赞美诗）
 5. *Elen síla lúmenn' omentielvo*（佛罗多的问候）
 6. *Cuio i Pheriannath anann!*（在 Cormallen 干杯）
 7. *Guren bêd enni*（我的心告诉我）
 8.*哎呀！ 人工智能！ 埃尔伯雷斯·吉尔托尼尔！*（尸罗山姆）
 9. *Nai hiruvalyë Valimar!*（凯兰崔尔对佛罗多说的最后一句话）
 10. *Ónen i-Estel Edain* (Gilraen's linnod)

 **答案：**
 1. Q. — *Namárië* 完全是昆雅语； 元音重，*-ë* 结尾，特征 Q. 声音
 2. S. — *Daro!*（辛达林语形式祈使句）； *lasto*（辛达语听）； *贝丝*（辛达语词）
 3.废话。 — 魔多的黑色言语； 完全陌生的结构
 4. S. — *A*（呼格助词，辛达林风格）； *Elbereth*（瓦尔达名字的辛达语形式）； *Gilthoniel*（辛达林语：*gil* star + *thon* kindle + *-iel*）
 5. Q. — *elen*（昆雅星形）； *-elvo* 所有格双后缀（昆雅语）； *lúmenn'* = 所有格（昆雅语）
 6. S. — *i Pheriannath* 有辛达林冠词 *i* + 鼻音突变 *ph*； *anann* = 辛达林语“长久”
 7. S. — *guren*（辛达语“我的心”，带有 1sg 后缀）； *bêd*（辛达语动词形式）； *enni*（辛达语“对我来说”）
 8. S.——与#4相同； *Ai!* 在两种语言中都用作感叹词
 9. Q. — *nai*（昆雅语“愿如此”）； *hiruvalyë*（昆雅语未来 2sg 的 *hir-* 查找）； *Valimar*（昆雅语名称）
 10. S. — *Ónen*（*anna-* 过去的辛达林语 1sg 给予）； *i-Estel*（辛达林语：冠词 + *Estel*）； *Edain*（*Adan* 的辛达语复数 = Man）
