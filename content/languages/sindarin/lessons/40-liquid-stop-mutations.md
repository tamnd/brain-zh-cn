## 简介

 本课完成了辛达语突变系统。 在第 11、15 和 30 课中，我们介绍了：
 - **软突变**（Lenition）——最常见； 由冠词、介词、许多助词触发
 - **鼻腔突变** — 由*an*（与格/有益）和复数冠词*in*触发
 - **混合突变** — 由与冠词融合的介词触发（*erin*、*uin*、*ammin*）

 现在我们添加两个最终突变：
 - **液体突变** — 由以 *-r*（液体辅音）结尾的单词或前缀触发
 - **停止突变** — 由以 *-t* 或 *-d* 结尾的单词触发（牙科停止）

 诚实的一句话：**液体突变和终止突变是五种突变中得到最少证实的**。 支持它们的证据是有限的，即使是对突变“作用”达成一致的学者有时也对其确切的触发因素和范围存在分歧。 本课程介绍了学术共识，指出了仍然存在的不确定性，并为学习者提供了实用的建议。

 ## 1. 五个突变：完整回顾

 在学习最后两个之前，我们的立场如下：

 | 突变| 主要触发因素 | 主要特点|
 |----------|-----------------|-------------|
 | 软（Lenition）| *i* （定义冠词 sg.），许多介词，形容词-名词位置，在 *a* （呼格）之后 | 大多数辅音为浊音/擦音 |
 | 鼻 | *an*（与格），*in*（复冠词）| 塞音→鼻音+浊音； *b*→*m*，*d*→*n*，*g*→*ng* |
 | 混合| *erin*、*uin*、*ammin*（准备+文章融合）| 软音和鼻音变化的混合 |
 | 液体| 以 *-r* 结尾的单词/前缀之后 | 类似于软停止； 详情如下 |
 | 停止| 以 *-t* 或 *-d* 结尾的单词之后 | 清音停音只会变成浊音 |

 ---

 ## 2.液体突变：定义和历史起源

 ### 这是什么？ 

从历史上看，当一个以液体辅音结尾的单词（或前缀）（特别是*-r*）后面跟着一个以可变辅音开头的单词时，就会触发**液体突变**。 第一个单词的最后 *-r* 影响了第二个单词的初始辅音，导致可预测的声音变化。 

“液体突变”这个名字来自于 *l* 和 *r* 的语音分类，即**液体**——具有流动、浊音质量的辅音。 其中，只有 *-r* 似乎触发了辛达林语中的这种特殊突变。 *-l* 可能在早期阶段就这样做了，但证据不太清楚。 

### 历史机制

 在原始精灵语（通用艾尔达语）中，单词连接处的声音是相互作用的。 当以 *-r* 结尾的单词与后面的单词有密切的语法联系时，*-r* 通过称为 **同化** 的过程影响后面的初始辅音 - 后面的声音变得与前面的声音更加相似。 

用古典历史语言学术语来说：*-r* 是浊辅音。 在清音塞音 (*p*、*t*、*c*) 之后，紧接着的浊辅音变为浊辅音 (*b*、*d*、*g*)。 这与驱动软突变的发声机制相同——差异在于确切的历史调节环境。

 ## 3. Liquid 突变：变化

 ### 辅音变化

 | 声母| 液体突变后|
 |--------------------|------------------------|
 | *p* | → *b* |
 | *t* | → *d* |
 | *c* | → *g* |
 | *b* | → *v* |
 | *d* | → *dh* |
 | *克* | → *'（滴）* |
 | *米* | → *v* |
 | *s* | → *h* |
 | *lh*| → *l* |
 | *右* | → *r* |
 | *h* | → *ch* |

 ### 与软突变的比较

 细心的学习者会立即注意到：**液体突变产生与软突变相同的变化**。 

这是学术界争论的焦点。 许多研究托尔金语言的语言学家得出的结论是，出于实用目的，液体突变和软突变**功能相同**——它们产生相同的辅音交替。 这种区别是历史性的（导致它们的机制不同），但**输出**是相同的。

如果这是真的，那么知道软突变的学习者已经知道液体突变——唯一的问题是何时应用它（触发）。 

### 可能的 *p* → *ph* 区别

 一些学者认为液体突变与软突变有一个微妙的区别：
 - 软突变：*p* → *b*（浊爆破音）
 - 液体突变：*p* → *ph*（清摩擦音）或 *p* → *b* — **有争议**

 如果液体突变给出 *p* → *ph* （不是 *p* → *b*），那么它至少在这一辅音上与软突变不同。 然而，这种区别是基于比较证据的理论重建，而不是明确证明的辛达语例子。 大多数新辛达语实用指南将液体突变视为与软突变相同，并将 *p* 问题作为学术脚注。 

---

 ## 4.液体突变：触发因素

 Liquid 突变是由某些以 *-r* 结尾的 **介词和前缀** 触发的：

 ### 已证实的触发器

 **1. *或* — “在上面，在上面”**

 > *或* (over) + 名词 → 名词首辅音的液体突变

 示例ᴺS.：
 - *or* + *barad* (塔) → *或 varad* (塔上方) — *b* → *v* = 与软突变相同
 - *or* + *tûr*（功率）→ *或 dhûr*（超功率） — *t* → *dh* = 与软突变相同

 **2. *ior* —“超过，超越”（变体/古老）**

 > *ior*（过度，古老）+名词→液体突变

 使用较少； 相同的变化。 

**3. *ar-* 前缀 — “旁边、旁边”或“高贵”（用作动词/名义前缀时）**

 动词复合词中的前缀 *ar-* 可能会触发后续词根的液体突变。 证据有限，而且大多是理论上的。 

### 实际挑战

 液体突变的挑战在于，它的**触发因素在经过验证的辛达林文本中很少见**。 介词 *or* 以某些复合形式出现，但很少直接单独出现在名词之前，以明显显示突变。 我们的大部分证据来自新辛达语重建。 

---

 ## 5. Liquid Mutation：诚实的评估

 学术景观如下：

 **位置 1（Jallings，*基本辛达林语*）**：液体突变 = 软突变。 同样的变化，不同的历史渊源。 在实践中，只要需要液体突变，就应用软突变。 

**位置 2（Fauskanger，*Ardalambion*）**：液体突变是一种独特的突变，但与软突变的不同之处仅在于理论 *p* → *ph* 与 *p* → *b* 问题。 在所有其他辅音中，它与软突变匹配。 这种区别很小，几乎没有实际影响。 

**位置3（Thorsten Renk，*Pedin Edhellen*）**：液体突变应具体情况具体分析； 经证实的证据太少，无法制定明确的规则。 

**给学习者的建议**：在任何会触发液体突变的地方应用软突变（主要是在 *or*、*ior* 和以 *-r* 结尾的前缀之后）。 如果您想保守且技术上准确，请将 *p* 标记为可能 → *ph* 而不是 *b*。 注意不确定性并继续前进。

 ## 6. 停止突变：定义和历史起源

 ### 这是什么？ 

从历史上看，当以“齿音塞子”（*-t* 或 *-d*）结尾的单词或前缀后面紧跟着一个以可变辅音开头的单词时，就会触发“停止突变”。 最后的牙科停顿影响了接下来的初始停顿。 

“stopmutations”这个名字来自*dentalstops*——辅音*t*和*d*，它们是在上牙后面的牙嵴处产生的“stops”（完全咬合的辅音）。 

### 历史机制

 在原始精灵语中，当一个单词以 *-t* 或 *-d* 结尾时，下一个单词的初始辅音通过 **接触同化** 或 **弱化** 的过程发生变化：
 - 清音塞音 (*p*, *t*, *c*) 在 *-d/-t* 之后变成 **浊音**: *p*→*b*, *t*→*d*, *c*→*g*
 - 浊音塞音（*b*、*d*、*g*）——根据大多数分析——**没有改变**（它们已经是浊音了）

 这是与软突变的主要区别：**停止突变仅影响清音停止**，而不影响浊音。 

---

 ## 7. 停止突变：变化

### 辅音变化

 | 声母| 停止突变后|
 |--------------------|---------------------|
 | *p* | → *b* |
 | *t* | → *d* |
 | *c* | → *g* |
 | *b* | 没有变化|
 | *d* | 没有变化|
 | *克* | 没有变化|
 | *米* | → *v* （有争议）|
 | *s* | →（没有变化，或者 → *h*？）（有争议）|
 | *lh*| 没有变化？ |
 | *右* | 没有变化？ |
 | *h* | 没有变化？ |

 与浊音塞音的**软突变**进行比较：
 - 软突变：*b* → *v*、*d* → *dh*、*g* → *'*（掉落）
 - 停止突变：*b* → *b*、*d* → *d*、*g* → *g*（无变化）

 **这是决定性的区别**：塞音突变不会影响浊塞音、摩擦音或鼻音。 仅清音塞音（*p*、*t*、*c*）发生变化。

 ## 8. 停止突变：触发器

 ### 有限证明

 终止突变是五种突变中**最少被证实**的。 以下是基于历史分析和有限文本证据提出的触发因素：

 **1. 数字 *cened* / *ad-* 前缀**

 数字前缀 *ad-* （意思是“两个，再次”——如 *adan* = “第二人”？或方向性“后退”）以及可能以 *-d* 或 *-t* 结尾的某些数字词会触发后续名词的停止突变。 

**2. 以 *-t*** 结尾的单词

 少数以 *-t* 结尾的功能词或前缀可能会触发停止突变。 证据非常有限。 

**3. 动词前缀 *o(d)-***

 前缀*od-*（与*od*相关，方向/空间：“来自，远离”）可能会触发以下动词词根的停止突变。 这是ᴺS。 重建。 

### 为什么很少看到停止突变

 1. **经证实的文本中很少有触发因素**：产生停止突变的条件在托尔金出版的辛达林语中很少出现
 2. **清音塞音的变化与软突变相同**：无论是软突变还是塞音突变，a *p* 都会变成 *b*； 你不能总是仅从结果中区分它们
 3. **浊音停止不变可能看起来像没有突变**：如果您在以*-d*结尾的单词后面看到*b*，是停止突变（b保持b）还是根本没有突变？ 结果是一样的。

 ## 9. 诚实学习者罕见突变指南

 对于实际学习者来说，这是一个坦率的优先顺序：

 ### 必须完全了解

 **软突变** — 随处使用：在单数冠词 *i* 之后，在大多数介词 (*o*, *na*, *an*) 之后，在许多复合短语中。 你会在托尔金的辛达语文本的每一页上遇到这个。 

### 必须熟知

 **鼻音突变** — 用在与格助词 *an* 和复数冠词 *in* 之后。 你会在句子中经常需要这个。 

### 应该从概念上理解

 **混合突变** — 在介词+冠词融合形式之后使用（*erin*、*uin* 等）。 中频。 

### 注意； 如有疑问时应用软突变

 **液体突变**——知道它存在； 知道它是由以 *-r* 结尾的单词触发的； 在实践中应用软突变，你将是正确的或非常接近。 

**停止突变**——知道它存在； 知道它只影响清音塞音（*p*、*t*、*c*），而浊音塞音保持不变； 适用于您对触发有信心的地方； 否则请注意不确定性。 

### “有疑问时”规则

 当您不确定要应用哪种突变时，**软突变是最安全的默认值**。 软突变是：
 1.最常见的
 2. 产生与许多辅音的其他突变重叠的变化
 3. 不太可能产生辛达语听起来陌生的形式

 ## 10. 完整突变参考：所有五个突变

 这是所有五个辛达语突变的主表。 完美记住软栏； 学好鼻音； 从概念上理解混合、液体和停止。

| 初始| 软| 鼻 | 混合| 液体| 停止|
 |--------|------|--------|--------|--------|-----|
 | *p* | *b* | *b* | *b* | *b*（*ph*？）| *b* |
 | *t* | *d* | *d* | *d* | *d* | *d* |
 | *c* | *克* | *克* | *克* | *克* | *克* |
 | *b* | *v* | *米* | *米* | *v* | *b*（不变）|
 | *d* | *DH* | *n* | *n* | *DH* | *d*（不变）|

 | *米* | *v* | *米* | *米* | *v* | *v*？ |
 | *s* | *h* | *h* | *h* | *h* | *啊*？ |
 | *lh*| *l* | *lh*| *l* | *l* | *lh*|
 | *右* | *r* | *右* | *r* | *r* | *右* |

 **主要观察结果**：
 1. 除了有争议的 *p* 情况外，软突变和液体突变是相同的

 3. 混合变异=清音软； 浊音鼻音（两者的“混合”）
 4. 鼻音突变将浊塞音变为鼻音对应音 (*b*→*m*, *d*→*n*, *g*→*ng*)

 ## 11.练习：液体和停止突变的例子

 **液体突变示例**（应用软突变；注释为ᴺS.）：

 1. *或* + *palan*（远）→ *或balan* ᴺS。 (*p* → *b*: 与软相同)
 2. *或* + *tirith*（观看）→ *或 dhirith* ᴺS。 (*t* → *dh*: 与软相同)
 3. *或* + *caras*（堡垒）→ *或garas* ᴺS。 (*c* → *g*: 与软相同)
 4. *或* + *barad*（塔）→ *或varad* ᴺS。 (*b* → *v*: 与软相同)
 5. *或* + *galadh*（树）→ *或'aladh* ᴺS。 (*g* → *'*: 与软相同)

 **停止突变示例**（仅*p/t/c*改变；浊音不变；ᴺS.）：

 1. [触发] + *peth* (字) → *beth* (p → b)
 2. [trigger] + *tol* (island) → *dol* (t → d) — 注意 *dol* 是一个现有的单词！ 结果可能不明确
 3. [触发] + *celeb* (银) → *geleb* (c → g)
 4. [触发] + *barad* (塔) → *barad* (b → b；不变！)
 5. [触发] + *galadh* (树) → *galadh* (g → g；不变！)

 第 4 项和第 5 项的对比：软突变将给出 *varad* 和 *'aladh*； 停止突变使它们保持不变。 这就是诊断的差异。

 ## 12. 为什么会存在这种复杂性？ 

学生可能会合理地问：为什么辛达语有五个突变？ 一个还不够吗？ 

答案就在于托尔金的**历史语言学方法**。 托尔金并没有将辛达语设计成一种方便的交流工具。 他将其设计为一种具有完整**内部历史**的语言——一种像真正的语言一样，通过数千年的声音变化而进化的语言。 每个突变代表一个*不同的历史过程*：

 - **软突变**源于某些句法环境中初始辅音的普遍弱化和发音（凯尔特语言中广泛的过程，激发了托尔金的突变）
 - **鼻音突变**源自单词边界处的早期鼻辅音簇
 - **混合突变**源自特定的语音环境（介词+冠词融合）
 - **液体突变**源于前面的 *-r* 对后续初始的影响
 - **停止突变**是由先前的牙科停止对后续初始的影响引起的

 每一个都是**有历史依据的**。 这就是为什么辛达林语尽管很复杂，但却感觉如此真实——它具有真实存在的语言的质感。 

---

 ## 需要记住的要点

1. **液体突变**由前面的*-r*触发； 产生与软突变相同的变化（可能有大多数学习者忽略的 *p* 区别）
 2. **停止突变**由前面的*-t/-d*触发； 仅更改清音塞音（*p*→*b*、*t*→*d*、*c*→*g*）； 浊音停止**不变**（这是与软突变的主要区别）
 3. **当有疑问时，使用软突变** - 它与液体突变完全重叠，并且对于清音停止它也匹配停止突变
 4. **总共五种突变**：软>鼻>混合>液体>停止（按照频率和实际重要性的顺序）
 5. **复杂性反映了历史**：每次突变都源于不同的历史语音过程，这就是为什么辛达林语具有真实语言的分层、现实深度
