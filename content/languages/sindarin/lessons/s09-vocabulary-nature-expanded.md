## 托尔金视野中的自然

 托尔金热爱植物学，是一位充满热情的博物学家，在英国乡村长大。 他的精灵自然词汇是辛达林语中最丰富和最彻底证明的领域之一。 原因是结构性的：许多经过证实的辛达语短语都来自地名，而中土世界的地名绝大多数与自然特征有关——山丘、河流、树木、星星、天气。 

*Amon Sûl*（风之山）、*Hithaeglir*（迷雾山脉，字面意思是“薄雾面纱的山峰”）、*Eryn Lasgalen*（绿叶之林）、*Nen Hithoel*（雾湖）、*Nîn-in-Eilph*（天鹅舰队）——这些名字中的每一个都是一堂等待解开的自然词汇课。 

本增刊按环境领域组织，旨在与中土世界地图一起阅读。 当你学习每个单词时，找到一个使用它的地名。

 ### 精灵对星星的崇敬

托尔金笔下的精灵不仅仅是太阳或月亮，更是星光下的生物。 精灵首先在星空下苏醒（在太阳和月亮存在之前）。 瓦尔达（Varda）是维拉中最受喜爱的人，她是*Elbereth Gilthoniel*——“星女王，星点燃者”（*gil* + *thon* kindle + *-iel*）。 伟大的精灵赞美诗（*A Elbereth Gilthoniel / silivren penna míriel / o menel aglar elenath...*）的整个开头是自然词汇的瀑布：星星，银色的光芒，宝石般的光芒，苍穹的荣耀。

 ### 读取地图上的水名

 从中土地图可以最好地理解辛达语水词汇的丰富性。 尝试找到这些名称并解析它们：
 - *Nen Hithoel* = *nen*（水/湖）+ *hîth*（雾）+ *-oel*（形容词后缀）=“雾灰色之湖”
 - *Nîn-in-Eilph* = *nîn*（水/水域）+ *in*（复数冠词）+ *eilph*（天鹅，复数*alph*）=“Swanfleet”= 天鹅的水边地方
 - *Anduin* = *and*（长）+ *duin*（大河）=“长河”
 - *Baranduin* = *baran*（金棕色）+ *duin* = 金棕色河流 = 白兰地酒

 ### 洛斯罗立安之花

 《金木》中提到的两种花已证明了辛达语的名字：
 - *Elanor*：金花，“太阳之星”。 山姆在洛斯罗立安看到这个名字后给他的女儿起了这个名字。 
- *Niphredil*：白花，“苍白的分散”。 它在多瑞亚斯的露西安·缇努维尔跳舞的地方盛开； 它也生长在洛斯罗立安。 

这两朵花是托尔金最仔细命名的植物学细节，它们的辛达语词源值得关注。

 ### *Yrch!* — 莱戈拉斯著名的警告

 在《指环王》中，当莱戈拉斯看到兽人靠近时，他会喊出一个词：*Yrch！*这是辛达林语中*orch*（兽人）的复数形式，显示了 i 情感：*o→y*。 这个词得到了充分的证实，在恐惧和警告中大声喊叫，并表明“i-affection”复数形式确实是活生生的复数形式——一个处于极端状态的辛达林语使用者会自动达到自然的复数形式。

 ### 将地名解析为词汇课

每个主要辛达语地名都是一个词汇句子。 练习解析：
 - *Hithaeglir* = *hîth*（薄雾）+ *aeleg*（刺痛/尖锐）+ *-ir*（复数？）=“雾刺峰”= 迷雾山脉
 - *Imladris* = *imlad*（深谷）+ *ris*（裂缝）=“裂缝的深谷”=瑞文戴尔
 - *Mithlond* = *mîth*（灰色）+ *lond*（避风港）=“灰色避风港”
 - *Caradhras* = *caran*（红色）+ *rass*（喇叭峰）=“红角”

 ## 练习：自然词汇练习

 将词汇与中土地图上您知道的地点联系起来：

 1. *Amon Sûl* 的字面意思是什么？ 该位置的通用语音名称是什么？ 
2. *Nen Hithoel* 具有三个组成部分：它们是什么以及它们的含义是什么？ 
3. *Eryn Lasgalen* 取代了名称 *Taur-e-Ndaedelos*。 解析两个名称并解释发生了什么变化以及原因。 
4. *orch* 的复数形式是 *yrch*。 是什么突变或元音变化产生了这个？ *in yrch* 会发生什么突变？ 
5. *Alfirin* 和 *niphredil* 都是白色的花。 他们的词源对它们的描述有何不同？ 
6. *Andúril*（剑）和*Anduin*（河）共享元素*and-*。 每个化合物的含义是什么？ 
7. 完整解析*Mithlond*并解释它描述的是一个什么样的地方。 
8. *naur* 一词出现在感叹号 *Naur an edraith ammen!* 和天气相关词汇中。 核心含义是什么？上下文如何区分用法？ 

**答案：**
 1. *Amon Sûl* = 风之丘 (*amon*) (*sûl*) = Weathertop
 2. *Nen*（水/湖）+ *hîth*（雾）+ *-oel*（形容词后缀，意思是“灰雾”）= 雾蒙蒙的灰色水湖； 雾气缭绕的缝隙中平静的湖水
 3. *Eryn Lasgalen* = 森林 (*eryn*) + *las* (叶子) + *calen* (绿色，形容词) = “绿叶之木”； *Taur-e-Ndaedelos* = 大森林 (*taur*) + *en* (of) + *ndaedelos* (*daedel* dread + *-os* 集体？) = "巨大恐惧/恐惧森林"; 索伦堕落后，黑暗的名字被欢乐的名字所取代
 4. I-affection（内元音变化）：*o→y*； *in* 之后的 *yrch* 上没有辅音突变，因为该单词以元音开头（该突变仅适用于初始辅音）
 5. *Alfirin* = *al-*（无）+ *firin*（凡人）=“不朽”——由它不是什么（凡人）定义； *niphredil* = *niph-*（苍白）=“苍白/苍白的分散”——由其颜色和外观定义
 6. 两者都使用*and* = long：*Andúril* = long-fire（长火焰）； *Anduin* = 长河（长长的大河）
 7. *Mîth*（灰色）+ *lond*（避风港）= 灰色避风港； 一个灰色水域的避风港，中土世界最西端的港口，船只驶往不朽之地
 8. *Naur* = 火； 在 *Naur an edraith ammen!* = 字面上的火（点燃作为防御）； 在天气/医疗环境中=“发烧”或“燃烧”（隐喻热量）
