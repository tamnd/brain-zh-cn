## 简介

 辛达林语的个人名字是一份礼物——一首压缩的诗，捕捉了有关一个人的本性、外表、血统或命运的基本信息。 托尔金笔下的精灵将命名视为一门高雅艺术，他为角色发明的名字是任何人工语言中最美丽的单词之一：*加拉德瑞尔*、*露西恩*、*凯勒布林博*、*格洛芬德尔*、*廷努维尔*。 每一个都是一个有意义的微型世界。 

在本课中，您将学习托尔金在《未完成的故事》中描述的精灵命名传统、主要的男性和女性名字元素及其注释、父名的工作原理以及精灵如何获得绰号。 本课程最后提供了创建您自己的精灵语个人名字的分步指导。

 ## 1. 精灵命名传统

 ### 三个名字

 托尔金在他的文章《The Shibboleth of Fëanor》以及《未完成的故事》和《中土世界的人民》中的相关注释中详细描述了精灵的命名习惯。 精灵最多可以有三个名字**：

 **1. *Ataressë* — 父亲的名字**

 出生时由父亲给予，通常用昆雅语（诺多族的威望语言）或辛达语（辛达语）。 父亲的名字反映了孩子在血统中的地位——通常它与父亲自己的名字相呼应，或者包含父系的元素。 该名称是“官方”名称，用于正式场合。 

示例：
 - *Elrond* 是 Elrond 的父亲名字（由他的父亲 Eärendil 赋予）
 - *Finduilas* 从她的父亲 Orodreth 那里继承了她的父亲名字

 **2. *Óressë* — 母亲的名字**

 当母亲的直觉掌握了孩子的内在本质或命运时，母亲给的名字通常晚于父亲的名字。 母亲的名字通常是**预言性的**——它们可能反映孩子的命运、性格，或者在某些情况下，反映母亲的悲伤或预知。 

著名的母亲名字：
 - *Maedhros* 从 Nerdanel 那里得到了他的母名（*Russandol*，“铜顶”，因为他不寻常的红发——不是典型的诺多林）
 - 米瑞尔 (Míriel) 给费艾诺 (Fëanor) 的母名：“费艾诺 (Fëanor)” = *Faenor*（昆雅语的辛达语形式 *Fëanáro*，“火之灵”）——她预见到了他燃烧的创造力

 **3. *Kilmessë* — 选择的名字 (Epessë)**

 由精灵自己在成年后选择，或由其他人给予并接受——基于行为或品质的**绰号**。 也称为*epessë*（“后名”）。 这是与传说中的人物最相关的名字：
 - *Erchamion* =“单手”（贝伦的 epessë，在卡哈洛斯失去一只手后）
 - *Cúthalion* =“强弓”（Beleg 的 epessë，因其弓箭技巧）
 - *Mithrandir* = “Grey Wanderer”（甘道夫的辛达林 epessë，由认识他的精灵提供）

 ### 辛达林语与昆雅语名称

 到了第三纪元，辛达语文化（辛达语、中土世界的诺多族、灰精灵）主要在日常生活中使用辛达语名字。 昆雅语被保留用于传说、仪式和高级对话。 当托尔金为我们提供精灵字符的辛达语形式时，这些是日常使用的名称：
 - 高等精灵 *阿塔尼斯* → 辛达林 *奈尔文* → epessë *加拉德瑞尔*
 - *Curufinwë* → 辛达语 *Curufin*

## 4. 名人分析——详细分析

 ### 1.*阿拉贡*
 > *ara-*（高贵、王者风范 — 与 *aran* 国王有关）+ *gorn*（受人尊敬/恐惧，来自 *gor-* 或可能是“浮躁的”）

 托尔金自己的注释：*阿拉贡*可能源自与“受人尊敬的国王”或“国王般的恐惧”相关的词根。 这个名字很合适：阿拉贡既高贵（*ara-*）又激发敌人的恐惧（*gorn*）。 他是“令人敬畏的国王”。 

### 2.*莱戈拉斯*
 > *lego* 或 *laeg*（敏锐、敏捷 — 可能是“绿色”，来自 *laeg* = 敏锐、新鲜/绿色）+ *lass*（叶子）

 托尔金的注释：**“Greenleaf”。** *Lasgalen*（如 *Eryn Lasgalen* = 绿叶木）使用相同的 *las/lasg/laeg* 根。 莱戈拉斯是森林里的一片绿叶——敏捷、自然。 

### 3.*加拉德瑞尔*
 > *galad*（光辉，光——来自*cal-/gal-*根）+ *riel*（戴花环的少女——从*rig-*到花环+*-iel*）

 托尔金的解释是：“戴着光辉花环的少女。” *galad* 元素连接到 *Galadrim*（树人）——两者共享光/生长根 *GAL-*。 加拉德瑞尔不仅仅是“美丽的精灵”，而且是光芒四射、富有创造力的光芒的化身——她的本性融入了她的名字。 

### 4.*埃尔隆德*
 > *el*（星星，精灵 — 来自 *EL-* 根）+ *rond*（拱顶、圆顶拱门、像天空一样弯曲的洞穴天花板）

 含义：**“星穹”**或“星穹”。 埃尔隆德的兄弟是*埃尔罗斯* (*el* + *ros* = “星形泡沫”)。 两兄弟都带有*el*（星星）元素——他们是诞生在天堂之下的星水手埃雅仁迪尔的孩子。 埃尔隆德的名字捕捉到了头顶繁星闪烁的穹顶天空。 

### 5.*名人*
 > *celeb*（银）+ *rim*（大数字？或此处来自 *rimbor* = 拳头）+ *bor*（手/拳头）

 托尔金的注释：**“银拳”。** *rim* 元素可能是 *rimbor*（拳头）的变体——一只强大而熟练的手。 凯勒布理鹏是第二纪元最伟大的铁匠。 他的拳头/手技艺精湛。 他的名字就是他的工艺身份。 

### 6.*格洛芬德尔*
 > *glor-*（金色荣耀，来自 *GLOR-* 词根，与金色和光彩有关）+ *find*（头发、发束、一绺头发）+ *-el*（存在/人）

 含义：**“金头发”。** 他的金色头发很有名。 这个三元素名称很不寻常，但每个部分都很重要：*glor-*（头发颜色的质量）、*find*（头发本身）、*-el*（拥有这根头发的人）。 

### 7.*哈尔迪尔*
 > *hal-*（高、高 — 来自 *KHAL-* 词根：举起、提升）+ *dîr*（男人、成年男性）

 含义：**“高个子男人”**或更准确地说是“尊贵者”。 哈尔迪尔是洛瑞恩的行军典狱官——地位和体格都很高的人。 这个名字非常实用。 

### 8.*Finduilas*
 > *find*（头发，发束）+ *ui*（永远，总是，永久 - 时间元素）+ *las*（叶子）

 托尔金自己的文本并没有给出任何明确的解释。 可能的含义：**“长叶头发”**（头发像永久的叶子）或诗意的复合词，表示持久美丽的东西。 这个名字有一种流畅、忧郁的气质，很适合芬杜伊拉斯——英年早逝的奥罗德雷斯的悲惨女儿。 

### 9.*梅格林*
 > *maed/maeg*（锐利、刺眼——眼睛、智慧）+ *glin*（闪光、闪烁——眼睛、剑；闪光和洞察力）

 托尔金的注释：**“锐利的目光。”** 梅格林以其敏锐的目光和双重思想而闻名——他的智慧是他最伟大的天赋，也是他的失败。 他的名字概括了他的看法和他危险的内在本性。 

### 10. *廷努维尔*
 > *tinn-* / *tin-*（火花，微小的星点——夜莺眼睛里的小光？）+ *dúv-*（与*dû*有关，夜晚，黄昏，黄昏的昏暗）+ *-iel*（少女）

 托尔金的注释：**“夜莺”**（字面意思是“暮光闪闪的女儿”或“黄昏火花的她”）。 当贝伦第一次看到露西恩在黄昏时分的森林中翩翩起舞时，她给她起了这个名字，她的歌声就像夜莺的歌声。 *-iel* 让它变得个性化和亲密——她是“夜莺少女”。

 ## 5. 父系结构

 ### 儿子：*-ion*

 > 父亲的名字 + *-ion* → “[父亲] 的儿子”

- *Legolasion* ᴺS。 = 莱戈拉斯之子
 - *埃尔隆迪恩* ᴺS。 = 埃尔隆德的儿子 — 尽管托尔金使用 *Elladan* 和 *Elrohir* 来表示埃尔隆德的儿子
 - *Eldarion* = Eldar 的儿子？ 不——*Eldarion*是阿拉贡的儿子； 分析：*Eldar*（精灵人）+*-ion* =“Eldar之子”（比喻：精灵血统的儿子）

 ### 的女儿：*-iel*

 > 父亲的名字 + *-iel* → “[父亲] 的女儿”

 - *Eärendiliel* ᴺS。 = 埃雅仁迪尔的女儿
 - *阿拉尼尔* ᴺS。 = 国王的女儿（将军：*aran* + *-iel*）

 ### 完整属格结构

 父名的另一种形式使用所有格：
 > *[孩子的名字] [ion/iell] en [父亲的名字]*

 - *Arwen iell en Elrond* ᴺS。 = 阿尔文，埃尔隆德的女儿

 *Ion*（儿子）和*iell*（女儿）是辛达语中儿子和女儿的意思：
 - *ion* = 儿子 — ᴺS。 （证明形式存在争议；一些学者更喜欢 *ион*/*yôn*）
 - *iell* = 女儿 — ᴺS。 （来自*yell*或相关表格）

 ### 创建你自己的绰号

 绰号应该：
 1. 描述一项基本品质、行为或外表特征
 2. 由两个要素组成（品质+人/物）
 3. 使用第 35 课中学到的模式（后缀 *-dir*、*-wen*、*-ion*、*-iel* 等）
 4. 大声说话时听起来自然

 ## 7. 命名练习：创建 5 个完整的精灵名字

 ### 过程

 **第 1 步**：选择角色的本质或最显着的品质
 **第 2 步**：找到表达此内容的辛达语单词
 **第 3 步**：选择适合角色的男性或女性结局
 **第四步**：组合、检查音韵、朗读
 **第5步**：标记ᴺS。 并注意其含义

 ### 工作示例 1：一位男性精灵，他是一位伟大的歌手

 1.品质：唱歌
 2. 元素：*lind-* / *glin-*（歌曲、歌唱）+ *-dir*（男人）或 *-ron*（伟大的）
 3. 男性化
 4. *Lindron* ᴺS。 =“伟大的歌手”或*Lindir*（已被证明是一个名字！）
 5. 使用 *Glindir* ᴺS。 =“Gleam-Singer，歌曲闪闪发光的人”以获得更诗意的东西

 ### 实例2：与星光相关的女性精灵

 1.品质：星光
 2. 元素：*elen*（星星）+ *-iel*（少女）→ *Eleniel* ᴺS。 （星少女）——或者
 *吉尔文*ᴺS。 (*gil* 明亮的星星 + *-wen* maiden) — 语音更流畅
 3. 女性化
 4. *吉尔文* ᴺS。 ——《星光少女》
 5. 马克ᴺS.; 注：*gil*（星星，明亮的火花）在辛达林语中得到证实； *-wen* 已得到证实

 ### 实施例3：银发男性精灵领主

 1.品质：银发
 2. 元素：*celeb*（银）+ *find*（头发）+ 男性结尾
 3. *名人发现* ᴺS。 或 *Celebfindir* ᴺS。 （银发男）
 4. 检查：*celeb* + *find* = *Celebfind* ᴺS。 — 不寻常的辅音簇 *-bf-* 可能会简化：*Celebfind* → 也许 *Celebinn* ᴺS.？ 
5. 替代方法：*Mithelass* ᴺS。 — 灰叶 (*mith* + *lass*) 如果他的头发是银灰色的

 ### 示例4：一位女性精灵，其本质是黄昏和神秘

1.品质：暮色、神秘
 2. 元素：*dû-*（夜晚，黄昏时的昏暗）+ *fain*（白色，光芒四射——矛盾：她在黑暗中闪闪发光）
 3. 阴性：+ *-iel* 或 *-wen*
 4. *Dúfainiel* ᴺS。 =“Twilight-Radiance Maiden”——类似于 *Tinnúviel* 图案的名字
 5. 或者更简单：*Dúwen* ᴺS。 =“夜之少女，暮光之女”

 ### 示例 5：以敏捷的骑术而闻名的男性精灵

 1.品质：千里马
 2. 元素：*roch-*（马）+ *-dir*（人）或*caer-*（迅捷/伟大）+ *roch-*
 3. *Rochdir* ᴺS。 =“骑士”（简单）
 4. 更令人回味：*Rochallor* — 等等，*Rochallor* 被证明是芬国昐的马的名字！ 将其用作个人姓名模板：*roch* + *-allor*（来自 *alor*？非标准）
 5. 最佳：*Rochion* ᴺS。 =“马之子”（比喻：致力于马的人）或*Brethilroch* ᴺS。

 ## 8. 快速参考：建立知名度备忘单

 ### 对于男性名字：
 - 从质量元素开始：*ar-*、*beleg-*、*celeb-*、*el-*、*glor-*、*hal-*、*thal-*
 - 结尾为：*-dir*（男人）、*-gon*（领主）、*-on*（伟大）、*-rond*（拱顶）、*-ron*（守门员）

 ### 对于女性名字：
 - 从光/自然元素开始：*cal-*、*fair-*、*lúth-*、*mith-*、*nim-*、*tin-*
 - 结尾为：*-el*（人）、*-iel*（女儿）、*-wen*（少女）、*-del*（心爱的人）

 ### 听起来的目标：
 - 流动：辅音簇如 *-nd-*、*-lm-*、*-rn-*、*-nth-*
 - 元音：*a*、*e*、*i* 最常见； *ó* 和 *û* 增添庄重感
 - 避免：在没有液体的情况下多次剧烈停止（*bktr* 等）； 辛达语流

 ## 需要记住的要点

 1. **三种姓名类型**：父亲姓名 (*ataressë*)、母亲姓名 (*óressë*)、选择的加词 (*kilmessë*)
 2. **母亲的名字是有预言性的**：最富有诗意的复杂名字往往预示着命运
 3. **男性与女性**：结尾后缀（*-dir/-ron* vs. *-el/-wen/-iel*）是主要性别标记
 4. **绰号是通过行为获得的**：中土世界最令人难忘的名字是 epessë
 5. **创建前分析**：拆开经过验证的名称比任何规则列表都更能说明问题
