## 颜色词汇

 辛达语有一套丰富的颜色术语，其中许多直接出现在中土世界地点和人物的名称中。 学习颜色还可以立即带来实际回报：您可以开始更深入地分析第 3 课中的地名。 

| 辛达林语 | 英语 | 期间 | 注释和外观|

 **关于颜色区别的注意事项：**

 **绿色** — 辛达林语至少有两个绿色单词。 *卡伦*是普遍的绿色——生长物、森林、夏尔的颜色。 *Laeg* 是鲜艳、新鲜的绿色——春天的新叶或清澈的水的鲜绿色。 绿色精灵（*Laegrim*）大概因其森林家园的明亮绿色而得名。 当您在地名（*Eryn Lasgalen*、*Calenardhon*）中看到 *galen* 时，它是 *calen* 的 lenit 形式。 

**白色** — *glân* 和 *nim/nimp* 都表示白色，但有细微差别。 *Glân* 意味着干净、纯白色——闪亮的白色。 *Nim/nimp* 是苍白、冷白色——像冬天或石头一样的白色。 *Nimrodel* 和 *Nimrais* 使用 *nim-*； 如果你想要“白塔”有一种明亮闪亮的感觉，*glân*可能是首选。 

**黑暗** — *morn* 和 *dûr* 均表示黑暗，但 *dûr* 具有特别阴沉、威胁的含义。 *Barad-dûr* 是幽暗之塔，黑暗而压抑的塔。 *早晨*简直就是黑暗（致命的黑暗，夜晚）。 其区别就像英语中的“黑暗”与“阴沉”。

 ## 颜色形容词跟在名词后面

 这是说英语的人在学习辛达林语时必须忘记的首要语法规则之一。 在英语中，形容词位于它们所修饰的名词的**之前**：

 - 英语：“**绿色**森林”
 - 辛达林语：*eryn **galen*** =“森林绿色”→“绿色森林”

 名词在前； 形容词如下。 这是标准的凯尔特语（和辛达林语）词序。

**更多示例：**
 - *阿蒙·卡兰* =“红山”=“红山”
 - *ered luin* =“蓝色山脉”=“蓝色山脉”（*l* 没有突变 - 见下文）
 - *barad nim* =“白塔”=“白塔”
 - *taur morn* =“黑暗森林”=“黑暗森林”

 **地名中：** 这就是为什么中土地名在名词后面有“描述性词”：
 - *Amon Sûl* =“风山”（不是“风山”）
 - *Eryn Lasgalen* =“绿叶森林”（不是“绿叶森林”）
 - *Ered Luin* =“蓝色山脉”=“蓝色山脉”

 ## 什么是软突变？ 

辛达林语有一个称为**辅音突变**的系统，其中**单词的第一个辅音根据语法上下文而变化**。 辛达林语的突变有多种类型，但最常见和最重要的是**软突变**，也称为**lenition**（来自拉丁语*lenis*=温柔、柔软）。 

**软突变**是形容词直接跟在名词后面时发生的初始辅音的变化。 在这个位置，辅音变得“更柔和”——发音不那么尖锐，通常是浊音或弱化。 

### 为什么会存在突变？ 

在辛达林语的早期阶段，名词末尾有一个元音，位于形容词之前。 名词的最后元音导致后面单词的初始辅音发生变化（通过同化过程）。 然后最后一个元音被侵蚀并消失——但它引起的辅音变化*仍然像化石一样*。 结果是：现代辛达林语中没有明显触发因素的变化，但仍然标记“这个形容词位于名词后面”。 

这正是威尔士语中发生的情况。 威尔士语*coch*（红色）在“y”（定冠词）之后以及许多复合位置中变成*goch*。 这种突变是一种历史残留物，已被语法化——它现在“标记”了位置，而不是由仍然存在的声音“引起”。 

### 软突变不是什么

 软突变并不意味着这个词变得更难理解——它意味着这个词可以预见地、有规律地改变它的形式。 一旦你了解了变化表，你就可以随时重建未突变的（基本的，“激进的”）形式。

 ## 软突变变化（预览）

 完整的软突变表有十一个变化。 以下是影响您刚刚学到的颜色词的五个单词的预览：

 | 激进（基础）| 软突变（lenied）| 示例|
 |----------------|------------------------|---------|
 | **b** | **v** | *巴兰* → *瓦兰* |
 | **c** | **克** | *卡伦* → *盖伦*; *卡兰* → *加兰* |
 | **d** | **DH** | *dûr* → *dhûr* （当 *dûr* 跟在名词后面时使用）|
 | **克** | ∅（消失！）| *glan* → *lân* |
 | **米** | **v** | *早上* → *沃恩*; *密斯* → *维斯* |
 | **l** | **l**（不变）| *鲁因* → *鲁因* |
 | **n** | **n**（不变）| *尼姆* → *尼姆* |
 | **r** | **r** （不变） | — |
 | **s** | **h** | *苍鹭* → *苍鹭* |
 | **t** | **d** | *泰德* → *泰德* |
 | **p** | **b** | *帕兰* → *巴兰* |

 最显着的变化是：
 - **g** 完全消失：*glân*（白色）→ *lân*； *galen* 已经是 *calen* 的简化形式（c→g 是 *c* 的突变）
 - **c** → **g**：字母 *c* 在软突变中总是变成 *g*
 - **b**, **m** → **v**: *b* 和 *m* 都变成 *v*
 - **d** → **dh**：浊塞音 *d* 软化为浊擦音 *dh*

 **等等——一个关键的澄清：**当*calen*（绿色）经历软突变时，它变成*galen*（c→g）。 但*glân*（白色）以*gl-*开头，软突变中的*g*消失，得到*lân*。 所以：

 - *barad **galen*** = "绿塔" (*calen* → *galen* 在名词之后，c→g)
 - *barad **lân*** = "白塔" (*glân* → *lân* 在名词之后，g→∅)
 - *amon **vorn*** = "dark hill" (*morn* → *vorn* 在名词之后，m→v)
 - *ered **luin*** =“蓝山”（*luin* → *luin*，l 不变）
 - *taur **varan*** =“黄褐色/棕色森林”（*baran* → *varan*，b→v）

 ## 触发：名词后的形容词

在本课中，软突变的触发因素是：**直接跟在其名词后面的形容词**。 这是根本触发因素之一。 还有许多其他触发器（第 11 课和第 13 课中介绍），但这是最直接有用的。 

**规则**：当**描述词**（形容词或任何修饰词）紧跟在**名词**之后时，描述词的声母会发生软突变。 

| 名词+形容词| 未变异的形容词 | 变异形容词| 短语含义|
 |-----------------|--------------------|--------------------------------|----------------|
 | *巴拉德* + *卡伦* | *卡伦* | *盖伦* | 绿塔|
 | *巴拉德* + *格兰* | *格兰* | *兰* | 白塔|
 | *阿蒙* + *卡兰* | *卡兰* | *加兰* | 红山|
 | *阿蒙* + *早上* | *早上* | *沃恩* | 黑暗山|
 | *牛* + *巴兰* | *巴兰* | *瓦兰* | 棕色森林|
 | *艾德* + *鲁因* | *卢因* | *luin*（不变）| 蓝山|
 | *米纳斯* + *尼姆* | *尼姆* | *vim* ... | 等等 — *n* → *n* 不变？ |

 实际检查： *n* 在软突变中没有改变。 所以 *nim* → *nim*: *minas nim* = “白塔”。 但是 *nimp* → *nimp* （不变）。 然而，*Nimrodel* 是用作专有名词的完整单词，而不是狭义形式。 

让我也检查一下 *morn*: *m* → *v* 的软突变。 所以：
 - *taur morn*（未突变，无触发）但如果直接作为形容词跟随：*taur vorn* ✓

 但是等等：在经过充分验证的复合词 *Mordor* 中，*môr* 是第一个元素（名词）而不是形容词。 在 *Barad-dûr* 中，*dûr* 位于 *barad* 之后：*dûr* 发生突变吗？ 软突变中的 *d* → *dh*。 未连字符的形式为 *barad dhûr*。 但经过验证的形式是 *Barad-dûr* 且不带连字符的 *d*。 这表明在**化合物名称**中，突变可能并不总是适用，或者化合物在突变之前已经化石化。 

这是一个重要的细微差别：**自由定语形容词**（句子中普通名词后面的形容词）确实采取软突变，而**正确的复合名称**可能会保留未突变的形式。 “名词后的形容词=软突变”规则适用于生动的、富有成效的语法——而不是冻结的专有名词复合词。

 ## 颜色词软突变的实例

 **1. *Eryn Lasgalen***（绿叶森林）
 - *eryn* = 森林
 - *las* = 叶子
 - *galen* = 绿色（从 *calen* 获取；c→g）
 - *galen*中的*g-*是*calen*的*c-*的软突变

 这是最重要的经过验证的例子之一：*Eryn Lasgalen* 中的 *galen* 绝对是 *calen* 的简化形式，确认 c→g 是 *c* 的软突变。 

**2. *Ered Nimrais***（白山）
 - *ered* = 山脉
 - *nim-* = 白色 + *rais* = 峰值
 - 复合词 *nimrais* 将 *nim-* 视为前缀； 整个单词在复合边界发生变化，而不是纯粹的形容词突变

 **3. 由 *Calenardhon* 暗示 → 重命名省**
 - *Calenardhon* = *calen* (green) + *ardhon* (realm, Region)：这里 *calen* 是第一个元素（前缀），而不是后面的形容词，所以没有突变

 ## 颜色练习句子

 构建这些名词+颜色短语，在需要时应用软突变：

**1.**“灰色流浪者”——*mithrandir* 使用 *mith*（灰色）+ *randir*（流浪者）； 但作为复合专有名称，不适用任何突变。 作为常用短语：*randir vith* =“灰色流浪者”（*mith* → *vith*, m→v）

 **2.** “红山” — *orod* + *caran* → *orod garan* (c→g)

 **3.** “黑暗森林” — *taur* + *morn* → *taur vorn* (m→v)

 **4.** “白塔” — *minas* + *glân* → *minas lân* (g→∅)

 **5.** “蓝山” — *ered* + *luin* → *ered luin* （l 不变）

 **6.** “绿色山谷” — *nan* + *calen* → *nan galen* (c→g)

 **7.** “银树” — *orn* + *celeb* → *orn geleb* (c→g) — 但经证明的专有名称 *Celeborn* 首先有 *celeb* （作为修饰词头复合词，不同的顺序）

 **8.** “棕色河流” — *sîr* + *baran* → *sîr varan* (b→v) — cf. *Baranduin* 使用 *baran* 作为第一个元素，因此没有突变

 **答案关键：**

 | 短语| 结果 | 应用突变 |
 |--------|--------|-----------------|
 | 《灰色流浪者》| *兰迪尔·维思* | m→v |
 | “红山”| *奥罗德加兰* | c→g |
 | “黑暗森林”| *金牛座* | m→v |
 | “白塔”| *米纳斯兰* | g→∅ |
 | “蓝山”| *艾瑞德·鲁因* | 无（l 不变）|
 | “绿色山谷”| *南盖伦* | c→g |
 | “银树”| *奥恩·盖勒布* | c→g |
 | “棕色河”| *瓦兰先生* | b→v |

 | 概念| 关键事实 |
