## 简介

 这是辛达语核心课程的最后一课。 我们以一个涉及我们所学一切的主题作为结束语：**省略**——当言语和诗歌中的单词一起流动时元音的下降或收缩。 省略是辛达语口语的“结缔组织”； 这就是使语言听起来流畅、富有音乐感而不是停顿的原因。 

我们还回顾了托尔金在辛达语文本中使用的三个标点符号——撇号（*'*）、连字符（*-*）和凸起点（*·*）——每个标点符号都有一个经常被混淆的独特功能。 

在省略分析之后，本课程以对整个 41 课课程的简短回顾和建议的后续步骤结束。 

---

 ## 1. 什么是省略？ 

**省略**是当两个单词以紧密的顺序说出时，一个或多个声音（通常是元音）的省略或缩写。 它通过消除尴尬的元音间断（来自不同单词的两个相邻元音）使语音更加流畅。 

在英语中，省略因缩写而为人们所熟悉：
 - “我是”→“我是”（省略 *a*）
 -“不”→“不”（省略 *o*）

 在诗歌中，省略对于保持韵律至关重要：
 - 莎士比亚：“th'expense ofspirit”——*the*的*e*在*expense*之前省略

 在**辛达林语**中，出现了省略号：
 1. 在**诗歌** (*linnad*) 中——保持音节计数和音乐流畅
 2. 在**散文**中——当某些常见的短词出现在元音词首词之前时
 3. 作为**语法化融合**——省略已成为语法形式的固定特征

 ## 2. 辛达林语中的三个标点符号

 托尔金在他的辛达语文本中使用了学生经常混淆的三个特殊标记：

 ### 撇号 *'*

 托尔金辛达林语中的撇号有**两种完全不同的功能**：

 **功能1：*g-***的软突变

 当软突变应用于初始 *g-* 时，*g* 消失并用撇号标记：
 - *galadh*（树）→软突变后的*'aladh*
 - *i galadh* → *i 'aladh* （树；g 在软突变下下降）

 这**不是省略**——这是辅音突变。 撇号标志着 *g* 曾经存在但已被删除的事实。 

**功能2：诗歌中的元音省略**

 在诗歌文本中，当两个元音可能发生冲突时，撇号可能会标记掉元音：
 - 在元音开头的单词之前以 *-a* 或 *-e* 结尾的单词：最后的元音消失
 - 用撇号标记以表明某些内容已被省略

 ### 连字符 *-*

 连字符标记两个元素之间的**紧密的语法联系**，这两个元素没有完全融合成一个单词，但作为一个紧密的单元来朗读：

 **附加到动词形式的前缀**：
 - *palan-díriel*（远视，凝视远方的她）— *palan-* 前缀 + *díriel* 分词
 - *na-chaered*（远距离）— *na-* + *chaered*（软/鼻突变：*c*→*ch*）

 **带有属格助词的化合物**：
 - *Bar-en-Danwedh* — 家 + *en* (of-the) + *Danwedh* (赎金)
 - *Tol-in-Gaurhoth* — 岛屿 + *in* (of-the) + *Gaurhoth*

 **名称中包含两个形容词或元素**：
 - *Iarwain-Ben-adar*（最年长且无父——汤姆·庞巴迪尔的完整辛达林头衔）

 连字符**不**标记省略或突变； 它标记**复合连接**或**前缀附件**。 

### 凸起的点 *·*

 托尔金有时在他的手稿中使用**凸起的点**（·）或**中间的点**来表示**紧密的接合点**——两个独立的单词在语法上紧密相连，几乎发音相同：

 - *i·Mbair Annui* —“西部土地”：*i*（文章）*·*（紧密结合点）*Mbair*（土地，鼻音突变：*b*→*mb*）
 - *an·i·laes* ᴺS。 —“to the leaf”：介词 *an*，然后是 *i*（冠词），然后是 *laes*（叶子），所有这些都在紧密结合点上

 现代辛达语编辑和学者使用凸起的点来表示这种紧密结合但没有完全融合的情况。 它出现在托尔金自己的一些手稿辛达语短语中，并已被采纳为学术惯例。

 ## 3. 辛达林语元音省略：基本规则

 ### 规则 1：冠词 *i* + 元音首词

当冠词 *i* (the) 位于以元音开头的单词之前时，在正式散文中**没有省略** - 两个元音处于中断状态：
 - *i aran* (国王) — *i* + *a* → 间断，两个元音均发音

 然而，在**快速语音**或**诗歌**中，*i* 可能会在后面的元音之前变成半元音 *y*，或者两者可能混合。 托尔金并没有在他所证实的文本中系统地标记这一点，因此这很大程度上是自然发音的问题。 

###规则2：介词+元音-词首

 有些介词在元音之前自然收缩或省略：

 **介词 *a* （以及呼格助词）**

 元音开头词之前的呼格/连词*a*：*a* + *Elbereth* = *A Elbereth* — 在这种情况下**无省略**； 托尔金在写*A Elbereth*时将*a*分开。 保留 *a* 是因为它标记了呼唤（呼唤某人）并且间断是故意的。 

**介词*o*（来自）**

 *o* + 元音首词：托尔金写下 *o Imladris*（来自瑞文戴尔），并以 *o* 站立； 没有省略。 再次，中断被维持。 

**介词 *na* (at, to, for)**

 *na-* 在随后突变的辅音之前：*na-chaered* = na + *caered* （带有软/鼻音突变 *c*→*ch*）。 这是连字符前缀形式，而不是省略。 

### 规则 3：诗意省略

 在**精灵诗句** (*linnad*) 中，省略用于调整节拍的音节数：
 - 元音开头单词之前以 *-a* 结尾的单词：*-a* 可能会被省略
 - 用撇号书写：*o'Imladris* 会省略 *I* 之前的 *o*

 托尔金并没有给我们提供许多完整的诗歌省略例子，但从他的诗歌形式的一般规则来看，其原理是清楚的。

 ## 4. *A Elbereth Gilthoniel*：逐行省略分析

 研究音韵接合点的最佳辛达语文本是赞美诗 *A Elbereth Gilthoniel*。 让我们检查一下它的所有接合现象实例。 

### 全文（来自*指环王*，“许多会议”）```
A Elbereth Gilthoniel,
silivren penna míriel
o menel aglar elenath!
Na-chaered palan-díriel
o galadhremmin ennorath,
Fanuilos, le linnathon
nef aear, si nef aearon!
```### 分析

 **第 1 行**：*Elbereth Gilthoniel*
 - *A* = 呼语助词（呼唤 Elbereth）； *E* 之前的中断 — 未省略； 停顿是有意义的
 - *Elbereth* = 瓦尔达的名字（辛达语）； 来自*el*（明星）+ *bereth*（女王）——星女王
 - *Gilthoniel* = Varda 的绰号来自 *gîl*（星星）+ *thon-*（点燃，生火）+ *-iel*（谁做了 X，阴性分词）=“她点燃了星星”
 - 此行中**无省略**； 无撇号的接缝处是为了重量和清晰度而精心设计的

 **第 2 行**：*silivren penna míriel*
 - *silivren* = 闪闪发光（如星星）：*silif-*（银色闪光）+ *-ren*（形容词后缀）
 - *penna* = 坡度、斜度； 这里是“倾斜/倾泻而下”
 - *míriel* = 像宝石一样闪闪发光：*mîr*（宝石）+ *-iel*（like-a，分词形式用作形容词）
 - **连接点**：*silivren penna* — 均以辅音结尾/开头； 没有省略。 *penna míriel* — *penna* 以 *-a* 结尾； *míriel* 以 *m* 开头：无元音冲突，无需省略。 
- **头韵**：*silivren...sil...* — *s/s* 头韵是有意的诗意音乐

 **第 3 行**：*o menel aglar elenath！*
 - *o* = 来自（介词）
 - *menel* = 诸天，苍穹
 - *aglar* = 荣耀，光辉（来自 *aglar* = 光辉，与 *agar* 血液有关？否 — *aglar* = 荣耀，来自 *'gl-* 根）
 - *elenath* = 所有星星（复数类别：*elen* + *-ath*）
 - **连接点分析**：*o menel* — *o* + *m*：无元音冲突。 *menel aglar* — *-el* 结尾 + *a-* 开头：间断 *el.a* — 在快速讲话中这可能会被省略，但托尔金将其写为两个单词。 *aglar elenath* — *-ar* 结尾 + *e-* 开头：间断 *ar.e* — 再次，没有省略。 
- **重要**：托尔金始终**不**忽略这​​些连接点； 间断是为了诗意的节奏而故意的

 **第 4 行**：*Na-chaered palan-díriel*
 - *Na-chaered*：*na-*（前缀：远处；方位）+ *chaered*（*caered* 在 *na-* 之后带有 *c*→*ch* 突变）
 - *caer-* 词根：巨大，广阔； *caered* = 遥远的距离
 - *Na-chaered* =“距离很远，很远”
 - *palan-díriel*：*palan-*（远而宽）+ *díriel*（凝视着 — *tir-* 观看，女性完成分词 *díriel* = 她凝视过）
 - **连字符的使用**：*na-* 和 *palan-* 都使用连字符来显示前缀附件，而不是省略
 - **突变**：*na-* 之后的 *c* → *ch*：这是混合/鼻突变（*na* 的牙齿前缀 *n-* + 以下 *c* → *ch*）

 **第 5 行**：*o galadhremmin ennorath*
 - *o* = 来自
 - *galadhremmin* = 树格状，与树木编织：*galadh*（树）+ *remmin*（网状，编织 — *rem* 网格 + *-min* 被动分词）
 - *ennorath* = 整个中部地区：*ennor* + *-ath*（类复数）
 - **Juncture**：*galadhremmin ennorath* — *-in* 结尾 + *e-* 开头：间断 *in.e* — 书写时不省略； 中断状态维持不变
 - **头韵/声音**：第 5 行中的 *gal/gal* 回声连接回第 1 行的 *Gilthoniel*

 **第 6 行**：*Fanuilos, le linnathon*
 - *Fanuilos* = 瓦尔达的另一个名字/绰号：*fan-*（云，白色？）+ *uilos*（永远的雪？或*uiles*，*ui* 永远 + *los* 雪）
 - *le* = e，你（第二人称代词，正式）
 - *linnathon* = 我会唱歌：*linna-*（唱歌）+ *-thon*（1sg 将来时后缀）— **完全证明的将来时形式**
 - **Juncture**: *Fanuilos, le* — 逗号暂停； 不是语音接合点的问题。 *le linnathon* — *le* 以元音 *e* 结尾； *linnathon* 以 *l* 开头：*e.l* — 元音 + 辅音，无需省略

**第 7 行**：*nef aear，si nef aearon！*
 - *nef* = 在这一侧
 - *aear* = 海
 - *si* = 现在，这里
 - *aearon* = 大海、海洋（*aear* + *-on* 增强）
 - **接合点**：*nef aear* — *nef* 以辅音 *f* 结尾； *aear* 以元音 *a* 开头：*f.a* — 不需要省略（辅音+元音连接）。 *si nef* — *si* 以 *i* 结尾； *nef* 以 *n* 开头：*i.n* — 无省略。 *nef aearon* — 与上面的 *nef aear* 相同。 
- **重复**：*nef aear, si nef aearon* — 第二个例子中几乎重复的短语加上增强的*aearon*（更大的海）创造了一个高潮结构：“在海的这一边，在大洋的这一边！”

 ### 摘要：*A Elbereth* 中的省略

 托尔金**没有**省略这首诗中的任何元音连接点。 所有间断点（元音结尾词后面跟着元音开头词）都被完整写出。 这可能反映出：
 1. **正式的赞美诗**——这是一首虔诚的歌曲； 省略可能感觉太随意
 2. **刻意的节奏**——间断有助于诗的韵律和重量
 3. **托尔金的作曲实践** - 他可能设计了这首诗，以便格律不需要省略

 ## 5. 属格结构中的省略

 ### *en* + 元音

 属格助词 *en* (of-the) + 随后的元音开头词创建了一个潜在的省略环境：

 标准形式：*en* + *Edhil*（精灵，属格：“精灵的”）→ *en Edhil* 或 *en·Edhil* 或 *en'Edhil*？ 

托尔金的做法：已证明的形式**不被省略**。 *en* 独立存在。 

然而，在**快速的非正式演讲**或**诗歌**中，*en* 在元音之前可能会变成 *'n*：
 - *Gwaith-i-Mírdain*：这里的属格是 *-i-*（不是 *en*）——使用的冠词形式，而不是助词； 表明所有格形式存在变异

 ### 融合与消除

 有些看起来像省略的形式实际上是**语法化的融合**——随着时间的推移，这两个词已经融合成一种形式，不再是实时的省略：

 - *erin* = *er* + *in*？ 否 — *erin* = *ar-*（在）+ *i-*（该）融合到 *erin*（在）中 — 已证明
 - *uin* = *o* (from) + *i* (the) → *uin* (from the) — *o* 和 *i* 融合
 - *ammin* = *am* (上/上) + *i* + *n* → 融合方位定形式

 这些是产生混合突变触发器的**历史融合**（第 30 课）。 它们不是“实时”省略——说话者不会有意识地删除元音； 现在的形式只是词汇形式。

 ## 6. 复合词的省略

 在辛达林语**复合词**（其中两个词干融合成一个词汇项）中，复合接缝处的元音可能：
 1. **简单地站在间隙**：*galadhremmin* — *galadh*（树）+ *remmin*（网） — *galadh* 的 *dh* + *remmin* 的 *r* 均保留； 没有元音省略，因为接缝是辅音-辅音
 2. **丢失一个元音**：如果两个元素都以元音结尾和开头，其中一个可能会收缩或下降

 复合词中元音接触的示例：
 - *Nindalf*（*nîn* 湿 + *dalf* 平坦表面） — *n* + *d* 接缝； 没有元音冲突
 - *Elanor* (*el* star + *anor* sun) — *el* + *an* → *elanor* — *anor* 的元音 *a* 紧跟在 *el* 的 *l* 之后； 他们处于中断状态：*el.a* 没有省略

 ## 7. 精灵诗句中的省略和韵律

 ### *林纳德*：精灵诗句形式

 *林纳德*（字面意思是“歌唱”）是辛达语中精灵诗句的术语。 托尔金描述了精灵诗歌的几个特点：
 1. **头韵**（跨行辅音重复）
 2. **内部韵律或谐音**（行内的元音模式）
 3. **基于音节计数的韵律**（尽管不如古典拉丁韵律那么严格）
 4. **具有重复模式的诗节结构**

 出于音节计数的目的，**省略可用作一种韵律工具**：如果一行有太多音节，诗人可以在连接处省略一个元音，以将计数减少一。 

### *Namárië* 中的省略（昆雅语参考）

 在托尔金的昆雅诗*Namárië*（加拉德瑞尔的挽歌）中，有一个精灵诗句中省略的明显例子：

 > *艾！ 劳里·兰塔尔·拉西·苏里宁*

这里 *laurië* 以 *-ie* 结尾，而 *lantar* 以 *l* 开头 — 辅音开头，无元音冲突。 但在后面的几行中，-e/-a 接触点是根据诗歌的节奏仔细导航的。 

虽然 *Namárië* 是昆雅语，而不是辛达林语，但它展示了托尔金的一般方法：通过词语选择和声音设计，**避免诗歌中的严重中断**，并以省略作为最后的手段。

 ## 8. 语法化省略：冠词和介词融合

 正如第 5 节所述，辛达林语中的一些“省略”已成为固定的语法形式。 这些值得编目：

 | 表格 | 分析| 意义| 突变触发|
 |------|----------|---------|--------------------|
 | *艾琳* | *ar-* + *i-n* | 上 | 混合突变|
 | *uin* | *o* + *i* + *n*？ | 从| 混合突变|
 | *阿明* | *上午-* + *我-* | 直到 | 混合突变|
 | *一个·我* | *一个* + *我* | 到/为了| 鼻突变|
 | *zh* | 的（所有格）| — | 下一个单词的软突变 |
 | *迪* | 在下面，在（+文章）下| — | — |

 这些形式并不是演讲者临时创造的省略形式；而是由演讲者创造的。 它们是**固定的词汇形式**，历史上恰好是由两个单词的融合产生的。

 ## 9. 凸起点的详细信息

 凸起的点*·*（或中间的点）出现在托尔金的辛达林语手稿中，其上下文如下：

 > *i·Mbair Annui*（“西部土地”）

 这里：
 - *i* = 冠词“the”
 - *·* = 闭合连接标记（冠词和名词在语法上结合在一起）
 - *Mbair* = lands（在复数冠词后带有鼻音突变*B-*：*bair* → *mbair*；*M* 表示冠词已被吸收）

 凸起的点表明 *i* 和 *Mbair* 在语法上是一个单位（冠词 + 名词短语），即使它们被写为两个标记。 它是一个**韵律和句法**标记，而不是语音省略标记。 

现代辛达语学者和新辛达语作家一致使用 *·* 来表示冠词绑定而不省略，特别是当后面的名词以因冠词而发生突变的辅音开头时。

 ## 10. 三种标点符号工具：总结

 | 马克| 名称 | 功能| 示例|
 |------|------|----------|---------|
 | **'* | 撇号| (1) 软突变下掉了*g*； (2) 诗意元音省略 | *i 'aladh*（树；g 掉落）； *o'Imladris* ᴺS。 （来自瑞文戴尔的诗意）|
 | *-* | 连字符 | 前缀附件； 复合连接； 紧密化合物（非完全融合）| *帕兰-迪瑞尔*； *na-chaered*； *Bar-en-Danwedh* |
 | *·* | 凸起点| 关闭关口； 不融合的冠词-名词绑定 | *我·姆拜尔* |

 **经验法则**：
 - 在非复合词的元音之前看到 *'* → 可能删除了 *g* （软突变）
 - 请参阅元素之间的 *-* → 前缀或闭合复合词
 - 参见 *·* → 冠词/助词 + 紧密句法键中的名词

 ---

 ## 11. 练习：识别这些文本中的省略和连接

 分析以下五个段落的省略号（撇号）、接合点（凸起点）和复合/前缀连字符。 注意每个标记代表什么。 

**第 1 段**：*Naur an edraith ammen！*
 （“为我们的拯救而火！”——甘道夫谈卡拉德拉斯）
 - 分析：*Naur* = 火（独立）。 *an* = for/to（介词）。 *edraith* = 救援，拯救（*e-* + *draith* 来自 *dar-* 来拯救）。 *ammen* = 我们（*am-* + *men* → 对于我们来说，1pl 对象）。 无省略标记； 没有撇号。 所有辅音接合处或间断均得以保留。 

**第 2 段**：*Ennyn Durin Aran Moria：pedo mellon a minno！*
 （“莫瑞亚领主都林之门：说朋友并进入！”）
 - *Ennyn* = 门（*annon* 的复数形式，i-情感：*a*→*e*）。 *Durin* = 专有名称。 *阿兰* = 国王/领主。 *Moria* = 黑色裂缝（*mor* 黑暗 + *ia* 裂缝）。 *pedo*=说话！ （至关重要的）。 *梅隆*=朋友。 *a* = 和。 *minno* = 输入！ （命令式*分钟-* + *-否*）。 此证明文本中没有省略号。

**第 3 段**：*Cuio i Pheriannath anann！*
 （“愿半身人长寿！”）
 - *Cuio* = 可以[他们]活着：*cuia-* + 虚拟语气*-o*。 *i* =（文章）。 *Pheriannath* = P→Ph（*i* 之后 *P* 的软突变）。 *anann* = 长，很长一段时间（*an-* + *ann* 长）。 没有省略； *Ph-* 显示软突变，而不是省略。 

**第 4 段**：*A tiro nin，Fanuilos！*
 （“哦，看着我，法努伊洛斯！”）
 - *A* = 呼语助词（呼唤）。 *提罗*=看！ 看！ （命令式 *tir-* + *-o*）。 *nin* = me（1sg 宾语代词：*ni* + *n* 附件）。 *Fanuilos* = 瓦尔达的名字。 无省略标记。 

**第 5 段**（ᴺS.作文）：*I 'aladh maer nâ faer sui i·'waew。*
 （“树快如风。”）
 - *I* = 文章。 *'aladh* = 树（撇号：*galadh* 的 *g* 在软突变下掉落）。 *maer* = 好/很好。 *nâ* = 是 (3sg copula)。 *faer* = 迅速。 *sui* = 如，像。 *i·'waew* = 风：*i*（文章）*·*（关闭关口）*'waew*（撇号：*g* 的*g* 风在*i* 之后的软突变下下降）。 两个撇号：都标记为*g*，而不是元音省略。

 ## 需要记住的要点

1. **三种功能不同的标点符号**：*'*（删除辅音或诗意省略）、*-*（前缀/复合词）、*·*（闭合连接点）
 2. **托尔金在他经过证实的辛达语文本中很少使用元音省略**——通常会保留间断，特别是在正式/诗歌语域中
 3. **语法化融合** (*erin*, *uin*) 历史上源自省略/融合，但现在是固定形式
 4. **诗歌中的省略**可作为一种格律工具——当音节数需要调整时
 5. **软突变撇号**（删除*g*）是迄今为止托尔金文本中*'*最常见的用法——不要将其与元音省略混淆
