## 托尔金证实的短语

 与许多人工语言相比，辛达林语最令人兴奋的特征之一是托尔金在《指环王》、其附录和他未发表的手稿（其中许多出现在《中土世界的历史》系列和《帕尔玛埃尔达兰伯龙》期刊中）中嵌入了真实的、语法上可分析的短语。 这意味着学习者的第一句话可能是真正的托尔金，而不是后来的粉丝的发明。 

本课中的每个短语要么在托尔金的著作中**直接证明**，要么是**最小重构** - 也就是说，它仅使用经过证明的语法规则组合的经过证明的单词，没有创造性的飞跃。 如果短语有任何不确定性，则会予以注明。 经过验证的辛达林语标记为 **S.**，经过验证的诺尔多林语（引入辛达林语的早期版本）标记为 **N.**，重建的新辛达林语标记为 **ᴺS.**

 ---

 ## 问候

 | 辛达林语 | 发音 | 翻译 | 笔记|
 |----------|----------------|-------------|--------|
 | *梅·戈万宁！* | /mai ɡɔˈvannɛn/ | 好遇见！ | **S.** 标准的精灵语问候语。 在《魔戒》中使用（格洛芬德尔以此向霍比特人打招呼）。 点燃。 “我们见面了。” |
 | *Vedui'！* | /ˈvɛdui/ | 冰雹！ 问候！ | **S.** 格洛芬德尔在布鲁伊南渡口向弗罗多使用。 撇号表示省略最后一个元音 (*vedui·e* → *vedui'*)。 |
 | *苏拉德！* | /ˈsuilad/ | 问候！ | **N./ᴺS.** 来自动词 *suila-*“问候、致敬”。 比 *vedui'* 更正式。 |
 | *Êl síla erin lû 电子管理 vîn* | /ɛːl ˈsiːlɑ ˈɛrin luː ɛ ɡɔˈvɑnɛd viːn/ | 一颗星星闪耀在我们相会的时刻| **S.** 经典精灵语问候语的辛达语表述。 *êl* = 星星，*síla* = 闪耀，*erin* = 在（+ 软突变）上，*lû* = 小时，*e-govaned* = 会议，*vîn* = 我们的。 |

 **关于*Mae govannen*的文化注释：** 精灵使用这种问候语是因为他们的聚会通常很少见且有意义——多年分离后的相遇，或者与同类的第一次会面。 语法对此进行编码：*govannen* 是 *govad-* 的过去被动分词（一起见面，走到一起），因此该短语的字面意思是“well have-we-been-met”或“well are we met”。 这种相互的、被动的结构强调会议是一次共同的活动，而不是一个人对另一个人做某事。

 ## 告别

 | 辛达林语 | 发音 | 翻译 | 笔记|
 |----------|----------------|-------------|--------|
 | *纳瓦尔！* | /ˈnavaɛr/ | 告别！ | **S.** 由 *na* (be) + *vaer*（*maer* 的良好、宽松形式）组成。 所以字面意思是“做好人！” 或“愿一切顺利！” ——对他人幸福的祝愿。 *na* 之后的 *m* → *v* 的突变是常规软突变（第 10-11 课）。 |
 | *诺瓦尔！* | /ˈnɔvaɛr/ | 告别！ | **S./ᴺS.** *navaer* 的变体； *no-* 形式代表一种替代的命令式/祈愿式。 托尔金的语言笔记中引用了这两种形式。 |

 **分解*纳瓦尔*：**
 - *na* = “be”（动词 *na-* 的祈使式/祈愿式，第 26 课）
 - *vaer* = *maer* 的 lenited 形式（好、很好、优秀）
 - 综合：“乖一点！” → 作为告别：“愿你一切安好！”
 - *m* → *v* 的软突变是您将遇到的第一个突变触发器：单词 *na* 导致后面的单词发生软突变。 （第 10-11 课中有更多相关内容。）

 ## 感谢和礼貌

 | 辛达林语 | 发音 | 翻译 | 笔记|
 |----------|----------------|-------------|--------|
 | *汉农勒* | /ˈhannɔn lɛ/ | 谢谢（对一个人）| **S.** 直接在 LotR 中得到证明。 *Hannon* = 我感谢（*hanna-* 的 1sg 礼物），*le* = thee/you（sg. 代词）。 佛罗多在镜子前对凯兰崔尔使用。 |
 | *勒汉农* | /lɛ ˈhannɔn/ | 谢谢（强调“你”）| **ᴺS.** 带有前置代词的变体，用于稍微强调 - “我感谢你。” |
 | *红莲 bêd enni* | /ˈɡurɛn bɛːd ˈɛnni/ | 我的心告诉我| **S.** 选自“The Road Goes Ever On”（托尔金的歌曲组）。 *Guren* = 我的心 (*gûr* + 所有格 *-en*)，*bêd* = 告诉/说，*enni* = 对我说。 |

**关于*hannon le*：** 这句话在托尔金的世界里具有非常直接、亲密的品质。 *Le* 是熟悉/亲密的第二人称代词 - 与 *mae govannen* 和其他密切接触的短语中出现的单词相同。 当佛罗多对加拉德瑞尔说“hannon le”时，选择“le”（熟悉）而不是更正式的结构反映了精灵特有的亲密感——精灵用亲密的语气称呼他们深深尊敬的人，就像人们可能在法语中对心爱的老师或父母使用“tu”一样。

 ## *《指环王》* 中的常用短语

 以下是文本中一些经证实的最著名的辛达语段落：

 | 辛达林语 | 发音 | 翻译 | 来源 |
 |----------|----------------|-------------|--------|
 | *Le abdollen* | /lɛ abˈdɔllɛn/ | 你迟到了| 附录中阿尔文对阿拉贡说。 *Le* = 你，*abdollen* =“迟到”（*ab-tol-* 的过去分词意思是“迟到/晚到”）。 |
 | *佩多·梅隆和米诺人* | /ˈpɛdɔ ˈmɛllɔn a ˈminnɔ/ | 说吧，朋友，然后进入 | 都林铭文之门。 *Pedo* = 说话！ （*ped-* 的命令），*mellon* = 朋友，*a* = 并且，*minno* = 输入！ （*min-* 的命令）。 |
 | *Cuio i Pheriannath anann！* | /ˈkuiɔ i ˈfɛriannɑθ ˈɑnɑnn/ | 愿半身人长寿！ | 科马伦庆祝活动。 *Cuio* = 直播！ （*cuia-* 的可选/命令），*i Pheriannath* = 半身人（*ph-* 是 lenited *p-*），*anann* = 长（adv.）。 |
 | *阿莱塔特，莱塔特！* | /ɑ ˈlaitɑ tɛ ˈlaitɑ tɛ/ | 赞美他们，赞美他们！ | 科马伦。 *A* = 感叹词，*laita* = 赞美！ （*laita-* 的命令），*te* = 他们。 |
 | *安达夫·莱图瓦尔梅特！* | /ɑnˈdɑvɛ lɑiˈtuvɑlmɛt/ | 我们要永远赞美他们！ | 同样的场景。 *Andave* = long（副词，来自 *and*），*laituvalmet* = 我们将赞美他们（复杂动词形式）。 注：一些学者将此短语归类为昆雅语或正式仪式中使用的昆雅语-辛达语混合语。 |

 ### 详细分析：*Pedo Mellon a Minno*

 这是托尔金作品中最著名的短语之一——都林之门之谜。 甘道夫的错误在于将其解读为描述（“说‘朋友’并输入”），而实际上它是一条指令（“说‘朋友’并输入”，即简单地说出单词*mellon*）。 

- **Pedo**：*ped-* 的命令式（命令形式）“说话，说”。 主要动词的命令式由添加 *-o*: *ped-* → *pedo* 构成。 （第 17 课全面介绍了这一点。）
 - **梅隆**：“朋友。” 源自词根*MEL-*“爱”。 与*mel-*（爱）和*meleth*（爱）相关。 相同的词根给出了昆雅语 *Melkor* （一个以不同的词根出现的词根 - 实际上 *Melkor* 不是来自 *mel-*，但 *mellon* 本身是明确的）。 
- **a**：连词“and”。 在辛达林语中很常见。 
- **minno**：*minna-* 的祈使句“进入，进入。” *Minna-* + 命令式 *-o* → *minno*。 

命令式结构（动词 + *-o*）再次出现在*cuio*（活！）、*sogo*（喝！）、*mado*（吃！）中——一种富有成效且规则的模式。

 ## Cormallen 的赞扬

 《国王归来》中的科马伦庆祝场景包含了经过验证的辛达林语的显着集中。 至尊戒被摧毁后，刚铎和洛汗的大军赞扬了持戒者：

 *阿莱塔特，莱塔特！ 安达夫·莱图瓦尔梅特！ Cormacolindor，a laita tárienna！*

 分解一下：
 - *A* — 感叹词“O！Ah！”
 - *laita te* —“赞美他们”（*laita* = 命令式，*te* = 他们，3pl 宾语代词）
 - *laita te* — 重复强调
 - *Andave* — “长，很长一段时间”（副词）
 - *laituvalmet* — “我们会赞扬他们”（将来时 1pl 带有 3pl 宾语后缀；可能是昆雅语形式）
 - *Cormacolindor* —“持戒者”（可能是昆雅语：*corma* = 戒指 + *col-* = 熊 + *-indo-r* pl.）
 - *a laita tárienna* —“并且赞美到最高点/赞美他们到最高点”（*tári* = 女王/身高，*-enna* = 与格后缀）

**语言注释：** 在仪式演讲中混合辛达林语和昆雅语形式是有意的。 刚铎努门诺尔人后裔的正式仪式经常使用昆雅语来表达高贵的风格，而通用语言则是辛达林语。 这个场景捕捉到了双语现象。

 ## 打电话和自我介绍

 | 辛达林语 | 翻译 | 笔记|
 |----------|-------------|--------|
 | *我...* | 我是... | **S.** *Im* 是强调第一人称代词（第 8 课）。 用于自我介绍：*Im Aragorn* =“我[am] Aragorn。” 请注意，辛达林语在简单的名义身份陈述中不需要动词“to be”。 |
 | *吉利尔* | 我很高兴| **ᴺS.** 来自*geli-*“高兴，高兴。” *Gelir* = 目前 1sg。 |
 | *Suilon le* | 我向你问好| **ᴺS.** *Suila-* + 1sg 后缀 *-on* = *suilon*; *le* = 你。 |

 ## 逐字分解*Mae Govannen*

 这个短语值得详细研究，因为它包含几个重要的语法特征：

 **梅** [迈]
 - 副词：“很好，很好，非常好”
 - 与*maer*相关（形容词“好，很好”），您也可以在上面的*Navaer*中看到（*vaer* = lenited *maer*）
 - 与昆雅语 *máre* 同源“好，好”

 **govannen** [ɡɔˈvannɛn]
 - *govad-* 的过去被动分词（与会面、聚集在一起）
 - *Go-* 是一个前缀，意思是“一起，在公司”（来自词根 *WO-*“一起”）
 - *-pad-* 或相关根给出“会议”的感觉
 - *-(n)nen* 是辛达林语过去被动分词结尾
 - 所以： *go-* + 动词词根 + *-nnen* → *govannen* = “together-met，一起见面”

 **整个短语：** 没有明确的主语代词 - 动词形式 *govannen* 暗示“我们”。 这句话的结构是被动/参与式：“（我们）很相遇（在一起）。” 它既是事实陈述（“我们相遇得很好”），又是愉悦的表达（“我们相遇真是太好了！”）。 

这种双重品质——陈述和情感表达在一起——非常精灵。 托尔金的精灵倾向于在语法中编码情感的精确语言，而不是添加单独的情感词。

 ## 练习对话

 应用第一课的发音规则，大声朗读这段简短的对话：

 > **阿兰迪尔：** *Mae govannen！*
 >
 > **Elenmir：** *Mae govannen！ 我是埃伦米尔。 伙计，我有能力吗？*
 >
 > **阿兰迪尔：** *我是阿兰迪尔。 汉农勒*
 >
 > **Elenmir：** *Gelir im。 纳瓦尔！*
 >
 > **阿兰迪尔：** *纳瓦尔！*

 翻译：
 - “很高兴认识！”
 - “很高兴认识你！我是埃伦米尔。你叫什么名字？” (*man* = 什么，*ieneth* = 名字，*lín* = 你的)
 - “我是阿兰迪尔。谢谢你。”
 - “我很高兴。再见！”
 - “告别！”

 请注意，*Navaer* 是两个说话者在分手时说的。 这是正常的模式——双方互相道别，而不仅仅是告别的人。 精灵的告别文化往往是相互的、温暖的。

 ## 总结

 现在，您已经拥有了一套真正的托尔金核心短语，用于开始、结束和引导基本的辛达语对话。 要记住的关键模式：

 1. **Mae govannen** =“很高兴见到”——通用的精灵语问候语
 2. **Hannon le** =“我谢谢你”——直接宾语代词 *le* 跟在动词后面
 3. **Navaer** =“告别”——*na-* + lenited *maer* 的命令式
 4. **Pedo mellon a minno** =“说朋友并进入” - 命令式 + 名词 + and + 命令式
 5. **Im ...** =“我是...”——自我介绍的强调代词，不需要动词
