## 概述

 问候是通向每种语言的大门，在世界语中，问候具有特殊的重要性，因为两个世界语学家之间的每一次相遇也是一种团结的行为——一种对不属于任何单一国家的语言的共同承诺。 当你说“萨鲁顿！”时 对于另一位发言者来说，您正在参与 140 年人们选择沟通而不是分离的传统。 因此，学习世界语问候和介绍不仅仅是实用的词汇练习，更是练习。 它是进入全球社会的切入点。 

从语法的角度来看，本课介绍世界语最重要的两个基础：完整的人称代词集和以 **-as** 结尾的现在时动词。 因为世界语动词不会改变形式来与主语一致——“mi estas”、“vi estas”、“li estas”都使用相同的结尾——你会发现，在本课之后，你已经可以写出数量惊人的语法正确的句子。 

## 学习目标

 在本课程结束时，您可以：
 - 在早上、白天和晚上使用标准问候语
 - 使用两种不同的结构（nomiĝas / nomo estas）介绍自己
 - 正确识别和使用所有九个世界语人称代词
 - 通过在词根上添加 **-as** 来结合现在时态的任何动词
 - 使用 **estas** 作为系词来做出“X is Y”陈述

 ## 语法重点

 ### 模式 1：现在时结尾 -as

 **结构：** 动词根 + **-as** = 现在时（适用于所有人称和数字）

 这是世界语语法最自由的特征之一。 在英语中，你必须记住“我是/你是/他是/我们是/他们是”——单个动词的六种不同形式。 在法语、西班牙语或德语中，每个动词都有一个包含六种共轭形式的表。 在世界语中，无论主语是谁或是什么，只要在动词词根上添加 **-as** 就可以了。 动词**永远不会改变**以与其主语一致。

动词词根是您在字典中找到的形式减去不定式结尾**-i**。 所以**lern-i**（学习）有词根**lern-**，每个人的现在时都是**lernas**。 同样，**far-i**（要做）→ **faras**； **est-i**（将）→ **estas**； **parol-i**（说话）→ **parolas**。 

| 主题 | 动词| 造句| 英语 |
 |---------|------|----------|---------|
 | 米| 勒纳斯 | 米勒纳斯世界语。 | 我正在学习世界语。 |
 | 六 | 勒纳斯 | Vi lernas 世界语。 | 您正在学习世界语。 |
 | 李 | 勒纳斯 | Li lernas 世界语。 | 他正在学习世界语。 |
 | 倪| 勒纳斯 | 尼尔纳斯世界语。 | 我们正在学习世界语。 |
 | 伊犁 | 勒纳斯 | Ili lernas 世界语。 | 他们正在学习世界语。 |

 常见错误：添加西班牙语或法语中的主语一致词尾 → ❌“mi lerno / vi lernas / li lernos”（混合时态）→ ✓“mi lernas / vi lernas / li lernas”（所有词尾相同）

 ### 模式 2：Estas 作为 Copula

 **结构：** 主语 + **estas** + 补语（名词或形容词）

 动词 **estas** (is/am/are) 的作用与英语中的“to be”作为连接动词完全相同。 它将主题与描述或标识联系起来。 在世界语中，**estas** 在现在时中是不变的——无论主语是谁，它的形式都不会改变。 用作补足语的名词前不需要冠词（世界语没有不定冠词，定冠词 **la** 仅在指代特定时才使用）。 

请注意，**vi** 涵盖单数“you”和复数“you”——世界语在称呼一个人或多个人之间没有语法区别。 语境和礼貌程度是通过其他方式传达的，而不是动词形式。 

| 造句| 英语 |
 |----------|---------|
 | 我是学生。 | 我是一名学生。 |
 | Vi estas instruisto。 | 你是一位老师。 |
 | 我爱你。 | 他是我的兄弟。 |
 | 我已经是世界语了。 | 她是一名（女）世界语学家。 |
 | Ni estas geamikoj。 | 我们是朋友（混合组）。 |

 常见错误：使用**esti**（不定式）而不是**estas**（有限现在时）→ ❌“Mi esti Studento”。 → ✓“我是学生。”

 ### 模式 3：自我介绍 — 两种结构

 **结构 A：** Mi nomiĝas [名称]。 
**结构 B：** Mia nomo estas [名称]。 

这两种结构在语法上都是正确的并且被广泛使用。 **Nomiĝas** 由动词 **nomi** （命名）加上反身后缀 **-iĝ-** （成为/处于某种状态）组成，意思是“我被称为/我为自己命名”。 它稍微优雅和惯用。 第二种结构 — **mia nomo estas** — 在 A1 阶段更加透明且更容易构建，因为它仅使用您已经知道的系词 **estas**。 您将在自然的世界语对话中听到和阅读这两种语言。 

| 造句| 英语 |
 |----------|---------|
 | 我的名字是 Kenji。 | 我的名字是健二。 / 我叫健二。 |
 | 我现在是健二。 | 我的名字是健二。 |
 | 通过 nomo Kio estas？ | 你叫什么名字？ |
 | 基尔 vi 提名？ | 你叫什么？ / 你叫什么名字？ |

 常见错误：逐字翻译“My name is” → ❌“Mia nomo estas nomiĝas Kenji”。 → ✓ 仅使用一种结构：“Mi nomiĝas Kenji”或“Mia nomo estas Kenji”。

 **在世界语俱乐部会议上**

 ——萨鲁顿！ 我提名为玛尔塔。 卡吉维？ 
- 你好！ 我叫玛尔塔。 你呢？ 

——博南维斯佩隆，玛尔塔！ 我是田中。 我是日本人。 
— 晚上好，玛尔塔！ 我是田中。 我来自日本。 

——嗬，有趣！ 我是巴西罗。 你能长久地学习世界语吗？ 
——哦，有趣！ 我来自巴西。 您学习世界语很久了吗？ 

— Ne，nur tri monatojn。 卡吉维？ 
——不，只有三个月。 你呢？ 

— 米勒纳斯·德·杜·哈罗伊。 基尔 vi fartas hodiaŭ，田中？ 
——我已经学习两年了。 田中，今天怎么样？ 

— 我的骨头，dankon！ 这是一个很糟糕的事情。 
——我很好，谢谢！ 这个俱乐部很棒。 

——杰斯！ Ĉiuj estas tre afablaj。 Ĉu vi konas Anan？ 
- 是的！ 每个人都非常友好。 你认识安娜吗？

— Ne ankoraŭ。 Kiu estas Ana？ 
- 还没有。 安娜是谁？ 

——这是尼亚·克鲁博的前言。 我是善意的人。 
— 她是我们俱乐部的主席。 她是一位非常好的老师。 

——博内格。 Mi ŝatus konatiĝi kun ŝi！ 
- 精彩的。 我很想见见她！

## 文化笔记

 世界语问候语“Saluton！” 有一种超越其字面意义的温暖。 在国际世界语聚会上——从每年吸引数千名参与者的Universa Kongreso（世界大会）到小型地区俱乐部——当两个陌生人用世界语互相问候的那一刻，他们体验到了社区所谓的“interna ideo”（内在想法）：相信跨文化界限的交流是可能的并且值得追求。 有些世界语学家将他们第一次成功地用世界语与外国人进行对话描述为一次深刻的联系体验。 

**Pasporta Servo**（护照服务）是世界语专家的全球网络，为旅行的演讲者提供免费住宿。 当您敲响一位从未见过的主人的门并受到“Bonvenon！Mi estas [名字]”的欢迎时，您就参与了一个完全围绕共享语言构建的信任社区。 这使得世界语中的问候语具有独特的意义——它们很少只是闲聊。
