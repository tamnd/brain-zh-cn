## 概述

 家庭词汇是通用的——每个人都有某种家庭，谈论亲戚是人们互相认识时做的第一件事之一。 在世界语中，家族词典完美地阐释了该语言中两种最优雅、最高效的手段：**-in-** 后缀，它可以从任何词根创建女性形式，以及 **ge-** 前缀，它指的是混合性别群体。 您无需记住“父亲”和“母亲”、“兄弟”和“姐妹”、“儿子”和“女儿”等单独的单词，而是学习一个词根并应用一个后缀。 整个家谱可以用几个词根和两个词缀来描述。 

本课还介绍了复数结尾 **-j**，当指代多个名词（以及与其一致的形容词）时，将其添加到所有名词（以及与其一致的形容词）中。 结合所有格形容词（mia、via、lia、ŝia...），您将获得谈论自己的家庭和询问他人的语法工具——这是定义早期语言学习的个人对话的基本技能。 

## 学习目标

 在本课程结束时，您可以：
 - 以男性和女性形式命名直系亲属和大家庭成员
 - 应用 **-in-** 后缀创建家庭名词的女性变体
 - 使用 **ge-** 前缀来指代混合性别的家庭成员群体
 - 通过在任何名词上添加**-j**来形成复数名词
 - 使用所有九个所有格形容词（mia、via、lia、ŝia、ĝia、nia、via、ilia、onia）

 ## 语法重点

 ### 模式 1：女性形式的 -in- 后缀

 **结构：** 阳性（或中性）名词词根 + **-in-** + **-o** = 阴性名词

 世界语对语法性别的处理方式非常简洁。 名词并不像法语或西班牙语那样本质上是阳性或阴性（其中“table”是阴性，“book”是阳性，没有语义动机）。 相反，世界语名词词根默认是中性的，或者如果它们描述人，则可能被认为是通用/男性的。 要指定女性，请在以 **-o** 结尾的名词前添加后缀 **-in-**。 这会为每个相关名词生成一个完全可预测的对。

该系统有少数以女性形式为基础的例外情况（例如，**virino** = 女人，从 **vir-** = 男人 + **-in-** = 女性→“女性”，以及 **knabo** / **knabino** = 男孩/女孩），但实际上，**-in-** 规则涵盖了您在 A1 级别遇到的绝大多数情况。 

| 男性/中性| 女性化| 英语对|
 |--------------------|----------|-------------|
 | 帕特罗 | 帕特里诺| 父亲/母亲|
 | 弗拉托| 弗拉蒂诺 | 兄弟/姐妹|
 | 菲洛 | 菲利诺 | 儿子/女儿|
 | 鳄梨| 阿维诺 | 祖父/祖母|
 | 昂克洛 | 在线 | 叔叔/阿姨|
 | 埃佐 | 埃吉诺 | 丈夫/妻子|
 | 乐器 | 仪器| (男)老师/(女)老师|
 | 亚美子 | 网野 | （男）朋友/（女）朋友|

 常见错误：将词根视为男性专用并在描述女性时省略 -in- → ❌“Mia patrino estas bona patro”（我的母亲是一位好父亲）→ ✓“Mia patrino estas bona patrino”（我的母亲是一位好母亲）

 ### 模式 2：混合性别群体的 ge- 前缀

 **结构：** **ge-** + 复数名词 = 包含男性和女性的群体

 前缀 **ge-** 解决了许多语言中的一个常见问题：如何指代包含男性和女性的群体而不默认使用男性复数（如法语和西班牙语）？ 在世界语中，**ge-** 明确表示混合性别群体。 **Gepatroj** 的意思是“父母”（父亲和母亲在一起）。 **Gefratoj** 的意思是“兄弟姐妹”（兄弟姐妹的统称）。 **Geamikoj** 的意思是男女“朋友”。 

**ge-** 前缀几乎总是与复数 **-j** 结尾一起使用，因为单个个体不能是“两种性别”。 唯一的例外是**gepatro**（性别中立的父母，理论上），很少使用； 在实践中，您会看到**gepatroj**（父母作为一对或一组）。 请注意，**ge-** 单词也需要复数所有格形容词形式。 

| ge-字| 英语 | 例句|
 |----------|---------|-----------------|
 | gepatroj | 家长 | Miaj gepatroj estas bonajhomoj。 |
 | 杰夫拉托 | 兄弟姐妹| Ni estas tri gfratoj。 |
 | 盖菲洛 | 孩子（自己的） | Ili havas du gefilojn。 |
 | 杰沃伊 | 祖父母 | Miaj geavoj loĝas en Italio。 |
 | 盖佐伊 | 已婚夫妇| La Geedzoj festas sian datrevenon。 |
 | 吉米科伊 | 朋友（混合）| Miaj gamikoj estas tre interesaj。 |

 常见错误：在指代一个人时使用 **ge-** 与单数名词→ ❌“Mia gepatro estas kuracisto”（不常见，暗示双性父母之一）→ ✓ “Mia patro/patrino estas kuracisto”表示父母之一； 两者的“Miaj gepatroj”

 ### 模式 3：复数 -j 和所有格形容词

 **结构：**名词 + **-j** = 复数； **mia、via、lia、ŝia、ĝia、nia、ilia** + 名词 = 占有

 每个世界语名词都通过添加**-j**（发音像英语“y”）来形成其复数：**libro**→**libroj**（书籍），**patro**→**patroj**（父亲），**infano**→**infanoj**（孩子）。 这是完全正常的，没有例外。 当形容词修饰复数名词时，也需要**-j**：**bonaj libroj**（好书），**grandaj familioj**（大家庭）。 

所有格形容词由代词添加 **-a** 构成：**mi** → **mia**（我的）、**vi** → **via**（你的）、**li** → **lia**（他的）、**ŝi** → **ŝia**（她）、**ĝi** → **ĝia**（其）、**ni** → **nia**（我们的）、**ili** → **ilia**（他们的）。 像所有形容词一样，它们在数量上与所修饰的名词一致：**mia libro**（我的书）→ **miaj libroj**（我的书）。 

| 占有欲| 示例（单数）| 示例（复数）|
 |------------------------|--------------------|-----------------|
 | 米娅（我的）| 米娅·帕特罗 | 米亚吉帕特罗伊 |
 | 通过（你的）| 通过弗拉蒂诺| 通过杰夫拉托 |
 | 利亚（他的）| 利亚·费洛 | 利亚吉·格菲洛伊 |
 | 尼亚（她）| 尼亚阿维诺 | ŝiaj geavoj |
 | 尼亚（我们的）| 尼亚家族| 尼亚吉·帕伦科伊 |
 | 伊利亚（他们的）| 伊利亚·埃佐 | 伊利亚吉·盖德佐伊 |

 常见错误：当名词为复数时，忘记所有格上的 **-j** → ❌ "mia gepatroj" → ✓ "miaj gepatroj"

 ## 对话

**显示家庭照片**

 — 你是通过照片拍摄的吗？ 
— 我可以看看你的照片吗？ 

——康普雷内布尔！ 我的家人。 请注意以下事项。 
- 当然！ 这是我的家人。 这是我的父母。 

— Kiaj afablajhomoj！ 基尔提名通过帕特罗？ 
——人们多么友善啊！ 你父亲叫什么名字？ 

— 提名为罗伯托。 Liestas kuracisto。 Kaj ĉi tie estas mia patrino，玛尔塔。 
——他的名字叫罗伯托。 他是一名医生。 这是我的母亲玛尔塔。 

— Kaj ĉu vi havas gfratojn？ 
— 你有兄弟姐妹吗？ 

— Jes，我有一个 fraton kaj du fratinojn。 我现在是保罗。 Li havas dudek kvin jarojn。 
— 是的，我有一个兄弟和两个姐妹。 这是我的兄弟保罗。 他今年二十五岁。 

— Kaj la fratinojn？ 
——姐妹们呢？ 

— Ana 是我的，我已经做好了准备。 Kaj la plej juna estas Sofia, ŝi havas dek kvin jarojn.
 — 这是安娜，她十八岁。 最小的是索菲亚，她十五岁。 

— 老大一家！ 是否有关系？ 
——一个大家庭！ 你的祖父母也住在这里吗？ 

— Ne, miaj geavoj loĝas en la kamparo, proksime de Krakovo.
 — 不，我的祖父母住在克拉科夫附近的乡村。

## 文化笔记

 几十年来，**-in-** 后缀系统在世界语界引起了有趣的争论。 一些发言者认为，“默认”的阳性或中性词根带有明确标记的阴性形式反映了该语言 19 世纪起源中固有的文化偏见。 作为回应，社区制定了替代惯例：一些现代说话者使用 **iĉ-** 作为明确的男性标记（因此 **patriĉo** = 父亲，**patrino** = 母亲，**patro** = 父母，无论性别），但这种用法不是标准世界语的一部分，仍然是个人和社区选择的问题。 

与此同时，**ge-** 前缀因其实用性而受到喜爱。 在英语等语言中，提及混合群体需要迂回说法（“兄弟姐妹”、“母亲和父亲”），但在世界语中，一个单词 - **gefratoj**、**gepatroj** - 表达了相同的想法。 当世界语学家在国际会议上谈论他们的家庭时，这种效率是真正值得赞赏的：这是一个小例子，说明有计划的语言如何改善自然语言中的歧义和不对称性。
