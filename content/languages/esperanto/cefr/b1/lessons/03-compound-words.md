## 概述

 世界语最具创造性和生产力的特征之一是它能够通过将两个或多个词根连接在一起来形成复合词。 与英语广泛借用希腊语、拉丁语和法语作为技术词汇的英语不同，世界语可以从熟悉的词根构建复杂的术语——使得任何了解其组成部分的学习者可以立即解释新单词。 该系统极大地减少了词汇负担，同时提高了表达能力。 

世界语中的复合规则很简单，但需要一些练习：最右边的元素是复合词的语法“中心词”，并带有词类结尾，而前面的元素以其裸根形式出现（没有语法结尾）。 顺序是修饰语在先，这意味着描述性元素出现在被描述的元素之前——这与受法语影响的英语借用复合词的趋势相反。 一旦你内化了这种模式，你将能够理解不熟悉的复合词并当场创造出你自己的精确单词。 

---

 ## 学习目标

 - 应用修饰符优先的复合规则来创建和解码复合词
 - 识别词根之间何时需要连接元音 (-o-) 以获得悦耳的声音
 - 使用最有生产力的复合形成元素（-ĉambro、-lando、-tempo、-farti 等）来扩展词汇量
 - 通过识别每个词根及其关系来分析长化合物

 ## 语法重点

 ### 1. 核心复利规则

 在世界语中，复合词是通过将词根并排放置来形成的。 **最右边的词根**是语法中心词——它带有词性结尾并决定含义类别。 所有前面的根都显示为**裸根**（它们的根形式没有 -o、-a、-i、-e 结尾）。 

> **vapor** + **ŝip** + **-o** =vaporŝipo（蒸汽船 - 一艘以蒸汽 [vaporo] 运行的船 [ŝipo]）
 > **fer** + **voj** + **-o** = fervojo（铁路 — 由铁 [fero] 制成的道路 [vojo]）
 > **man** + **skrib** + **-o** = manskribo（手写 — 手写 [skribo] [mano]）

 顺序始终是**修饰符 + 中心词**，这反映了德语，并且与法语/英语借用复合词相反：
 > 英语：“bookshop”（书修饰商店）——与世界语顺序相同！ 
> 但是：“卧室”= 世界语中的 dorm**o**ĉambro（dormĉambro — 卧室）

 请注意，英语复合词也恰好是修饰语优先，因此世界语复合词通常感觉直观。 

---

 ### 2. 连元音 -o-

当两个辅音簇发生碰撞或复合音难以发音时，通常会在词根之间插入 -o-。 这与名词结尾的 -o 相同，但在这里它充当纯语音连接器：

 > **dom** + **o** + **best** + **-o** = **domobesto** — 驯养动物（domo + besto，加入 -o-）
 > **labor** + **o** + **don** + **-a** = **laborodona** — 付出劳动，有利可图（加入 -o-）

 然而，许多化合物无需连接器即可正常工作：
 > **skrib** + **maŝin** + **-o** = **skribmaŝino** — 打字机（无需连接器）
 > **tag** + **manĝ** + **-o** = **tagmanĝo** — 午餐

 在大多数情况下，连接元音是可选的，并且由悦耳的声音（听起来令人愉快的声音）引导。 当有疑问时，先尝试一下； 如果发音困难，加-o-。 

---

 ### 3. 富有成效的核心词

 某些单词经常作为复合词的右侧元素（中心词）出现，从而形成多产的族：

 **-ĉambro**（房间）：
 > 宿舍**ĉambro**（卧室）、ban**ĉambro**（浴室）、劳动**ĉambro**（书房/办公室）、lud**ĉambro**（游戏室）、kuir**ĉambro**（厨房——尽管通常只是“kuirejo”）

 **-lando**（国家/土地）：
 > hejm**lando**（家乡）、trans**lando**（异国土地）、supre**lando**（高地）、mal**proksim**a lando（遥远的土地）

 **-manĝo**（餐）：
 > maten**manĝo**（早餐）、tag**manĝo**（午餐）、vesper**manĝo**（晚餐）、posta**manĝo**（餐后甜点）

 **-节奏**（时间/季节）：
 > prin**tempo**（春天）、labor**tempo**（工作时间）、liber**tempo**（空闲时间/休闲）、ferio**tempo**（假期时间）

 **-ejo**（活动场所）：
 > lern**ejo**（学校）、labor**ejo**（工作场所）、kuir**ejo**（厨房）、ludejo（游乐场）、libr**ejo**（书店）

 **-isto**（专业人士）：
 > instruo → instru**isto** （老师）、muziko → muzik**isto** （音乐家）、kuiri → kuir**isto** （厨师/厨师）

 ---

 ### 4. 长化合物及其分析

 世界语允许包含三个、四个甚至更多词根的复合词。 阅读它们的关键是识别每个根并从右到左进行：

 > **fervojstacio** = fero + vojo + stacio → 火车站（位于铁路上）
 > **subterakvoj** = sub + tero + akvo + -j → 地下水
 > **poŝtelefonujo** = poŝo + telefono + ujo → 手机壳（存放口袋电话的容器）
 > **internacilingvo** = internacia + lingvo → 国际语言

 创建长化合物时，请注意清晰度。 如果三根复合词不明确，请使用连字符或用介词重新表述：
 > **ĝardenlaboro** — 园艺工作（透明）
 > **ruĝa vinhavo** — 红酒容器（比 rugxvinhavujo 改写更好）

 ## 练习

 ### 练习 1：解码化合物

 将每个复合词分解为词根并给出英语含义。 

1. 塔格卢莫
 2. 马滕鲁奥
 3. 勒诺利布罗伊
 4. 尼普康斯特鲁霍
 5. dormkamero（变体拼写）
 6. 温特罗维特罗
 7. 国际访问

*答案：*
 1. tago + lumo = 日光
 2. mateno + ruĝo = 黎明（早晨发红）
 3.lerno + libro + -j = 教材（学习书籍）
 4. ŝipo + konstrui + ejo = 造船厂

 7. internacia + lingvo + isto = 国际语言专家（Esperantist）

 ### 练习 2：构建复合体

 将给定的词根组合成一个世界语复合词。

 2. akvo（水）+ falo（瀑布）→ \_\_\_（瀑布）
 3. hejmo (回家) + iri (去) → \_\_\_ (回家/回家)
 4. nokto（夜）+ manĝo（餐）→ \_\_\_（深夜餐）
 5. aero（空气）+haveno（港口/港口）→ \_\_\_（机场）

 *答案：* 1. sunbrulo 2. akvofalo 3. hejmiro（形容词/名词） 4. noktomanĝo 5. aerohaveno / aeroporto

 ## 文化笔记

 Zamenhof 的复合系统直接受到德语的启发，其中像 *Donaudampfschifffahrtsgesellschaft* （多瑙河轮船公司）这样的词展示了无限的生产力——尽管众所周知，但也有潜在的不透明性。 世界语实现了更实际的平衡：词根往往很短并且在国际上可以识别，并且严格的修饰语优先规则可以防止英语复合词中可能出现的歧义（“dog trainer”是指训练狗的人，还是训练其他人的狗？）。 在世界语中，*hundotrejnisto* 明确表示“训练狗的人”。 这种精确性就是世界语技术和科学界能够快速、明确地创造新术语的原因，特别是在计算机科学等领域——*komputilo*（计算机）、*interreto*（互联网）、*poŝtelefono*（移动电话）——所有这些都是透明地从熟悉的根源构建的。
