## 概述

 世界语后缀的工作方式与前缀完全相同：具有一致含义的单个元素附加到任何兼容的词根以生成新单词。 本课程涵盖四个高效后缀：**-ej-**（与某物相关的地方）、**-ist-**（从业者或专家）、**-in-**（女性性别标记）和 **-ul-**（具有某种品质特征的人）。 这四个后缀一起使您可以从单个词根构建丰富的词族。 

世界语词缀最优雅的特征之一是它们可以“链接”。 从词根 *instrui* （教学）开始，您可以构建 *instruisto* （教师）、*instruistino* （女教师）和 *instruistejo* （学校/教学场所）。 每个步骤都是透明且可预测的——除了知道每个词缀的含义之外，不需要记忆。 

## 学习目标

 - 使用 **-ej-** 从活动或目的根源派生地点名词
 - 使用 **-ist-** 派生专业或专业名词
 - 使用 **-in-** 来标记任何人或动物名词的阴性形式
 - 使用 **-ul-** 从形容词或名词词根派生人称名词
 - 链接多个词缀以构建多步骤单词派生

 ## 语法重点

 ### 模式 1：-ej- — 与动作或事物相关的地点

 添加到动词或名词词根的后缀 **-ej-** 创建一个单词，意思是“该活动发生的地方”或“与该事物相关的地方”。 

**来自动词：**
 - lerni（学习）→ **lern**ejo（学校）
 - kuiri（做饭）→ **kuir**ejo（厨房）
 - bani（洗澡）→ **ban**ejo（浴室、澡堂）
 - preĝi（祈祷）→ **preĝ**ejo（教堂，礼拜场所）
 - Labori（工作）→ **labor**ejo（工作场所，办公室）
 - ludi（玩）→ **lud**ejo（游乐场、游戏区）
 - trovi（寻找）→ **trov**ejo（失物招领处，找到东西的地方）

 **来自名词：**
 - kuracisto（医生）→ **kuracist**ejo（诊所、医生手术）
 - libro（书）→ **libr**ejo（书店、图书馆——尽管也使用 *biblioteko*）
 - Birdo（鸟）→ **bird**ejo（鸟舍，鸟类保护区）
 -porko（猪）→ **pork**ejo（猪圈）

 **完整句子示例：**
 - La infanoj iras al la **lernejo** ĉiun matenon。 — 孩子们每天早上都去上学。 
- Nia **kuirejo** estas malgranda，sed funkcia。 — 我们的厨房虽小但功能齐全。 
- En la **banejo** estas varma akvo。 — 浴室里有热水。 
- Li Laboras en **kuracistejo** proksime al la centro。 — 他在中心附近的一家诊所工作。

**常见错误：** 将 -ej- 添加到已经表示地点（*urbo*、*domo*）而没有新的特定用途的地点词中。 *Urboejo* 的含义并不比*urbo* 多多少。 当特定活动或目的定义了该地点时，请使用 -ej-。 

### 模式 2：-ist- — 实践者、专家、信徒

 **-ist-** 创建一个名词，意思是“其职业、专长或坚定承诺为 X 的人”。 

**来自领域或学科的名词：**
 - muziko → **muzik**isto（音乐家）
 - dento → **dent**isto（牙医）
 - komerco → **komerc**isto（商人、贸易商）
 - arto → **艺术**isto（艺术家）
 - 科学（o）→ **科学**isto（科学家）
 - sporto → **sport**isto（运动员，运动员）
 - ŝtato → **ŝtat**isto（政治家）
 - historio → **histori**isto（历史学家）

 **来自动词：**
 - instrui → **instru**isto（老师）
 - kuri → **kur**isto（跑步者、信使）
 - vendi → **vend**isto（销售人员、供应商）
 - 程序i → **程序**isto（程序员）

 **从意识形态根源来看：**
 - 社会 → **社会**isto（社会主义）
 - 世界语 → **世界语**isto（世界语学家）

 **完整句子示例：**
 - Mia patrino estas **dentisto** — ŝi Laboras tri tagojn semajne。 — 我的母亲是一名牙医 — 她每周工作三天。 
- Li estas fama **muzikisto** kaj ludas la violonon。 — 他是一位著名的音乐家，会拉小提琴。 
- Nia **instruisto** ĉiam klarigas aferojn tre pacience。 — 我们的老师总是非常耐心地解释事情。 
- 基尔**世界语**，我是世界语协会的成员。 — 作为一名世界语学家，我每年都会参加代表大会。 

### 模式 3：-in- — 女性性别标记

 **-in-** 将名词标记为阴性。 它附加在任何人或动物名词后以指定女性性别。 基本形式（不带 -in-）在许多词根中是中性的或传统上阳性的。 

**个人/家庭的话：**
 - frato（兄弟姐妹/兄弟）→ **frat**ino（姐妹）
 - patro（父母/父亲）→ **patr**ino（母亲）
 - filo（孩子/儿子）→ **fil**ino（女儿）
 - amiko（朋友）→ **amik**ino（女性朋友）
 - kuz®（表弟）→ **kuz**ino（女表弟）
 - reĝo（君主/国王）→ **reĝ**ino（女王）
 - instruisto (老师) → **instruist**ino (女老师)

 **动物的话：**
 - kato（猫）→ **kat**ino（母猫）
 - hundo（狗）→ **hund**ino（母狗，母狗）
 - leono（狮子）→ **leon**ino（母狮）
 - kokо（鸡）→ **kok**ino（母鸡）

 **完整句子示例：**
 - Mia **fratino** 在 Varsovio 学习医学。 — 我姐姐在华沙学医。 
- La **leonino** ĉasis la antilopo。 — 母狮猎杀羚羊。 
- Nia **instruistino** venis el Brazilo kaj parolas portugalan kaj Esperanton。 — 我们的女老师来自巴西，会说葡萄牙语和世界语。 
- La **reĝino** malfermis la novan muzeon。 — 女王为新博物馆揭幕。 

**关于性别中立的注释：** 一些现代世界语学家使用后缀 **-iĉ-** （非官方）来表示明确的男性形式，使基本形式完全性别中立。 这是社区中正在进行的讨论，但传统系统（-in- 表示女性，否则为基本形式）在大多数文本中仍然是标准的。 

### 模式 4：-ul- — 以某种品质为特征的人

 **-ul-** 创建一个名词，意思是“以 X 为特征的人”——通常来自形容词，但也来自名词。 它侧重于人的“定义特征”。 

**来自形容词：**
 - juna（年轻）→ **jun**ulo（年轻人）
 - maljuna（老）→ **maljun**ulo（老人）
 - riĉa（富人）→ **riĉ**ulo（富人）
 - malriĉa（穷人）→ **malriĉ**ulo（穷人）
 - bela（美丽）→ **bel**ulo（美丽的人，美女）
 - stulta（愚蠢）→ **stult**ulo（傻瓜，愚蠢的人）
 - kolera（生气）→ **koler**ulo（生气的人，性急的人）
 - malsana（生病）→ **malsan**ulo（病人，病人）

 **来自名词：**
 - strango（陌生人）→ **strang**ulo（一个奇怪的/奇怪的人）
 - Hero（英雄）→ **hero**ulo（英雄人物）

 **词链——世界语词缀的真正力量：**

从*instrui*（教导）开始：
 1. instrui → **instru**isto（教师——男性或一般）
 2. instruisto → **instruist**ino（女老师）
 3. instruisto → **instruist**ejo（学校，教学场所）
 4. instruistino → **instruistin**ejo （技术上来说：女教师的位置——不寻常，但语法上有效）

 从*juna*（年轻）开始：
 1. juna → **jun**ulo（年轻人）
 2. junulo → **junul**ino（年轻女子）
 3. junulo → **junul**aro（青年作为一个集体） — *-ar-* 第 6 课涵盖

 ## 练习

 ### 练习 1：构建带有后缀的单词

 添加正确的后缀（-ej-、-ist-、-in-、-ul-）以创建所需的含义。 

1. sport + ___ = 运动员、运动员
 2. kuir + ___ = 厨房
 3. riĉ + ___ = 有钱人
 4. reĝ + ___ = 皇后
 5. vend + ___ = 销售人员
 6. preĝ + ___ = 教堂
 7. malsan + ___ = 病人
 8. 兄弟会 + ___ = 姐妹
 9. histori + ___ = 历史学家
 10.bel + ___ = 一个美丽的人

 *答案：1.isto 2.ejo 3.ulo 4.ino 5.isto 6.ejo 7.ulo 8.ino 9.isto 10.ulo*

 ### 练习 2：链式推导

 从每个词根构建派生词链。 每个链至少使用 2 个后缀。 

1. 从*kuiri*（做饭）开始：→厨师（人）→女厨师→厨房
 2. 从*kanti*（唱歌）开始：→歌手→女歌手→音乐厅（提示：kanto = 歌曲）
 3. 以*juna*（年轻）开头：→ 年轻人→ 年轻女子→ 一群青年（*-ar-*）

 *答案示例：*
 1. kuiristo → kuiristino → kuirejo
 2. 坎蒂斯托 → 坎蒂斯蒂诺 → 坎特霍
 3. 朱努洛 → 朱努洛 → 朱努拉罗

 ### 练习 3：描述人物

 对于每个描述，使用 -ul- 或 -ist- 创建一个世界语名词。 

1. 一个永远勇敢的人 (kuraĝa)
 2. 计算机编程人员（programi）
 3. 非常生气的人（kolera）
 4. 致力于科学的人（scienco）
 5. 穷人 (malriĉa)

 *答案：1.kuraĝulo 2.programisto 3.kolerulo 4.sciencisto 5.malriĉulo*

 ## 文化笔记

 后缀 **-ist-** 在世界语文化中具有特殊意义。 **世界语**——世界语的演讲者和倡导者——是社区最常见的自我认同之一。 扎门霍夫本人在早期的信件中署名“Doktoro Esperanto”（充满希望的信件），围绕该语言发展起来的社区自然而然地采用 **-isto** 来命名自己。 如今，**世界语**不仅意味着语言知识，而且通常还意味着对该语言所代表的国际理解理想的承诺。 当演讲者在大会上见面并用“Saluton, samideano!”互相问候时。 （你好，思想家同行！），共同的“ist”身份是将他们团结起来的一部分。
