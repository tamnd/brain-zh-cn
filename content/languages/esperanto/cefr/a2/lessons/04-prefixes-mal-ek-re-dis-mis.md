## 概述

 世界语的词缀是其最大的优势之一。 少量的前缀和后缀，每个都有明确且一致的含义，可以附加到任何兼容的词根上，以创建数千个新单词。 本课涵盖五个重要的前缀：**mal-**（相反）、**ek-**（突然开始或短暂动作）、**re-**（再次或返回）、**dis-**（分开或分散）和 **mis-**（错误或不正确）。 

仅掌握这五个前缀就可以极大地扩展您的活跃词汇量。 您无需分别记住 *bona* 及其对立面，而是自动导出 *malbona*。 您无需查找“misunderstand”一词，而是即时组合 *mis-* + *kompreni*。 这种高效的构词能力是世界语学习效率如此高的特征之一。 

## 学习目标

 - 将 **mal-** 应用于形容词和动词以表达完全相反的意思
 - 使用**ek-**来表达动作的突然开始
 - 使用**re-**来表达重复或返回动作
 - 使用**dis-**来表达分散或分离
 - 使用 **mis-** 表示操作执行不正确

 ## 语法重点

 ### 模式 1：mal- — 相反的前缀

 **mal-** 创建任何形容词或动词的精确反义词。 它对几乎所有形容词和许多动词都有效。 

**带形容词：**
 - 博纳 → **马尔**博纳（好→坏）
 - 格兰达 → **马尔**格兰达（大→小）
 - varma → **mal**varma（暖→冷）
 -rapida→**mal**rapida（快→慢）
 - juna → **mal**juna（年轻→年老）
 - riĉa → **mal**riĉa（富→穷）
 - facila → **mal**facila （简单 → 困难）
 - proksima → **mal**proksima（近→远）
 - alta → **mal**alta（高 → 矮）
 - nova → **mal**nova（新→旧/二手）

 **带动词：**
 - ami → **mal**ami（爱→恨）
 - ĝoji → **mal**ĝoji（高兴→悲伤）
 - 费米 → **mal**费米（关闭 → 打开）
 - komenci → **mal**komenci （开始 → 结束/终止）
 - helpi → **mal**helpi（帮助→阻碍）

 **完整句子示例：**
 - La vetero estas **malvarma** hodiaŭ — ni restu hejme。 — 今天天气很冷 — 我们待在家里吧。 
- Li **malamis** sian najbaron，ĉar ĉiam li ludis laŭtan muzikon。 ——他讨厌他的邻居，因为他总是大声播放音乐。 
- **Malfermu** la fenestron，bonvolu — estas varme ĉi tie！ — 请打开窗户 — 这里很暖和！

**常见错误：** 使用 *mal-* 来否定并非真正对立的事物。 *ne bona*（不好）与*malbona*（非常糟糕）不同。 使用 *ne* 进行简单否定，仅当相反确实存在时才使用 *mal-*。 

### 模式 2：ek- — 突然开始或短暂动作

 **ek-** 标志着一个动作的开始（起始方面）或一个短暂、突然的事件。 它在运动、声音和情感动词中尤其常见。 

- iri → **ek**iri（出发，开始）
 - kuri → **ek**kuri（开始奔跑）
 - Ridi → **ek**ridi（大笑）
 - dormi → **ek**dormi（入睡）
 - brili → **ek**brili（开始发光，闪光）
 - paroli → **ek**paroli（开始说话，大声说话）
 - plori → **ek**plori（泪流满面）
 - kanti → **ek**kanti （突然唱歌）
 - ami → **ek**ami（坠入爱河）

 **完整句子示例：**
 - Kiam mi diris la ŝercon，ĉiuj **ekridi**s。 ——当我讲完这个笑话后，大家都笑了起来。 
- Ŝi **ekamis** lin je la unua renkontiĝo。 ——第一次见面，她就爱上了他。 
- La suno **ekbrilis** 后 la pluvo。 ——雨后，阳光开始灿烂。 
- **Ekiris** la grupo je la oka matene。 ——早上八点，一行人出发了。 

**常见错误：** 使用 *ek-* 表示持续或习惯性的行为。 *Li ĉiam ekiras frue* 很尴尬 - ek- 意味着开始，而不是常规模式。 说*Li ĉiam foriris frue*（他总是很早就离开）。 

### 模式 3：重新——再次或回来

 **重新**有两个相关的含义：*再次*做某事（重复）或*返回*做某事（返回运动）。 

**重复意义：**
 - Legi → **re**legi （重读）
 - fari → **re**fari（重做）
 - komenci → **re**komenci （重新开始）
 - konstrui → **re**konstrui（重建）
 - skribi → **re**skribi（重写）

 **返回/返回感：**
 - veni → **re**veni（回来，返回）
 - doni → **re**doni（回馈，归还一些东西）
 - iri → **re**iri（返回）
 - porti → **re**porti（带回来）
 - telefoni → **re**telefoni（回电）

 **完整句子示例：**
 - 发布 vojaĝo ŝi **revenis** ĝoja kaj energiplenа。 — 旅行结束后，她快乐而充满活力地回来了。 
- Mi **relegis** ĉi tiun libron trifoje — ĝi estas tiel bona。 ——这本书我重读了三遍——太好了。 
- **Redonу** al mi mian krajонon，bonvolu！ ——请把我的铅笔还给我！ 
- Post la fajro, la komunumo **rekonstruis** la pontоn。 — 火灾发生后，社区重建了桥梁。 

### 模式 4：dis- — 分开、分散、向各个方向

 **dis-** 意味着分散、分离或同时向多个方向发送。 

- sendi → **dis**sendi（广播，发送给许多人）
 - rompi → **dis**rompi（粉碎，碎成碎片）
 - iri → **dis**iri（驱散，向各个方向走）
 - doni → **dis**doni（分发，分发）
 - ŝiri → **dis**ŝiri（撕裂）
 - fali → **dis**fali（崩溃、崩溃）
 - lerni → **dis**lerni（广泛传播知识）

 **完整句子示例：**
 - Post la kunveno ĉiuj **disiri**s en malsamajn direktojn。 ——会议结束后，大家分散到不同的方向。 
- La Organizo **dissendis** la novaĵojn al ĉiuj membroj。 — 该组织向所有成员广播新闻。 
- La malforta branĉo **disrомpis** sub la pezo de la neĝo。 ——脆弱的树枝在雪的重压下折断了。 
- La instruisto **disdonis** la ekzamenajn paperojn。 ——老师分发试卷。 

### 模式 5：mis- — 错误地、不正确地

 **mis-** 表示操作执行不正确、错误或以错误的方式执行。 

- kompreni → **mis**kompreni（误解）
 - uzi → **mis**uzi （滥用、滥用）
 - diri → **mis**diri（说错话）
 - traduki → **mis**traduki（误译）
 - lerni → **mis**lerni（学习不正确）
 - kalkuli → **mis**kalkuli（错误计算）
 - konduti → **mis**konduti（行为不端）
 - skribi → **mis**skribi（拼写错误，写错）

**完整句子示例：**
 - Mi **miskomprenis** lademandon kaj respondis malĝuste. — 我误解了问题并回答了错误。 
- Ĉi tiu vorto estas **misskribita** en la teksto。 ——这个词在文中拼写错误。 
- La komputilprogramo **miskalkulos** la rezulton，se vi enigos malgustajn datumojn。 — 如果您输入不正确的数据，计算机程序将错误计算结果。 

## 对话

 **Tomo Kaj Yuki revenas de vojaĝo.**
 （Tomo 和 Yuki 旅行归来。）

 | 世界语 | 英语 |
 |------------|---------|
 | **Tomo：** 很好，你**revenis**！ La vojaĝo estis longa。 | 我们终于回来了！ 旅程很漫长。 |
 | **由纪：** 天哪，我很感兴趣。 Mi ekamis tiun urbon tuj！ | 是的，但是非常有趣。 我立刻就爱上了那个城市！ |
 | **Tomo：** Mi ankaŭ。 Sed la vetero estis **malvarma** — 矿井 atendistion。 | 我也是。 但天气很冷——我没想到会这样。 |
 | **由纪：** 哈，我 **错误** la veterprognozon。 Estis skribita“varma”，sed mi Legis“malvarma”。 康特拉耶！ | 哈，我看错天气预报了。 它写的是“温暖”，但我读的是“冷”。 相反！ |
 | **Tomo：** Mi **miskomprenис** la hoteliston kaj prenis **malĝustan** ĉambron。 | 我误会了酒店老板，走错了房间。 |
 | **由纪：** 你好吗？ 凯昂维法里斯？ | 真的吗？ 你做了什么？ |
 | **Tomo：** 我 **rekomencis** la procezon kaj klarigis la situacion。 很好，**disdonis** ĉambrajn ŝlosilojn al ĉiuj ĝuste。 | 我重新开始这个过程并解释了情况。 最后他们正确地将房间钥匙分发给了每个人。 |
 | **Yuki：** Almenaŭ ĉio 骨finiĝis。 Ĉu vi **relegus** la libron pri tiu urbo？ 我是善意的。 | 至少一切都结束了。 你会重读那本关于那个城市的书吗？ 非常好。 |
 | **托莫：** 埃布尔。 Sed unue mi bezonas **ekdormi** — mi estas **malenergia**！ | 或许。 但首先我需要入睡——我精力不足！ |

7. 寒冷的天气让每个人都感到悲伤。 

*答案示例：*
 1. Ŝi ekamis la urban ĉe la unua vido。

### 练习 3：构建词族

 从词根 *diri*（也就是说）开始，使用学到的前缀和您知道的任何后缀创建单词。 用每个单词写一个句子。 

示例：diri→misdiri（说错话）→Mi Misdiras kelkfoje kiam mi estas nervoza。 

尝试：re + diri、mal + diri（这有意义吗？）、dis + diri。 

## 文化笔记

前缀 **mal-** 对于世界语来说是如此重要，以至于它已成为某种文化的试金石。 著名的世界语口号“**Malami neniun**”（不恨任何人）和**malamo kontraŭ malamo**（仇恨对仇恨）的理想出现在世界语文学和演讲中，可以追溯到扎门霍夫的早期著作。 扎门霍夫本人也饱受反犹太主义之苦，并梦想中立的共同语言可以减少群体间的敌意。 “马拉米”这个词——字面意思是“不去爱”——抓住了这种愿景的某些部分：仇恨是爱的逆转，理论上可以逆转的东西。
