## 概述

 世界语的构词系统是其最著名的特征。 与英语不同，英语主要通过借用其他语言来获取新词汇（通常对学习者来说是不透明的：“偶然性”、“普遍存在”、“短暂”），世界语通过结合学习者已经知道的一小组词根和词缀来构建新单词。 这意味着像*malakcepteblaĵo*（不可接受的事物）这样的词对于任何知道*mal-*（相反）、*akcepti*（接受）、*-ebl-*（可能）、*-a*（形容词）和*-ĵo*（具体事物）的人来说是完全透明的。 在 B2 阶段，你不仅需要掌握单个词缀，还需要掌握它们形成的“链”和“堆栈”——四个、五个或六个语素的组合，产生精确的抽象词汇。 

本课程直接建立在 -ig-/-iĝ- 课程 (L3) 的基础上，并连接到文体语域课程 (L7)。 先进的构词法使得世界语既成为一种精确的哲学语言，又成为一种高效的日常语言。 学完本课后，您将能够在缺乏现成术语时当场创造新词，通过分解识别不熟悉的复杂单词的含义，并使用 B2 学术和政治话语所需的全套抽象词汇。 

## 学习目标

 在本课程结束时，您可以：
 - 通过分析每个语素的贡献，分解和解释具有四个或更多词缀的世界语单词
 - 从给定的根生成正确且惯用的派生链（例如，libera → libereco → liberigi → liberiĝi → liberigisto）
 - 有效地使用 -ebl-、-eco、-ado、-ismo、-ano 和 pli- 模式来表达抽象概念
 - 解释世界语采用国际词汇的标准，并根据世界语形态调整借词

 ## 语法重点

 ### 多词缀堆叠：构建复杂单词

 **结构：** [前缀] + 词根 + [后缀] + 词类结尾

 世界语允许几乎无限的词缀堆叠，只要结果在逻辑上是透明的。 实际上，超过 3-4 个语素的大多数单词都保留用于技术、哲学或幽默用途，但 B2 学习者需要了解解码和生成它们的原理。

B2级别复杂堆叠的关键词缀是：
 - **mal-**（相反）：malbona（坏），malgranda（小）
 - **-eg-**（增强，极端程度）：bona→bonega（优秀），granda→grandega（巨大）
 - **-et-**（小型，温和程度）：varma → varmeta（稍微温暖）
 - **re-**（再次，返回）：skribi → reskribi（重写）
 - **-ad-**（连续或重复动作）：skribi → skribadi（继续书写）
 - **pli-**（比较动作前缀）：bona → plibona（更好）→ plibonigi（改进）
 - **-ebl-**（可能，-能够）：fari → farebla（可行），akcepti → akceptebla（可接受）
 - **-ec-**（-ness，-ity，质量）：Libera → libereco，bona → Bonaco
 - **-ism-**（意识形态、制度）：社会主义、世界主义
 - **-an-**（成员，居民）：Esperanto → esperantano（世界语学家，成员），urbo → Urbano（城市居民）
 - **-ist-**（专业、习惯性执业者）：muziko → muzikisto、kuracisto（来自 *kurac-* 的医生）
 - **-ĵo**（具体事物/实例）：belo → belaĵo（美丽的事物），neebla → neeblaĵo

 全面分析的堆叠示例：

 *malakceptebla* = mal-（相反）+ akcepti（接受）+ -ebl-（可能）+ -a（形容词）→“不可接受，不可接受。” 五个语素，完全透明。 

*plibonigado* = pli-（更多/比较）+ bona（好）+ -ig-（使役）+ -ad-（正在进行）+ -o（名词）→“不断改进的持续过程。” 六个语素。 

*re-skrib-ad-i* = re-（再次）+ skribi（写入）+ -ad-（连续）+ -i（不定式）→“不断重写，反复写入和重写。”

 *lib-er-ec-o* = libera（自由）+ -ec-（质量/状态）+ -o（名词）→“自由”。 三个语素，很常见的词。 

| 世界语 | 英语 | 语素分析 |
 |------------|---------|--------------------|
 | 马拉克塞特布拉 | 不可接受| mal + akcepti + -ebl + -a |
 | 普利博尼亚多 | 不断自我完善| pli + bona + -ig + -iĝ + -ad + -o |
 | 马尔法西莱吉 | 使极其困难| mal + facila + -eg + -ig + -i |

 **常见错误：** ❌ *malbonigadi*（意为“继续做坏事”）——虽然语法上可以解析，但这种组合非常不自然，会让母语人士感到困惑。 规则：仅仅因为堆叠是*可能的*并不意味着它是*惯用的*。 更喜欢更简单的结构，除非复杂性确实增加了精度。

 ### 生产衍生链

 **结构：** 选择一个根→应用每个主要的衍生路径→生成完整的族

 在 B2 阶段，您应该能够从单个词根生成完整的词族。 这不仅仅是一个词汇技巧——它表明你对词法系统有深刻的理解。 这是来自 *libera* 的完整链：

 | 表格 | 语素| 意义|
 |------|------------|---------|
 | 自由| 自由 + -a | 自由（形容词）|
 | 自由| 自由 + -e | 自由地（副词）|
 | 利雷科 | 自由 + -ec + -o | 自由，自由|
 | 利贝里吉| 自由 + -ig + -i | 自由，解放(tr.) |
 | 自由 | 自由 + -iĝ + -i | 变得自由（解释）|
 | 自由 | 自由 + -ig + -ist + -o | 解放者（专业/习惯）|
 | 自由| 自由 + -ig + -ant + -o | 一个正在解放的人（现在）|
 | 自由| 自由 + -ig + -int + -o | 解放者（过去）|
 | 马利伯(ec)o | mal + liber + (-ec) + -o | 不自由，囚禁|
 | 马尔利贝里吉 | mal + 自由 + -ig + -i | 监禁、剥夺自由|
 | 自由主义| 自由 + -al + -ism + -o | 自由主义（派生形容词词根需要 -al- 中缀）|

 相同的链模式几乎适用于任何根。 来自*sana*（健康）：saneco、sanigi、saniĝi、sanigisto（治疗者）、malsana、malsaneco、malsanulo（病人）、malsanulejo（医院）、malmalsanigi...等等。 一旦你内化了一个根的模式，你就拥有了所有根的模式。

| 世界语 | 英语 |
 |------------|---------|
 | La unueco de la movado estas nia plej granda forto。 | 运动的团结是我们最大的力量。 |
 | Beli 是 rara verbo； 经常犯下罪恶。 | Beli 是一个罕见的动词； 人们最常使用 beligi 或美化自己。 |
 | La plibonigado de la lingvo estas laŭstatuta celo de la Akademio。 | 语言的持续改进是学院的法定目标。 |
 | Ebleco, ke ili akceptos nian proponon. | 他们有可能接受我们的建议。 |

 **常见错误：** ❌ *La liberigistoj liberigadis la kaptitojn dum jaroj.* → ✓ *La liberigistoj daŭre liberigis la kaptitojn dum jaroj.* — *-ad-* on *liberigi* 听起来像是一种风格上的矫揉造作； *daŭre* + 简单动词更自然地表示“持续做”很长一段时间。

 ### 国际词汇和新词

 **结构：** 外来词顺应=语音顺应+世界语词类词尾

 世界语借用了国际词汇——主要来自拉丁语、希腊语、法语、德语、英语和其他广泛使用的语言——但又进行了系统的改编。 标准：当没有可派生的世界语形式对于该概念来说是自然的或透明的，并且该词真正是国​​际性的（跨多个语系得到认可）时，该词被借用。 适配规则：

 1. 如果外来后缀被世界语词类词尾替换，则将其去掉：*nation* → *nacio*、*possible* → *ebla*（不，这个是派生的）、*telephone* → *telefono*
 2. 使音系适应世界语的音素：英语“sh”→ *ŝ*（*shopping* → *ŝoping*，口语，或正式的 *aĉeto*），“ch”→ *ĉ* 或 *ĥ*，具体取决于来源
 3. 添加世界语词尾：*-o* 为名词，*-a* 为形容词，*-i* 为动词

 世界语界的新造词（新词）：世界语学院将*Oficialaj Aldonoj*（官方增补）发布到官方词汇表中。 社区新词通过 *Monato*、在线论坛和 PIV (Plena Ilustrita Vortaro) 传播。 当你遇到一个不熟悉的世界语单词时，在查找之前尝试将其分解 - 形态通常会告诉你 80% 的含义。 

| 世界语 | 来源 | 笔记|
 |------------|--------|--------|
 | 因雷托| 内部造币| internet: inter- (之间) + reto (网) — 不是借自英语 |
 | 综合 | 内部造币| 计算机：komput- + -il-（工具）+ -o |
 | 电话 | 国际贷款| 电话：直接适配|
 | 电视| 内部造币| 电视：tele-（远）+ vid-（看）+ -o |
 | 博客| 英语贷款| 博客：直接借贷，社区接受|
 | 报告 | 内部造币| 电子邮件：ret-（网络/网络）+ poŝto（邮寄/邮件）|

 **常见错误：** ❌ 当高效的世界语派生工作正常时，不必要地借用：*ĉekumi* 表示“检查”→ ✓ *kontroli* 或 *verifi* 已经建立。 为世界语形态无法有效涵盖的真正新概念保留借词。 

## 真实文本

 La vorto *komputilo* estis kreita ne per prunto el la angla, sed per interna derivado1: komput- (el latino *computare*, kalkuli) + -il- (ilo, Instrumento) + -o。 世界语原则的基本原则：如果您需要了解世界语的基本概念，请使用该语言。 Aliaj internacie konataj vortoj — kiel *interreto* (inter + reto) kaj *retpoŝto* (reto + poŝto) — montras la saman logikon。 Tamen ne ĉiuj novaĵoj estas tiel facile tradukеblaj³。 La vorto *博客* — uzata en la interreto por persona retejo kun regulaj enskriboj — estas akceptita de multiaj esperantistoj kiel simpla prunto, ĉar *ret-ĵurnalo* aŭ similaj kunmetaĵoj ŝajnas al iuj tro longaj. 这说明了国际间的紧张关系。

*“komputilo”这个词不是借用英语，而是通过内部派生创建的：komput-（来自拉丁语 computare，计算）+ -il-（工具、仪器）+ -o。 这是世界语原则的一个例子：如果该语言已经具有表达概念所需的词根和词缀，请使用它们而不是借用。 其他国际知名单词——例如 interreto（inter + reto）和 retpoŝto（reto + poŝto）——也显示了相同的逻辑。 然而，并非所有新事物都那么容易翻译。 “博客”一词——在互联网上用于具有常规条目的个人网站——被许多世界语学者视为简单的借用词，因为“ret-ĵurnalo”或类似的复合词对某些人来说似乎太长了。 这说明了内部衍生的纯粹性与国际贷款的实用性之间持续存在的紧张关系。*

 ¹ *per interna derivado* —“通过内部推导”，与借用相对
 ² *anstataŭ pruni* — “而不是借用” — *pruni* = 借用（一个词），*prunto* = 贷款

## 文化笔记

 世界语的系统构词引起了语言学家和语言规划者的特别兴趣。 德国犹太语言学家拉扎勒斯·路德维希·扎门霍夫（Lazarus Ludwig Zamenhof）会说俄语、波兰语、意第绪语、德语、希伯来语、法语、拉丁语和希腊语，他通过观察欧洲语言共享哪些派生过程，然后将其规范化，设计了词缀系统。 其结果是一个系统对于罗曼语、日耳曼语和斯拉夫语的使用者来说感觉很自然，同时保持逻辑上的一致性。 

高级世界语词汇的最佳单一资源是 *Plena Ilustrita Vortaro de Esperanto* (PIV)，这是一本综合性的世界语图解词典，现已在 vortaro.net 上在线提供。 PIV 包含超过 16,000 个词根条目，以及完整的派生表、来自经过验证的文献的例句以及词源注释。 对于认真的 B2-C1 学习者来说，每周花 30 分钟探索 PIV 条目（不仅仅是查找单词，而是阅读完整的推导系列）是最有效的词汇扩展策略之一。
