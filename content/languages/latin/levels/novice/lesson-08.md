## 概述

 **第三变格**是最大且变化最多的拉丁语变格。 与第一和第二变格不同，第三变格名词有许多不同的主格单数形式。 **属格单数**始终是您的锚点：它以 **-is** 结尾并揭示真正的词干。 

**规则：** 从单数属格中去掉 **-is** 以获得词干，然后添加词尾。 

*mīles, mīlitis* → 词干 *mīlit-*
 *rēx, rēgis* → 词干 *rēg-*
 *nox，noctis* → 茎 *noct-*

 第三变格包括阳性、阴性和中性名词。 每个名词都必须记住性别——没有单一可靠的标记。 

---

 ## 辅音词干：*mīles、mīlitis、m.*（士兵）

 | 案例 | 单数| 复数|
 |------|----------|--------|
 | **标称。** | 英里 | 军事**ēs** |
 | **将军** | 军事**是** | 军事**嗯** |
 | **数据。** | 军事***ī** | 军事**ibus** |
 | **根据** | 军事**em** | 军事**ēs** |
 | **Abl.** | 军事**e** | 军事**ibus** |
 | **声音** | 英里 | 军事**ēs** |

 需要记住的关键结局：
 - **属格单数**：-is
 - **宾格单数**：-em
 - **属格复数**：-um
 - **与格/夺格复数**：-ibus

 ---

 ## 中性辅音干：*corpus, corporis, n.*（主体）

 中性规则适用：主格 = 宾格 = 呼格。 

| 案例 | 单数| 复数|
 |------|----------|--------|
 | **标称。** | 语料库 | 企业**a** |
 | **将军** | 企业**是** | 企业**嗯** |
 | **数据。** | 企业**ī** | 企业**ibus** |
 | **根据** | 语料库 | 企业**a** |
 | **Abl.** | 企业**e** | 企业**ibus** |
 | **声音** | 语料库 | 企业**a** |

 注意：中性第三变格的主格/宾格/呼格复数 = **-a** （与第二变格中性复数相同，并且类似于第一变格名词复数 - 上下文区分它们）。 

---

 ## I 词干规则

 第三变格名词的一个子组是**i-词干**。 它们与复数属格（**-ium** 而不是 **-um**）和某些夺格形式中的辅音词干不同。 识别 i 词干对于避免错误很重要。 

**如何识别i-stem：**

 1. **巴黎音节名词**：主格和属格具有**相同的音节数** (*cīvis/cīvis*, *nūbēs/nūbis*) → i-stem
 2. **以主格结尾 -s 或 -x** 的名词，其中词干以 **两个辅音** 结尾（*urbs/urbis* — 词干 *urb-* 以 rb 结尾；*pars/partis* — 词干 *part-* 以 rt 结尾）→ i-词干
 3. **主格中以 -al、-ar、-e** 结尾的中性名词 (*animal, mare*) → i-stem

 ---

 ## I-词干阳性/阴性: *cīvis, cīvis, m./f.* (citizen)

 | 案例 | 单数| 复数|
 |------|----------|--------|
 | **标称。** | 公民 | cīv**ēs** |
 | **将军** | cīv**是** | cīv**ium** ← i 词干标记 |
 | **数据。** | cīv**ī** | cīv**ibus** |
 | **根据** | cīv**em** | cīv**ēs** |
 | **Abl.** | cīv**e** （或 cīv**ī**） | cīv**ibus** |
 | **声音** | 公民 | cīv**ēs** |

 与辅音词干的主要区别：**属格复数-ium**（不是-um）。 

---

 ## 中性 I-词干：*mare、maris、n.*（海）

 中性 i 词干还有其他独特的形式：

 | 案例 | 单数| 复数|
 |------|----------|--------|
 | **标称。** | 母马| mar**ia** ← -ia 复数 |
 | **将军** | 三**是** | mar**ium** ← -ium |
 | **数据。** | 玛**ī** | 玛**ibus** |
 | **根据** | 母马| 玛**亚** |
 | **Abl.** | mar***ī** ← -ī abl。 SG。 | 玛**ibus** |
 | **声音** | 母马| 玛**亚** |

 中性 i 词干：nom./acc./voc. 复数 **-ia** （不是 -a），属格复数 **-ium**，夺格单数 **-ī** （不是 -e）。

 ## 十个例句及分析

 **1.** *Mīlitēs rēgem sequuntur.*
 - *Mīlitēs* — 名义。 pl。 （第 3 宣告 -ēs 结尾）
 - *rēgem* — 根据。 SG。 （词干 *rēg-* + -em）
 - **翻译：** 士兵跟随国王。 

**2.** *Nōmen rēgis Mārcus est.*
 - *Nōmen* — 名义。 SG。 中性
 - *rēgis* — 创。 SG。 （占有）
 - **翻译：** 国王的名字是马库斯。 

**3.** *Per noctem mīlitēs iter faciēbant.*
 - *noctem* — 根据。 SG。 （经过的时间）
 - *iter faciēbant* — 成语：“我们正在旅行”
 - **翻译：** 士兵们整夜行军。 

**4.** *Vōx fēminae pulchra Erat。*
 - *Vōx* — 名义。 sg.，主题； *fēminae* — gen。 SG。 
- **翻译：** 女人的声音很美。 

**5.** *Corpus mīlitis in agro inventum est.*
 - *语料库* — 名义。 SG。 中性; *mīlitis* — 将军。 SG。 
- **翻译：** 士兵的尸体在野外被发现。 

**6.** *Cīvēs oppida multa aedificābant。*
 - *Cīvēs* — 名义。 pl。 i-茎
 - *oppida* — 根据。 pl。 中性（2nd decl.）
 - **翻译：** 公民正在建造许多城镇。 

**7.** *夜里的奢华。*
 - *Lūx* — 名义。 新加坡; *nocte* — abl。 SG。 与*在*
 - **翻译：** 灯光在夜晚闪耀。 

**8.** *情绪不稳定。*
 - *Tempus* — 名义。 SG。 中性; *fugit* = 第三个中士。 出版社 （*fugere*，逃离）
 - **翻译：** 时间过得真快。 （著名格言）

 **9.** *Nautae per mare iter longum faciēbant。*
 - *母马* — 根据。 SG。 （中性 i 词干）； *根据* + 依据。 
- **翻译：** 水手们正在海上进行长途旅行。 

**10.** *Opus hominum multa tempora durat。*
 - *作品* — 名义。 SG。 中性; *hominum* - gen。 pl.； *tempora* — 根据。 pl。 （时间范围）
 - **翻译：** 人类的工作持续了很多年。

 ## 练习

 **A。 给出每个名词的词干：**
 1. *pax, pacis* → ?
 2. *同人、人类* → ?
 3. *iter，行程* → ?
 4. *nox，noctis* → ？ 

**B。 在复数属格和夺格单数中下降 *tempus*。**

 **C。 翻译：**
 1. *Rēgēs oppidum capiunt.*
 2. *伟大的暂时作品*
 3. *Cīvium nomina nota sunt.*
 4. 士兵们听到国王的声音。 
5. 阳光照耀黑夜。 

**答案A：** 1. *pāc-* | 2. *古人类-* | 3. *行程-* | 4.*晚上-*

 **答案键 B：** Gen. pl. *临时*； 阿布。 SG。 *时间*

 **答案C：**
 1. 国王占领了城镇。 
2. 这是花费大量时间的工作。 
3. 公民的姓名为人所知。 
4. *Mīlitēs vōcem rēgis audiunt.*
 5. *Lūx sōlis per noctem fulget.*

 ---

 ## 总结

 - 第三变格：单数属格总是 **-is**； 剥去它以找到茎。 
- **辅音词干**：gen。 pl。 **-嗯**，abl。 SG。 **-e**。 
- **I-词干**（parisyllabics + 双辅音词干）：gen。 pl。 **-ium**； 中性 i 词干加 abl。 SG。 **-ī** 和名义/acc./voc。 pl。 **-ia**。 
- 中性名词遵循通用中性规则：nom。 = 根据。 = 声音。 
- 在第三个变格中，性别是**不可预测的**——用每个名词来记住它。
