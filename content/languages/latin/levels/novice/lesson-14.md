## 概述

 您已经知道第一/第二变格形容词 (*bonus/bona/bonum*)。 一大群重要的拉丁形容词遵循**第三变格**模式。 这些形容词根据它们有多少种不同的主格单数形式进行分类：三终止、两终止或一终止。 在其他情况下，所有词都有共同的第三变格结尾，但具有 **i-词干** 特征。 

---

 ## 1. 两个终结：*fortis、forte*（勇敢、坚强）

 M 和 F 主格单数的一种形式，N 的另一种形式。这是最常见的类型。 

| 案例 | 男/女 | 尼 |
 |------|-----|---|
 | 名义。 SG。 | 堡垒**是** | 堡垒**e** |
 | 将军 sg. | 堡垒**是** | 堡垒**是** |
 | 数据。 SG。 | 堡垒***ī** | 堡垒***ī** |
 | 附件。 SG。 | 堡垒**em** | 堡垒**e** |
 | 阿布。 SG。 | 堡垒***ī** | 堡垒***ī** |
 | 标称/附件 pl。 | 堡垒**ēs** | 堡垒**ia** |
 | 将军。 | 堡垒 ** 姆 ** | 堡垒 ** 姆 ** |
 | 数据/Abl。 pl。 | 堡垒**ibus** | 堡垒**ibus** |
 | 附件。 pl。 | 堡垒**ēs** | 堡垒**ia** |

 ## 2. 三终止：*celer、celeris、celere* (swift)

 所有三种性别都有单独的主格。 阳性词以 *-er* 结尾（无 *-is*），而阴性词和中性词则遵循双终止模式。 

| 案例 | 中号 | F | 尼 |
 |------|---|---|---|
 | 名义。 SG。 | 芹菜 | 芹菜**是** | 芹菜**e** |
 | 将军 sg. | 芹菜**是** | 芹菜**是** | 芹菜**是** |
 | 数据。 SG。 | 芹菜***ī** | 芹菜***ī** | 芹菜***ī** |
 | 附件。 SG。 | 芹菜**em** | 芹菜**em** | 芹菜**e** |
 | 阿布。 SG。 | 芹菜***ī** | 芹菜***ī** | 芹菜***ī** |
 | 标称/附件 pl。 | 芹菜**ēs** | 芹菜**ēs** | 芹菜**ia** |
 | 将军。 | 芹菜**ium** | 芹菜**ium** | 芹菜**ium** |
 | 数据/Abl。 pl。 | 赛勒**ibus** | 赛勒**ibus** | 赛勒**ibus** |

 **其他三词尾形容词：** *ācer、ācris、ācre*（敏锐、敏锐）； *puter、putris、putre*（腐烂）。 

---

 ## 3. 一次性终止：*fēlix, fēlicis*（幸运、幸福）

 所有三种性别都具有**相同的主格单数**。 单数属格显示词干 (*fēlic-*)。 

| 案例 | 男/女 | 尼 |
 |------|-----|---|
 | 名义。 SG。 | fēl**ix** | fēl**ix** |
 | 将军 sg. | fēlic**是** | fēlic**是** |
 | 数据。 SG。 | 费利克***ī** | 费利克***ī** |
 | 附件。 SG。 | 费利克**em** | fēl**ix** |
 | 阿布。 SG。 | 费利克***ī** | 费利克***ī** |
 | 标称/附件 pl。 | 费利克**ēs** | 费利克**ia** |
 | 将军。 | fēlic**ium** | fēlic**ium** |
 | 数据/Abl。 pl。 | 费利克**ibus** | 费利克**ibus** |

 **注意：** 在单终止形容词中，中性宾格单数 = 主格 (*fēlix*)，遵循中性规则。 

---

 ## 4. 夺格单数：*-ī* 与 *-e*

 这种区别对于区分夺格中的第三变格形容词和第三变格名词很重要：

 | 背景 | 结束| 示例|
 |--------|--------|---------|
 | **形容词**（定语/与名词一致）| **-ī** | *mīlite fortī*（与勇敢的士兵）|
 | **实质性**（形容词用作名词）| **-e** | *ā fortī*（勇敢的人） |
 | **表语**（在连接动词之后）| **-e** | *rēx fēlice* 会很不寻常； 表语通常跟在名词 |

 实践中：当第三变格形容词与名词一致时，使用 *-ī*。 当单独用作名词时，有时会看到*-e*。 语法各不相同； 要记住的最重要的规则是，**第三变格形容词在夺格单数中使用 -ī **，与大多数第三变格名词（使用 *-e*）不同。 

---

 ## 5.属格复数：始终*-ium*

 与大多数第三变格名词（可能使用 *-um* 或 *-ium*）不同，**第三变格形容词在复数属格中始终采用 *-ium***：
 - *fortium*（勇敢的人）
 - *fēlicium*（幸运者）
 - *全能*（全部）

 ## 例句

 1. *Mīles fortis hostēs vīcit.* — 勇敢的士兵击败了敌人。 
2. *Iter breveerat.*——旅程很短。 
3. *Omnes discipulī Latīnam linguam amant.* — 所有学生都喜欢拉丁语。 
4. *Vir prūdēns multa tacet.* — 谨慎的人对很多事情保持沉默。 
5. *Labour gravis est.* — 工作很严肃/辛苦。 
6. *Puer fēlix per agros currit.* — 快乐的男孩穿过田野。 
7. *Exercitus ingēns ad urbem vēnit.* — 一支庞大的军队来到了这座城市。 
8. *Dux audāx periculum non timet.* — 勇敢的领导者不惧怕危险。 
9. *简单易行； difficile est facere.* — 很容易说； 行动起来很困难。 
10. *Vox fēminae dulciserat.* — 女人的声音很甜美。

 ## 练习

 **A。 确定类型（一/二/三终止）并以夺格单数形式衰落：**
 1. *全能，全能*
 2. *ingēns, ingentis*
 3. *芹菜，芹菜，芹菜*

 **B。 使形容词与名词一致（给出正确的形式）：**
 1. *puer* + *fortis*（单数主格）
 2. *puellārum* + *fēlix*（复数属格）
 3. *mīlitibus* + *audāx*（与格/夺格复数）
 4. *iter* + *breve*（宾格单数）

 **C。 翻译：**
 1. *Omnium virōrum fortissimus est Caesar。*（注 *fortissimus* = 最勇敢；解析 *omnium*。）
 2. *Vir prūdēns brevia verba dīcit.*
 3.一切困难的事情都是从头开始的。 （提示：使用 *omnium rērum difficilium* + *est incipere*。）
