## 概述

 **与格**和**夺格**是初学者最容易混淆的两种情况 - 不是因为它们看起来很相似（它们的结尾通常不同），而是因为它们都有几种不同的用途。 本课程列出了这两种情况在新手级别最重要的用途。 

---

 ## 与格

 ### 1. 间接宾语的与格

 最常见的用法：与格标记**对谁**或**为谁**做某事的人。 在英语中，这通常用“to”或“for”来表达。 

*Nauta puellae rōsam dat.*
 — 水手送给女孩**一朵玫瑰。 

*魔导师 puerīs librōs 发出。*
 — 老师**为男孩们**买书。 

间接宾语是直接宾语的“接受者”。 模式：给予、展示、告诉或发送的动词+直接宾语（宾格）+间接宾语（与格）。 

### 2. 占有的与格

 与 *esse* 的形式一起使用来表达某人 **拥有**。 “拥有者”出现在与格中； 所拥有的东西就是主体。 

*Mihi nomen est Mārcus。*
 — **对我来说** 一个名字是马库斯 → 我的名字是马库斯。 

*军用短剑。*
 — **对于士兵** 有一把剑 → 士兵有一把剑。 

*Tibi multī amīcī sunt.*
 — 你有很多朋友。 （字面意思是：“有很多朋友是为你而存在的。”）

 ### 3. 参考与格（感兴趣的与格）

 与格表示**为谁的利益或劣势**而发生某项行动，或者**在其眼中**某些事情是真实的人。 

*Mihi 胎盘。*
 — 这对我来说很令人高兴 → 我喜欢它。 / 这让我很高兴。 

*Hoc tibi dīcō。*
 — 我这样说**是为了你的利益**/针对你而言。 

*综合性不是真理。*
 — 真相是**众所周知**/每个人都认为的。

 ## 烧蚀情况

 夺格是最通用的拉丁语格，涵盖了英语用许多不同的介词（“by”、“with”、“from”、“in”、“at”）表达的关系。 它可与**与**和**不**介词一起使用。 

### 4. 伴奏的夺格

 **与** *cum*（总是），表达**谁陪伴**主体。 

*Cum amīco venit.*
 — 他**和一位朋友一起来**。 

*Mīlitēs cum rēge iter fēcērunt.*
 — 士兵们**与国王**一起踏上旅程。 

注意：*cum* 作为附加词附加在人称代词上：*mēcum*（与我）、*tēcum*（与你）、*nōbīscum*（与我们）。 

### 5. 手段的夺格（工具）

 表示所使用的**工具或仪器**。 **拉丁语中没有介词**。 

*格拉迪奥·帕格纳特。*
 — 他**用剑**（通过剑）战斗。 

*足足足。*
 — 我们**用脚/用脚**（通过脚）行走。 

*Aqua nāvigant。*
 — 他们**通过水**/**在水上**航行。 

### 6. 方式夺格

 表达**如何**执行操作。 通常需要形容词； 当与形容词（*magnā cum cūrā* 或 *cum magna cūrā*）一起使用时，*cum* 插入**名词之前**，但当形容词存在时，*cum* 是可选的。 

*Magnā cum cūrā 抄写。*
 ——他**写得很仔细**。 

*Summā virtūte pugnavērunt。*
 ——他们**以最大的勇气**。 

*Silentiō intravit。*
 — 他**默默地**（没有形容词→不需要*暨*）。 

### 7. 夺格与介词 — 休息和分离

 这些介词总是采用夺格：

| 介词 | 意义| 示例|
 |-------------|---------|---------|
 | *ā / ab* | 从，远离| *ab oppidō* — 来自城镇 |
 | *dē* | 从; 下来; 关于 | *dē monte* — 从山上下来 |
 | *ē / 前* | 出于，从| *ex silvā* — 走出森林 |
 | *在* | 中，上（休息）| *在agrō* — 在田间|
 | *子* | 下（休息）| *sub arbore* — 在树下 |
 | *暨* | 与 | *cum mīlitibus* — 与士兵们 |
 | *正弦* | 没有 | *sine aqua* — 没有水 |
 | *亲* | 前; 代表 | *pro patriā* — 代表祖国 |
 | *超级* | 上面（休息）| *超级地形* — 地球之上 |

 请记住：*在* + 烧蚀 = **地点**（静止时）； *in* + 宾格 = **运动到**。 

### 8. 时间夺格 When

 表达**时间点**（无介词）。 名词通常是时间词（*nox、diēs、annus、hora*）。 

*Eā nocte mīlitēs vēnērunt。*
 —**那天晚上**士兵来了。 

*Primā lūce profectī sumus。*
 ——我们**天一亮就出发了**。 

*Eōdem annō Caesar in Galliam vēnit.*
 — **同年** 凯撒进入高卢。 

### 9. 分离的夺格

 表达**与谁或什么**分离或移除某物。 通常与 *ab、dē、ex* 或释放、缺乏、去除等动词一起使用。 

*Mīlitēs hostibus liberāvit.*
 — 他将士兵们从敌人手中解放出来。 

*插入符水色。*
 — 他缺乏 **水** （*carēre* 之后没有介词）。

 ## 十二个例句——分析与格和夺格的用法

 **1.** *Puellae librum dō。*
 - *puellae* = 间接宾语的与格 → **翻译：** 我给了女孩一本书**。 

**2.** *Mihi nomen est Iūlia。*
 - *Mihi* = 占有的与格 → **翻译：** 我的名字是 Julia。 

**3.** *Tibi multī amīcī sunt.*
 - *Tibi* = 占有的与格 → **翻译：** 你有很多朋友。 

**4.** *Mihi Placet haec via.*
 - *Mihi* = 参考/兴趣的与格 → **翻译：** 这条路让我高兴/我喜欢这条路。 

**5.** *Cum amīcīs in oppidō ambulābat.*
 - *amīcīs* = 伴奏的夺格（带有 *cum*）
 - *oppidō* = 夺格与*in*（地方在哪里）
 - **翻译：**他正在镇上**和朋友**散步。 

**6.** *Gladiō miles hostēs vīcit。*
 - *Gladiō* = 意思的夺格（无介词）
 - **翻译：** 士兵用剑**征服了敌人。 

**7.** *Magnā cum cūrā magister puerōs docēbat.*
 - *Magnā cum cūrā* = 方式夺格
 - **翻译：** 老师正在非常细心地教导男孩们**。 

**8.** *Mīlitēs ex oppidō cucurrērunt.*
 - *ex oppidō* = 夺格与 *ex*，分离/运动
 - **翻译：** 士兵们**跑出城镇**。 

**9.** *Sine aqua homō vīvere nōn potest。*
 - *aquā* = 烧蚀与 *sine*
 - **翻译：** 没有水，人就无法生存。 

**10.** *Eā nocte rēx epistulam scrīpsit。*
 - *Eā nocte* = 时间的夺格，当
 - **翻译：** **那天晚上**国王写了一封信。 

**11.** *Pro patriā mīlitēs pugnavērunt。*
 - *patriā* = 夺格与 *prō*（代表）
 - **翻译：**战士们**代表祖国**而战。 

**12.** *Silentiō in silvam intravit。*
 - *Silentiō* = 方式夺格（无形容词，无 *cum*）
 - **翻译：** 他**默默地进入了森林**。

 ## 总结表：与格与夺格用法

| 使用 | 案例 | 介词？ | 翻译信号|
 |-----|------|-------------|--------------------|
 | 间接宾语| 与格| 没有 | “to/for”（与给予/讲述动词一起）|
 | 占有| 与格| 没有 | “有”（与*esse*）|
 | 参考/兴趣 | 与格| 没有 | “为了/为了”（好处或坏处）|
 | 伴奏| 烧蚀| *暨*（必填）| “与”（一个人）|
 | 手段/工具| 烧蚀| 没有 | “与/通过”（一件事）|
 | 方式| 烧蚀| *暨*（可选，带有形容词。）| “与[形容词]”|
 | 地点 | 烧蚀| *中，子* | “在、在、在”|
 | 与 | 的分离/运动 烧蚀| *ab、ex、dē* | “从，出”|
 | 时间当| 烧蚀| 没有 | “in、on、at”（时间名词）|

 ## 练习

 **A。 确定每个与格或夺格的用法：**
 1. *Rēgī servus donum deedit.* (*Rēgī*)
 2. *Mīlitīgladiuslonguserat.* (*Mīlitī*)
 3. *Cum rēgīnā fēminae ambulābant.* (*rēgīnā*)
 4. *Magnā virtūte pugnavit.* (*Magnā virtūte*)
 5. *Ex oppidō cīvēs fugiēbant.* (*oppidō*)
 6. *Eō annō bellum fuit.* (*annō*)

 **B。 翻译成拉丁文：**
 1. 女孩给水手送水。 
2.我们有很多朋友。 （使用占有的与格）
 3、战士们以极大的勇气与敌人作战。 
4. 没有国王，公民就无法和平生活。 
5. 那天晚上我们听到了声音。 

**答案A：**
 1. *Rēgī* = 与格间接宾语
 2. *Mīlitī* = 占有的与格
 3. *rēgīnā* = 伴奏的夺格（带有 *cum*）
 4. *Magnā virtūte* = 方式的夺格
 5. *oppidō* = 夺格与 *ex*，分离
 6. *annō* = 时间的夺格，当

 **答案B：**
 1. *Puella nautae aquam dat.*
 2. *Nōbīs multiti amīcī sunt.*
 3. *Mīlitēs hostēs magna virtūte pugnavērunt.*
 4. *Sine rēge cīvēs in pāce vīvere nōn possunt.*
 5. *Eā nocte vōcēs audīvimus.*

 ## 总结

 - **与格**涵盖：间接宾语（“to/for”）、占有（与*esse*）和参考/兴趣。 
- **夺格**涵盖：伴奏（*cum*）、方式（无介词）、方式（*cum* + adj.）、地点（*in/sub*）、动作（*ab/ex/dē*）和时间（无介词）。 
- 区分夺格用法的关键：**介词的存在**和**语义上下文**（人与物、运动与静止、时间与地点）。 
- *Cum* 始终在场伴奏； 当形容词出现时，它是可选的（但常见的）； 它不存在手段。 
- 结合第 2 课和第 6 课，您现在已经掌握了所有六个拉丁格及其主要功能。
