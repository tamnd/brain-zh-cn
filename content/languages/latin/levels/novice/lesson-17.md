## 概述

 拉丁形容词有三种**比较程度**：正面（好）、比较级（更好）和最高级（最好）。 源自形容词的副词也有比较级和最高级形式。 拉丁语提供了两种表达“than”的结构：助词 *quam* 和比较的夺格。 

---

 ## 1.形容词比较级——规则构成

 将 **-ior** (M/F) 和 **-ius** (N) 添加到正词干（通过从单数属格中删除 *-is* 找到）：

 | 积极| 茎| 比较（男/女/女）|
 |----------|------|----------------------|
 | 长，-a，-um | 长-| 长**ior** / 长**ius** |
 | 富蒂斯，强| 堡垒- | 堡垒 ** ior ** / 堡垒 ** ius ** |
 | 费利克斯，费利西斯 | 费利克- | fēlic**ior** / fēlic**ius** |
 | audāx、audācis | audāc- | audāc**ior** / audāc**ius** |

 **比较级的变格：**它们作为**第三变格形容词**（两个词尾：M/F相同，N不同），但*没有*i-词干特征（gen. pl.是*-um*，不是*-ium*；abl. sg.是*-e*，不是*-ī*）。 

型号：*longior、longius*（更长）

 | 案例 | 男/女 | 尼 |
 |------|-----|---|
 | 名义。 SG。 | 朗吉奥尔 | 长吉斯 |
 | 将军 sg. | 长吉奥里斯 | 长吉奥里斯 |
 | 数据。 SG。 | 长吉奥里 | 长吉奥里 |
 | 附件。 SG。 | 长吉奥雷姆 | 长吉斯 |
 | 阿布。 SG。 | 长焦雷 | 长焦雷 |
 | 标称/附件 pl。 | 长 | 长吉奥拉 |
 | 将军。 | 长吉奥鲁姆 | 长吉奥鲁姆 |
 | 数据/Abl。 pl。 | 长吉奥里布斯 | 长吉奥里布斯 |

 ## 2. 最高级形容词 — 规则构成

 大多数形容词：将 **-issimus/-issima/-issimum** 添加到词干：

 | 积极| 最高级|
 |----------|-------------|
 | 长肌| 长**issimus/a/um** |
 | 富通 | 堡垒**伊西穆斯/a/um** |
 | 菲利克斯 | fēlic**issimus/a/um** |

 **特殊规则 - 以 *-er* 结尾的形容词：** 将 *-rimus/-rima/-rimum* 添加到主格阳性（不是词干）：

 | 积极| 最高级 |
 |----------|-------------|
 | acer、acris | āc**errimus/a/um** |
 | 芹菜，芹菜| celer**rimus/a/um** | celer**rimus/a/um** |
 | 吝啬鬼，吝啬鬼| 守财奴**rimus/a/um** |

 **特殊规则 - 以 *-ilis* 结尾的形容词：** 添加 *-limus/-lima/-limum*：

 | 积极| 最高级|
 |----------|-------------|
 | 设施| fac**illimus/a/um** | fac**illimus/a/um** |
 | 艰难梭菌 | 艰难梭菌**limus/a/um** |
 | 相似| 类似**limus/a/um** |

 最高级形容词作为常规第一/第二变格形容词而下降（*-us/-a/-um*）。 

---

 ## 3. 不规则比较

 这些一定要熟记； 比较级和最高级是由完全不同的词干构成的：

 | 积极| 比较（男/女/女）| 最高级 |
 |----------|------------------------|-------------|
 | 奖金，-a，-um（好）| 梅利奥尔/梅利乌斯| 擎天柱/a/um |
 | malus，-a，-um（坏）| 佩尔/佩尤斯| 悲观/a/um |
 | 马格努斯，-a，-嗯（很棒）| 梅奥/梅尤斯 | 马克西姆斯/a/um |
 | parvus，-a，-um（小）| 次要/减| 最小/a/um |
 | multus，-a，-um（很多/很多）| — / 加 | plūrimus/a/um |

 **关于 *plūs* 的特别说明：** 在单数形式中，*plūs* 用作 **中性名词**，而不是形容词。 它需要一个 **部分所有格**：
 - *plūs vīnī* — 更多的酒（字面意思是“更多的酒”）
 - *plus temporis* — 更多时间

 在复数形式中，*plūs* 通常用作形容词：*plūrēs mīlitēs*（更多士兵）。 

---

 ## 4. 副词构成

 **肯定副词：** 从第一/第二变格形容词：将 *-ē* 添加到词干 (*longus* → *longē*)。 从第三变格形容词：添加 *-iter* 或使用中性宾格 (*fortiter*、*feliciter*、*facile*)。 

**比较副词：** = 比较形容词的**中性宾格单数**：
 - *longior* → 副词比较级：*longius*（更多/相当长）
 - *改善* → *melius*（更好）
 - *maior* → *magis* （更多 — 不规则）

 **最高级副词：** 最高级词干 + *-ē*:
 - *longissimus* → *longissimē*（大多数/非常长）
 - *optimus* → *optimē*（最好，非常好）
 - *fortissimus* → *fortissimē*（最勇敢）

 | 学位| 形容词| 副词|
 |--------|------------|--------|
 | 积极| 长肌| 龙|
 | 比较| 朗吉奥尔 | 长吉斯 |
 | 最高级 | 最长肌| 长

 ---

 ## 5. 表达“Than”的两种方式

**一个。 *quam* 结构：** 比较后使用 *quam*（“比”）； 比较的两个事物必须处于**相同的情况**：

 *Caesar fortior est quam Pompeius。* — 凯撒比庞培更勇敢。 
（*凯撒* 名义。= *庞培* 名义。）

 **B。 比较的夺格：** 省略 *quam*； 将第二个元素放在**消格**中：

 *Caesar Pompeiō fortior est.* — 凯撒比庞培更勇敢。 
（*Pompeiō* 夺格代表“比庞培”）

 只有当比较标准是主格或宾格时，比较的夺格才可能。 如果是另一种情况，则需要*quam*。

 ## 例句

 1. *Via haec longior est quam illa.* — 这条路比那条路长。 
2. *Caesaromnibus mīlitibus fortiorerat.* — 凯撒比所有士兵都勇敢。 
3. *Liber Marcī brevior est librō Lūcī.* — Marcus 的书比 Lucius 的书短。 
4. *Melius est tacēre quam stulta dicere.* — 沉默胜于说蠢话。 
5. *Maxima urbs Imperīerat Rōma。* — 帝国最伟大的城市是罗马。 
6. *Puer celerius cucurrit quam puella。* — 男孩比女孩跑得更快。 
7. *Optimus magister discipulōs laudat.* — 最好的老师赞扬他的学生。 
8. *Plūs temporis dabimus.* — 我们将给予更多时间。 
9. *Pessimus servus ā dominō pūnītur.*——最坏的奴隶会受到主人的惩罚。 
10. *Nihil melius est amīcitiā.* — 没有什么比友谊更好的了。
