## 不完美时态

 ### 不完美意味着什么

 **不完美**（拉丁语 *imperfectum* =“未完成”）表示：

 - **正在进行的或连续的过去动作**：“我在爱”，“她在统治”
 - **习惯性或重复过去的行为**：“他曾经警告过”，“他们每天都会唱歌”
 - **尝试采取的行动**：“我试图采取”（意向未完成式）

 不完美**永远**不是一个已完成的事件。 这就是**完成**时态的作用（第 9 课）。 

### 不完美标志：-bā-

 未完成时是通过在现在词干和人称词尾之间插入 **-bā-** （在 -m 和 -nt 词尾之前缩短）形成的。 

**公式：** 现在词干 + **bā/ba** + 人称词尾（-m、-s、-t、-mus、-tis、-nt）

 ##将来时

 ### 两种不同的未来阵型

 拉丁语有**两种**不同的未来模式，具体取决于词形变化：

 - **第一和第二变位**：未来符号 **-bi-** （在 1sg 中具有变体 -bō-，在 3pl 中具有变体 -bu-）
 - **第三、第三-io和第四个结合**：未来符号**-a-**（1sg）/ **-ē-**（其他人）

 这种区别至关重要——混淆它们是一个常见的错误。

 ## 十个例句 — 三种时态混合

 **1.** *Puella silvam amābat。*
 - *amābat* = 不完美的第三条。 → **翻译：** 这个女孩曾经很喜欢森林。 / 这个女孩很喜欢森林。 

**2.** *Mīlitēs oppidum capiēbant。*
 - *capiēbant* = 不完美的 3rd pl.，3rd-io → **翻译：** 士兵们正在占领城镇。 

**3.** *Magister saepe puerōs monēbat.*
 - *monēbat* = 不完美的第三个词，第二个连词。 （习惯）→ **翻译：** 老师经常用来警告男孩们。 

**4.** *Nōs erāmus agricolae； nunc cīvēs sumus.*
 - *erāmus* = 不完美的第一 pl。 *埃塞*的； *总和* = 现在
 - **翻译：** 我们是农民； 现在我们是公民。 

**5.** *Mox rēx oppidum vincet。*
 - *vincet* = 未来的第 3 个 sg.、第 3 个 conj. → **翻译：** 很快国王就会征服这座城镇。 

**6.** *Nāvigābisne ad Rōmam？*
 - *Nāvigābis* = 未来第 2 个 sg.，第 1 个 conj.； *-ne* = 是/否问题助词
 - **翻译：** 你会航行到罗马吗？ 

**7.** *Amīcī meī semper vērum dīcent.*
 - *dīcent* = 未来第 3 层，第 3 个连接。 → **翻译：** 我的朋友永远都会说实话。 

**8.** *Dum puella canēbat, nauta dormīēbat.*
 - *canēbat* = 不完美的第三个连词，第一连词。 （*canere*，唱歌）； *dormīēbat* = 不完美的第三个语法，第四个连词。 
- **翻译：** 当女孩唱歌时，水手正在睡觉。 

**9.** *Erimus Rōmae olim。*
 - *Erimus* = 未来的第一层。 *埃塞*的
 - **翻译：** 有一天我们会到达罗马。 

**10.** *Servī labourābant； 多米努斯·多米埃巴特.*
 - 都是不完美的、习惯性的/同时发生的过去式
 - **翻译：** 奴隶们正在工作； 主人正在睡觉。

 ## 总结

 - **不完美**（符号**-bā-**）表达过去正在进行的、习惯性的或尝试的行动。 
- **未来**有两种模式： **-bi-/-bō-/-bu-** 用于第一/第二变位； **-a-/-ē-** 代表第三/第四。 
- *Esse* 未完成时：*eram、erās、erat、erāmus、erātis、erant*。 
- *Esse* 未来：*erō、eris、erit、erimus、eritis、erunt*。 
- 留意 *olim、mox、semper、saepe* 等副词 — 它们通常表示要使用哪种时态。
