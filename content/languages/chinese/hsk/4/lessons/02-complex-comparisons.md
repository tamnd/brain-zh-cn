## 概述

 与比（A比B+形容词）的基本比较是A2技能。 在 B1 处，您需要表达大多少、多多少倍以及近似等价。 这些模式是...的N倍、比...V倍和大约允许您进行定量和分级的比较，这些比较经常出现在学术写作、新闻和日常说服中。 棘手的部分是控制词序：中文的比较结构将程度词放在动词后面，而不是像英语那样放在动词前面。 

## 学习目标

 - 用是...的N倍构建乘法比较。 
- 用“比 + 倍数/得少”来表达显着程度的差异。 
- 应用大约和超出来表达近似相等和可测量的差距。 
- 添加程度补语时避免常见的词序错误。

 ## 语法重点

 ###模式1：A是B的N倍

 **结构：** A + 是 + B + 的 + [数字] + 倍

 用这个来表示 A 是 B 的 N 倍（一样多）。请注意，是...的 N 倍意味着恰好是 B 的 N 倍，而不是更多的 N 倍。 因此，“两倍”的意思是“两倍”，而不是“两倍多”。 

| 中文 | 拼音| 英语 |
 |--------|--------|---------|
 | 中国人口是加拿大的三倍。 中国的人口是三十多倍。 | 中国人口是加拿大的30多倍。 |
 | 本项目的预算是上次的两倍。 这个象母是玉素石上的两杯。 | 这个项目的预算是上次的两倍。 |
 | 他的薪水是我的三倍。 他的心水是我的三倍。 | 他的工资是我的三倍。 |

 **常见错误：** 不要说“是B多N倍”来表示“N倍”。 这是英语“三倍以上”的频繁干扰。 在中文里，是...的三倍，意思是总数的三倍。 

### 模式2：A比B + 动词 + 指甲 / 多了

 **结构：** A + 比 + B + [动词/形容词] + 指甲 OR A + 比 + B + [动词/形容词] + 多了

 这表达了显着差异：“多于/少于”。 “多了”和“多了”在这里可以互换，但“多了”听起来更口语化一些。 

| 中文 | 拼音| 英语 |
 |--------|--------|---------|
 | 今年的收入比去年多。 今年的岁月比现在多多了。 | 今年的收入比去年多很多。 |
 | 这条路比那条路短多了。 Zhe tiáo lù bǐ nà tiáo lù duǎn duō le。 | 这条路比那条短得多。 |
 | 她的中文比我好。 他的中文比我好得多。 | 她的中文比我的好得多。 |

 **常见错误：** 避免在比结构中插入“很”或“非常”。 “比我很好”是错误的。 所有程度的表达都在形容词之后：比我好。 

### 模式 3：大约和外部

 **结构A：** A + 和/跟 + B + 大约（A和B大致相同）
 **结构 B:** A + 跟 + B + 外 + [amount/很大/不大] (A 和 B 不同...)

 | 中文 | 拼音| 英语 |
 |--------|--------|---------|
 | 这两款手机的价格差不多。 这量宽受吉的加格查布多。 | 这两款手机的价格差不多。 |
 | 他们人的水平差不多。 他门两块人的水瓶香茶不都。 | 两人的技术水平相差不大。 |
 | 他们的年龄已经接近十岁了。 他门的年令香茶江津十岁。 | 他们的年龄相差近十年。 |

 **对比：**“大约”作为独立副词也可以表示“大约”（大约三点钟=大约三点钟），因此上下文决定了它是表达近似相等还是近似数量。 

## 对话

 **背景：** 两个朋友比较他们收到的工作机会。

A：你最后决定接受哪家公司的offer了？ 
你zuìhòu juédìng jiēshòu nǎ jia gōngsī de Offer le？ 
*您最终接受了哪家公司的offer？*

 B：我选了那家公司，虽然薪水和大公司差不多，但股权更好。 
我虚了那家楚闯公司，随然心水和大公司查不多，但古泉更浩。 
*我选择了创业公司。 薪资和大公司差不多，但股权更好*

 A：真的吗？我以为公司工资一般比大公司低。 
真得妈？ 我一味出了公斯公子一板比大公斯二。 
*真的吗？ 我认为初创公司的薪水通常比大公司低得多。*

 B：以前是这样，但现在竞争激烈，差距没那么大了。 
以前是这样，但现在在jìngzhēng jīliè，chājù méi name dà le。 
*以前是这样，但现在竞争激烈，差距不再那么大了。*

 A：那福利呢？ 
Nà fúlì ne？ 
*福利怎么样？*

 B：大公司的福利是公司的两倍之多，这方面的差距还是很明显的。 
大公寺的福利市楚楚公寺的两杯不值，这方面查觉海市亨明宪。 
*大公司的收益是初创公司的两倍以上。 这方面的差距还是非常明显的。*

 A：那你不补吗？ 
Nà nǐ bù hòuhuǐ？ 
*你不后悔吗？*

 B：不修复。成长空间比大公司大离婚，这对我来说更重要。 
不侯回。 城隍空建比大公司大得多，这对我来说更中要。 
*不后悔。 成长空间比大公司大得多，这对我来说更重要。*

 A：有道理。每个人的程度和目标都不一样，没办法直接比。 
有道砚。 Měi gè rén de chéngdù hé mùbiāo bù yīyang, méi bànfǎ zhijiē bǐ。 
*有道理。 每个人的情况和目标都不同； 你不能直接比较。*

 ## 练习

 **练习1：完成比较句。**

 1. 上海的房价是十年前的___（五倍）。 
2. 这家餐厅的菜比那家___（好多了）。 
3. 两个学生的成绩___（相差很多）。 

**练习 2：纠正错误。**

 2. 北京是上海的人口增长。→ ___
 3. 他们的年龄大约三岁。→ ___

## 文化笔记

 中国商业和学术环境中的比较很少是随意的。 引用特定的倍数（是...的N倍）或有数据支持的程度差异表明可信度。 在谈判中，中国专业人士经常战略性地使用“大约”来软化立场，而不是陈述客观事实。 说我们的价格和市场价格大约可以意味着从“几乎相同”到“大致相同”的任何意思，故意留下解释的空间。 认识到这种歧义与正确生成语法同样重要。
