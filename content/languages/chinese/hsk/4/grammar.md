HSK 4 语法涵盖大约 80 种核心模式，可实现自信的中高级交流。 此级别侧重于复杂比较、形式语域、高级条件结构、主题注释句子和细致入微的方面用法。 

## 1. Advanced Comparisons

 ### 1.1 比与具体差异

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | A + 比 + B + Adj + 一点儿 | 他比我高一点儿。 He is a little taller than me. |
 | A + 比 + B + Adj + 指甲 | 她比他聪明。| She is much smarter than him. |
 | A + 比 + B + Adj + 多了 | 今天比昨天热多了。| 今天比昨天热多了。 |
 | A + 比 + B + 多/少 + V + 数字 | 他比我多学了两年。 他比我多学了两年。 |

 ### 1.2 不如 (bùrú) — 不如

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | A + 不如 + B (+ Adj) | 我不如他跑得快。| I don't run as fast as him. |
 | A + 不如 + B | 这个方案优于那个。| This plan is not as good as that one. |

 ### 1.3 是……的结构——强调情况

 用于强调**过去**动作的时间、地点、方式或代理人。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | 是 + [Emphasis] + V + 的 | 他是昨天来的。| 他昨天才来。 （强调时间） |
 | 是 + [Emphasis] + V + 的 | 我是在北京学的中文。| I learned Chinese (specifically) in Beijing. |
 | 是 + [Emphasis] + V + 的 | 这是谁写的？| 这是谁写的？ (whose writing is this?) |

 ## 2. Complex Conjunctions

 ### 2.1 不仅……而且 — 不仅……而且

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 不但 + A，而且 + B | 他不但会中文，而且会日语。| He not only knows Chinese but also Japanese. |
 | 不仅……还 | 她不仅聪明，还很勤奋。| 她不仅聪明而且勤奋。 |

 ### 2.2 虽然…却/但是 — 虽然…然而

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 虽然 + 让步，却 + 对比 | 虽然累，却很开心。| 虽然累，但是还是很开心。 |
 | 尽管……仍然| 尽管下雨，仍然来了。| 尽管下雨，（我）还是来了。 |

 ### 2.3 既…又 — 两者…并且

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 既 + A + 又 + B | 这道菜既好吃又便宜。 This dish is both delicious and cheap. |
 | 又...又| 他又高又帅。| 他既高又帅。 |

 ### 2.4 宁可/不愿…也 — Willrather…Tor

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 宁可 + A + 也 + B | 宁可早到，也不迟到。 宁愿早到也不愿迟到。 |
 | 取代 + A + 也不 + B | | 取代一个人，也不会就。 宁愿孤独也不愿安定下来。 |

 ## 3. Topic-Comment Sentences

 该主题首先确定正在讨论的内容，然后是对此的评论。 

### 3.1 Basic Topic-Comment

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 主题+评论 | 那本书，我看完了。| 那本书——我已经读完了。 |
 | 主题+评论 | 中文，他学了三年了。| Chinese — he has studied it for three years. |
 | 主题+评论 | 这件事，我不太清楚。| 这件事——我不太清楚。 |

 ### 3.2 Double-Subject Sentences

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | S1 + S2 + 副总裁 | 那栋楼，门坏了。| 那栋楼——门坏了。 |
 | S1 + S2 + 调整 | 大象，鼻子很长。| 大象——它的鼻子很长。 |

 ## 4.越...越模式

 ### 4.1 越来越多 — 越来越多

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 更多 + 调整 | 天气渐冷。| The weather is getting colder and colder. |
 | 越来越+V | 他越来越不想上班。| 他越来越不想去上班。 |

 ### 4.2 越…越 — 越多…越多

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 越 + V + 越 + Adj | 越学越音乐家。| 你研究得越多，它就会变得越有趣。 |
 | 越 + Adj + 越 + V | 越贵越不想买。| 越贵我越不想买。 |

 ## 5. 形式否定模式

 ### 5.1 不…也不 — 也不…也不

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 不是 A + 也不是 B | 他不吃也不喝。 他既不吃也不喝。 |

 ### 5.2 非…不可 — 必须； 忍不住

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | + 非 V + 不可 | 这件事非做不可。| 这件事必须要做（不能回避）。 |
 | 非要 + V | 他非去。| 他坚持要去。 |

 ### 5.3无论/不管…都/也 — 无论如何…仍然

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | 无论+条件+都 | 无论多忙，都要吃饭。| 不管多忙，都要吃饭。 |
 | 不管 + 条件 + 也 | 不管他说什么，我也不信。| 不管他说什么，我都不会相信。 |
 | 无论…还是… + 都 | 无论刮风还是下雨，都要来。 不管刮风还是下雨，一定要来。 |

 ## 6. 表达遗憾和一厢情愿

 ### 6.1 如果/如果……就好了 — If Only

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 要是+条件+就好了| 要是我有钱就好了。| 要是我有钱就好了。 |
 | 如果……多好啊 | 如果你在就多好啊。| 如果你在这里该多好啊。 |

 ### 6.2 早知道…就 — 如果我早知道的话

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 早知道+条件+就+结果| 早知道这么难，就不来了。| 如果我知道这么难，我就不会来了。 |

 ## 7.目的和目标表达

 ### 7.1 为了 — 为了/为了……的目的

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 为了 + 目标 + V | 为了减肥，她每天都跑步。 为了减肥，她每天都跑步。 |
 | 为了+人/物| 为了家人，他努力工作。| 为了家人，他努力工作。 |

 ### 7.2 方便 — So As To / For the Convenience

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 通知 + 目标 | 请早点通知我，以便我安排时间。| 请尽早通知我，以便我安排行程。 |

 ---

 ## 8. 用倒是/是让步

 ### 8.1 倒是 — 诚然/确实如此，但是……

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | Adj + 倒是 + Adj，但是… | 便宜倒是便宜，但质量不好。| 它很便宜，但质量很差。 |
 | 声明 + 倒是 | 他倒是来了，可没说话。| 他确实来了，但什么也没说。 |

 ## 9. 枢轴句和使役句

 ### 9.1 让/叫/使 + 人称 + V — 使役

 | 动词| 注册 | 示例| 英语 |
 |------|----------|---------|---------|
 | 让 | 中立 | 让老师我重写。| 老师让我重写。 |
 | 叫| 口语| 妈妈叫我回家。| 妈妈叫我回家。 |
 | 使| 正式 | 这让我很感动。| 这让我深受感动。 |
 | 令 | 正式/书面| 这令人惊讶。| 这令人惊讶。 |

 ## 10. 什么…都/也 — 任何/每个

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 什么 + 都/也 + V | 他什么都吃。| 他什么都吃。 |
 | 谁+都/也+V | 谁都知道。| 每个人都知道。 |
 | 哪里 + 都/也 + V | 哪里都可以。| 任何地方都可以。 |
 | 怎么 + 都/也 + V | 怎么也找不到。| 怎么也找不到啊 |

 ## 11.时间表达和时态

 ### 11.1 已经…了 — 已经

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 已经+V+了| 他已经离开了。 他已经离开了。 |
 | 都 + 已经 + V + 了 | 都已经十点了。| 已经十点了。 |

 ### 11.2 快/快要/将要…了 — 关于

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 快+V+了 | 快下课了。| 课程即将结束。 |
 | 快要 + V + 了 | 我快要毕业了。| 我即将毕业了。 |
 | 将要 + V | 他将要出发。 他正要离开。 （正式）|

 ### 11.3 曾经 — 曾经（过去）

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 曾经 + V + 过 | 我曾经去过那里。| 我曾经去过那里。 |

 ## 12. 重点结构

 ### 12.1 连…都/也 — 甚至…（仍然/也）

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 连 + N + 都 + V | 连孩子都知道。| 连小孩子都知道。 |
 | 连 + V + 都 + V-Neg | 连吃饭都没时间。| 连吃饭的时间都没有。 |

 ### 12.2 就用于强调/限制

 | 用途 | 示例| 英语 |
 |--------|---------|---------|
 | 限制| 就这一个。| 就这一个。 |
 | 强调条件| 我就是不去。| 我干脆拒绝去。 |

 ## 13. 句末助词

 | 粒子| 语气| 功能| 示例|
 |----------|------|----------|------|
 | 了 | — | 状态改变| 下雨了。现在下雨了。 |
 | 吧| — | 建议/假设| 走吧。走吧。 / 你是老师吧？|
 | 呢 | — | 持续状态/温和问题| 他呢？他在哪儿？ /还在呢。还在这儿。 |
 | 嘛| — | “显然”/解释 | 他不来嘛，你知道的。显然他不会来。 |
 | 啊| — | 惊讶、感叹、软化| 真漂亮啊！太漂亮了！ |

 ## 14. 关键模式总结

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | 不仅…而且| 既努力，又聪明。| 不仅勤奋，而且聪明。 |
 | 是……的| 是在哪儿学的？| 你从哪里学来的？ |
 | 连…都 | 连这个也不知道。| 连这个都不知道 |
 | 无论…都 | 无论怎样都要坚持。| 不管怎样，一定要坚持。 |
 | 更多 | 越来越好。| 越来越好。 |
 | 非…不可| 去非不可。| 必须走。 |
 | 为了| 为了成功而努力。| 为成功而努力。 |
