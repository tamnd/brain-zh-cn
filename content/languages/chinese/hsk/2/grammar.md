HSK 2 语法涵盖了更流利地处理日常情况所需的大约 55 个核心模式。 该级别以 HSK 1 结构为基础，引入了比较、方面标记、补语和更广泛的助词。 

## 1. 比较

 ### 1.1 比 (bǐ) 比较 — “A 比 B 更多……”

 标准比较结构在被比较的两个项目之间使用“比”。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | A + 比 + B + 形容词 | 他比我高。| 他比我高。 |
 | A + 比 + B + 形容词 + 程度 | | 她比我高多了。 她比我高很多。 |
 | A + 比 + B + V + 更多/更少 | 我给他多喝了一杯。 我比他多喝了一杯。 |

 > **注意：** 不要在比句中使用“很”。 ~~他比我~~不正确。 

### 1.2 没有 (méiyǒu) 否定比较 — “A 不像……B”

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | A + 没有 + B + 形容词 | 这条路没有那条路宽。 这条路没有那条那么宽。 |
 | A + 没有 + B + V + 这么多 | 我没有他跑得快。 我跑得没他快。 |

 ### 1.3 一样 (yīyàng) — “相同”

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | A + 跟/和 + B + 一样 (+ Adj) | 这个跟那个一样贵。 这个和那个一样贵。 |
 | A + 跟 + B + 不一样 | 我跟他不一样。| 我和他不一样。 |

 ## 2. 方面标记

 ### 2.1 了 (le) — 已完成的动作

 了在动词之后表示动作已经完成。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 了 (+ 对象) | 我吃了饭。| 我吃过了。 |
 | V + 了 + 数量 + MW + N | 他买了三本书。| 他买了三本书。 |
 | 没 + V (没有了) | 我没吃饭。| 我没吃过。 |

 > 句末的了也可以表示状态的变化（情境了）：下雨了“It’s raining now.”

 ### 2.2 过 (guò) — 体验方面

 过记录了一次生活经历——过去某个时间至少做过一次的事情。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 过 (+ 宾语) | 我去北京。| 我去过北京（在某个时候）。 |
 | 没 + V + 过 | 我没吃过北京烤鸭。| 我从来没有吃过北京烤鸭。 |
 | V + 过 + 吗？ | 你学过中文吗？| 你学过中文吗？ |

 ### 2.3 着 (zhe) — 持续状态

 着标志着由一个动作（某事物是怎样的）导致的持续状态。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 着 (+ 宾语) | 门开着。| 门是开着的。 |
 | V1 + 着 + V2 | 他笑着说道。 他笑着说道。 |
 | 穿着/拥有 | 她穿着红色的衣服。| 她穿着红色的衣服。 |

 ## 3. 势补和度补

 ### 3.1 得 (de) 程度补语

 得将动词与动作完成情况的描述联系起来。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 得 + Adj/VP | 他说得很好。| 他讲得很好。 |
 | V + 得 + Adj + 了 | 你来得太晚了。| 你来得太晚了。 |
 | 否定：V + 得 + 不 + Adj | 他写得不好看。| 他写得不工整。 |

 ### 3.2 结果补语

 直接跟在主要动词后面的第二个动词或形容词显示结果。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V+好 | 作业完成了。 作业做完了。 |
 | V + 完成 | 我读完了这本书。| 我读完了这本书。 |
 | V + 到 | 他找到了工作了。| 他找到了一份工作。 |
 | V + 见 | 我听到了。| 我听到了。 |
 | V + 错 | 你写错了。| 你写错了。 |

 ### 3.3 潜在的补充——能/不能成功

 在动词和结果补语之间插入“得”或“不”。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 得 + 结果 | 听得懂| 能够理解（通过听）|
 | V + 不 + 结果 | 听不懂 | 无法理解（通过听）|
 | 看得见 / 看见 | 你看得见吗？| 你能看到吗？ |
 | 做完/做未完成| 这么多，做还没完。| 这么多——无法完成。 |

 ## 4.情态动词

 ### 4.1 想 (xiǎng) — 想要 / 愿意

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | S + 想 + V | 我想去北京。| 我想去北京。 |
 | S + 不想 + V | 他不想学习。| 他不想学习。 |

 ### 4.2 要 (yào) — 想要/需要/愿意

| 用途 | 示例| 英语 |
 |--------|---------|---------|
 | 想要| 我要一杯咖啡。 我想要一杯咖啡。 |
 | 需要/必须| 你要好好休息。| 你需要好好休息。 |
 | 意愿（未来）| 明天要下雨。| 明天会下雨。 |

 ### 4.3 会 (huì) — 知道如何/将（可能）

 | 用途 | 示例| 英语 |
 |--------|---------|---------|
 | 能力| 我会说中文。| 我会说中文。 |
 | 可能性| 他会来的。| 他（可能）会来。 |

 ### 4.4 能 (néng) — 可以（物理上/情境上）

 | 用途 | 示例| 英语 |
 |--------|---------|---------|
 | 身体能力| 我能吃两碗饭。| 我可以吃两碗米饭。 |
 | 情况许可 | 这里能抽烟吗？| 这里可以抽烟吗？ |

 ### 4.5 应该 (yīnggāi) — 应该/应该

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | S + 应该 + V | 你应该早点睡觉。| 你应该早点睡觉。 |
 | S + 不应该 + V | 他不应该这样做。| 他不应该这样做。 |

 ## 5. 持续时间和频率

 ### 5.1 表达持续时间

 时间跨度位于动词之后（或者宾语之后，如果有的话）。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 时间跨度 | 我等了两个小时。 我等了两个小时。 |
 | V + O + V + 时间跨度 | 他学了三年文学。 他学了三年中文。 |

 ### 5.2 频率表达式

 | 表达 | 意义| 示例|
 |------------|---------|---------|
 | 每天 | 每天 | 我每天都跑步。|
 | 经常 | 经常| 她经常迟到。 
| 偶尔 | 有时| 我偶尔做饭。|
 | 有时 | 偶尔| 他偶尔穿着。 
| 从来不| 从不（习惯性） | 我从来不喜欢。|

 ## 6. 位置和方向

 ### 6.2 方位词（方位词）

 |------|---------|---------|

 | 右侧 | 右侧| 右边 | 右边 |

 ## 7. 连接词和连接词

 ### 7.1 因为…所以 — 因为…因此

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 因为 + 原因，所以 + 结果 | 因为下雨，所以我没去。| 因为下雨，所以我没有去。 |
 | 可以省略一个：因为…… | 因为太贵了，我没买。| 因为太贵了，所以没买。 |

 ### 7.2 虽然…但是 — 虽然…但是

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 虽然+让步，但是+对比| 虽然很贵，但是很好。| 虽然贵，但是也不错。 |

 ### 7.3 如果…就 — 如果…那么

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 如果+条件，就+结果| 如果明天下雨，就结束了。| 如果明天下雨，我们就不去了。 |

 ### 7.4 一边…一边 — 同时

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 侧面+V1+侧面+V2| 他一边一边吃一边看电视。| 他一边看电视一边吃饭。 |

 ## 8. 粒子把（简介）

 把将宾语移到动词之前，以强调处置或处理某物。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | S + 把 + O + V + 补 | 我把书放在桌子上。 我把书放在桌子上。 |
 | S + 把 + O + V + 了 | 她把饭吃完了。| 她把米饭吃完了。 |

 > 完全掌握结构是在 HSK 3 中。在 HSK 2 中，认识其基本形式。

 ## 9. 问题与回答

 ### 9.1 带有“吧”的修辞/建议问题

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | 声明+吧？ | 你是学生吧？| 你是学生，对吧？ |
 | 建议+吧| 我们走吧。| 我们走吧。 |

### 9.2 还是 (háishi) — 或者（疑问句中）

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | A + 还是 + B？ | 你喝茶还是咖啡？| 您想要茶还是咖啡？ |

 > 在疑问句中使用“还是”，在陈述句中使用“或者”(huòzhě)：茶或者咖啡都可以。 

### 9.3 怎么了 — 出了什么问题？ 

| 表达 | 用途 |
 |------------|--------|
 | 怎么了？| 发生了什么？ 怎么了？ |
 | 你怎么了？| 你这人怎么回事？ |
 | 没什么。/没什么。| 没有什么。 / 没什么。 |

 ## 10. 关键句型总结

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | A 比 B + 调整 | 他比我忙。| 他比我还忙。 |
 | V+了 | 我来了。| 我到了。 |
 | V + 过 | 我去。| 我去过那里。 |
 | V + 着 | 门开着。| 门是开着的。 |
 | V + 得 + 形容词 | 写得很好。| 写得很好。 |
 | 因为……所以| 因为累了，所以睡了。| 因为累了，所以就睡了。 |
 | 虽然……但是| 虽然难，但是有趣。| 虽然很难，但是很有趣。 |
 | 侧面…侧面| | 一边唱一边跳。 边唱歌边跳舞。 |
