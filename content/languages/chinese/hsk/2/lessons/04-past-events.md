## 概述

 中文不像欧洲语言那样有时态。 相反，完成和改变都用助词“了”（le）来标记。 本课重点介绍标记已完成动作的动词“了”、如何用“没有”否定已完成的动作，以及诸如“刚才”和“已经”等表示最近发生的事件的时间词。 

## 学习目标

 - 用V+了+（宾语）来描述已完成的动作
 - 用“没有”+V（没有了）正确否定过去的行为
 - 使用“刚才”和“已经”来指定时间
 - 区分动词的“了”和句尾的“了”（状态改变）

 ## 语法重点

 ### 模式1：V +了+（对象）

 **结构：** 主语+动词+了+（宾语）

 动词后立即出现助词“了”，表示动作已完成。 当有直接宾语时，动词和宾语之间出现“了”。 

| 中文 | 拼音| 英语 |
 |--------|--------|---------|
 | 我吃了早饭。| Wǒ chī le zǎofàn。 | 我吃了早餐。 |
 | 她买了一件新衣服。| Tā mǎi le jiàn xīn yīfu。 | 她买了一件新衣服。 |
 | 他告诉了我这件事。| Tā gàosù le wǒ zhè jiàn shì。 | 他告诉了我这件事。 |
 | 我们完成了任务。| Wǒmen wánchéng le rènwù。 | 我们完成了任务。 |

 **常见错误：** 不要在“是”、“有”、“觉得”等静态动词或形容词后面加“了”。 了标志着动作动词的完成，而不是状态动词。 

### 模式 2：没有 + V（否定过去的行为）

 **结构：** 主语 + 没有 + 动词 + （宾语）

 要表示某事没有发生，请在动词前使用“没有”。 至关重要的是，不要使用否定形式的“了”。 两者不能共存于同一个子句中。 

| 中文 | 拼音| 英语 |
 |--------|--------|---------|
 | 我没有吃早饭。| Wǒ méiyǒu chī zǎofàn。 | 我没有吃早餐。 |
 | 她没有告诉我。| Tā méiyǒu gàosù wǒ。 | 她没有告诉我。 |
 | 他昨天没有来。 Tā zuótiān méiyǒu lái。 | 他昨天没来。 |
 | 我没有买那本书。| Wǒ méiyǒu mǎi nà běn shū。 | 我没有买那本书。 |

 **常见错误：**永远不要说我没有吃过或没有买过。 用“没有”来否定时，完全去掉“了”。 

### 模式 3：刚才和已经

 **结构：** 刚才 + S + V + 了（刚刚） / S + 已经 + V + 了（已经）

 刚才（just now）指的是最近刚刚过去的时刻，通常是在最后几分钟内。 已经（已经）强调某事在当前时刻或期望之前完成。 

| 中文 | 拼音| 英语 |
 |--------|--------|---------|
 | 刚才他来了，但你不在。| 刚才打来了，但你不在家。 | 他刚刚来了，你却不在。 |
 | 我已经吃了，不饿了。| Wǒ yǐjīng chī le, bù è le。 | 我已经吃过饭了，我不饿。 |
 | 火车已经开走了。| Huǒchē yǐjīng kāi zǒu le。 | 火车已经开走了。 |
 | 刚才发生了什么？| 刚菜发盛了？ | 刚刚发生了什么？ |

 ## 对话

 **A：**你刚才去哪儿了？我找到你好久了。 
**你刚才去哪了？ 我着了你好几.**
 *你刚才去哪儿了？ 我找你很久了。*

 **B：** 对不起，我去便利店买了点东西。 
**对不齐，wǒ qù biànli diàn mǎi le diǎn dōngxī.**
 *抱歉，我去便利店买了一些东西。*

 **A：**会议已经开始了，老板问你去哪儿了。 
**Huìyì yǐjīng kāishǐ le, lǎobǎn wèn nǐ qù nǎr le.**
 *会议已经开始。 老板问你去哪儿了。*

 **B：**啊？我忘了今天有会议！你为什么没有告诉我？ 
**À？ 我望了金田有慧仪！ 你wèishénme méiyǒu gàosù wǒ?**
 *什么？ 我忘了今天还有会议！ 你为什么不告诉我？*

**A:** 我昨天发了短信，你没看吗？ 
**我作天发了度心，你没看妈？**
 *我昨天给你发了一条消息。 你没看到吗？*

 **B:** 没有！我没有收到。快走吧！ 
**美由！ Wǒ méiyǒu shōu dào。 快走吧！**
 *不！ 我没有收到。 我们快点走吧！*

 ## 练习

 ### 练习 1：填空

 在需要的地方添加了，或者没有则否定：

 1. 我昨天______吃晚饭，不饿。（我昨天吃了晚饭，我不饿。）
 2. 他______告诉我这件事。（他没有告诉我这件事。）
 3. 你______ 完成作业了吗？（你已经完成作业了吗？）
 4. 她刚才 ______ 来，但是你不在。（她刚才来了，但你不在。）
 5. 我找了______半个小时，还是没找到。（找了半个小时还是没找到。）

 ### 练习 2：翻译成中文

 1.我已经给他打过电话了。 
2. 她昨天没有来上课。 
3. 刚才发生了什么？ 
4. 我们上周完成了这个项目。 

### 练习3：用中文回答

 1. 你今天早上吃了早饭吗？（今天早上你吃早餐了吗？）
 2. 你今天有没有忘带什么东西？（今天你忘记带什么东西了吗？）
 3. 昨天你做了什么有趣的事？（昨天你做了什么有趣的事情？）

 ## 文化笔记

 在中国人的日常对话中，“你吃了吗”这个问题是最常见的问候语之一，尤其是在老一辈人中。 它起源于食品安全真正受到关注的时代，询问饮食是真正关心的表达。 今天它的作用就像问“你好吗？” 而不是字面上询问膳食。 如果有人对你这么说，典型的反应是“吃了，你呢”（是的，我吃了。那你呢？），而不是真实描述你的上一顿饭。
