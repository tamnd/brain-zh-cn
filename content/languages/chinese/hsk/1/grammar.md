HSK 1 语法涵盖了基本交流所需的大约 66 个核心模式。 这些是所有汉语句子的基石。 

## 1. 基本句子结构

 ### 1.1 主语-动词-宾语 (SVO) — 默认词序

 汉语遵循严格的主语+动词+宾语顺序。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | S + V + O | 我吃米饭。| 我吃米饭。 |
 | S + V + O | 她看书。| 她读一本书。 |
 | S + V + O | 他们学习中文。| 他们学习中文。 |

 ### 1.2 动词是 (shì) —“成为”

 用于等同名词和名词短语。 **切勿与形容词一起使用**（使用“很”代替）。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | N + 是 + N | 我是学生。| 我是一名学生。 |
 | N + 是 + N | 她是老师。| 她是一位老师。 |
 | N + 不是 + N | 他不是医生。| 他不是医生。 |

 > **重要提示：** 不要说 ~~我很好~~ 来表示“我很好”。 请改用“我很好”。 

### 1.3 动词有（yǒu）——“有”和“有/有”

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | S + 有 + N | 我有一本书。| 我有一本书。 |
 | S + 有 + N | 他有两个孩子。| 他有两个孩子。 |
 | 地点 + 有 + N | 桌子上有书。| 桌子上有书。 |
 | S + 没有 + N | 我没有钱。| 我没有钱。 |

 ### 1.4 形容词谓语句子

 形容词充当谓语。 使用“很”（即使它并不强调——这是结构上必需的）。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | S + 很 + 形容词 | 他很高兴。| 他很高。 |
 | S + 很 + 形容词 | 今天很冷。| 今天很冷。 |
 | S + 不 + 形容词 | 这不好。| 这不好。 |

 ## 2. 问题形成

 ### 2.1 是/否问题是吗 (ma)

 在陈述句末尾添加“吗”以构成是/否问题。 

| 声明| 问题 | 英语 |
 |------------|---------|---------|
 | 你是学生。| 你是学生吗？| 你是学生吗？ |
 | 他吃了。| 他吃了吗？ 他吃了吗？ |
 | 天气好。| 天气好吗？| 天气好吗？ |

 ### 2.2 A-not-A 问题

 以肯定形式和否定形式重复动词。 

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | V + 不 + V | 你去不去？| 你去（或不去）？ |
 | 是不是| 他不是学生？| 他是学生（或不是）？ |
 | 有没有| 你没有时间？| 你有时间吗？ |

 ### 2.3 疑问词（疑问词 yíwèncí）

 | 词| 拼音| 意义| 示例| 英语 |
 |------|--------|---------|---------|---------|
 | 什么| 什梅 | 什么| 这是什么？| 这是什么？ |
 | 谁| 谢伊 | 谁| 他是谁？| 他是谁？ |
 | 哪 | nǎ | 其中 | 你是哪个国民？| 你来自哪个国家？ |
 | 哪里/哪里 | nǎr / nǎlǐ | 哪里 | 你去哪儿？| 你要去哪里？ |
 | 几| 吉 | 多少 (≤10) | 你有几本书？| 你有多少本书？ |
 | 多少 | 多少 | 多少/多少 | 多少钱？| 多少钱？ |
 | 怎么办？ 禅美 | 如何/为什么 | 怎么去？| 怎么走？ |
 | 怎么样 | zěmeyàng | 怎么样| 你怎么样？| 你好吗？ |
 | 为什么| 威森姆 | 为什么| 你为什么学中文？| 你为什么学习中文？ |
 | 什么时候 | shénme shíhou | 当 | 你什么时候来？| 你什么时候来？ |

 ### 2.4 用呢 (ne) 软化疑问句

 用来问“关于……呢？” 或继续话题。 

| 示例| 英语 |
 |---------|---------|
 | 我很好，你呢？| 我很好，你呢？ |
 | 这个呢？| 这个呢？ |

 ## 3. 否定

 ### 3.1 不 (bù) — 否定动词和形容词

 | 用途 | 示例| 英语 |
 |--------|---------|---------|
 | 否定动词| 我不去。| 我不去。 |
 | 否定形容词 | 天气不好。| 天气不好。 |
 | 否定是 | 他不是老师。| 他不是老师。 |
 | 否定想要/喜欢 | 我不喜欢。| 我不喜欢它。 |

 > **变调：**不是是第四声(bù)，但在另一个第四声之前变为第二声(bú)：不是(bú shì)、不要(bú yào)。 

### 3.2 没有 (méiyǒu) / 没 (méi) — 否定有和已完成的动作

 | 用途 | 示例| 英语 |
 |--------|---------|---------|
 | 否定有| 我没有钱。| 我没有钱。 |
 | 否定已完成的动作 | 我没吃。| 我（还没）吃饭。 |
 | 否定过去的事件| 他没来。| 他没有来。 |

 > **规则：** 使用没表示过去时/完成的动作。 使用“不”表示习惯性否定或未来计划。 
> - 我不吃肉。= 我不吃肉（习惯）。 
> - 我没吃肉。= 我没吃肉（这次）。 

## 4. 结构粒子的 (de)

 ### 4.1 占有（名词/代词 + 的 + 名词）

| 图案| 示例| 英语 |
 |---------|---------|---------|
 | 代词 + 的 + N | 我的书| 我的书|
 | N + 的 + N | 老师的书| 老师的书|
 | 姓名 + 的 + N | 王红的手机| 王红的电话|

 ### 4.2 修饰名词短语（形容词/动词短语 + 的 + N）

 | 示例| 英语 |
 |---------|---------|
 | 天气好| 好天气|
 | 聪明的学生| 聪明的学生|

 > **快捷方式：** 在亲密关系之前，“我的”可以缩写为“我”：
 > - 我妈妈（我的妈妈），我朋友（我的朋友）

 ## 5. 体助词了 (le)

 ### 5.1 已完成的动作（动词+了）

 | 示例| 英语 |
 |---------|---------|
 | 我吃了。| 我吃了。 / 我已经吃过饭了。 |
 | 他来了。 他来了。 / 他来了。 |
 | 我买了一本书。| 我买了一本书。 |

 ### 5.2 状态改变（句子+了）

 | 示例| 英语 |
 |---------|---------|
 | 他来了。 他（已经）来了。 （=他现在在这里）|
 | 春天了。| 春天来了。 |
 | 我懂了。| 我现在明白了。 |

 ## 6. 数字和量词

 ### 6.1 数+量词+名词结构

 在中文中，数字和名词之间总是需要量词（量词）。 

| 数字 + MW + 名词 | 拼音| 英语 |
 |--------------------|--------|---------|
 | 一个人| yī gè rén | yī gè rén 一个人 |
 | 两本书 | liǎng běn shū | liǎng běn shū | liǎng běn shū 两本书|
 | 三杯水| 三倍数 | 三倍数 | 三倍数 | 三倍数 三杯水|
 | 四张纸 | sì zhāng zhǐ | sì zhāng zhǐ 四张纸|

 ### 6.2 常用量词

 | 兆瓦| 拼音| 用于 | 示例|
 |----|--------|----------|---------|
 | 个 | 格| 一般（人、事） | 一个苹果一个苹果|
 | 本 | 本| 书籍、卷册 | 一本书 一本书 |
 | 张 | 张 | 扁平物体| 一张票一张票 |
 | 杯| 贝 | 杯子| 一杯茶一杯茶 |
 | 件 | 吉安 | 衣服、物品| 一件事一件事|
 | 只| zhī | 动物 | 一只猫一只猫|
 | 条 | 条| 长的柔性物体| 一条鱼一条鱼|
 | 块| 嚯 | 件; 元（非正式）| 五块钱五元|
 | 口 | 苦| 家庭成员 | 三口人三个家庭成员 |

 ### 6.3 二对二 — 二对两

 | 二 (èr) | 两 (liǎng) |
 |---------|----------|
 | 数数：一二三| 量词前：两个人 |
 | 序数：第二| 时间表达：两点|
 | 数学：二加二 | 钱：两块钱|

 ## 7. 时间表达式

 ### 7.1 时间短语位于动词之前

 在中文中，时间表达总是在动作之前（与英语相反）。 

| ✓ 正确 | ✗ 不正确 |
 |------------|------------|
 | 我明天去。(我明天就去。) | ~~我明天去。~~ |
 | 他昨天来了。（他昨天来了。） | ~~他昨天来了。~~ |

 ### 7.2 报时

 | 图案| 示例| 英语 |
 |---------|---------|---------|
 | N点| 三点| 3点|
 | N点N分| 三点二十分| 3:20 |
 | N点半| 三点半| 3:30 |
 | N点一刻| 三点一刻 | 3:15（四分之一）|
 | 差N分N点| 差五分三点| 2:55（5 至 3）|

 ### 7.3 日期格式（年→月→日→时间）

 中文日期的单位是从最大到最小：
 - 2026年5月30日下午三点=2026年5月30日下午3点
