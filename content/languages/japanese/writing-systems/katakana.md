 ## 外来词转录模式

 ### 末端辅音
 单词末尾的英语辅音会附加元音：

 | 英语 | 片假名 | 规则|
 |---------|----------|------|
 | 蛋糕| ケーキ (kēki) | ケーキ (kēki) | 最终 k → ki |
 | 牛奶| ミルク (miruku) | 最终 k → ku |
 | 测试| テsuto（tesuto）| 最后 t → 到 |
 | 检查| チェック (chekku) | 最终 ck → kku（双生）|
 | 床 | ベッド（beddo）| 最后的 d → do （如果重音则为 dd）|
 | 巴士 | 巴苏 (basu) | 词尾 s → su |
 | 俱乐部 | 克拉布 (kurabu) | 最后的 b → bu |

 ### 双辅音 → ッ
 英语双辅音/重读辅音变成 ッ（小 tsu）：
 - 黄油 → バター 或 バッター (battā)
 - 披萨 → ピザ 或 ピッツァ (pittsa)
 - 咖啡 → コーヒー（kōhī，“f”+长元音）
 - pocket → ポketto (poketto) — “t”加倍
 - 门票 → chiketto (chiketto)
 - 足球 → サッカー (sakka)

 ### R 和 L → ラ行
 日语没有单独的 L 和 R — 都变成了 ラ行 (ra/ri/ru/re/ro)：
 - 大米 → raisu (raisu)
 - 喜欢 → raiku (raiku)
 - 爱 → ラブ (rabu)
 - 光 → raito (raito)
 - 大厅 → ロビー (robī)
 - 图书馆 → raiburari (raibararī)
