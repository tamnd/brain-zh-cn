 ## 笔顺注释

 书写汉字时，请遵循以下一般规则：
 1. **从上到下** — 笔画通常从上到下
 2. **从左到右** — 水平笔画从左到右； 纵横多笔交叉时，一般先横
 3. **先外后内** — 对于封闭的汉字（如国），先绘制框架，然后绘制内容，然后关闭底部
 4. **先中心后边** — 对于垂直对称的汉字（如小），先画中心笔划

 ## 阅读模式

 ### On-yomi (音読み) vs Kun-yomi (训読み)
 - **On-yomi** 主要用于两个或多个汉字组合时：学校 (gak**KOU**)、电车 (den**SHA**)
 - **Kun-yomi** 用于日语母语阅读的单个汉字：学ぶ (mana**bu**)、车 (**kuruma**)
 - 今日 (kyou)、今年 (kotoshi)、大人 (otona) 等一些汉字有不遵循标准模式的特殊读法 (jukuji-kun)

 ## 学习技巧

 ### N5 汉字的部首
 许多 N5 汉字都有共同的部首（构件）来暗示含义：

 | 激进 | 含义提示| 示例 |
 |---------|-------------|---------|
 | 人 (i) | 人 | 休（休息=靠在树上的人）|
 | 口 | 嘴巴/开口| 话、言、语 |
 | 日 | 太阳/天 | 明、时、昨 |
 | 木| 树/木| 休、本、校、东 |
 | 水 (氵) | 水 | 海、泳、饮 |
 | 火 (灬) | 火| 燃, 照 |
 | 女 | 女人| 好、姊、始 |
 | 山 | 山| 岛，炭 |
 | 土 | 地球| 地、场 |
 | 金 (釒) | 金属| 鉄、银、银行 |

 ### 助记符示例
 - **山**（山）：简单画出的三座山峰
 - **川**（河）：三行如流水
 - **木**（树）：上面有树枝，下面有根的树干
 - **日**（太阳/天）：一个代表太阳的矩形，其中心线
 - **月**（月亮/月）：新月形状
 - **人**（人）：用两条腿走路的人
 - **山** + **川** = 风景； 自然词汇围绕这些部首聚集
 - **休**（休息）：一个人（人）靠在木（树）上=休息
 - **明**（明亮）：日（太阳）+月（月亮）= 两个发光的物体=明亮
 - **林**（树林）：两棵木（树）； **森**（森林）：三棵树（N5级意识）
