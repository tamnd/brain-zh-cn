## 概述

 动词是日语句子的引擎。 与英语动词不同，日语动词总是出现在句子的末尾，并且它们以可预测的模式变化。 对于学习者来说，好消息是日语只有两个真正的不规则动词（する和来る），所有其他动词都遵循两种一致的模式之一，称为**u-动词（第1组）**和**ru-动词（第2组）**。 一旦确定了动词属于哪一组，您就可以放心地将其变位为所有必要的形式。 

在本课程中，您将学习礼貌的**ます形式**——日常对话、JLPT听力任务以及与您不太熟悉的人交谈时使用的标准形式。 您将练习四种基本的动词变化形式：现在/将来肯定句 (〜ます)、现在/将来否定句 (〜ません)、过去肯定句 (〜ました) 和过去否定句 (〜ませんでした)。 您还将学习标记动作动词直接宾语的关键**を助词**，为您提供构建完整、自然句子的工具，例如“我昨天吃了寿司”或“我周日不学习”。 

## 学习目标

 在本课程结束时，您可以：

 - 识别动词是 u 动词还是 ru 动词并解释其区别。 
- 将动词变形为所有四种礼貌ます形式（肯定/否定×现在/过去）。 
- 正确使用を助词来标记动作动词的直接宾语。 
- 用时间和频率表达构建描述日常活动的完整句子。

 ## 语法点

 ### 语法点 1：U 动词和 Ru 动词 — 识别两组

 **U-动词（第 1 组）：** 字典形式以非る音节结尾，或以る结尾但前面的音节包含 /a/、/u/ 或 /o/ 声音
 **汝动词（第2组）：**字典形式以る结尾，前面的元音是/i/或/e/
 **不规则：** する（要做），来る（くる，来）

 这种差异很重要，因为这两个基团的共轭方式不同。 形成ます词干：u动词将最后的u行音改为i行（例如，书く → 书き、饮む → 饮み、话す → 话し、行く → 行き）； ru动词简单地删除る（例如，食べ**る**→食べ，见**る**→见，起き**る**→起き）。 然后所有动词都附加相同的词尾：ます / ません / ました / ませんでした。 

棘手的情况是以る结尾的动词，但实际上是u动词：**帰る**（かえる，返回），走る（はしる，运行），切る（きる，切割）。 这些看起来像 ru 动词，但共轭为 u 动词：帰ります、走ります。 这些都需要上下文和记忆。 一个可靠的提示：如果る之前的声音不是/i/或/e/（即，它是/a/、/u/或/o/），那么它绝对是一个u动词。

| 动词类型| 示例| ます茎| ます形式|
 |------------|---------|----------|---------|
 | U-动词 | 饮む (nomu) | 饮み (nomi) | 饮みます |
 | U-动词 | 书く (kaku) | 书き（柿子）| 书きます |
 | U-动词 | 话す (hanasu) | 话し (hanashi) | 话します |
 | Ru-动词| 食べる (taberu) | 食べ (tab) | 食べます |
 | Ru-动词| 见る (miru) | 见 (mi) | 见ます |
 | 不规则 | する (suru) | し (shi) | します |
 | 不规则 | 来る (kuru) | き (ki) | きます |

 **常见错误：**
 ❌ 帰ります→ 词干帰れ（视为ru动词）
 ✓ 帰ります → 词干帰り（u-动词，尽管以る结尾）

 ### 语法点 2：礼貌动词变形——四种核心形式

 **结构：**
 - 现在/将来肯定句：[ます词干] + ます
 - 现在/将来否定：[ます词干] + ません
 - 过去肯定句：[ます词干] + ました
 - 过去否定式：[ます词干] + ませんでした

 **を助词**标记直接对象——动作所针对的对象。 在英语中，我们只是将宾语放在动词之前； 在日语中，を（发音为“o”）位于宾语之后：ご饭**を**食べます（我吃米饭）。 を仅与动作动词连用，不能与あります/います连用。 

句子通常遵循以下顺序：**时间 + 主语は + 宾语を + 动词**。 除动词外的所有内容都可以根据上下文重新排列或省略，但动词必须保留在末尾。 时间表达如“毎日”、“昨天”、“毎朝”通常出现在句首或主语之后。 

| 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 毎朝、朝ご饭を食べます。 まいあさ、あさごはんをたべます。 Maiasa，asagohan wo tabemasu。 | 我每天早上都吃早餐。 |
 | 昔日、日本语を勉强しました。 きのう、にほんごをべんきょうしました。 Kinou，nihongo wo Benkyou shimashita。 | 我昨天学了日语。 |
 | 今日はテreneビを见ません。 | きょうはテreビをみません。 Kyou wa terebi wo mimasen。 | 我今天不看电视。 |
 | 先周、映画を见ませんでした。 せんしゅう、えいがをみませんでした。 Senshuu，eiga wo mimasen deshita。 | 上周我没有看电影。 |
 | 何时に起きますか。 なんじにおきますか。 | Nanji ni okimasu ka。 | 你几点起床？ |
 | 毎晩、十时に寝ます。 まいばん、じゅういちじにねます。 Maiban，juuichi-ji ni nemasu。 | 我每天晚上 11 点睡觉。 |

 **食べる和饮む的完整词形变化表：**

 | 表格 | 食べる（taberu，ru动词）| 饮む (名词，u-动词) |
 |------|------------------------|---------------------|
 | 现在肯定| 食べます | 饮みます |
 | 呈现负| 食べません | 饮みません |
 | 过去肯定句| 食べました | 食べました | 饮みました |
 | 过去的负面| 食べませんでした | 饮みませんでした |

 **常见错误：**
 ❌私は本をあります — を不能与あります/います一起使用
 ✓ 私は本があります（存在）或私は本を読みます（带有を的动作）

 ## 语法练习

 将括号中的动词变位为正确的ます形式。 

1. 毎日、牛乳を___。(饮む，现在肯定)*(答案:饮みます)*
 2. 昨日、宿题を___。（书く，过去肯定句）*（答案：书きました）*
 3.先周の土曜日、学校に___。(行く,过去否定)*(答案:行きませんでした)*
 4. 今日はテreneビを___。(见る, 表示否定) *(答案: 见ません)*
 5. 毎朝、七时に___。(起きる, 现在肯定) *(答案: 起きます)*
 6. 昨日、日本语を___。(勉强する，过去肯定句) *(答案:勉强しました)*
 7.先周、友达が___。(来る,过去肯定句)*(答案:来ました)*
 8. 今日、どこにも___。(行く,现在否定) *(答案:行きません)*
 9. 昨晚晩、何も___。(食べる，过去式否定)*(答案:食べませんでした)*
 10. 毎周、図书馆で本を___。(読む,现在肯定句)*(答案:読みます)*

 ## 翻译练习

 ### 英语 → 日语

 1.“我每天早上都喝咖啡。” → 毎朝、コーヒーを饮みます。 
2.“昨天我没有去上学。” → 昔日、学校に行きませんでした。 
3.“你几点睡觉？” → 何时に寝ますか。 
4.“我学了三个小时日语。” → 三时间、日本语を勉强しました。 
5.“她不吃肉。” → 彼女は肉を食べません。 

### 日语 → 英语

 1. 毎日、电车で学校に行きます。→“我每天都坐火车去学校。”
 2. 昨天、友达と映画を见ました。→“昨天和朋友看了电影。”
 3.今日は何もしませんでした。→“我今天什么也没做。”
 4. 何时に起きますか。→ “你几点起床？”
 5.先周、日本语を勉强しませんでした。→“我上周没学日语。”

 ## 文化笔记

 日本人的日常生活（日常生活，nichijou seikatsu）有一种独特的节奏，这是由长时间的工作和上学时间形成的。 许多日本学生在上学前遵循清晨俱乐部活动的惯例，上课直到下午晚些时候，然后再次参加课后俱乐部活动。 **gambaru (頑張る)** 的概念——尽力而为、坚持不懈——是日常生活的核心，通常被称为鼓励：頑張ってください（请尽力而为）。 在日本，您会经常听到并使用这句话。 

日本人对他们的**节俭（节约）**感到非常自豪——节俭，不浪费东西。 这体现在日常习惯中：吃完盘子里的所有食物（残さないで食べる，nokosenaide taberu），离开房间时关灯，仔细分类垃圾以便回收。 吃饭时，饭前说**いただきます（itadakimasu）**，饭后说**ごちそうさまでした（gochisousama deshita）**并不是随意的礼貌——这是孩子们在上学前学习的基本社会和文化习惯。
