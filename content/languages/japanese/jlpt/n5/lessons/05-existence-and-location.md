## 概述

 日语中两个最基本的动词是あります和います，两者都大致翻译为“存在”或“存在”。 关键区别很简单，但绝对重要：**あります**用于无生命的物体、植物和抽象事物，而**います**用于有生命的存在——人、动物和任何能够自行移动的东西。 正确区分这一点是 N5 级别最重要的技能之一，将立即提高您的日语发音的自然水平。 

在本课中，您还将掌握表示位置**（有事物存在的地方）的 **に 助词与表示动作位置**（有事物发生的地方）的 **で 助词。 您将学习使用相对位置词来描述物体的位置，如右（右）、左（左）、前（前）、后ろ（后面）、上（上方）、下（下方）和中（内部）。 在本课程结束时，您将能够指路、描述房间、告诉某人人和物体在哪里，并在阅读理解任务中理解基于位置的句子。 

## 学习目标

 在本课程结束时，您可以：

 - 正确使用あります和います来表达无生命和有生命实体的存在。 
- 使用に助词来标记某物存在的位置。 
- 使用助词“で”来标记动作发生的位置。 
- 用Xの右/左/前/后ろ/上/下/中描述物体的相对位置。

 ## 语法点

 ### 语法点 1：あります vs います — 无生命与有生命的存在

 **结构（あります）：** [位置] に + [无生命物] が + あります
 **结构（います）：** [地点] に + [生命体] が + います
 **询问：** [事物/人] は + [地点] に + あります/いますか

 规则很简单但很重要：**あります**适用于无法自行移动的事物——物体、植物、建筑物、金钱、时间、事件。 **います** 指的是可以移动的生物——人、动物、昆虫。 这种区别来自日本的自主运动概念：狗可以决定运动；而狗可以决定运动。 椅子不能。 

主语后面的助词が是“存在が”——它标记了被断言存在的事物。 助词“に”标记位置。 顺序很灵活，但最自然的句子结构将地点放在第一位：テーブルの上に本があります（桌子上有一本书）。 当问“X在哪里？”时，X成为带有は的主题：本はどこにありますか（书在哪里？）。 

在随意的言语中，这种区别有时是模糊的——一些说话者用“ある”来表示非常低的动物性实体，而机器人或车辆有时会根据上下文采用任一动词。 在 N5 中，始终遵循严格的规则：生物 = います，其他一切 = あります。

 **常见错误：**
 ❌犬があります / 椅子がいます
 ✓ 犬がいます（有生命）/椅子があります（无生命）——在标准日语中，有生命/无生命的区别是不容协商的

 ###语法点2：に（存在位置）vsで（动作位置）

 **结构（に）：** [地点] に + [事物/人] が + あります/います
 **结构（で）：** [地点] で + [动作动词]

 助词 **に** 标记一个静态位置——某物或某人只是存在或定位的地方。 助词 **で** 标记动作或事件发生的位置。 测试：如果你可以用“there is/are”替换动词，请使用に。 如果动词描述一项活动（吃饭、学习、工作、玩耍），则使用で。 

比较：図书馆に本があります（图书馆里有书——静态存在，に）与図书馆で勉强します（我在图书馆学习——在那里发生的行动，で）。 一个人可以同时出现在：田中さんは図书馆にいます（田中桑[存在]在图书馆）与田中さんは図书馆で本を読みます（田中桑在图书馆读书——动作）。 

一个有用的记忆技巧：に = 将某物固定在某个位置； で = 动作发生的阶段。 两者在日语口语和书面语中都非常常见，因此尽早掌握这种区别会带来巨大的好处。 

| 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 公园に猫がいます。 | こうえんにねこがいます。 Kouen ni neko ga imasu。 | 公园里有一只猫。 （存在）|
 | 公园で游びます。 | こうえんであそびます。 Kouen de asobimasu。 | （一）去公园玩。 （行动）|
 | 駅の前にコンビがあります。 えきのまえにコンビがあります。 Eki no mae ni konbini ga arimasu。 | 车站前有便利店。 |
 | 学校で日本语を勉强します。 がっこうでにほんごをべんきょうします。 日本语学校的情况。 | 我在学校学习日语。 |
 | このresutoranで食事しましょう。 このresutoranでしょくじしましょう。 食事岛精的河野餐厅。 | 我们去这家餐厅吃饭吧。 |
 | 部屋に谁もいません。 へやにだれもいません。 嘿你敢莫伊森。 | 房间里没有人。 |

 **常见错误：**
 ❌図书馆に勉强します
 ✓ 図书馆で勉强します — 学习是一种行动，所以で为必填； に 这里仅表示你存在于图书馆

 ## 翻译练习

 ### 英语 → 日语

 1.“椅子下面有一只猫。” → 椅子の下に猫がいます。 
2.“医院在哪里？” → 病院はどこにありますか。 
3.“我在学校吃午饭。” → 学校で昼ご饭を食べます。 
4.“房间里有三个人。” → 部屋に三人います。 
5.“这附近有便利店吗？” → この近くにコンビはありますか。 

### 日语 → 英语

 1.本棚の横に窓があります。→“书架旁边有一扇窗户。”
 2. 公园で友达と会います。→“我在公园遇见了我的朋友。”
 3.机の上に何もありません。→“桌子上什么都没有。”
 4.田中さんはどこにいますか。→“田中同学在哪里？”
 5.駅と病院の间に银行があります。→“车站和医院之间有一家银行。”

 ## 文化笔记

 日本的公寓和住宅通常使用称为 **LDK（起居、用餐、厨房）** 的房间计数系统与独立房间的数量相结合来描述：例如，1K 是一间只有厨房的工作室，2LDK 有两个房间加上一个起居-餐厅-厨房。 当你第一次去日本某人的家时，通常会在走进去时说**お邪魔します (Ojama shimasu)**——字面意思是“我闯入了”。 这句话表明您正在进入某人的私人空间，被认为是基本的礼貌。 离开时，你说お邪魔しました（Ojama shimashita，过去时）。 

**uchi（内，内部/家庭群体）**与**soto（外，外部/外群体）**的概念是日本社会互动的基础。 你的 uchi 是你的内部群体：家人、亲密的同事、队友。 索托是那个圈子之外的每个人。 这延伸到语言上：当对外人谈论自己的（uchi）家庭时，你使用谦卑的形式；当谈论别人（soto）的家庭时，你使用尊重的形式。 您在本课中学到的“内”和“外”这两个词作为物理位置词，因此也是日本文化中深刻的社会概念。
