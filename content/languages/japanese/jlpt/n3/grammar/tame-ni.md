## 图案：〜ため(に)

 **结构：**
 - **目的：** 动词（字典形式）+ために→“为了～”
 - **原因：** 动词（普通过去时た）+ ために / 名词 + の + ため(に) → “因为〜，由于〜”

 **正式性：** 正式/标准（在正式写作中优于から/ので）

 **含义：** 目的：“为了～”/原因：“因为～，由于～”

 ## 说明

 关键区别由ために之前的**动词时态**决定：
 - **字典形式**（非过去时）+ ために → **目的**：“X 的完成是为了实现 Y”
 - **过去式 (た)** + ために 或 **名词 + の + ため** → **原因**：“因为 X 发生，Y 结果”

 在正式写作（报告、报纸、学术文本）中，〜ため比から或ので更适合表达原因。 听起来更加客观和中立。

 ## 常见错误

 ❌雨が降るために、试合が中止になった。（错误 — 用于原因的字典形式）
 ✓ 雨が降ったために、试合が中止になった。— 对于原因，动词必须是过去式 (た)。 

❌疲れたために、早く帰りたい。（不自然——个人欲望的ため听起来很生硬）
 ✓ 疲れたので、早く帰りたい。— 由于个人主观原因使用ので。 

## 与相似模式的比较

 | 图案| 使用| 正式|
 |--------|-----|------------|
 | 〜ために（目的）| 目标/意图| 正式/中立|
 | 〜ために（原因）| 客观原因| 正式（最好是书面形式）|
 | 〜から | 主观个人原因| 休闲-中性|
 | 〜ので | 礼貌、客观的理由| 礼貌/中立 |
 | 〜によって | 原因/手段/代理人 | 正式（被动语境）|
