N3 词汇（累计约 3,750 个单词，N4 基础上约 2,250 个新单词）标志着从基本日常词汇到抽象、专业和学术语言的转变。 许多 N3 单词都是源自汉语的复合名词（汉语、kango）——学习识别汉字成分变得至关重要。 本页按语义领域组织所有 N3 级词汇。 

## N3 词汇与 N4 有何不同

 在 N4 中，大多数词汇都是具体的：食物、方向、动作、简单的形容词。 N3介绍：
 - **抽象名词**：意识（意识）、感覚（感觉）、判断（判断）
 - 休闲词的**正式/书面等价物**：利用する (riyou suru) 代替使う (tsukau)
 - **复合动词**：取り上げる（toriageru，采取/地址），乘り越える（norikoeru，克服）
 - **四字熟语**：开始出现在N3阅读段落中

 ## 15. 经常混淆的 N3 单词

 这些配对出现在考试中并且通常会被混淆：

 | 字A | 字 B | 差异|
 |--------|--------|------------|
 | 理由 (riyuu - 原因/原因) | 原因（genin — 原因/因素）| 理由=给定的理由； 原因=根本原因|
 | 目的 (mokuteki — 目的) | 目标（mokuhyou — 目标/目标）| 目的=你为什么做某事； 目标=你的目标|
 | 机会 (kikai — 机会) | 场合 (baai — 案例/情况) | 机会=做某事的机会； 场合 = 当/如果情况发生 |
 | 変える（kaeru — 改变某事）| 変わる（kawaru — 更改 intrans）| 及物与不及物对 |
 | 増やす（fuyasu — 增加某物）| 増える（fueru — 增加 intrans）| 及物与不及物对 |
 | 続ける（tsuzukeru — 继续某事）| 続く（tsuzuku — 继续翻译）| 及物与不及物对 |
 | 利用する（riyou suru — 使用/利用）| 使う（tsukau - 使用）| 使用更加正式/实用 |
 | 実现する (jitsugen suru — 实现/实现) | 完成する (tassei suru — 完成) | 実现=成为现实； 达成=完成目标|
 | 申明 (shuchou — 断言) | 意见（类似 - 意见）| 援助更强大、更有力 |
 | 同事 (douryou — 同事) | 仲间 (nakama — 朋友/同伴) | 同事=专业背景； 仲间 = 非正式群体 |

 ## JLPT问题类型的词汇

 ### 汉字阅读问题（汉字読み）
 重点：経済、政治、环境、判断、认识、影响、统计

 ### 汉字书写问题（汉字书き）
 重点：感谢、确认、连络、相谈、说明

 ### 上下文填充 (文脉规定)
 练习：上下文中使用的抽象名词——条件、基准、极限界、可能性

 ### 同义词/替换（言い换え类义语）
 常见对：利用する→使う、判断する→决める、依頼する→お愿いする

### 使用问题（最合适）
 检查：哪个句子正确使用了“期待”、“満足”、“不安”等。
