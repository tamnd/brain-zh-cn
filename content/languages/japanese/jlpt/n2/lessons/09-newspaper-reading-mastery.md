## 概述

 JLPT N2 阅读部分测试您在时间压力下从密集、正式的日语文本中提取含义的能力。 与 N3 的文本相对简单不同，N2 的段落广泛使用被动语态对冲、名词化、正式连词和学术疏远手段。 试图解析每个单词的读者将耗尽时间； 无法识别关键结构标记的读者会误解作者的立场。 本课程通过教授日语书面散文的正式语法和策略性阅读技巧来解决这两个挑战。 

本课程中的段落以实际的报纸社论和政策报告风格为蓝本，即 N2 考试中出现的确切类型。 您将学习识别〜とされている和〜によると〜という等对冲手段，了解被动结构如何在新闻写作中创造客观性，并使用名词化来识别复杂句子的逻辑结构。 结合最后一节中的定时阅读策略，本课应该会显着提高理解准确性和阅读速度。 

## 学习目标

 在本课程结束时，您可以：
 - 在上下文中识别和解释学术对冲模式〜とされている和〜によると〜という。 
- 了解被动结构和名词化在正式书面日语中的作用。 
- 高效地将略读→扫描→详细阅读策略应用于N2长度的段落。 
- 使用练习的结构识别有效地管理 N2 阅读部分的时间。

 ## 语法点

 ### 1. 〜とされている（被认为/据说是——学术对冲）

 **结构：** 动词/形容词/名词 + とされている / とされてきた

 〜とされている是一个非人称被动结构，意思是“被认为”、“据说”或“X被视为Y”。 它将信念或评估归因于普遍共识、学术界或既定观点，但没有具体说明来源。 这种模式在学术写作和报纸报道中极为常见，将信息呈现为公认的智慧或共识，而无需作者亲自担保。 

过去进行时形式〜とされてきた表明这已经是随着时间的推移而形成的观点：“长期以来一直认为是这样。”

 **注册：** 学术、新闻、官方。 正式书面日语必备。 

**与〜と言われている的区别：** 〜と言われている（“据说”）强调口头传播/谣言； 〜とされている强调正式分类或官方/学术共识。

 ### 2. 〜によると〜という / 〜によれば〜とのことだ（根据X，据说Y）

 **结构：**
 - 出处 + によると / によれば + 句子 + という / とのことだ / とされている

 这种模式将报告的信息归因于特定来源——报纸、政府报告、专家或调查。 这是引用消息来源的标准新闻手段。 结尾“という”或“とのことだ”标记该信息为报告语音； 在使用という时省略来源属性标记可以使句子看起来像是事实陈述而不是引用。 

**注册：** 新闻性、正式。 对于理解和书写新闻风格的日语至关重要。

 ### 3.新闻风格中的被动结构（〜れる/られる表示客观性）

 在新闻日语中，被动语态不仅用来表示主体受到影响，而且作为一种**创造客观性和距离**的文体手段。 作者避免说“我认为”或“政府做了X”，而是使用被动语态来表达事实和客观的信息。

常见模式：
 - 〜と指摘されている→“有人指出”
 - 〜と见られている→“它被视为/被视为”
 - 〜と悬念されている→“表示担忧”
 - 〜と判断される→“判断”
 - 〜と考えられる→“认为”

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 今回の失策は、事前调整の不足によるものと见られている。 こんいのしっさくは、じぜんちょうせいのふそくによるものとみられている。 konkai no shissaku wa, Jizen chōsei no Fusoku ni yoru mono to mirarете iru. | 最近的政策失败被认为是由于事先协调不足造成的。 |
 | 新税制の施行により、中小企业への影响が悬念されている。 しんぜいせいのしこうにより、ちゅうしょうきぎょうへのえいきょうがけねんされている。 Shin-zeisei no shiko ni yori、chūshō kigyō e no eikyō ga kenen sarete iru。 | 人们担心新税制的实施将影响中小企业。 |
 | このデータから、社会的格差がさらに拡大する恐れがあると指摘されている。 このデータから、しゃかいてきかくさがさらにかくだいするおそれがあるとしてきされている。 kono dēta kara，shakaiteki kakusa ga sara ni kakudai suru osore ga aru to shiteki sarete iru。 | 从这些数据来看，已经指出社会差距存在进一步扩大的风险。 |

 ### 4. 标题和密集句的名词化

 日本学术和新闻写作广泛使用名词化——使用〜こと、〜の和正式名词结尾将动词和从句转换为名词短语。 这创造了密集、信息丰富的句子，这是正式书面日语的特征。 

**正式文本中常见的名词化模式：**
 - 动词（普通形式）+こと/の→做X的行为
 - 动词（普通形式）+ため→为了/因为
 - 〜化（后缀）→成为X的过程（例：少子化、高齢化、グローバル化）
 - 〜性（后缀）→ X 的性质/性质（例如：持続可能性、実效性、重要性）

 **密集句分析示例：**

 「少子高齢化の急速な进展に伴う労働力不足の深度化が、日本経済の持続可能性に対する最大の旁威の一つとされている。」

 阅读： しょうしこうれいかのきゅうそくなしんてんに ともなうろうどうりょくぶそくのしんこくかが 、にほんけいざいのじぞくかのうせいにたいするさいだいのきょういのひとつとされている。 

翻译：“随着出生率下降和人口老龄化的迅速进展，劳动力短缺的加剧被认为是日本经济可持续性的最大威胁之一。”

 **结构分解：**
 - 少子高齢化の急速な进展 = 少子高齢化的迅速进展
 - 〜に伴う（第3课复习）=伴随
 - 労働力不足の深刻化 = 劳动力短缺加剧（以〜化命名）
 - 日本経済の持続可能性=日本经济的可持续性（〜性名义化）
 - とされている = 被认为是（学术对冲）

 ## 真实文本样本

 ### 文本 1：经济学编辑风格

 > 政府の最新统计によると、今年度の実质GDP增长率は前年比0.8%増にとどま ったという。専门家の间では、この数値は景気回复の钝化を示すものとされており 、财政出动の是非をめぐって议论が活発化している。一方面で、物価上升が家计に与 える影响を悬念する声も多く、金融政策の见直しが吃紧の课题と见なされている。 

**阅读：** せいふのさいしんとうけいによると、こんねんどのじっし つGDPせいちょうりつはぜんねんひ0.8ぱーせんとぞ うにとどまったという。せんもんかのあいでは、このすう ちはけいきかいふくのどんかをしめすものとされており、 ざいせいしゅつどうのぜひをめぐってぎろんがかっぱつか している。いっぽうで、ぶっかじょうしょうがかけいにあ たえるえいきょうをけねんするこえもおおく、きんゆうせ いさくのみなおしがきっきんのかだいとみなされている。 

**分析：**
 - **によると〜という** → 归因于政府统计数据
 - **とされており** → 学术共识对冲（延续形式）
 - **をめぐって** → 第 3 课复习：围绕/关于有争议的话题
 - **悬念する声も多く** → 关注的新闻被动：“有很多声音表达关注”
 - **と见なされている** → “被认为是”——另一种形式的学术疏远

 **全文：** “根据最新的政府统计数据，本财年实际GDP增速据称仍较上年增长0.8%。专家们认为，这一数字表明经济复苏放缓，财政刺激是否合适的争论变得激烈。另一方面，许多声音表达了对物价上涨对家庭影响的担忧，货币政策的审查被视为紧迫的挑战。”

 ###文本2：社会政策报告风格

 > 急速な少子高齢化の进展に伴い、社会保障费の増大が财政を圧定向しているとされて いる。政府の试算によれば、このまま対策を讲じなければ、2040年には社会保障 费がGDPの30%を超える可能性があるという。知识者の间では、社会保障制度の 抜本的改革なくして财政の持続可能性は确保し得ないとの意见が広まりつつある。

**阅读：** きゅうそくなしょうしこうれいかのしんてんにともない、 しゃかいほしょうひのぞうだいがざいせいをあっぱくしてい るとされている。せいふのしさんによれば、このまままたい さくをこうじなければ、にせんしじゅうねんにはしゃかいほ しょうひがGDPのさんじゅぱーせんとをこえるかのうせい があるという。しきしゃのあいでは、しゃかいほしょうせい どのばっぽんてきなかいかくなくしてざいせいのじぞくかの うせいはかくほしえないとのけんかいがひろまりつつある。 

**分析：**
 - **に伴い** → 第 3 课复习：同时/相应的变化
 - **とされている** → 学术对冲：“被认为是这样”
 - **によれば〜という** → 来源归属：根据政府估计
 - **し得ない** → 第6课复习：不可能（正式〜うる否定）
 - **〜つつある** → 渐进变化：逐渐蔓延

 **全文：** “随着出生率下降和老龄化的迅速进展，社会保障成本的不断增加被认为对公共财政构成压力。据政府估计，如果不采取措施，到2040年，社会保障成本有可能超过GDP的30%。如果不对社会保障体系进行根本性改革，就无法确保财政可持续性的观点正在逐渐在知情观察人士中传播。”

 ## N2阅读策略

 ### 策略 1：浏览 → 扫描 → 详细信息（3 遍方法）

 **第 1 遍 — 浏览（30 秒）：** 只读每个段落的第一句和最后一句。 确定主题和作者的总体立场。 

**第 2 遍 — 扫描（60 秒）：** 寻找表明逻辑结构的话语标记：
 - 侧面で / その侧面で（另一方面）→对比
 - したがって / そのため（因此）→ 结论/原因
 - しかし / ただし（但是/但是有条件）→反转
 - また / さらに（也/此外）→ 添加
 - なぜなら（因为）→解释

 **第 3 关 — 细节（剩余时间）：** 阅读回答每个问题所需的具体段落。 不要重新阅读全文。 

### 策略 2：确定作者的立场

 在 N2 社论风格的段落中，作者的观点通常嵌入在：
 - 被动结构：〜とされている，〜と见られている — 这些目前的共识，不一定是作者的观点
 - 直接断言结尾：〜である、〜といえる、〜べきだ — 这些代表作者的结论
 - 先问后答的结构：提出问题→提出反对意见→在最后一段中断言作者的立场

 ### 策略 3：N2 阅读部分的时间管理

 N2 阅读部分通常有 3-5 个长度逐渐增加的段落。 建议时间分配：

 | 文本类型 | 推荐时间|
 |------------|------------------|
 | 短文（200-300 个字符）| 5-7 分钟 |
 | 中等段落（400-500 个字符）| 8-10 分钟 |
 | 长篇文章（600-700 个字符）| 12-15 分钟 |
 | 比较/信息段落| 10-12 分钟 |

 **关键规则：** 不要在单个问题上花费超过 2 分钟。 如果您无法确定答案，请标记您的最佳猜测并继续。 

## 对话

 ### 对话 1：正式工作场所 — 讨论新闻文章

 **背景：** 两名研究人员在会议前讨论一篇经济学文章。 

[研究员A]:今朝の新闻によると、政府は新しい経済対策を発表したとのことです。 
[阅读]： けさのしんぶんによると、せいふはあたらしいけいざいたいさくをはっぴょうしたとのことです。 
[中文]：据今天上午的报纸报道，据报道政府宣布了一项新的经济政策。 

[研究员B]: そうですね。専门家の间では、密码の金融缓和政策は限界に近いとされているようです。 
[阅读]： そうですね。せんもんかのあいでは、げんこうのきんゆう かんわせいさくはげんかいにちかいとされているようです。 
[中文]：确实如此。 专家认为，当前的货币宽松政策显然已接近极限。 

[研究员A]:格差の拡大をめぐって、议论が続いていますね。この问题は简単には解决し得ない课题です。 
[阅读]： かくさのかくだいをめぐって、ぎろんがつづいていますね。 このもんだいはかんたんにはかいけつしえないかだいです。 
[中文]：围绕差距扩大的争论仍在继续。 这个问题是一个不容易解决的挑战。 

[研究员B]:财政出动の是非についても、意见が分かれているとされています。

## 文化/注册备注

 日本报纸社论采用精心校准的归因和对冲系统。 与英语社论可能直接陈述作者的观点不同，日语社论通常通过累积引用（〜によると）、共识主张（〜とされている）和被动语态观察（〜と见られている）来表达他们的论点，仅在最后一段中以如下模式揭示作者的直接推荐 〜べきだ或〜が求められる。 N2 学习者只阅读一篇文章的前几段，经常会错误地识别文章的主要论点，因为关键断言出现在最后。 

学术日语还表现出一种称为“对冲承诺”的特征：作家避免绝对的主张，而是使用多层距离——〜と考えられている，〜とされている，〜の可能性が示指引されている。 这不是不诚实或含糊不清——而是承认当前证据的局限性的智力精确性。 N2 考试问题经常测试读者是否能够区分文本所呈现的“既定共识”与“一种可能的解释”与“作者自己的立场”。 训练自己识别这些区别是阅读部分最有价值的技能之一。
