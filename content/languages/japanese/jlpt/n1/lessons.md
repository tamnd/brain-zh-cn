 ## 第 1 课：古典日语根源——为什么现代日语是这样的

 ### 为什么这对 N1 很重要

 N1 语法包含的模式只有当您了解它们的来源时才有意义。 像〜べし、〜ごとし、〜たる、〜なり这样的模式并不是“奇怪的N1语法”——它们是古典日语（文语）的幸存元素，仍然保留在正式和文学的现代用法中。 

### 古典日语（文语）与现代日语（口语）

 | 特色| 古典 (中文) | 现代（口语） |
 |--------|-----------------|------------|
 | 动词“是”| なり / たり | だ / です |
 | 定语| 〜き / 〜し | 〜い / 〜な |
 | 终端| 〜し / 〜り | 〜い / 〜だ |
 | 连词 | 〜て/〜で（相同）| 〜て / 〜で |
 | 负面| 〜ず | 〜ない |
 | 有条件| 〜ば（同）| 〜ば |
 | 应该/必须| 〜べし | 〜なければならない |

 ### N1 中幸存的关键古典形式

 **1. 〜ず（否定-古典）**
 现代〜ない。 仍用于正式写作和固定短语：
 - 绝えず = 绝えなく = 连续（设定短语）
 - 关わらず (かかわらず) = 不管
 - 果てしなく (はてしなく) = 无尽
 - あいまいならず = 不含糊（正式写作）

 **2. ～べし → ～べき → ～べきだ**
 古典义务形式。 现代的〜べきだ是语法化的生存：
 - 法令风格：すべからく〜べし（一无论如何应该〜）
 - 文学：约束は守るべし（一个人应该遵守承诺）
 - 现代正式语：これは注意すべき点だ（这是需要注意的一点）

 **3. 〜たる（〜たり的定语形式）**
 古典系词处于定语位置。 创建正式的“being [X]”短语：
 - 堂々たる = 端庄 (堂々 + たる)
 - 悠然たる = 沉着，从容
 - 歴然たる = 明显、清楚

 **4. 〜なり（古典系动词）**
 用于：
 - 文学/古文散文：これなり（就是这样）
 - 现代固定短语：〜なりの = 以自己的方式（なり保留了不同的含义）
 - “一旦”含义：帰ったなり = 返回时（动作完成的那一刻）

 ### N1 经典阅读

 N1阅读段落有时会引用或参考经典文本。 关键技能：
 1. 将ず识别为否定形式（食べず = 食べないで）
 2. 将べし视为义务（食すべし = 应该吃）
 3. 将にて识别为で（正式位置/意思）
 4. 将句末的なり识别为断言系动词
 5. 将ごとし识别为ようだ（就像/好像）

 ### 练习练习

 翻译这些受古典影响的句子：
 1. 光阴矢のごとし、勉学に励むべし。—光阴似箭； 一个人应该努力学习。 
2. 真のoriダーたる者は、民の声に耳を倾けねばならぬ。——真正的领导者必须倾听人民的声音。 
3. 努力せずして成功なし。——没有努力，就没有成功。 
4. 彼は帰ったなり、一言も话さなかった。——他回来后，从那一刻起就没有再说过一句话。

 ## 第 2 课：阅读真实文本 — 社说和评论

 ### N1 阅读挑战

 N1阅读段落通常是：
 - 800–1,500+ 个字符长
 - 内容高度抽象
 - 充满学术名词和正式连词
 - 以复杂的段落结构撰写，并带有对冲索赔
 - 测试对作者意图的理解，而不仅仅是表面事实

 ### N1阅读文本的类型

 **社说 (しゃせつ) — 社论**
 报纸评论文章。 结构：
 1.问题介绍（问题提起）
 2. 背景语境（background context）
 3. 分析/不同视角（分析・视点）
 4. 结论/号召性用语（结论·提言）

 **评论 (ひょうろん) — 文学/文化批评**
 对文学、文化、社会的学术批判。 结构：
 1. 论文陈述（协助）
 2. 用例子支持证据（根拠・例）
 3. 反论承认（反论）
 4. 强化结论（结论）

 **学术论文の要旨 (ようし) — 学术摘要**
 压缩学术写作。 每句话都信息密集。 

### 重点阅读策略

 **策略1：首先找到救济（主要主张）**
 在学术日语中，主要主张经常出现：
 - 第一段（引言）
 - 最后一段的结尾（结论）
 - 在つまり、すなわち、要するに等连词之后

**策略 2：跟踪对比标记**
 N1 段落经常通过对比进行争论。 关键标记：
 - しかし / しかしながら = 然而
 - ところが = 然而（更强的对比）
 - 一方(で) = 另一方面
 - それに対して = 对比
 - ひるがえって = 现在转向另一边

 **策略 3：识别对冲语言**
 日本学术界谨慎地对冲主张。 学习阅读：
 - 〜とも言える=可以这么说
 - 〜と考えられる=认为
 - 〜という见方もある=也有这样的观点
 - 〜とは言い切れない = 无法明确地说
 - 〜に他ならない = 只不过是（强烈的断言 - 没有对冲）

 **策略 4：抽象名词带有论点**
 N1 段落通过抽象名词进行争论。 识别：
 - 主语名词（论证的内容）
 - 谓语名词（关于主语的断言）
 - 连接关系（它们如何关联）

 ### 编辑分析示例

 **文本片段**（改编报纸风格）：
 > 现代社会における隔离问题は、単なる个人の问题に留まらず、社会整体が抱える构建的な课题にほかならない。経済的格差の 拡大がその温床となっているものの、要因はそれのみか、人间关系の希薄化、価値観の多様化なども复雑に络み合っている。 

**词汇分析**：
 - 隔离问题 (こりつもんだい) = 隔离问题
 - 构造的な课题（こうぞうてき）=结构挑战
 - 〜にほかならない = 只不过是
 - 温床 (おんしょう) = 繁殖地； 温床
 - 希薄化 (きはくか) = 变得稀疏/稀释
 - 络み合う (からまりあう) = 缠结; 交织

 **主要论点**：社会孤立是一个结构性社会问题，而不仅仅是个人问题； 除了经济差距之外，多种因素相互交织。 

###推荐阅读练习时间表

 | 日 | 活动 | 目标|
 |-----|----------|--------|
 | 周一/周三/周五 | 阅读一篇朝日新闻社论（社说）| 10分钟； 充分理解 |

 ## 第三课：四字熟语——100个最重要的成语

 四字成语直接考N1词汇部分。 考试通常包括 2-4 个问题，每个测试明确地在四字熟语上进行。 有关完整列表，请参阅专用的 [yojijukugo 页面](../yojijukugo/)。 

### 理解四字熟语结构

 大多数四字熟语具有以下结构之一：

 **类型1：ABCD（四种独立含义的组合）**
 - 一期一会 (いちごいちえ) = 一次、一次见面 = 一生一次的相遇
 - 花鸟风月 (かちょうふうげつ) = 花、鸟、风、月 = 自然之美

 **类型 2：AB + CD（两个配对概念）**
 - 自业自得 (じごうじとく) = 自己的行为，自己的收获 = 种瓜得瓜，种豆得豆
 - 喜怒哀楽 (きどあいらく) = 喜悦、愤怒、悲伤、快乐 = 情绪范围

 **类型3：否定（不/无/非+概念）**
 - 无我梦中 (むがむちゅう) = 无我、梦想、中心 = 专注； 迷失自我
 - 不言実行 (ふげんじっこう) = 不说话，行动 = 行动胜于雄辩

 **类型 4：历史/古典起源**
 - 四面楚歌 (しめんそか) = 四面楚歌 = 被敌人包围（来自中国历史）
 - 卧薪尝胆（がしんしょうたん）=睡柴尝胆=吃苦报仇

 ### 如何学习四字熟语

 1. **分组学习**：按第一个字符（一〜，自〜，大〜）或主题（勇气，本性，关系）分组
 2. **先理解字面意思**：拆字显隐喻
 3. **了解用法上下文**：有些是正面的（一期一会），有些是负面/批评的（言语道断），有些是中性的（喜怒哀楽）
 4. **N1考试形式**：给出四字熟语，选择正确的意思； 或根据上下文，选择正确的四字熟语

 ## 第 4 课：谚语和惯用句 (谚・惯用句)

 ### N1 谚语特点

 N1 测试的谚语是：
 - 比日常谚语不太常见
 - 通常具有微妙或违反直觉的含义
 - 可能有古典起源（中国谚语改编成日语）
 - 有时作为阅读理解主题出现（引用/分析谚语的文章）

 ### 著名的被误解的谚语

 **情けは人の为ならず**
 日语中最容易被误解的谚语。 人们常常认为它的意思是“善良不是为了别人（所以不要善良）”，但它实际上意味着“善良不仅是为了别人——它最终会回报给你。” 语法：〜のため + ならず =“不只是为了”。 

**出る杭は打たれる**
 从字面上看，“突出的钉子被敲下来”——描述了一种社会现实（墨守成规），而不是规定它。 当在论文中使用时，它常常作为一个问题来批评。

 ## 第 5 课：文学语法模式——现代正式写作中的古典影响

 ### 从古典到现代的光谱```
文語 (archaic)  ←────────────────→  口語 (colloquial)
〜ごとし         〜べき         〜に他ならない    〜なんだ
〜たる           〜をもって     〜ものとする      〜じゃないか
〜なり           〜はおろか     〜にほかならない  〜よね
```N1 测试中左范围——具有古典影响力的正式书面语言。 

### 如何识别阅读中的形式/文学模式

 **正式登记的信号词**：
 - 〜において（在；在——正式地点）
 - 〜に际して（之际）
 - 〜に鉴みて（鉴于；考虑）
 - 〜に即して（符合；遵循）
 - 〜を通じて / 〜を通して（通过；经由）
 - 〜にわたって（跨越；覆盖）
 - 〜に至るまで（一直到；甚至）
 - 〜をはじめとして（开始于；包括）

 **文学登记的信号词**：
 - 〜ごとく/ごとし（比如 - 古典）
 - 〜ことなく（不做）
 - 〜ならではの（独特）
 - 〜ものを（应该有——后悔）
 - 〜たる（存在——古典定语）
 - いわんや〜をや（更不用说——古典）
 - 〜しかり（就是这样——古典）

 ### 语境中的文学语法 — 示例段落

 > 言葉は単なるコミュニケーションの道具に过ぎないのではなく、 の民族记忆を宿したる容器にほかならない。一つの语が失われると き、それと共に消え去るものは、辞书の一个项目のみか、世代を超 えて受け継がれてきた知恵と感情の集积にほかならないのである。 

**语法分析**：
 - 〜に过ぎないのではなく=不仅仅是〜
 - 宿したる = 宿した（古典定语〜たる）
 - 〜にほかならない = 只不过是（×2）
 - 〜のみか=不仅〜而且
 - 受け継がれてきた = 已遗传（代代相传）

 **翻译**：“文字不仅仅是交流的工具，更是承载着一个民族记忆的器皿。当一个词丢失时，随之消失的不仅仅是字典的词条，更是代代相传积累的智慧和情感。”

 ## 第 6 课：法律和官方日语

 ### 为什么合法日语是 N1

 日语的合法用途：
 1. 古典语法模式（〜べし→〜すべし，〜ものとする）
 2. 精准的技术词汇（批准、实施、条项、遵守）
 3. 高度正式的连词和情态表达
 4.广泛被动语态（〜とされる、〜とみなされる）
 5. 名词重结构以避免歧义

 ### 关键法律语法模式

 **〜ものとする**（规定）：
 合同和法律中法律义务的标准形式。 
- 料金は月末に支払うものとする。— 每月月底付款。 
- 违反者には处罚则を科するものとする。— 违反者将受到处罚。 

**〜とみなす** （认为；视为）：
 法律虚构——将某事视为事实。 
- 日本国内に住所を有する者は、日本居住者とみなす。— 在日本有地址的人将被视为日本居民。 

**〜に基づく / 〜に基づいて** （基于；依据）：
 对于引用法律权威至关重要。 
- 法律に基づく处分（基于法律的措施）

 **〜をもって**（自以下日期起生效）：
 - 本日をもって有效を生ずる。— 该规定从今天起生效。 
- これをもって本件を解决したものとする。— 至此，此事即视为已解决。 

**〜に限る**（仅限于；仅）：
 - 参加资格は会员に限る。— 资格仅限于会员。 

**〜に际し(て)**（在……之际；当）：
 - 契约缔结に际して、以下の事项を确认する。— 缔结契约时，请确认以下事项。

 ## 第 7 课：学术日语 — 论文、论文和对冲语言

 ### 学术日语中的对冲系统

 学术日语受到严重限制。 避免直接提出主张——一切都按照经过论证、经过深思熟虑或可能真实的方式呈现。 

### 对冲词汇

| 表达 | 断言水平| 示例|
 |------------|--------------------|---------|
 | 〜と考えられる | 中等 — 考虑过 | 效果的と考えられる |
 | 〜と言えよう | 中等——可以说| 重要と言えよう |
 | 〜とも言える | 弱——可以说| 问题とも言える |
 | 〜に他ならない | 强——无非| 欠陥に他ならない |
 | 〜とは言い切れない | 不确定——不能说| 失败とは言い切れない |
 | 〜と思われる | 弱——似乎是| 有效と思われる |
 | ～であると推测される | 弱 — 推断 | 増加であると推测される |
 | 〜の可能性が高い | 中等 — 可能 | 原因である可能性が高い |
 | 〜は明らかである | 强——很明显| 相关性は明らかである |

 ### 学术写作中的话语连接器

 **引入主题**：
 - 本稿では〜を论じる。— 在本文中，我讨论〜。 
- 〜について预告する。— 我检查〜。 
- 〜を検讨するにあたり。— 在检查〜。 

**介绍证据**：
 - 〜によれば。— 根据〜。 
- 〜が示すように。— 如〜所示。 
- 〜を踏まえると。— 基于/考虑到～。 

**对比和资格**：
 - しかしながら。— 然而（しかし的正式版本）。 
- とはいえ。— 即便如此； 尽管情况确实如此。 
- もっとも。— 也就是说； 然而; 诚然。 
- ただし。— 然而； 前提是； 除了。 

**结论**：
 - 以上を踏まえると。— 基于以上。 
- 要するに。— 简而言之； 总结一下。 
- すなわち。— 即； 换句话说。 
- つまり。— 简而言之； 换句话说。 
- 结论として。— 结论。 

### 学术段落示例（带分析）

 > 近年的调查によれば、日本における若者の読书离れが深刻化していることが揭示了されている。 この趋势の背景には、デジタルメディアの普及のみか、社会的要因が复雑に络み合っていると 考えられる。もっとも、読书量の减少が即座に知的能力の低下を意味着するとは言い切れない。 しかしながら、深い思考を促す习惯の资金という観点からは、忧虑すべき问题にほかならない。 

**结构分析**：
 - 第1句：证据介绍+对冲断言（示意图されている）
 - 第2句：〜のみか和对冲的因果分析（考えられる）
 - 第3句：反资格（もっとも + とは言い切れない）
 - 第4句：强化结论（しかしながら + にほかならない）

 ## 第 8 课：微妙的细微差别 — 区分相似的 N1 模式

 本课程解决 N1 语法问题，这些问题需要在具有相似但不同含义的模式之间进行选择。 

### Cluster 1: 〜まで vs 〜すら vs 〜さえ vs 〜でさえ

 均表示“均匀”，但不同之处在于：

 | 图案| 最适合 | 示例|
 |---------|----------|---------|
 | さえ | 中性“偶数”； 条件 (さえ〜ば) | 子供さえ知っている |
 | すら | 文学； 更强的意外性| 専门家すら知らない |
 | まで | 程度; “到目前为止”| 泣くまでした |
 | でさえ | で + さえ; 与粒子| 彼でさえ失败した |

 **测试一下自己**：哪个最自然？ 
- 「あの天才___分からなかった」→ すら / でさえ（文学/强调）
 - 「练习___すれば上手になる」→ さえ（条件模式）

 ###集群2：〜てならない vs 〜てたまらない vs 〜てしかたがない

 均表示“无法帮助；无法忍受”，但不同之处在于：

 | 图案| 细微差别| 示例|
 |--------|--------|---------|
 | てならない | 不自觉的情绪； 正式 | 故郷が恋しくてならない |
 | てたまらない | 身体/情绪难以忍受； 强| 暑くてたまらない |
 | てしかたがない | 辞职：“没办法”| 眠くてしかたがない | 眠くてしかたがない

 ### 第 3 组：〜ものを vs 〜のに vs 〜ところを

 都表达了对比/条件情况：

 | 图案| 细微差别| 示例|
 |--------|--------|---------|
 | ものを | 文学的遗憾； 应该有| 言ってくれればよかったものを |
 | のに | 不满意； 矛盾| 言ったのに、分からない |
 | ところを | 情况/情况| 忙しいところを申し訳ない |

 ### 第 4 组：〜ばかりに vs 〜だけに vs 〜からこそ vs 〜ゆえに

 都表达了因果关系，但有不同的细微差别：

 | 图案| 细微差别| 示例|
 |--------|--------|---------|
 | ばかりに | 正是因为——糟糕的结果| 正直に言ったばかりに怒られた | 正直に言ったばかりに怒られた |
 | だけに | 因为——结果是自然的/预期的 | 努力したdけに、合格が嬉しい |
 | からこそ | 正是因为——强调| 好きだからこそ厳しく言う |
 | ゆえに | 所以; 因为——正式/文学| 无知ゆえに过ちを犯す |

 ### 第 5 组：〜にほかならない vs 〜に过ぎない vs 〜に他ならぬ

 | 图案| 细微差别| 示例|
 |--------|--------|---------|
 | にほかならない | 除了； 强调真实身份| これは差にほかならない |
 | に过ぎない | 无非就是； 只是; 不屑一顾 | これは始まりに过ぎない |
 | に他ならぬ | 古典/文学版にほかならない | 〜に他ならぬ事実 |

 ## 第 9 课：N1 听力 — 演讲、辩论和复杂采访

 ### 是什么让 N1 听力变得困难

- **速度**：原生速度，不适合学习者
 - **复杂性**：多发言者讨论、复杂主题
 - **注册**：学术、政治和技术语言
 - **隐性信息**：意味着什么，而不仅仅是所说的
 - **长格式**：需要持续关注的 3-5 分钟的独白

 ### N1 听力问题类型

 **类型 1：基于任务的理解（课题理解）**
 根据所提供的信息了解要采取什么行动。 
策略：聆听动作动词和条件。 

**类型2：要点理解（ポイント理解）**
 确定较长话语的要点。 
策略：聆听结论和重点标记。 

**类型3：综合理解（统合理解）**
 两位发言者讨论不同的立场； 确定同意/不同意。 
策略：跟踪谁的意见是谁的； 注意〜と思う、〜じゃないか标记。 

**类型4：延伸聆听（即时応答 / 长めの话）**
 快速的简短对话或长篇讲座。 
策略： 简而言之：立即响应时间。 长期而言：结构化笔记。 

### 听力词汇——言语中的话语标记

 **主题介绍**：
 - 今日は〜についてお话しします。 
- まず最初に〜について触れたいと思います。 

**过渡标记**：
 - さて / それでは（那时；现在）
 - ところで（顺便说一句；改变话题）
 - それはさておき（先放在一边）
 - それはともかく（不管怎样）

 **演讲中的重点**：
 - 重要なのは〜という点です。 
- ポイントは〜ということです。 
- つまり、〜ということになります。 

**言语中的对冲**：
 - 〜と言っても过言ではないでしょう。— 说～也不为过。 
- 〜と言ってよいのではないかと思います。—我想我们可以这样说〜。

 ## 第 10 课：N1 考试策略 — 每部分得分 100+

 ### 了解 N1 评分系统

 - **总分**：180 分
 - **通过门槛**：总分 100 分，加上部分最低分数
 - **最低部分**：语言知识（词汇/语法+阅读）：19/60； 听力：19/60
 - **未通过最低部分考试则无法通过整个考试** - 即使您的总分超过 100

 ### 逐节策略

 **词汇 (语汇) — ~35 个问题**

 时间分配：110 分钟的语言知识部分中的 20-25 分钟。 

问题类型：
 1. 汉字读法：选择划线字的正确读法
 2. 书き取り：为划线的读物选择正确的汉字
 3. 文脉规定：选择符合上下文的词
 4.言い换え类义语：选择与划线词意思最接近的词
 5. 用法（usage）：选择正确使用该词的句子

 策略：
 - 跳过并返回：如果不确定，标记并返回
 - 四字熟语题：每次考试通常有 2-3 个； 值得专门研究
 - 最难的用法：专注于搭配和注册

 **语法 (文法) — ~35 个问题**

 包括：
 - 文の文法1：选择完成句子的语法模式（15个问题）
 - 文の文法2：句子重新排序（5题）
 - 文章の文法：选择段落中空白的模式（5个问题）

 策略：
 - 句子重新排序：高价值； 练习 4 选择排序模式
 - 区分相似的模式：考官对此进行了大量测试（例如，〜ものを vs 〜のに）
 - 在回答之前阅读完整的文章上下文 文章の文法

 **阅读（読解）——变化**

 5种阅读类型：
 1. 短文（短文）：~150 个字符 — 3 个问题
 2. 中文（中等段落）：~500 个字符 — 3 个问题
 3. 长文（长文）：~1,000 个字符 — 4 个问题
 4.统合理解：比较两个文本——2个问题
 5.长文情报検索：在复杂的文本中查找信息

策略：
 - 时间管理：阅读是最难的时间限制
 - 在阅读文章之前先阅读问题
 - 主要主张问题：找到论文句子
 - 文学の考え（作者观点）：使用第7课中的对冲/断言分析
 - 如果时间紧张，则跳过“统合理解”——需要阅读两篇课文

 **听力（聴解）— ~60 分钟**

 无法查看或快退。 答案是现在或永远。 

策略：
 - 根据介绍句预测内容
 - 在长段落中做简短的笔记（关键名词和数字）
 - 为了综合理解：边听边写 A/B/两者
 - 最后的问题有时会问你没有听到的内容 - 注意否定

 ### 考试前 1 个月强化计划

 **第 1 周：评估**
 - 参加一场完整的模拟考试（限时）
 - 确定最薄弱的部分和问题类型
 - 列出已知的语法差距

 **第 2 周：语法强化**
 - 使用 Shin Kanzen Master Grammar 复习所有约 200 个 N1 模式
 - 重点关注第 8 课中困惑的配对
 - 每天做 20 多道语法练习题

 **第 3 周：词汇和汉字**
 - 回顾四字熟语主列表
 - 复习特别读物（熟字训）
 - 复习谚语前 30 名

 **第 4 周：阅读和模拟考试**
 - 两次定时模拟考试
 - 错误答案的详细审查
 - 识别阅读中的时间管理问题
 - 每日轻度听力复习
