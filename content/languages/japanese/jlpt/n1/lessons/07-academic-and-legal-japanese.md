## 概述

 法律和学术日语是整个语言中最独特的语域之一。 日语口语遵循模糊性、暗示性和共享语境的原则，而法律日语却恰恰相反：它力求完全消除歧义，使用语法结构精确地指定每个条件、例外和应用范围。 该登记册借鉴了明治时期（1868-1912 年）编纂的正式书面语言 (*bungo*) 的传统，当时日本立法者紧急将欧洲法律模式（尤其是德国和法国民法典）改编为日语。 日本现行的民法典（*民法*）、刑法典（*刑法*）和宪法（*宪法*）中至今仍在使用该编纂过程中的语法结构，几乎没有任何变化。 

学术日语与法律日语有一些共同的语法特征，特别是偏爱非人称被动语态结构、使用〜において/〜に关して作为主题框架，以及使用中日抽象名词而不是日语本土动词。 然而，学术日语也有自己的惯例：对冲表达（〜と考えられる，〜の可能性がある），引文公式（〜によれば，〜に仿い），以及逻辑连接词（ゆえに，したがって，したがって， これに対し）。 在 N1 级别，阅读理解段落经常取自学术摘要、法律条款和官方公告——不仅需要词汇，还需要理解这些语域所采用的语法逻辑。 

本课程介绍了法律、学术和官方文本中出现的五种关键语法模式，注释了示例法律和学术段落，并构建了这一要求严格的语域所需的词汇。 

## 学习目标

 在本课程结束时，您可以：
 - 在法律和正式的假设语境中理解并使用〜ものとする和〜ものとして造句。 
- 使用〜に即して/〜に即した来表达原则在具体案例中的应用。 
- 用〜に反して/〜に反する造句来描述违规或违反行为。 
- 使用〜をもって来表达正式的手段、开始或结论。 
- 使用〜いかんによらず和〜いかんにかかわらず来表达无条件适用。

 ## 语法点

 ### 1. 〜ものとする / 〜ものとして（规定/假设）

 **结构：**
 - 动词（普通形式）+ ものとする
 - 名词+である+ものとする
 - 动词（普通形式）+ ものとして + [后续从句]

**古典起源：**
 这种模式源自古典助词もの（物/者），它名词化前面的子句，并在其正式规定功能中与する（to do/be）相结合。 在古典和明治时代的法律文本中，〜ものとす（*mono to su*）是规定规则和程序的标准公式。 现代形式〜ものとする保留了这种规定性功能，基本没有变化。 

**说明：**
 〜ものとする作为法律、合同和法规中规定规则、义务和程序的规范公式。 它的意思是“特此规定”或“情况应如此”。 它是第三人称规定——不是个人决定，而是机构或法律决定。 

〜ものとして则相反，出于计算、论证或计划的目的引入假设。 它的意思是“将 X 视为情况如此”或“为此目的假设”。 它经常出现在学术写作中建立公理或简化假设时。 

**例句：**

 | 日语 | 阅读| 英语 |
 |----------|---------|---------|
 | 本契约に定めのない事项については、民法その他の关系法令の规定によるものとする。 ほんけいやくに さだめのない じこうについては、みんぽうそのほかの かんけいほうれいの きていによるものとする。 本协议未尽事宜，适用民法及其他有关法律、法规的规定。 |
 | 期日までに通知がない场合は、承诺したものとみなす。 きじつまでにつうちがないばあいは、しょうだくしたものとみなす。 逾期未收到通知的，视为已同意。 |
 | 本稿では、言语习得は段阶的プロセスであるものとして论を进める。 ほんこうでは、げんごしゅうとくはだんかいてきプロセsuであるものとしてろんをすすめる。 | 在本文中，论证是基于语言习得是一个阶段性过程的假设进行的。 |
 | 当事者双方が合意した内容は、书面によって确认されたものとする。 とうじしゃそうほうがごういしたないようは、しょめんによって かくにんされたものとする。 双方约定的内容视为书面确认。 |
 | 反证がない限り、当该书类は真正に成立したものとする。 はんしょうがないかぎり、とうがいしょるいはしんせいにせいりつしたものとする。 除非有相反证据，该文件应被视为已有效成立。 |

 **❌ 常见错误 → ✓ 更正：**
 - ❌彼は休休を取るものとする。— 使用个人主题与合法处方公式会产生语域冲突。 
- ✓ 休憩の申请は所定の书式を用いるものとする。— 非人称、机构主语正确为〜ものとする。 

**与N2等效〜ことにする比较：**
 〜ことにする表达个人决定：「早起きすることにした」（我决定早起）。 〜ものとする是制度规定，不是个人决定。 它们是不可替代的。

 ### 2. 〜に即して / 〜に即した（根据/专门应用于）

 **结构：**
 - 名词+に即して+[从句]
 - 名词 + に即した + 名词

 **古典起源：**
 即 (*soku*) 具有“与……对齐”、“被压迫”或“直接对应”的经典含义。 在复合词中，“即”出现在“即时”、“座即に”、“即兴”中。 〜に即して将这种定向排列应用于抽象原则、规则或情况。 

**说明：**
 〜に即して的意思是“直接对应”、“专门应用于”或“根据……的具体情况”。 它比〜に従って（根据）或〜に基づいて（基于）更强：〜に即して意味着不仅遵循规则，而且“应用”原则到当前的具体情况。 法律用法：将法规应用于具体情况。 学术用途：将理论框架应用于特定数据集或案例研究。 

**例句：**

| 日语 | 阅读| 英语 |
 |----------|---------|---------|
 | 各事案具体的状况に即して、适切な対处法を判断するものとする。 かくじあんの ぐたいてきじょうきょうに そくして、てきせつな たいしょほうを はんだんするものとする。 应根据案件的具体情况确定适当的应对措施。 |
 | 现実に即した指导法を采用することで、学习效果が向上した。 げんじつにそくしたしどうほうをさいようすることで、がくしゅうこうかが こうじょうした。 | 通过采用与现实世界条件紧密结合的教学方法，学习成果得到了改善。 |
 | 法の精神に即して解釈するならば、本条项はこのように読むべきである。 ほうのせいしんにそくしてかいしゃくするならば、ほんじょうこうはこのように よむべきである。 | 如果根据法律精神解释，本条款应解读如下。 |
 | 时代の変化に即した规则の修改が急务だ。 じだいのへんかにそくしたきそくのかいていがきゅうむだ。 迫切需要修订法规以适应社会变化。 |

 **与〜に従って比较：**
 〜に従って = 被动遵守规则（“遵循给定的规则”）。 〜に即して = 积极的、针对具体情况的原则应用。 法官依法行事（〜に従って）； 法官将法律适用于具体事实（〜に即して）。

 ### 3. 〜に反して / 〜に反する（相反/违反）

 **结构：**
 - 名词 + に反して + [从句]
 - 名词 + に反する + 名词

 **说明：**
 反 (*han*) 的意思是“反对”、“违背”。 〜に反して表示行动、结果或状态与期望、协议、规则或原则相反。 它有两个主要范围： (a) 违反正式规则或协议； (b) 结果与预期或意图相反。 

**例句：**

 | 日语 | 阅读| 英语 |
 |----------|---------|---------|
 | 本に反する规定行为は、损害损害の対象となる。 ほんきていにはんするこういは、そんがいばいしょうのたいしょうとなる。 违反本规定的行为将受到损害索赔。 |
 | 予想に反して、実験结果は仮说を支持するものだった。 よそうにはんして、じっけんけっかはかせつをしじするものだった。 与预期相反，实验结果支持了假设。 |
 | 条约的精神に反した行动は、国际的批判を招くだろう。 じょうやくのせいしんにはんしたこうどうは、こくさいてきひはんをまねくだろう。 违背条约精神的行为将招致国际社会的批评。 |
 | 公序良俗に反する契约条款は无效とされる。 こうじょりょうぞくにはんするけいやくじょうこうはむこうとされる。 违反公共政策和道德的合同条款应被视为无效。 |
 | に反して参加者が少なく、主催者は落胆した。 きたいにはんしてさんかしゃがすくなく、しゅさいしゃはらくたんした。 与预期相反，出席人数很少，令主办方感到失望。 |

 **与〜に违反して的比较：**
 〜に违反して是专门的法律/监管违规行为。 〜に反して更广泛——它涵盖违反期望、精神或原则。 两者都遵循法律； 只有〜に反して与予想或期待搭配使用。

 ### 4. 〜をもって（通过/截至 - 正式完成或开始）

 **结构：**
 - 名词 + をもって + [关于手段或时间的从句]
 - 名词 + をもって + 〜とする / 〜に代える

 **古典起源：**
 もって (*motte*) 是持つ (*motsu*，持有/拥有) 的经典动名词形式：“拥有 X”或“凭借 X”。 在古典和文学日语中，〜をもって表示完成任务的工具或手段，也表示开始或结束的正式时刻。 

**说明：**
 〜をもって有三个主要用途：（1）手段/方法——“通过”，高度正式地替代〜で； (2) 时间边界——“截至”、“生效日期”； (3) 正式替换 — 〜をもって〜に代える：“X 应充当 Y。”

 **例句：**

 | 日语 | 阅读| 英语 |
 |----------|---------|---------|
 | 本书面语をもって、正式な解约通知といたします。 ほんしょめんをもって、せいしきなかいやくつうちといたします。 本文件特此构成正式取消通知。 |
 | 本日をもって、当プロジェクトを终了とします。 ほんじつをもって、とうプロジェクトをしゅうりょうとします。 截至今日，该项目已结束。 |
 | 诚意をもって交渉に临む所存です。 せいいをもってこうしょうにのぞむしょぞんです。 我们打算以诚意进行谈判。 |
 | 同封の书类をもって入学手続きに代えるものとする。 どうふうのしょるいをもってにゅうがくてつづきにかえるものとする。 所附文件应作为注册程序。 |
 | 完全会一致をもってされたことを报告する。 ぜんかいいちをもってけつぎされたことをほうこくする。 我报告该决议获得一致通过。 |

**与〜によって比较：**
 〜によって在一般写作中表达手段/原因。 〜をもって 是正式注册语，保留用于官方文件、法律文本和正式公告。

 ### 5. 〜いかんによらず / 〜いかんにかかわらず（无论如何/无论）

 **结构：**
 - 名词 (の) + いかんによらず
 - 名词 (の) + いかんにかかわらず
 - 名词 (の) + いかんを问わず (近义词)

 **古典起源：**
 いかん (*ikan*) 是一个中日复合词：如何（普通话*rú hé*），意思是“情况如何”或“情况如何”。 〜いかんによらず因此意味着“无论事情如何”。 这种模式进入了日本正式的法律和官僚语言，并且基本上保持不变。 

**说明：**
 い〜かんによらず和〜いかんにかかわらず都表示无条件适用。 〜いかんにかかわらず 稍微正式一些，在法律上也更精确。 互补的〜いかんによっては（取决于如何）出现在有条件适用的相同法律文本中。 

**例句：**

 | 日语 | 阅读| 英语 |
 |----------|---------|---------|
 | 国民のいかんにかかわらず、本法の规定は适用される。 こくせきのいかんにかかわらず、ほんほうのきていはてきようされる。 本法的规定无论国籍如何均适用。 |
 | 理由のいかんによらず、提出上限を超えた申请は吸收しない。 りゆうのいかんによらず、ていしゅつきげんをこえたしんせいはじゅりしない。 逾期提交的申请，无论何种原因，均不予受理。 |
 | 结果のいかんにかかわらず、参加者全员に修了证を授与する。 けっかのいかんにかかわらず、さんかしゃぜんいんにしゅうりょうしょうをじゅよする。 无论结果如何，所有参与者都将获得结业证书。 |
 | 経纬のいかんを问わず、当事者间の合意が最优先される。 けいいのいかんをとわず、とうじしゃかんのごういがさいゆうせんされる。 无论背景情况如何，双方之间的协议优先。 |

 **与〜にかかわらず (N2) 比较：**
 〜にかかわらず可以在许多情况下替代，但〜いかんにかかわらず更正式，特别暗示某物的*方式*或*性质*的变化。 在法律文本中，〜いかんにかかわらず是标准的。 

## 真实文本

 ### 示例 1：合同条款（契约条项）

 > **第十三条（损害损害）**
 > 本契约规定に反する行为によって相手方に损伤を与えた场合、当事者は、その损伤赔 赔偿の责めを负うものとする。损害赔偿の范囲は、通常损害に限るものとし、特别损害につ いては、当事者がその事を予见し、または予见できた场合に限り、これを赔偿するもの とする。 理由のいかんを问わず、本条に定める补偿额は、当契约の総额を顶部とする。 

**注释：**
 - **本契约の规定に反する行为によって** — 〜に反する（违反）+〜によって（通过）
 - **当事者は…责めを负うものとする** — 〜ものとする 规定法律义务
 - **通常有障碍に限るものとし** — 〜に限る（仅限）+ 〜ものとし（规定）
 - **理由のいかんを问わず** — 无条件损害赔偿上限
 - **当该契约の総额を上限とする** —上限公式； 机构处方

 **译：** 第十三条（损害）：一方违反本协议规定的行为给另一方造成损害的，该方应当承担赔偿责任。 损害赔偿范围仅限于普通损害赔偿； 特殊损害，仅在当事人预见或者能够预见有关情况的情况下才予以赔偿。 无论何种原因，本条规定的赔偿金额不得超过相关合同的总价值。 

### 示例 2：学术摘要（学术论文要旨）

 > 本研究は、现代日本语における受动构文の使用频度の変化を、コーパsuデータに即して分析したものである。 から二〇二〇sにかけての新闻コーパsuを対象に、受动文の出现率を手段的方法をもって调查した结果、正式书き この倾向は、教科书规范に反するものでなく、むしろ文语的形式性への 接近と解釈される。今后の研究においては、话しことばデータを加え、书きことばとの乖离のいかんを精查する必要がある。 

**注释：**
 - **kopasudeータに即して** — 直接对语料库进行分析/根据语料库进行分析
 - **计量的方法をもって调查した** —“通过”定量方法
 - **教科书规范に反するものでなく** — 否定违规：不违反规范
 - **乖离のいかんを精查する** — 调查分歧的性质/程度

 ## 对话

 ###对话1：合同审查会议（契约审查会议）

 **弁护士：**第七条の「本契约に定めのない事项は民法によるものとする」という规定ですが、これは通常の记载ですか。 

**社内法务负责：**はい、标准的な补充条款です。当事者双方明确说明に合意した事项以外は、法律の规定が自动的に适用されるものとする、という趣旨です。 

**弁护士:** ただ、本件の特殊性に即して考えると、民法の一般原则だけでは不十分な缘が生じうるのではないでしょうか。 

**社内法务承担：** ご指摘はもっともです。特に损害赔偿の范囲については、第十三条をもっ て特别规定として优先适用するという解釈で问题ないと考えております。 

**弁护士:** 理由のいかんにかかわらず上限を设ける条项については、公序良俗に反するという称がなされる可行性はありませんか。 

**社内法务承担：** その点は精查済みです。上回避が着しく不当でない限り、裁判所は当事者 の合意を尊重する倾向があり、本件では问题ないものと判断しております。

###对话2：论文答辩（学位论文审查）

 ## 翻译练习

 ### 英语 → 日语（法律/学术）

 1. “无论所造成的损害的性质或金额如何，本条的规定均应适用。” → 本条规定は、発生した损害の性质又は金额のいかんにかかわらず、适用されるものとする。 

2、“组委会根据各申请人的具体情况确定评审标准。” → 委员会は、各申请者具体的事情に即して、评审基准を判断するものとする。 

3.“违反本协议精神的行为应构成立即终止的理由。” → 本契约の精神に反する行为は、即时解除の事由となるものとする。 

4、“自签字之日起，双方特此接受本协议规定的所有权利和义务。” →署名の日をもって、当事者双方は书に规定するすべての権利および义务を受诺したものとする。 

5.“出于分析目的，本研究假设变量在统计上是独立的，并据此进行。” → 本研究は、分析目的のため、変数间は统计的に独立であるものとして论を进める。 

### 日语 → 英语（法律/学术）

 1. 「本程序に定めのない事项については、别途协议の上、决定するものとする。」 → 本规则未尽事宜，由双方另行协商确定。 

2. 「时代の変化に即した规定の整备は、行政の责任においてなされるべきである。」 → 制定符合社会变迁的法规是行政部门应承担的责任。 

3. 「公益に反する行为は、たとえ合法であるとしても、伦理的批判を免れない。」 → 即使合法，违反公共利益的行为也无法逃脱伦理批评。 

4. 「本证明书をもって、在籍を证明するものとする。」 → 此证明书作为入学证明。 

5. 「提出书类のいかんを问わず、申请の最终判定は审查委员会が行う。」 → 无论提交的文件如何，申请的最终决定均由审查委员会做出。 

## 文化历史注释

日本法律文本的独特语法起源于明治政府在 1870 年至 1900 年间实施的非凡的法律现代化计划。日本面临着巨大的外部压力，要求修改佩里准将到来后强加的“不平等条约”——这些条约否认日本关税自主权，并对外国人强加治外法权，因为西方列强认为日本的法律体系“不文明”。 为了展示法律的现代性，日本立法者与德国和法国法律学者（特别是德国法律学者赫尔曼·罗斯勒（Hermann Roesler）的《商法典》和法国学者古斯塔夫·布瓦索纳德（Gustave Boissonade）的早期刑法典）合作，以日语起草了一套全面的西式法律体系。 这些起草者选择的语法结构——〜ものとする、〜をもって、〜いかんにかかわらず——是因为它们能够用一种以前缺乏西方成文法传统的语言传达法律的精确性。 这些相同的结构仍然保留在日本现行的法律法规中，成为明治变革的活语言纪念碑。 

学术日语沿着一条平行但截然不同的轨迹发展。 明治大学系统以德国研究型大学为蓝本，早期的日语学术写作深受德国学术风格的影响——德国学术风格偏爱长名词性句子、被动结构和抽象名词，而不是个人口头陈述。 这就产生了日本学术散文的特征：偏爱中日复合名词而不是日语动词，使用非人称被动结构（〜と考えられる，〜が示指向される），以及避免第一人称主张。 像西天音（Nishi Amane）这样的明治时代学者（他创造了“哲学”*tetsugaku*“哲学”和“理性”*risei*“理性”等术语）建立了日本学术话语的词汇。 今天的 N1 阅读文章直接借鉴了这一传统。 

## 自检

 1. 〜ものとする和〜ことにする之间的主要区别是什么？ 在适当的上下文中提供每个示例。 
*（预期：〜ものとする = 机构/法律处方；〜ことにする = 个人决定。例如：「提出期限を五月一日とするものとする。」 vs 「五月一日に提出することにした。」)*

 2. 「现実に即した政策を立案する必要がある」中，即した修饰什么，为什么这里用〜に即した而不是〜に合った？ 
*（预期：即した修改政策。〜に即した意味着积极与特定的现实世界条件保持一致；〜に合った只是意味着“适合/适合”，而不暗示具体情况的应用。）*

 3. 合同上写着：「本日をもって契约を解除する」。 〜をもって在这里有什么法律效力，〜で可以代替它吗？ 
*（预期：〜をもって将今天标记为准确的正式生效日期。〜で在法律文件中在风格上不合适 - 〜をもって表示正式登记。）*

 4. 将下列合同条款翻译成英文：「利用者の责に帰すべき事由のいかんにかかわらず、当社は本サービsuに起因する损害についての责任を负わない。」
 *（预计：无论是否由用户原因造成，我们对因本服务而造成的任何损害均不承担任何责任。）*

 5. 〜に反して和〜に违背して有什么区别？ 在什么情况下两者都可以使用，在什么情况下只能使用一种？ 
*（预期：〜に违反して=具体的法律/监管违规。〜に反して=更广泛的-包括违反期望，精神或原则。两者：法律に反して/法律に违反して。仅〜に反して：予想に反して， 期待に反して。）*
