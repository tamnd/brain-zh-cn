## 概述

 报告性演讲是 N4 及以上级别的核心沟通技能之一。 在现实生活中，您经常需要转达某人所说的话、分享您的想法、报告您所听到的内容或引用来源。 日语使用**引语助词と**将报道的内容与言う、思う、闻く、考える和感じる等动词联系起来。 统一所有这些模式的关键规则是，と之前的内容必须始终采用**简单形式**，无论句子的其余部分多么礼貌。 

本课还解决了一个常见的混淆来源：〜と言う（直接/间接报告的言语）、〜によると〜そうだ（引用来源）和〜らしい（明显的道听途说）之间的重叠。 您将练习将直接引语转换为间接引语，了解时态在报道子句中的作用，并培养正确地将信息归因于其来源的能力。 本课的练习特别关注在と引用子句中使用ます形式的错误——这是最常见的 N4 错误之一。 

## 学习目标

 在本课程结束时，您可以：
 - 使用〜と言う/言った 报告具有正确的简单形式嵌入的直接语音和间接语音。 
- 用〜と思う/思った表达自己的观点和推论。 
- 使用〜と闻く/闻いた和〜によると〜そうだ来引用消息来源并转发传闻信息。 
- 使用简单形式+と将直接引语句子（带「」引号）转换为自然间接引语。

 ## 语法点

 ### 1. 〜と言う / 〜と言った（转述言语 — 直接和间接）

 **结构：** [普通从句] + と言う / と言った / と言っている / と言っていた

 **说明：**

 这里的助词と是**引号**，而不是连词“and”。 と之前的所有内容都是所说内容的*内容*，并且该内容必须采用明文形式。 跟在と后面的动词表示与言语行为的关系：

 - と言う =“这么说”（当前/习惯）
 - と言った =“这么说”（报道过去）
 - と言っている =“正在说/一直在说”
 - と言っていた =“这么说”

 将直接引语转换为间接引语时，引文中的所有ます和です形式都会变成其普通形式的等价形式。 报告内容的时态反映的是“行为发生的时间”，而不是报告的时间。 

**直接→间接语音转换示例：**

 | 直接言语| 间接引语|
 |--------------|----------------|
 | 「明日行きます」と言った。 明日行くと言った。 
| 「美味しくないです」と言った。 | 美味しくないと言った。 
| 「学生でした」と言った。 | 学生だったと言った。 |
 | 「雨が降りますか」と闻いた。 | 雨が降るかと闻いた。 |

 **更多示例：**

| 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 田中さんは来周来ると言った。 たなかさんはらいしゅうくると言った。 田中先生和伊塔一起来了。 | 田中说他下周会来。 |
 | 先生は试験は难しくないと言っていました。 せんせいはしけんはむずかしくないといっていました。 老师是 shiken wa muzukashikunai 到 itte imashita。 | 老师说考试不难。 |
 | 彼女は「疲れた」と言った。 かのじょは「つかれた」と言った。 Kanojo 对 itta 来说是“tsukareta”。 | 她说：“我累了。” |
 | 母は早く帰ってくると言っていた。 はははやくかえってくるといっていた。 哈哈 wa hayaku kaette kuru to itte ita。 | 我妈妈说她会早点回来。 |
 | 彼はもう帰ったと言いました。 かれはもうかえったといいました。 Kare wa mō kaetta 到 iimashita。 | 他说他已经回家了。 |
 | 山田さんが结婚すると言っている。 やまださんがけっこんするといっている。 山田桑 ga kekkon suru 到 itte iru。 | 山田说他要结婚了。 |

 **常见错误：**
 ❌ 彼は来ますと言った → ✓ 彼は来ると言った
 （在报告的言语中，ます不要在と之前）

 ### 2. 〜と思う / 〜と思った（思考——意见和推论）

 **结构：** [普通形式] + と思う（现在的想法）/ と思った（过去的想法）

 **说明：**

 と思う 用于：
 1. 表达**发言者当前的观点**：私はこれは正しいと思う（我认为这是正确的）
 2. 根据证据表达**初步结论**：明日は雨だと思う（我认为明天会下雨）
 3. 以第三人称表达某人的想法：彼はいいと思っている（他觉得不错）

 重要提示：当と思った与第一人称一起使用时，它可以表示“我认为”（可能已经改变的过去的信念），或者它可以表达你形成意见的那一刻。 

**示例：**

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 私はこの映画は面白いと思います。 わたしはこのえいがはおもしろいとおもいます。 Watashi wa kono eiga wa omoshiroi to omoimasu。 | 我觉得这部电影很有趣。 |
 | 明日は雨が降ると思う。 あしたはあめがふるとおもう。 Ashita wa ame ga Furu to omou。 | 我想明天会下雨。 |
 | 彼は来ないと思った。 かれはこないとおもった。 Kare wa konai 到 omotta。 | 我以为他不会来。 |
 | 彼女はもう家に着いたと思います。 かのじょはもういえについたとおもいます。 Kanojo wa mō ie ni tsuita to omoimasu。 | 我想她已经到家了。 |
 | 先生はそれは正しいと思っている。 せんせいはそれはただしいとおもっている。 Sensei wa sore wa tadashii to omotte iru。 | 老师认为这是正确的。 |

 **常见错误：**
 ❌ 私は明日は雨が降りますと思います → ✓ 私は明日は雨が降ると思います
 （と思う之前的简单形式，总是）

 ---

 ### 3. 〜と闻く / 〜と闻いた / 〜によると〜そうだ（听说/根据）

 **结构：**
 - [普通形式] + と闻く/闻いた =“我听到/听说”
 - [出处] + によると + [普通形式] + そうです/そうだ = “根据[出处]，...”

 **说明：**

 と闻く/闻いた 报告您从某人那里听到的信息，说话者对听到的信息承担一些个人责任。 这是个人传闻：有人直接告诉你的。 によると〜そうだ 用于引用外部的、通常是正式的来源（新闻、书籍、天气预报、第三方）——这些信息感觉更离谱。 两者都需要在と之前使用普通形式。 

**示例：**

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 田中さんが転职すると闻いた。 たなかさんがてんしょくするときいた。 田中先生从天色到 kiita。 | 听说田中要换工作了。 |
 | 先生から、试験が来周あると闻きました。 せんせいから、しけんがらいしゅうあるときいた。 老师来了，就让他来吧。 | 我听老师说下周有考试。 |
 | 天気予报によると、明日は雪が降るそうです。 てんきよほうによると、あしたはゆきがふるそうです。 天哪，你已经到了，ashita wa yuki ga Furu so desu。 | 根据天气预报，明天有雪。 |
 | ューによると、地震があったそうだ。 にゅーすによると、じしんがあったそうだ。 Nyūsu ni yoru to, jishin ga atta so da。 | 据消息称，发生了地震。 |
 | 友达によると、あの店は高いそうですよ。 ともだちによると、あのみせはたかいそうですよ。 Tomodachi ni yoru to, ano mise wa takai so desu yo. | 据我朋友说，那家店显然很贵。 |

 **常见错误：**
 ❌ 天気予报によると、明日は雪が降ると言いました → ✓ 天気予报によると、明日は雪が降るそうです
 （与によると一起使用そうだ表示外部来源；言いました暗示特定的人说话）

 ## 语法练习

 将括号内的直接引语转换为间接引语，或填写缺失的助词/形式。 答案如下。 

1. 田中さんは「明日来ます」と言った。→ 田中さんは明日_____と言った。 
2.先生から、宿题が多い___闻きました。 
3. nyusu___よると、明日は晴れる___です。 
4. 彼女は「映画が好きです」と言っていた。→ 彼女は映画が_____と言っていた。 
5. 私はこの本はつまらない___思います。 
6. 母は「早く起きなさい」と言った。→ 母は早く起きる___に言った。（使用よう）
 7. 友达が教えてくれた。その店は安い___。（そうだ）→ ___
 8. 「电车が遅れています」と駅员が言った。→ 电车が遅れている___駅员が言った。 
9.先生は试験は来周___言っている。（ある）
 10.田中さんが転勤する___闻きましたが、本当ですか。 

**答案关键：**
 1. 来ると（普通形：くる）
 2. とを（と）
 3. によると / そうです
 4. 好きだ（普通形式）
 5. と（と思います）
 6.ように（よう+に）
 7.友达が教えてくれたが、その店は安いそうだ。 
8. と（电车が遅れていると）
 9. あると（あると言っている）
 10. と（転勤すると）

 ## 翻译练习

 ### 英语 → 日语

 1. 田中说他不会来参加聚会。 
2.我想明天会很冷。 
3.据消息，该店昨天关门了。 
4.听说考试很难。 
5. 她说：“我从未去过日本。”

**答案：**
 1.田中さんはパーティーに来ないと言った。 
2. 明日は寒いと思います。 
3. nyusuによると、その店は昨日闭まったそうだ。 
4.试験はとても难しかったと闻きました。 
5. 彼女は「日本に行ったことがない」と言っていた。 

### 日语 → 英语

 1.先生は「来周は休みだ」と言いました。 
2.友达から、あの映画は面白くないと闻いた。 
3. 天気予报によると、今夜は雪が降るそうです。 
4. 私は彼女はもう大学を卒业したと思っていた。 
5. 彼は「分からない」とだけ言った。 

**答案：**
 1. 老师说：“下周放假。”
 2. 我听朋友说那部电影没什么意思。 
3. 根据天气预报，今晚显然会下雪。 
4. 我以为她已经大学毕业了。 
5. 他只说：“我不明白。”

 ## 文化笔记

 日本报道的言论反映了对**信息来源和可靠性**的深刻文化敏感性。 通过在语法上区分你个人观察到的内容、你逻辑上得出的结论、你从朋友那里听到的内容以及权威来源报道的内容，日语在每个句子中都编码了**认知责任**——谁对信息负责。 这就是为什么你不能简单地说“そうだ”来表示所有事情：日语语法要求你明确说明你从哪里获得信息。 

这也解释了为什么と思います在日语对话中如此普遍。 通过将陈述表述为“我认为”，演讲者表现出了理智上的谦逊和对错误的开放态度。 这并不是弱沟通——在日本文化中，这是**体贴的沟通**，表明你尊重听众持有不同观点的权利。 过于自信的绝对陈述可能会显得咄咄逼人，而恰到好处的“と思います”则可以保持社会和谐完好无损。

 ## 自检

 1. 为什么在转述言语中，即使在礼貌的句子中，也必须始终将简单形式置于と之前？ 
2.田中さんが来ると闻いた和田中さんが来るそうだ有什么区别？ 
3.转换：「もう食べました」と言った→间接引语。 
4. 什么时候用によると〜そうだ而不是と闻いた？ 
5. と思った与と思っている有何不同？ 为每一个提供一个例子。
