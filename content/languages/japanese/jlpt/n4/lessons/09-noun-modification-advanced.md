## 概述

 名词修饰是指能够使用一个完整的句子或从句来描述一个名词——就像英语使用“我买的书”或“打电话的人”等关系从句的方式一样。 在日语中，所有修饰都在它所修饰的名词之前**，并且修饰子句必须采用**简单形式**。 这种名词前修饰系统是英语和日语之间最重要的结构差异之一，掌握 N4 可以极大地提高阅读理解和表达范围。 

本课涵盖三个相互关联的主题。 首先，**关系从句修饰**：将普通从句直接放在名词之前以限制或描述它。 其次，**〜という + 名词**：一个重要的模式，意思是“被称为/被命名/被称为”，用于引入新的或解释的概念。 第三，**名词化**：〜の和〜こと模式将动词短语转换为名词短语，使得可以将动作作为事物来谈论。 其他模式包括〜ところ（阶段/点）、ことがある（经验和偶尔发生）、ことにする（决定）和ことになる（已经决定/事实证明）。 

## 学习目标

 在本课程结束时，您可以：
 - 通过在名词之前放置简单形式的动词短语来构建自然关系从句。 
- 使用〜という + 名词来介绍和解释命名概念、引用和定义。 
- 区分名词化词 の 和 こと 并根据上下文选择正确的。 
- 在自然会话和正式语境中使用ことがある、ことにする和ことになる。

 ## 语法点

 ### 1.普通形式+名词（关系从句修饰）

 **结构：** [普通句子] + 名词

 **说明：**

 在日语中，要使用从句（关系从句）修饰名词，只需将整个从句（以**普通形式**）直接放在名词之前即可。 没有关系代词（没有“那个”、“哪个”或“谁”）。 修饰后的名词称为**中心名词**（主名词，shu-meishi）。 该子句可以以任何方式描述名词：时间、内容、身份、描述、目的或任何其他关系。 

关键规则：
 - 该子句必须采用**简单形式**（现在/过去，肯定/否定）
 - 修饰从句直接位于名词之前——它们之间没有助词
 - 相同的模式适用于所有动词类型和形容词
 - 当中心名词是嵌入子句的*主语*时，在子句中使用が（而不是は）

 **图：** [私が昨日食べた] + ケーキ = “我昨天吃的蛋糕”

| 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 私が昔読んだ本はとても面白かった。 わたしがきのうよんだほんはとてもおもしろかった。 Watashi ga kinō yonda hon wa totemo omoshirokatta。 | 我昨天读的书很有趣。 |
 | 田中さんが作った料理はおいしい。 たなかさんがつくったりょうりはおいしい。 田中先生 tsukutta ryōri wa oishii。 | 田中做的饭菜很好吃。 |
 | 东京に住んでいる友达に会った。 とうきょうにすんでいるともだちにあった。 东京 ni sunde iru tomodachi ni atta。 | 我认识了一位住在东京的朋友。 |
 | 私が行きたiresutoranはここです。 わたしがいきたいれすとらんはここです。 Watashi ga ikitai Resutoran wa koko desu。 | 我想去的餐厅就在这里。 |
 | 彼が言ったことは本当だと思う。 かれがいったことはほんとうだとおもう。 Kare ga itta koto wa hontō da to omou。 | 我认为他说的是真的。 |
 | 试験に合格した学生はkurasuの半分だった。 しけんにごうかくしたがくせいはくらすのはんぶんだった。 Shiken ni gōkaku Shita gakusei wa kurasu no hanbun datta。 | 通过考试的学生占全班的一半。 |

 **常见错误：**
 ❌ 私が 昔日読みましたの本 → ✓ 私が 昔日読んだ本
 （关系从句中的名词前切勿使用ます形式；始终为普通形式+删除任何系词）

 ### 2. 〜という + 名词（“被称为/被称为/那......”）

 **结构：** [单词/名称/概念] + という + 名词

 **说明：**

 という字面意思是“被称为/命名/说的”。 它将名称、引言、概念或信息与对其进行分类或标记的名词连接起来。 它有三个主要用途：

 1. **命名**：「テトorisu」というゲームを知っていますか（“你知道一个叫俄罗斯方块的游戏吗？”）
 2. **引用/报道**：「早く来い」という声が闻こえた（“我听到一个声音说‘快来’”）
 3. **概念定义**：梦という言葉の意义（“‘梦’这个词的意思”）

 という也用于介绍不熟悉或已定义的概念，这对于解释和学术语言至关重要。 

| 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 「ゆっくり」という言葉の意思是は何ですか。 「ゆっくり」というこどばのいみはなんですか。 “Yukkuri”到 iu kotoba no imi wa nan desu ka。 | “yukkuri”这个词是什么意思？ |
 | タナカという人から电话がありました。 たなかというひとからでんわがありました。 田中与 iu hito kara denwa ga arimashita。 | 接到了一个叫田中的人的电话。 |
 | 明日试験があるという话を闻いた。 あしたしけんがあるというはなしをきいた。 Ashita shiken ga aru to iu hanashi o kiita。 | 我听到一个故事，明天有考试。 |
 | 「頑張れ」というメッセージをもらった。 「がんばれ」というめっせーじをもらった。 “Ganbare” 为 iumesējiomoratta。 | 我收到一条消息说“尽力而为”。 |

 **常见错误：**
 ❌ タナカというの人 → ✓ タナカという人
 （という直接在名词前面，不带の）

 ### 3. 名词化词：〜の vs. 〜こと（动词转名词）

 **结构：**
 - [普通形式] + の + 助词（の 是随意的，与感觉/经验动词一起使用）
 - [普通形式] + こと + 助词（こと为形式形式，用于抽象概念和形式表达）

 **说明：**

 の 和 こと 都可以将动词短语变成名词短语。 它们通常可以互换，但每个都有特定的首选上下文：

 | 特色 | 〜の | 〜こと |
 |---------|------|--------|
 | 注册 | 休闲、口语 | 正式，书面|
 | 感觉动词（看、听、感觉）| ✓ 首选 | ✗ 不太自然 |
 | 抽象概念、规则、决策| ✗ 不太自然 | ✓ 首选 |
 | 情感动词后 (嬉しい, 好き) | 都可以 | 都可以 |
 | 固定表达（ことがある、ことにする）| ✗ | ✓ 必须使用こと |
 | 正式写作的主题 | ✗ | ✓ |

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 彼が泳ぐのを见た。 | かれがおよぐのをみた。 | Kare ga oyogu no o mita。 | 我看见他在游泳。 （感官→の）|
 | 日本语を勉强することは楽しい。 にほんごをべんきょうすることはたのしい。 Nihongo o Benkyō suru koto wa tanoshii。 | 学习日语很有趣。 (正式/一般 → こと) |
 | 泳ぐのが好きです。 | およぐのがすきです。 Oyogu no ga suki desu。 | 我喜欢游泳。 （感官/个人喜好→の） |
 | 毎日运动することが大切です。 まいにちうんどうすることがたいせつです。 Mainichi undō suru koto ga taisetsu desu。 | 每天锻炼身体很重要。 （规则/原则 → こと）|

 **常见错误：**
 ❌ 见るこが好き → ✓ 见るのが好き（感官偏好）
 ❌ ここでタバコを吸うのは禁止です → ✓ ここでタバコを吸うことは禁止です（正式规则/禁止 → こと）

 ### 4. ことがある（经验/偶尔发生）

**结构：**
 - [普通过去式] + ことがある =“曾经（曾经）有过这样做的经历”（经验性的）
 - [普通非过去] + ことがある =“有时/偶尔做”（习惯性偶尔）

 **示例：**

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 日本に行ったことがあります。 にほんにいったことがあります。 Nihon ni itta koto ga arimasu。 | 我曾经去过日本。 |
 | 寿司を食べたことがない。 | すしをたべたことがない。 | 寿司 o tabeta koto ga nai。 | 我从来没有吃过寿司。 |
 | 时々授业を休むことがある。 ときどきじゅぎょうをやすむことがある。 Tokidoki jugyō o yasumu koto ga aru。 | 我有时会逃课。 （偶尔）|
 | 彼に会ったことがありますか。 かれにあったことがありますか。 Kare ni atta koto ga arimasu ka。 | 你见过他吗？ |

 ---

 ### 5. ことにする / ことになる（决定/决定）

 **结构：**
 - [普通非过去] + ことにする =“决定做”（说话人的决定）
 - [普通非过去时] + ことになる = “it has was said that / it Thors out that”（外部决定或自然结果）
 - [普通非过去] + ことにしている =“让它成为一种习惯/规则”（持续的自我决定）

 **说明：**

 ことにする 表达**个人的、深思熟虑的决定**。 ことになる 表达**外部决定的或自然产生的结果** - 说话者不是决策者。 这种区别具有社会意义：ことになりました经常在公司/团体环境中宣布决定时使用，因为它听起来不太像个人选择（更谦虚/偏向）。 

| 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 毎日日本语を勉强することにした。 まいにちにほんごをべんきょうすることにした。 Mainichi Nihongo o benkyō suru koto ni shita。 | 我决定每天学习日语。 |
 | 転职することにしました。 てんしょくすることにしました。 天色 suru koto ni shimashita。 | 我决定换工作。 |
 | 来年、东京に転勤することになりました。 らいねん、とうきょうにてんきんすることになりました。 雷年，东京，天金，琴声，成下。 | 已决定明年调往东京。 |
 | 毎朝、30分走ることにしている。 まいあさ、30ぷんはしることにしている。 Maiasa，sanjuppun hashiru koto ni shite iru。 | 我给自己定下了每天早上跑步30分钟的规则。 |

 ---

 ### 6.〜ところ（阶段/动作要点）

 **结构：**
 - [字典。 形式] + ところ =“即将”（行动前）
 - [〜ている] + ところ = “在中间”（正在进行）
 - [普通过去] + ところ = “刚刚完成”（动作后）

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 今から出かけるところです。 いまからでかけるところです。 今生今世。 | 我正要出去。 |
 | 今、勉强しているところです。 いま、べんきょうしているところです。 今，我们就这样吧。 | 我现在正在学习中。 |
 | ちょうど食べたところです。 ちょうどたべたところです。 Chōdo tabeta tokoro desu。 | 我刚吃完饭。 |

 ## 语法练习

 用の、こと、という、ことにした、ことになった、ところ或ことがある填空。 答案如下。 

1. 田中___人が来ました。（介绍一个人的名字）
 2. 彼が话している___を闻いた。（感官-听到他说话）
 3.毎日运动する___は健康にいいです。（形式原则）
 4.日本に行った___がありますか。（经验）
 5.来年、会社を辞める___。（个人决定）
 6.时々夜中に起きる___ある。（偶尔的习惯）
 7.今、ご饭を食べている___です。（中间）
 8. 东京に転勤する___なりました。（外部决定）
 9. が黒板先生に书いた___をノートに写した。（写的是什么——の或こと）
 10. 「努力は里切らない」___言葉が好きだ。（引用/俗语称为...）

**答案关键：**
 1. という（田中という人）
 2.の（话しているのを闻いた）
 3. こと（运动することは）
 4. こと（行ったことが）
 5. ことにした（辞めることにした）
 6. ことが（起きることがある）
 7. ところ（食べているところ）
 8. ことに（転勤することになった）
 9. こと（书いたことをノートに — 稍微正式/抽象）
 10. という（「努力は里切らない」という言葉）

 ## 翻译练习

 ### 英语 → 日语

 1. 我老师推荐的书很有趣。 
2.我从来没有吃过马肉。 
3.我决定每天早上6点起床。 
4. 奈良市因鹿而闻名。 
5. 我现在正要离开。 

**答案：**
 1.が勧めた先生本はとても面白かった。 
2.马刺しを食べたことがない。 
3. 毎朝6时に起きることにした。 
4.奈良という都市は鹿で有名です。 
5.今から出かけるところです。 

### 日语 → 英语

 1. 昔田中さんが言ったことを覚えていますか。 
2. 外国に住んだことがありますか。 
3.来月から吸烟することにしました。 
4.今年から部长が変わることになりました。 
5. 走るのが得意ですか？ 

**答案：**
 1. 你还记得田中昨天说的话吗？ 
2.您曾经在国外生活过吗？ 
3. 我决定从下个月开始戒烟。 
4. 决定从今年开始更换经理。 
5.你擅长跑步吗？

 ## 文化笔记

 日语关系从句——将整个句子放在名词之前——反映了**中心词结尾的句子结构**，这与英语等中心词在先的语言有着根本的不同。 在英语中，“the book I read”在前是中心名词，在后是修饰语。 在日语中，私が読んだ本将整个修饰语（私が読んだ =“我读过”）放在中心名词（本=“书”）之前。 这种结构差异意味着说日语的人必须从末尾向后规划句子，在构建描述性上下文的同时牢记关键名词。 

ことにする（个人决定）和ことになる（外部/自然结果）之间的区别具有重要的文化维度。 在日本的工作场所和社会环境中，通常认为使用ことになりました（已决定，决定者未指定）而不是ことにしました（我决定）来表达决定更为谦虚，因为它偏离了个人代理。 宣布新政策的经理可能会说〜ということになりました（已决定...）而不是〜ということにしました（我已决定...），将决策视为集体或自然的结果，减少潜在的阻力并保持团队和谐。 这种语法选择体现了日语中“集体决策（nemawashi）”优于个人权威的价值观。
