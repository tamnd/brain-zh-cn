##〜てしまう

 | 物业 | 价值|
 |----------|--------|
 | 图案| 〜てしまう |
 | 日本语能力考试等级 | N4 |
 | 类型 | 完成 |
 | 正式| 中性|
 | 注册 | 两者 |

 ## 含义

 **〜てしまう**表示一个动作已经完全完成，或者一个动作意外地发生，通常会导致遗憾或失望的感觉。 它强调事件的终结性。 

## 结构式```
[Verb Te-form] + しまう
```* **变形：** 采用任何动词的*Te形式*（例如，食べる → 食べて、読む → 読んで）并附加しまう。 
* **礼貌用语：** 在礼貌用语中，使用 **〜てしまいます** (te-shimaimasu)。 
* **随意/口语收缩：** 在随意言语中，**〜てしまう** 通常收缩为 **〜ちゃう** (chau)，而 **〜でしまう** 收缩为 **〜じゃう** (jau)。 

## 详细说明

 1. **完成：** 表示一个动作完全完成。 例如，“我把它全部吃了”（全部食べてしまった）。 它强调没有留下任何东西或者该过程已完全结束。 
2. **遗憾/无意：** 常用来表达说话者无意或不喜欢的事情发生。 例如，“我不小心丢了钱包”（财布をなくしてしまった）。 
3. **正式性：** 虽然基本形式是中性的，但缩写形式（〜ちゃう）是严格随意的，应避免在正式写作或商务场合中使用。 
4. **细微差别：** “完成”和“遗憾”之间的选择完全取决于上下文。 如果你完成了一项艰巨的任务，听起来就像“终于完成了！” 如果你打破了一个花瓶，听起来就像“哎呀，我打破了它”。 

## 例句

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 本を全部読んでしまった。 | ほんをぜんぶよんでしまった。 Hon o zenbu yonde shitta。 | 我读完了整本书。 |
 | 电车で寝てしまって、乘り过ごした。 でんしゃでねてしまって、のりすごした。 Densha de nete shimatte，norisugoshita。 | 我在火车上睡着了，错过了车站。 |
 | 宿题を忘れてしまいました。 | しゅくだいをわすれてしまいました。 Shukudai o wasurete shimaimashita。 | 我（遗憾的是）忘记了我的作业。 |
 | | ケーキを全部食べちゃった。 けーきをぜんぶたべちゃった。 Kēki o zenbu tabechatta。 | 我把蛋糕都吃了（哎呀）。 |
 | 伞を电车に忘れてしまった。 | かさをでんしゃにわすれてしまった。 Kasa o densha ni wasurete shimatta。 | 我把伞落在火车上了。 |

 ## 与相似模式的比较

 ### 〜てしまう vs 〜た（过去时）
 〜た只是简单地陈述发生的事件，而〜てしまう则添加了**最终性**或**情绪反应**（遗憾或解脱）的细微差别。 

| 图案| 细微差别| 示例|
 |--------|--------|---------|
 | 〜た | 事实| 食べました（我吃了。） |
 | 〜てしまう | 遗憾/完成| 食べてしまいました（全吃了/误吃了。） |

 ### 〜てしまう vs 〜ておく
 〜ておく指的是为未来做准备，而〜てしまう指的是完成某件事或意外的结果。 

## 常见错误

 ❌ 宿题を忘れてしまったです。 → ✓ 宿题を忘れてしまいました。 
* **解释：** しまう是动词。 您必须结合动词本身 (shimaimashita)，而不是在字典形式中添加“desu”。 

❌ 彼は帰るしまった。 → ✓ 彼は帰ってしまった。 
* **说明：** 您必须使用前面动词 (kaette) 的 *Te 形式*，而不是字典形式。 

→ ✓ 嬉しいから、宿题を终わらせてしまった（或终えてしまった）。 
* **说明：** 确保前面的动词正确为及物/不及物。 

## 练习练习

 **填空：**

 1.友达の秘密を（话す）＿＿＿＿＿＿＿＿。（不小心说出了朋友的秘密。）
 2.映画が（终わる）＿＿＿＿＿＿＿＿。（电影已完全结束。）
 3. 昨天の夜、全部（食べる）＿＿＿＿＿＿＿＿。（昨晚吃完了。）
 4. バスに财布を（忘れる）＿＿＿＿＿＿＿＿。（我把钱包落在公交车上了。）
 5. 难しい问题が（解く）＿＿＿＿＿＿＿＿。（我设法解决了这个难题。）

 **答案关键：**
 1.话してしまった（Hanashite shimatta）
 2. 终わってしまった (Owatte shitta)
 3. 食べてしまった（Tabete shimatta）
 4. 忘れてしまった (Wasurete shimatta)
 5.解いてしまった（Toite shimatta）

 ## JLPT等级笔记

 在JLPT N4中，这种模式经常出现在听力部分，以识别说话者的**情绪状态**（遗憾）。 在阅读部分中，寻找它来了解角色是否完成了任务或者他们是否在哀叹错误。 记住：如果你看到〜ちゃう，那只是这个图案的休闲版！
