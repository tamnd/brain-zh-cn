##〜はずだ

 | 物业 | 价值|
 |----------|--------|
 | 图案| 〜はずだ |
 | 日本语能力考试等级 | N4 |
 | 类型 | 期待|
 | 正式| 中性|
 | 注册 | 两者 |

 ## 含义

 **〜はずだ** (hazu da) 模式表达基于理由或证据的强烈期望或逻辑结论。 它翻译为“应该”、“应该”或“预计如此”。 当说话者由于他们所掌握的信息而确信某件事是真实的或将会发生时，就会使用它。 

## 结构式```
[Verb dictionary form] + はずだ
[Verb た-form] + はずだ
[い-adj] + はずだ
[な-adj] + な + はずだ
[Noun] + の + はずだ
```* **动词：** 使用字典形式表示现在/将来的期望，使用过去 (た) 形式表示过去的期望。 
* **い-形容词：** 直接附加到形容词上。 
* **な形容词：** はずだ之前必须跟**な** (na)。 
* **名词：** はずだ之前必须跟**の**（否）。 

## 详细说明

 **〜はずだ** 当你的信念有逻辑基础时使用。 与简单的猜测不同，这种模式意味着您有理由认为某件事是这样的。 例如，如果您知道火车定于 8:00，您会说“8时に来るはずだ”（应该在 8:00 到达）。 

细微差别是“逻辑确定性”之一。 如果您对自己的预期错误感到惊讶，您可以使用过去时 **〜はずだった** (hazu datta)，意思是“我预计 [it will be/happen]，but...”

 这种模式在形式上是中性的。 在礼貌用语中，使用 **〜はずです** (hazu desu)。 在随意的演讲中，你可以去掉末尾的 **だ** (da)，但保留它会强调你的信念。 

需要注意的是，**〜はずだ**一般用于客观期望。 当谈论自己的内心感受或主观愿望时，其他模式如*〜つもりだ*（tsumori da）更合适。 

## 例句

 | 日语 | 阅读| 罗马字 | 英语 |
 |----------|---------|--------|---------|
 | 彼はもうすぐ来るはずだ。 かれはもうすぐくるはずだ。 Kare wa mō sugu kuru hazu da。 | 他应该很快就会来。 |
 | 键は机の上にあるはずです。 かぎはつくえのうえにあるはずです。 Kagi wa tsukue no ue ni aru hazu desu。 | 钥匙应该在桌子上。 |
 | 昨天は休みだったはずだ。 きのうはやすみだったはずだ。 Kinō wa yasumi datta hazu da。 | 昨天本来应该休息一天的。 |
 | この问题は简単だったはずだ。 このもんだいにかんたんだったはずだ。 Kono mondai wa kantan datta hazu da。 | 这个问题本来应该很简单。 |
 | 彼女は学生のはずです。 かのじょはがくせいのはずです。 Kanojo wa gakusei no hazu desu。 | 她应该是一名学生。 |
 | 电车は10分前に出たはずだ。 でんしゃはじゅっぷんまえにでたはずだ。 Densha wa juppun mae ni deta hazu da。 | 火车本应在 10 分钟前出发。 |
 | 彼は忙しいはずだ。 かれはいそがしいはずだ。 Kare wa isogashii hazu da。 | 他应该很忙。 |

 ## 与相似模式的比较

 ### 〜はずだ vs 〜にちがいない (ni chigainai)

 | 图案| 细微差别|
 |---------|--------|
 | **〜はずだ** | 基于证据的逻辑预期。 |
 | **〜にちがいない** | 强烈的主观信念/确定性。 |

 * **〜はずだ:** “根据时间表，商店应该开门。” （逻辑）
 * **〜にちがいない:** “他在出汗；他一定很紧张。” （强烈的个人结论）

 ### 〜はずだ vs 〜つもりだ (tsumori da)

 * **〜はずだ** 是对世界或他人的客观期望。 
* **〜つもりだ**是您自己的意图或您对自己行为的主观信念。 
* *示例：*“行くはずだ”（预计他会去）与“行くつもりだ”（我打算去）。 

## 常见错误

 ❌ 彼は学生はずだ。 → ✓ 彼は** 学生のはずだ**。 
* *更正：*名词必须在はず前用**の**连接。 

❌ 彼は忙しいなはずだ。 → ✓ 彼は** 忙しいはずだ**。 
* *更正：* い形容词在はず之前不要带“na”。 

❌ 私は明日行くはずだ。 → ✓ 私は明日**行くつもりだ**。 
* *更正：* 使用 *tsumori* 表示您自己的意图，而不是 *hazu* （这意味着外部期望）。 

## 练习练习

 **填空：**

 1. 彼は今日（来る）______。 
2. この店は（有名）______はずです。 
3. 昨日は（雨）______はずだったのに、晴れていた。 
4. 彼は（先生）______はずだ。 
5. この料理は（おいしい）______はずだ。 

**答案关键：**
 1. **来るはずだ** （根据日程预计）
 2. **有名な**（な-形容词需要“na”）
 3. **雨だった**（过去时期望）
 4. **先生の** （名词需要“不”）
 5. **おいしい** (い-adj 直接连接)

 ## JLPT等级笔记

 在JLPT N4中，你经常会在阅读理解或句子排序任务中看到这一点。 寻找诸如“时间表”、“通常”或“我听说……”之类的关键词，它们为期望提供了“证据”。 如果您看到以“我打算...”开头的句子，请避免选择 *hazu*。
