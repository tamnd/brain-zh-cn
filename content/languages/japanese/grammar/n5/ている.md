 ## 含义

 〜ている 根据动词类型有两种主要解释。 

### 1.持续行动（渐进）

 与动作动词一起使用来表达当前正在进行的动作——相当于英语中的“-ing”。 

- 食べている — 正在吃（吃的动作仍在发生）
 - 走っている — 正在奔跑

 ### 2. 结果状态

 与描述状态变化的动词一起使用。 动作已经完成，ている表示由此产生的状况。 

- 结婚している — 已结婚（结婚的结果）
 - 知っている — 知道（学习/了解的结果）
 - 死んでいる — 死了（死亡的结果）

 ## Te-Form 构成提醒（第 1 组动词）

 | 动词结尾 | Te 形式变更 | 示例|
 |-------------|----------------|---------|
 | う、つ、る | → って | 买う → 买って |
 | む、ぶ、ぬ | → んで | 饮む → 饮んで |
 | く | → いて | 书く → 书いて |
 | ぐ | → いで | 泳ぐ → 泳いで |
 | す | → して | 话す → 话して |

 第2组动词：去掉る，加て（食べる → 食べて）。 
不规则：する → して、くる → きて。

 ## 仅使用结果状态含义的动词

 有些动词描述一次性的变化而不是正在进行的动作，所以ている总是指结果状态，而不是仍在进行中的过程。 

| 动词| ている 的意思 |
 |------|----------------|
 | 知る（了解）| 知っている — 知道 |
 | 结婚する（结婚） | 结婚する 结婚している — 已结婚 |
 | 死ぬ（死）| 死んでいる — 死了 |
 | 着る（穿衣服）| 着ている — 穿着 |
 | 住む（定居/搬入）| 住んでいる — 生活（在一个地方）|

 ---

 ## 随意收缩：ている → てる

 在日常谈话中，い经常被省略。 

- 食べている → 食べてる
 - 知っている → 知ってる
 - 何してる？ — 你在做什么？ 

这种缩写在日常对话中极为常见，但在正式写作或演讲中应避免。 

---

 ## 常见错误

 - **使用简单形式而不是te形式：** ✗ 食べいる → ✓ 食べている
 - **将结果状态与过去时混淆：** 结婚している并不意味着“结婚了”——它的意思是“（目前）已婚”。 使用“结婚した”来表示已完成的事件。 
- **忘记写作中的い：** てる在随意的演讲中没问题，但在正式场合总是写ている / ています。 
- **将渐进意义应用于状态变化动词：** 知っている并不意味着“即将知道”——它只是意味着“知道”。
