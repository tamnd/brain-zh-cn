 ## 基本旅行短语

 | 日语 | 阅读| 英语 |
 |----------|---------|---------|
 | 〜はどこですか？ | 〜はどこですか | ～在哪里？ |
 | 〜までいくらですか？ | 〜までいくらですか | 多少钱~？ |
 | 〜行きの电车はどれですか？ 〜ゆきのでんしゃはどれですか | 哪趟车去~？ |
 | ここで止まりますか？ | ここでとまりますか | 就停在这里吗？ |
 | 次は何駅ですか？ つぎはなんえきですか | 下一站是哪站？ |
 | 迷いました。 | まよいました | 我迷路了。 |
 | 助けてください。 | たすけてください | 请帮我。 |
 | | 日本语があまり分かりません。 にほんごがあまりわかりません | 我不太懂日语。 |
 | 英语を话せますか？ えいごをはなせますか | 你能说英语吗？ |
 | 写真を撮っていいですか？ しゃしんをとっていいですか | 我可以拍张照片吗？ |

 ## 例句

 - 新宿駅はどこですか？新宿站是吗？ — 新宿站在哪里？ 
- 东京まで新干线で何时间かかりますか？东京制作新干线de nanjikan kakarimasu ka？ — 乘坐新干线到东京需要多少小时？ 
- 次の电车は何时に来ますか？Ttsugi no Densha wa nanji ni kimasu ka? — 下一趟火车几点到？ 
- 地図を见せてください。Chizu wo misete kudasai。 — 请给我一张地图。 
- ホテルまでタkushiーでいくらかかりますか？Hoteru 制作了 takushii de ikura kakarimasu ka？ — 乘坐出租车到酒店要多少钱？
