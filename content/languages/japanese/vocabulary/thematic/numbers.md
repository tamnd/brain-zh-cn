日语数字很简单，但计数器系统（针对不同类别的物体使用不同的计数词）是该语言的独特挑战之一。

 **与英语的主要区别**：日语以 10,000（万人）为单位对数字进行分组，而不是 1,000。 所以 100,000 = “十万”，1,000,000 = “百万”。 

## 日语本土数字 (Hitotsu–Toono)

 用于没有特定计数器的一般计数（1-10 个物体）：

 | 数量 | 日语 | 罗马字 |
 |--------|----------|--------|
 | 1 | 一つ | 一津 |
 | 2 | 二つ | 五达 |
 | 3 | 三つ| 三津 |
 | 4 | 四つ| 四津 |
 | 5 | 五つ| 逸 |
 | 6 | 六つ | 陆津 |
 | 7 | 七つ | 七日 |
 | 8 | 八つ | 八津 |
 | 9 | 九つ | 古古津 |
 | 10 | 10 十 | 也是|

 ## 计数器系统

 日语对不同类别的物体使用不同的“计数器”（助数词，josushi）。 在数字后附上适当的计数器。 

### 〜本 (hon) — 长圆柱形物体

 | 日语 | 罗马字 | 对于|
 |----------|--------|-----|
 | 一个| 日本 | 铅笔、钢笔、瓶子、树木、道路、电影（电影）|
 | 二本 | 日本| |
 | 三本 | 三邦| |
 | 四本| 永洪 | |
 | 五本 | 戈本| |

 例如：ペン一本（一支笔）、ビール二本（两瓶啤酒）、木三本（三棵树）

 ### 枚〜 (mai) — 扁平物体

 | 日语 | 罗马字 | 对于|
 |----------|--------|-----|
 | 一枚 | 一迈 | 纸张、票据、邮票、盘子、薄的东西|
 | 二枚 | 尼迈 | |
 | 三枚 | 三麦| |

 例如：切手一张（一张邮票）、纸二张（两张纸）、Tshiャツ三张（三件T恤）

 ### 〜册 (satsu) — 装订书

 | 日语 | 罗马字 | 对于 |

 | 二册 | 日津 | |
 | 三册 | 三萨| |

 ### 〜番 (ban) — 序数 (数字位置)

 | 日语 | 罗马字 | 意义|
 |----------|--------|---------|
 | 一番| 一番| 1号； 第一的; 最|
 | 二番| 尼班 | 2号； 第二 |
 | 三番| 三板| 3号； 第三|

 作为副词，一番也有“最好/最”的意思：一番好き = 最喜欢
