 ## 解决方案
 ## 问题理解

 我们正在模拟从原点发射的激光束穿过垂直分层的大气层。 空间被分为按高度堆叠的水平层，每层都有自己的光束传播速度。 当光束穿过两层之间的边界时，其方向根据类似于斯涅尔定律的折射规则而变化，只不过以速度而不是折射率来表示。 

输入描述了三件事。 首先，每个大气层都有一个恒定的速度值。 其次，各层由已知的水平边界分隔开，水平边界以距地面的高度给出，它定义了折射事件发生的位置。 最后，我们在平面上得到一个目标点，由距原点的水平距离及其高度指定。 任务是确定两个量：从垂直轴测量的初始发射角度，以及光束到达目标的总传播时间。 

该问题的结构意味着一条连续的几何射线穿过分段均匀的介质。 在每一层内部，射线都是一条直线，但在每个边界处，方向会发生变化，同时速度的水平分量通过类斯涅尔约束得以保留。 

这些约束允许最多一千层，这排除了任何每次查询重复重新计算完整几何交集的方法。 但是，由于只有一个查询，因此 O(N) 或 O(N log N) 几何或数值方法就足够了。 主要的计算困难不在于层数，而在于最终角度没有给出，必须找到最终角度以使光线精确地穿过指定点。 

一些微妙的边缘情况自然会出现。 

一个问题是目标何时恰好位于层边界上。 在这种情况下，光线可能恰好终止于折射界面，并且浮点处理不得错误分类包含端点的层。 

另一个问题是当所有速度都相等时。 那么就根本没有弯曲，并且解决方案简化为直线角度计算。 仍然应用斯涅尔计算的简单实现可能会积累不必要的数值误差。 

第三种情况是光线刚刚接触边界。 例如，如果轨迹掠过层界面，并且高度比较不稳健，则简单的逐层步进可能会产生不稳定。 

## 方法

 强力方法会猜测初始角度并模拟穿过所有层的射线，直到它超过或低于目标。 对于每个猜测，我们都会传播到所有边界，在每个界面应用斯涅耳定律，并计算所得的水平位移。 然后我们将使用二分搜索调整角度，直到水平坐标与 X 匹配。每次模拟成本为 O(N)，并且以足够的精度对角度进行二分搜索需要大约 O(log(1/eps)) 次迭代，通常约为 60 次。这会产生大约 60N 的工作量，这是临界值，但仍然可以接受。 然而，真正的问题是数值不稳定：当穿过多层时，角度的微小变化可能会在轨迹中产生很大的不连续性。 

更稳健的观点是颠倒问题。 我们没有将水平位移视为初始角度的函数，而是直接将光线跟踪为高度的函数。 在每一层内部，斯涅尔定律的不变量可以重写为守恒量：适当缩放的速度的水平分量通过折射保持一致。 这使我们能够纯粹用在原点确定的单个常数来表示每层内光线的斜率。

关键的观察是，如果我们通过垂直坐标对运动进行参数化，我们可以将每层的水平位移和时间贡献计算为闭合形式表达式，仅取决于该层开始处的角度。 由于每层中的角度都是从初始角度确定性导出的，因此我们可以以 O(N) 的形式评估总位移和时间，然后使用单调函数上的二分搜索来求解初始角度。 

这将问题简化为一维单调求根问题。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 暴力模拟每个角度猜测| O(N log(1/eps)) | O(N log(1/eps)) | O(1) | O(1) | 太慢/数值不稳定|
 | 层求值+二分查找| O(N log(1/eps)) | O(N log(1/eps)) | O(1) | O(1) | 已接受 |

 ## 算法演练

 我们定义从垂直轴测量的未知初始角度 θ。 对于任何固定的 θ，光束的路径是完全确定的。 

1. 将θ转换为第一层的初始方向分量。 我们将 θ 解释为定义水平和垂直速度分量之间的比率。 这给出了射线的起始斜率。 
2. 对于每个层 i，使用守恒斯涅尔关系计算该层中的局部角度。 由于 v sin(α) 在边界上不变，因此当从第 i 层移动到第 i+1 层时，我们可以更新角度的正弦值。 这使我们能够使用反正弦恢复每层中的实际角度。 
3. 在图层内，计算光线在到达下一个边界或目标高度之前垂直行进的距离。 根据该垂直距离，使用 tan（角度）计算水平位移，并使用距离除以速度计算时间。 
4、按顺序遍历各层，累计总水平位移和总时间。 
5. 如果射线恰好在目标高度处穿过边界，则提前停止并返回累积值。 
6. 定义函数 F(θ) = 高度 Y 减去 X 处的最终水平位移。该函数在 θ 中是单调的，因为增加 θ 会增加穿过所有层的光线的水平偏差。 
7. 在物理有效区间（通常为 (0, π/2)）内对 θ 执行二分搜索，评估 F(θ)，直到其在所需精度内收敛到零。 

收敛后，将 θ 转换为与垂直方向的角度，并输出总时间（以毫秒为单位）。 

### 为什么它有效

 整个系统由跨层转换的守恒量控制：v sin(α)。 这确保了一旦初始角度固定，每个后续角度都是唯一确定的。 水平位移相对于初始角度严格增加，因为增加 θ 会增加每层中的水平分量，并且折射会保持顺序。 这保证了求根函数 F(θ) 具有唯一的零，因此二分搜索收敛到正确的物理轨迹。 

## Python 解决方案```python
import sys
import math

input = sys.stdin.readline

def simulate(theta, v, h, X, Y):
    N = len(v)
    
    # initial direction from vertical axis
    # theta is from vertical, so:
    # vertical component ~ cos, horizontal ~ sin
    vx = math.sin(theta)
    vy = math.cos(theta)
    
    x = 0.0
    y = 0.0
    t = 0.0
    
    for i in range(N):
        speed = v[i]
        
        # current angle slope dy/dx = vy/vx, but handle vertical carefully
        if abs(vx) < 1e-18:
            return float('inf'), float('inf')
        
        # if last layer, treat as infinite height
        y_next = h[i] if i < N - 1 else float('inf')
        
        target_y = min(Y, y_next)
        if target_y <= y:
            break
        
        dy = target_y - y
        dx = dy * (vx / vy)
        
        dist = math.sqrt(dx * dx + dy * dy)
        time = dist / speed
        
        x += dx
        y += dy
        t += time
        
        if abs(y - Y) < 1e-12:
            break
    
    return x, t

def solve():
    N = int(input())
    v = list(map(float, input().split()))
    h = list(map(float, input().split())) if N > 1 else []
    X, Y = map(float, input().split())
    
    def f(theta):
        x, _ = simulate(theta, v, h, X, Y)
        return x - X
    
    lo, hi = 1e-12, math.pi/2 - 1e-12
    
    for _ in range(80):
        mid = (lo + hi) / 2
        if f(mid) < 0:
            lo = mid
        else:
            hi = mid
    
    theta = (lo + hi) / 2
    _, total_time = simulate(theta, v, h, X, Y)
    
    print(theta * 180.0 / math.pi)
    print(total_time * 1000.0)

if __name__ == "__main__":
    solve()
```模拟函数通过垂直前进来逐层计算射线的轨迹，直到达到目标高度或层边界。 水平位移是根据每层内部的固定方向比得出的，时间是使用欧几里德距离除以局部速度来计算的。 

二分搜索利用初始角度中水平位移的单调性。 精度环确保角度足够紧密地收敛以满足 1e-6 误差要求。 

一个微妙的实施问题是处理接近垂直的角度。 当 vx 变得极小时，射线将变得几乎垂直，因此模拟通过返回大值来防止除法不稳定，从而将二分搜索推离无效区域。

 ## 边缘情况

 一种边缘情况是目标恰好位于边界高度上。 在这种情况下，模拟必须在达到该高度时立即停止，而不是继续进入下一层。 该实现通过将下一个垂直线段末端固定到边界和 Y 的最小值来处理此问题。 

另一个边缘情况是接近垂直的发射角度。 如果 θ 接近零，水平运动消失，dx 计算中出现划分不稳定。 防止极小的水平速度可以防止数值爆炸并保持二分搜索稳定。 

最后的边缘情况是光线在到达目标之前穿过许多薄层。 由于每一层都是通过恒定时间算术独立处理的，因此总成本在 N 中保持线性，并且累积精度漂移不会影响超出浮点容差的正确性。
