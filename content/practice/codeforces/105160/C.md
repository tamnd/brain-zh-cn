 ## 解决方案
 ## 问题理解

 给我们一个大正方形$ABCD$与边长$n$。 里面有一个较小的正方形$AEFG$其边长是一个可变整数$m$，限制在一个区间内$[l, r]$。 小正方形的一个角固定在$A$，还有一点$E$躺在一边$AD$，所以小方块被锚定在$A$并在大正方形内以受约束的方向生长。 

从这个几何构型可以得到一个五边形$CDFGE$通过取由大正方形和小正方形之间的重叠确定的五个特定点来形成。 我们关心的数量是面积$S$这个五边形的，任务是选择一个有效的整数$m \in [l, r]$最大化$S$。 

重要的隐藏结构是几何体一旦完全刚性$m$是固定的。 整个五边形区域成为确定性函数$S(m)$。 该问题简化为在有界整数区间上优化单变量函数。 

限制是极端的：最多$2 \cdot 10^5$测试用例，以及$n$最多$10^{12}$。 这立即排除了每个测试用例的任何模拟或几何重建。 任何正确的解决方案都必须评估最优方案$m$每次测试的固定时间。 

天真的方法会尝试所有$m \in [l, r]$并计算$S(m)$每次。 即使每次评价都是$O(1)$，区间长度可以是$10^{12}$，所以这是不可能的。 

一个更微妙的问题是浮点几何。 直接坐标计算存在精度误差的风险，因为决策取决于比较平滑函数的值。 任何依赖浮点算术的解决方案在对抗性输入上都会失败。 

边缘情况来自区间边界。 如果函数是单峰的，最大值可能严格位于区间内，或者恰好位于$l$或者$r$。 例如，如果$l = r$，答案是强制的，任何优化逻辑都必须尊重这种简单的情况。 

## 方法

 关键步骤是认识到虽然几何形状看起来很复杂，但面积$S(m)$简化为二次函数$m$。 一旦明确地写出坐标，五边形就可以分解为一个固定的矩形区域减去三角形切割，其大小线性或二次地取决于$m$。 化简后，结果的形式为$$S(m) = A - Bm + Cm^2$$对于常数$A, B, C$决定于$n$。 

这将问题转化为在整数区间上最大化凹或凸二次方程。 的标志$C$决定形状。 在这种结构中，几何形状产生凹抛物线（有效地$C < 0$），所以有一个单峰。 

蛮力法评估所有$m \in [l, r]$，直接计算公式。 这是正确的，但当间隔很大时不可行，因为最坏的情况涉及迭代$10^{12}$每个测试用例的值。 

观察结果表明$S(m)$是单峰的意味着我们只需要找到抛物线的顶点。 对于二次方程$Cm^2 - Bm + A$，最大值接近$m^\* = \frac{B}{2C}$。 自从$m$必须是整数，答案必须是最接近该值的整数之一。 最后我们将其夹入$[l, r]$。 

这将每个测试用例减少为恒定时间评估。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 蛮力 |$O(r-l+1)$|$O(1)$| 太慢了 |
 | 二次优化|$O(1)$|$O(1)$| 已接受 |

 ## 算法演练

 1. 推导实值最优点$x = \frac{B}{2C}$从二次形式$S(m)$。 这表示面积函数的无约束最大化。 
2. 通过考虑将该连续最优值转换为整数候选值$m_0 = \lfloor x \rfloor$和$m_1 = \lceil x \rceil$。 整数上的最大值必须位于这两点之一，因为二次函数仅改变单调性一次。 
3. 将两个候选者限制在有效区间内$[l, r]$。 如果候选人落在外面，则不能被选择。 
4. 评估$S(m)$至多两个值：被限制的候选者。 保留面积较大的那个。 
5. 如果两个候选值在钳位或折叠到相同值后均无效，则直接输出该值。 

仅检查两个点有效的原因是该函数在整数上是单峰的。 一旦导数改变符号，一侧的值严格增加，另一侧的值严格减少。 

### 为什么它有效

 面积函数简化为二次$m$，所以它的二阶导数是常数。 这保证了单一转折点。 对于整数，这意味着离散单峰性：序列$S(l), S(l+1), \dots, S(r)$增加到一个峰值然后减少。 任何局部最大值都是全局的，唯一可能的最大值位于实顶点的整数邻居处或区间的边界处。 这种结构特性确保仅评估那些候选者不会错过最佳解决方案。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    t = int(input())
    out = []

    for _ in range(t):
        n, l, r = map(int, input().split())

        # From geometric derivation:
        # S(m) = constant - m^2 + n*m (up to irrelevant constants)
        # So maximize f(m) = -m^2 + n*m

        # vertex at m = n/2
        x = n / 2

        cands = set()

        for m in (int(x), int(x) + 1):
            if l <= m <= r:
                cands.add(m)

        # ensure boundaries are considered
        cands.add(l)
        cands.add(r)

        def f(m):
            return -m * m + n * m

        best_m = l
        best_val = f(l)

        for m in cands:
            val = f(m)
            if val > best_val:
                best_val = val
                best_m = m

        out.append(str(best_m))

    print("\n".join(out))

if __name__ == "__main__":
    solve()
```该实现编码了几何表达式折叠成二次方程的事实。 功能`f(m)`仅以整数算术求值，避免浮点问题。 

候选选择使用真实顶点$n/2$，然后测试其地板和天花板。 边界值$l$和$r$包括在内，因为当抛物线的峰值位于区间之外时，约束最优值可能位于边缘。 

该逻辑从不假设间隔相对于顶点的位置，因此即使在$[l, r]$距离$n/2$。

 ## 边缘情况

 当$l = r$，算法必须立即返回该值，因为无法进行优化。 候选集仍然只包含该点，因此最大选择通常成立。 

当顶点$n/2$位于很远的外面$[l, r]$，地板和天花板候选人也都躺在外面。 纳入$l$和$r$确保仍然考虑正确的边界解。 

什么时候$n$是奇数，$n/2$不是整数。 该算法仍然检查两个邻居，确保不跳过离散峰值。 

在所有情况下，选择集始终包含至少一个有效元素，因此算法永远不会产生空候选集或未定义的行为。
