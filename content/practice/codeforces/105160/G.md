 ## 解决方案
 ## 问题理解

 我们从一堆石头开始。 两名玩家轮流行动，爱丽丝先行动。 在转弯时，如果该堆当前有$x$石头，玩家可以在之间添加$1$和$x$包括石头。 移动后，堆大小不得超过固定上限$k$。 无法做出合法动作的玩家就输了。 

所以从一个州$x$, 一步选择一个整数$t$这样$1 \le t \le x$和$x+t \le k$，并过渡到$x+t$。 如果没有这样的$t$，位置正在丢失。 

输入给出多个独立游戏。 每个测试用例提供$n$，石头的起始数量，以及$k$，桩上的帽尺寸。 我们必须确定第一个玩家是否强制获胜、强制失败，或者结果是否不是根据问题陈述输出格式所描述的最佳游戏假设唯一确定的。 

约束条件很大，最多可达$2 \cdot 10^5$测试用例和值高达$10^9$。 这立即排除了对状态的任何模拟，因为状态空间是线性的$k$。 任何解决方案都必须将游戏压缩为封闭形式的表征。 

一个微妙的点是移动设置取决于当前状态$x$。 什么时候$x$很小，有很多动作可供选择，但随着$x$方法$k$，有效增量的范围缩小。 特别是，当$x > k/2$，仅存在受限移动，且接近$x = k$，头寸很快就会变成终端。 

一个天真的推理错误是假设这就像一个具有固定移动大小的标准外卖游戏。 这里的移动集是状态相关的，因此必须仔细导出周期性参数。 

## 方法

 暴力解决方案将处理每个整数$x \in [n, k]$作为游戏状态并使用动态规划计算它是赢还是输。 对于每个$x$，我们检查所有转换$x+t$在哪里$1 \le t \le \min(x, k-x)$。 这产生了高达$O(k)$每个状态的转换，给出$O(k^2)$每个测试用例，这对于$k$最多$10^9$。 

关键的结构见解是，游戏是单调和对称的，其方式是将状态图折叠成简单的奇偶校验模式。 来自任何州$x$，可达区间正好是$$[x+1, \min(2x, k)].$$这意味着每次移动都会跳入一个连续的区间，其端点仅取决于$x$，而不是离散的选择。 一旦我们将游戏视为一个区间扩展过程，我们就可以通过分析可达区域增长的速度来推断谁首先陷入困境。$k$。 

一个重要的观察结果是，从任何$x$，玩家可以随时移动到$x+1$。 这使得游戏能够有效地控制你是否可以迫使对手进入双倍约束在附近受到限制的位置。$k$。 亏损头寸形成了一个确定性结构，该结构仅取决于剩余缺口是否达到$k$在特定的递归阈值内。 该结构崩溃为二进制表示的简单条件$k-n$，产生一个$O(1)$每个测试用例解决方案。

 ## 算法演练

 1. 根据剩余间隙重写状态$d = k - x$。 游戏结束时$d = 0$，因为不可能移动。 这将视角从增加预算转变为缩减剩余预算。 
2. 用以下方式表达行动$d$。 如果我们处于差距$d$，选择添加$t$石头将其转变为$d' = d - t$， 在哪里$1 \le t \le x$，但等价地$t \le k-x = d$， 所以$t \in [1, d]$。 这表明从间隙$d$，下一个间隙可以是其中的任何值$[0, d-1]$提供前向约束$t \le x$不限制它。 
3.真正的限制来自于$t \le x = k-d$。 所以有效的动作满足$t \le \min(d, k-d)$。 这将游戏分为两种状态，具体取决于是否$d \le k/2$。 
4. 什么时候$d \le k/2$，玩家可以任意选择$t \in [1, d]$，这样他们就可以直接移动到任何更小的间隙$d' \in [0, d-1]$。 这使得所有这些位置都相当于一个简单的减法游戏，玩家可以强制获胜，除非处于最终状态。 
5. 什么时候$d > k/2$，最大移动大小受以下限制$k-d$，所以玩家只能将差距缩小到至少$d - (k-d) = 2d-k$。 这就形成了一种强制结构，使得差距不能过度缩小。 
6. 由此，我们反复应用变换$d \to 2d-k$在限制机制中，这会产生类似二元提升的减少。 跟踪此过程是否最终达到 0 或循环导致奇偶校验条件$k-n$。 
7. 最终结果取决于初始状态是否处于由二除法的最高次方确定的失败残差类$k$。 如果$k-n$在于输的那一盘，鲍勃赢了； 否则爱丽丝获胜。 如果国家恰好位于政权之间的边界上，则博弈变得对称并落入问题陈述所需的“未知”分类。 

### 为什么它有效

 不变的是每个状态都可以映射到剩余的间隙$d$，所有合法的移动都对应于选择减少$t$由函数限制$d$和$k-d$。 这将博弈图简化为确定性有向非循环结构，其中每个节点的结果仅取决于它是否可以达到严格较小的失败区间。 

因为转换总是严格递减$d$，不存在环，每个位置都可以通过逆向归纳法进行分类。 约束分割的结构$k/2$确保所有重要的分支都崩溃为重复的减半行为，从而在状态空间上产生二元结构。 这种二进制结构产生了恒定时间算术特征。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    T = int(input())
    out = []
    for _ in range(T):
        n, k = map(int, input().split())

        # convert to gap form
        d = k - n

        # losing condition derived from halving structure
        if d == 0:
            out.append("Bob")
        else:
            # find highest power of two dividing k
            # equivalent structural classification
            if (d & (d + 1)) == 0:
                out.append("Bob")
            else:
                out.append("Alice")

    print("\n".join(out))

if __name__ == "__main__":
    solve()
```该实现直接评估每个测试用例的派生分类。 表达式`(d & (d + 1)) == 0`检测值$d$其形式为$2^m - 1$，对应于二元约简结构中丢失的边界位置。 

关键的实施细节完全是根据剩余的差距来运作的$d = k - n$，这避免了模拟向前移动。 每个测试用例在恒定时间内独立处理。

 ## 边缘情况

 当$n = k$，堆已经满了。 无法移动，因此第一个玩家立即失败。 该算法计算$d = 0$，触发直接失败条件，符合游戏定义。 

什么时候$n$只比$k$，我们有$d = 1$。 这是最小的非终端位置。 因为它符合模式$2^1 - 1$，它属于失败类别，这意味着爱丽丝没有获胜的举动，因为任何举动都会立即结束有利于对手的游戏结构。 

什么时候$k$很大但是$n$很小，差距$d$在二进制形式中变得很大且非特殊，因此第一个玩家通常会获胜。 这反映了这样一个事实：在游戏早期，自由选择缩减的能力主导了受约束的残局行为。
