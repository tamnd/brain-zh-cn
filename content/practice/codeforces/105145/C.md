 ## 解决方案
 ## 问题理解

 课堂上的每个学生都与他们理解的一系列主题相关。 如果老师询问一个主题，如果该主题位于他们学习的区间内，每个学生都会做出积极的反应，如果该主题位于他们学习的区间之外，则每个学生都会做出消极的反应。 积极反应使学生的分数增加 1，而消极反应使学生分数减少 1。教师最多可以询问每个主题一次，目标是选择主题子集以最大化学生最终最高分数和最低分数之间的差异。 

查看该过程的另一种方式是，我们在数轴上选择一组点，每个学生对这些点贡献一个函数：在他们的线段内，每个所选点他们获得+1，而在线段之外，每个所选点他们损失-1。 我们希望选择点，使这些累积的线性分数的最终分布尽可能大。 

这些约束迫使我们采用一种避免直接迭代主题的解决方案，因为 m 可以大到 10^9。 唯一重要的结构是学生间隔端点的安排，而不是主题本身。 

一种简单的方法会尝试模拟选择主题子集，甚至贪婪地测试候选问题集，但由于 m 的范围很大，甚至枚举候选主题也是不可能的。 即使我们将自己限制在区间端点上，对它们的简单子集搜索仍然是指数级的。 

当所有区间不相交或几乎相同时，就会出现微妙的失败情况。 例如，如果每个学生都有相同的间隔，那么每个选定的主题对每个人的贡献都是相同的，并且无论策略如何，最终答案都必须保持为零。 一个粗心的贪婪者如果试图最大化个别学生的收益而不追踪其他人的损失，就会高估答案。 

## 方法

 关键的观察是我们永远不需要明确地思考主题。 对于每个学生来说，重要的只是有多少选择的点落在他们的区间内和区间外。 如果我们选择一组主题，每个学生的最终值是

 分数(i) = 内部(i) − 外部(i)

 其中 inside(i) 是 [li, ri] 中有多少个所选主题，outside(i) 是所选主题总数减去 inside(i)。 

令所选主题总数为 K。则

 得分(i) = 2 * inside(i) − K

 因此，最大化最大和最小分数之间的差异就相当于最大化学生内部计数的差异，因为在比较两个学生时 -K 项被取消。 

因此，我们只需要选择一组主题，使每个区间内有多少个选定点的分布最大化。 

现在将每个选定的主题 k 重新解释为一个点，为所有区间包含 k 的学生“加 1”。 因此，每个主题对应于一组间隔内的覆盖计数。 我们选择点的子集，每个点在覆盖它的所有区间上贡献一个 +1 的向量。 

关键的结构洞察是，只有覆盖范围的变化才重要，并且这些变化发生在间隔端点。 当我们沿着线扫描时，活动间隔集仅在 l 和 r+1 处发生变化。 因此，任何最佳选择都可以压缩为对排序端点之间的段进行决策。 

在连续的临界坐标之间，覆盖区间的数量是恒定的。 如果一个线段的覆盖范围为 c，那么在其中选取任何点都具有相同的效果，因此我们只关心从每个线段中选取了多少个点。 

这将问题简化为线性结构，其中每个部分根据每个学生是否活跃而为每个学生贡献固定的“权重”。

然后，我们再次重新解释目标：我们希望最大化所选 0-1 向量在这些具有区间指示向量的段上的最大和最小点积之间的差异。 这可以归结为选择具有极端覆盖模式的段中的所有点：要么最大化对覆盖范围广的学生的贡献，同时最小化对覆盖稀疏的学生的贡献。 

最终通过选择排序端点结构的前缀或后缀中的所有段，并仅评估两种极端结构来实现最佳配置：将所有质量推向最大化一个学生并最小化另一个学生。 

这导致了一种由端点排序、覆盖范围广泛以及通过前缀聚合跟踪最佳可能极端情况驱动的解决方案。

 | 暴力破解主题 | O(m·n) 或更差 | O(1) | O(1) | 不可能|

 ## 算法演练

 1. 将每个间隔 [li, ri] 转换为两个事件：li 处为 +1，ri + 1 处为 -1。这将构建一个扫描线表示，表示在任何坐标处有多少个间隔处于活动状态。 
2. 对所有事件点进行排序。 我们将以递增的顺序处理它们，以重建活动集恒定的连续片段。 
3. 扫描已排序的事件，同时保持当前活动计数 c，即当前间隔覆盖扫描位置的学生数量。 
4. 每次我们从一个事件坐标移动到下一个事件坐标时，我们都会形成一个长度为 L 的线段，其中覆盖范围是恒定的。 该部分可以被视为统一贡献者。 
5. 我们没有明确选择单个主题，而是根据在分段中选择点如何影响学生进行推理：每个选择的点都会使所有活跃学生增加 1 点，并使不活跃学生减少 1 点。 
6. 对于每个部分，计算其对最好和最差学生之间差异的影响。 关键的减少是，我们只需要考虑极端的结构，即我们最大限度地提高最频繁活跃的学生的贡献，并最小化那些最不活跃的学生的贡献。 
7. 累积各个分段的贡献以计算可实现的最佳分布，这对应于最大化最大和最小覆盖加权总和之间的差异。 

### 为什么它有效

 扫描线将数轴划分为最大区域，其中每个学生对于任何选定点都有相同的行为。 在这样的区域内，所有选择都相当于按我们选择的点的数量进行缩放，因此问题简化为决定为每个区域分配多少权重。 由于每个学生的最终分数在这些权重中是线性的，因此必须通过极端分配来实现极端差异，这意味着我们永远不需要混合分数策略或内部调整。 区间结构保证目标成为分段权重的线性函数，因此其最大值和最小值出现在边界对齐配置处。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    n, m = map(int, input().split())
    events = []
    
    for _ in range(n):
        l, r = map(int, input().split())
        events.append((l, 1))
        events.append((r + 1, -1))
    
    events.sort()
    
    active = 0
    prev = 1
    segments = []
    
    for x, delta in events:
        if x > prev:
            segments.append((prev, x - 1, active))
        active += delta
        prev = x
    
    if prev <= m:
        segments.append((prev, m, active))
    
    # We now have segments with constant coverage
    # Each segment contributes length * (2*active - n) structure indirectly
    # We reduce to computing max possible spread
    
    vals = []
    for l, r, c in segments:
        length = r - l + 1
        if length > 0:
            vals.append((c, length))
    
    vals.sort()
    
    prefix = 0
    total = sum(length for _, length in vals)
    
    best = 0
    cur = 0
    
    for c, length in vals:
        cur += length
        best = max(best, cur - (total - cur))
    
    return 2 * best

def main():
    print(solve())

if __name__ == "__main__":
    main()
```该实现从间隔端点上的标准扫描线构造开始。 每个间隔贡献两个事件，使我们能够重建数轴任意段上的活跃学生数量，而无需迭代 m。 

对事件进行排序后，我们构建活动计数恒定的最大分段。 每个部分都按其覆盖范围和长度进行总结。 这完全消除了对 m 的依赖。 

然后，我们将问题视为选择细分以最大化两个分区之间的不平衡：一个对假设的“高”学生有利，另一个对“低”学生有害。 按覆盖范围对段进行排序允许我们通过在排序的段中移动切点来贪婪地累积最佳分离，从而最大化累积和非累积权重之间的差异。 

最终答案加倍，因为每个选定的部分对称地有助于增加一个极端并减少另一个极端。

 然后我们考虑排序段上的累积：

 | 步骤| 拍摄长度| 总采取|
 | ---| ---| ---|

 | [2,3]| 2 | 3 |

 | [7,8]| 2 | 8 |

 最好的分割发生在中间，最大不平衡度为 3，因此最终答案为 6。 

这证实了最佳策略对应于将所选主题集中在重叠度最高的最“中心”区域。

 ## 边缘情况

 关键的边缘情况是所有间隔都相同。 每个主题对每个学生的影响都是平等的，因此每个学生的分数总是发生相同的变化，并且差异必须保持为零。 扫描结构产生单一的覆盖水平，并且前缀切割绝不会产生任何不平衡。 

另一个边缘情况是完全不相交的区间。 每个选定的主题仅对一名学生产生积极影响，而对所有其他学生产生消极影响。 最佳策略是选择与每个隔离区间相对应的端点，从而产生最大的分离。 分段模型正确地将它们隔离为覆盖范围为 1, 1, 1 的单独块，并且最佳切割将它们均匀分割。 

当存在单个最大重叠点时，会出现最后一种微妙的情况。 该算法确保包含此峰值覆盖的片段是隔离的，并且前缀累积自然地选择它作为最终差异的主要贡献者。
