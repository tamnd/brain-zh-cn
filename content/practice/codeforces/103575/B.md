 ## 解决方案

 让字母表成为${x_1 < x_2 < \cdots < x_t}$具有多重性$n_1,\ldots,n_t$和$\sum_{i=1}^t n_i = n$。 算法 L 相对于这个有序字母表按照严格的字典顺序生成排列。 排列的等级$a_1 \ldots a_n$因此 是按字典顺序排列更小的不同多集排列的数量。 

修复前缀$a_1 \ldots a_{i-1}$。 在位置$i$，假设剩余的可用重数是$m_1,\ldots,m_t$总计$m = n-i+1$。 如果一个符号$x_k$被放置在位置$i$，完成次数为多项式系数$$\frac{(m-1)!}{m_1!\cdots (m_k-1)!\cdots m_t!}.$$总结一下这一切$x_k < a_i$给出职位的贡献$i$到排名。 这正是适用于重复符号的标准多项式莱默码。 

现在考虑排列$$314159265.$$底层有序符号是$$1 < 2 < 3 < 4 < 5 < 6 < 9,$$具有多重性$$n_1 = 2,\quad n_5 = 2,\quad n_2 = n_3 = n_4 = n_6 = n_9 = 1.$$我们逐步计算排名。 

在$a_1 = 3$, 符号小于$3$是$1$和$2$。 

如果$1$被放置在第一位，剩余的重数给出一个计数$$\frac{8!}{2!\,1!\,1!\,2!\,1!\,1!} = \frac{40320}{2} = 20160.$$如果$2$放在第一位，出现相同的分母，给出另一个$20160$。 因此第一个位置贡献$40320$。 

修复后$3$，剩余的多重集是$$\{1^2,2,4,5^2,6,9\}.$$在$a_2 = 1$, 没有小于的符号$1$，所以贡献是$0$。 

在$a_3 = 4$，其余符号小于$4$是$1$和$2$。 

如果$1$放置后，剩余的多重集具有大小$6$只有一个重复的$5$, 给予$$\frac{6!}{2!} = 360.$$如果$2$放置时，会出现相同的值，因此贡献为$720$。 

在$a_4 = 1$，没有贡献产生。 

加工后$a_5 = 5$，剩余的多重集是${2,6,9}$连同另外一项$5$已经占了，并且符号小于$5$是$1$和$2$。 

如果$1$放置后，其余三个符号是不同的，有助于$3! = 6$。 

如果$2$被放置，出现相同的值，给予贡献$12$，因此总共$48$。 

在$a_6 = 9$, 符号小于$9$在剩余的多重集中${2,5,6,9}$是$2$,$5$， 和$6$。 

每个选择都会留下三个不同的符号，贡献$3! = 6$，因此总共$18$。 

在$a_7 = 2$，没有更小的可用符号起作用。 

在$a_8 = 6$，唯一较小的可用符号是$5$，并放置它留下一个完成，贡献$1$。 

在$a_9 = 5$，没有进一步的贡献发生。 

总结所有贡献，$$40320 + 720 + 48 + 18 + 1 = 41107.$$军衔为$314159265$因此在算法 L 下是$$\boxed{41107}.$$
