 ## 解决方案
 ## 问题理解

 我们在扩展复平面上给出了三个输入输出点对，其中每个点都是一个由其实部和虚部表示的复数。 这些对定义了以下形式的独特线性分数变换$f(z) = \frac{az + b}{cz + d}$，复系数满足$ad - bc \neq 0$。 一旦它对三个不同点的作用固定，这样的函数就完全确定了。 

对于每个测试用例，我们知道这个转换如何映射三个复数$z_1, z_2, z_3$三个复数值$w_1, w_2, w_3$，我们被要求计算第四个点的图像$z_0$在同样的变换下。 

输入将每个复数编码为两个整数，因此每个测试用例为三个映射提供十二个整数，为三个映射提供两个整数$z_0$。 输出是一对实数，代表实部和虚部$f(z_0)$，具有非常高的精度要求。 

约束主要由测试用例的数量决定，最多$10^5$。 这立即排除了任何涉及使用朴素高斯消元法和重常数因子求解一般四变量线性系统的每个测试用例的方法，或涉及扩展形式的复杂有理表达式的符号操作的任何方法。 该解决方案必须将每个测试用例的算术工作量减少到恒定。 

一个微妙的问题是数值稳定性。 直接扩展$\frac{az+b}{cz+d}$解决后$a,b,c,d$可以累积浮点误差。 另一个危险是尝试显式计算$a,b,c,d$即使所有输入都是小整数，也可以通过行列式重复形成大的中间复数。 预期的解决方案避免了明确地求解系数。 

朴素推理的一个典型失败案例是尝试计算$a,b,c,d$通过根据三个约束加上归一化求解四个方程。 尽管可以通过利用交叉比率的不变量来更清晰地计算变换，但这会引入不必要的代数复杂性和浮点不稳定性。 

## 方法

 线性分数变换保留交叉比。 对于任意四个不同的点$z, z_1, z_2, z_3$，我们有$$\frac{(z - z_1)(z_2 - z_3)}{(z - z_3)(z_2 - z_1)}
=
\frac{(f(z) - f(z_1))(f(z_2) - f(z_3))}{(f(z) - f(z_3))(f(z_2) - f(z_1))}$$该恒等式完全表征了转换行为，而无需引入$a,b,c,d$。 

蛮力方法可以明确解决$a,b,c,d$使用三个已知映射加上归一化约束，例如设置$d = 1$或固定比例。 这导致了一个由三个具有三个未知数的复杂方程组成的系统。 解决这个问题需要重复复杂的算术和除法，并且对于$10^5$测试用例中，即使常数因子的低效率也会变得显着。 更重要的是，当分母变小时，就会出现浮点不稳定。 

关键的观察是，我们从不需要变换本身，只需要它在某一点上的作用。 交比恒等式给出了未知数中的直接方程$w_0 = f(z_0)$，可以在每个测试用例的恒定时间内用代数方法求解。 

重写交叉比率方程可得到有理式$w_0$，清除分母后可以重新整理成线性方程。 这避免了求解任何系统，并将一切简化为复杂算术运算的固定序列。

 | 交比直接计算| O(T)| O(1) | O(1) | 已接受 |

 ## 算法演练

 我们表示$z_i$和$w_i$作为复数并以复数形式计算所有算术。 

1. 计算交比系数$$k = \frac{(z_0 - z_1)(z_2 - z_3)}{(z_0 - z_3)(z_2 - z_1)}$$这是必须与图像空间中的相应表达式匹配的不变值。 
2. 用未知数表达相同的不变量$w_0$:$$k = \frac{(w_0 - w_1)(w_2 - w_3)}{(w_0 - w_3)(w_2 - w_1)}$$3. 叉乘消除分母：$$k (w_0 - w_3)(w_2 - w_1) = (w_0 - w_1)(w_2 - w_3)$$4. 展开两边但保持表达式分组$w_0$:$$k(w_2 - w_1)w_0 - k(w_2 - w_1)w_3 = (w_2 - w_3)w_0 - (w_2 - w_3)w_1$$5. 移动所有涉及的术语$w_0$到一侧：$$(k(w_2 - w_1) - (w_2 - w_3))w_0 = k(w_2 - w_1)w_3 - (w_2 - w_3)w_1$$6. 求解所得线性方程：$$w_0 = \frac{k(w_2 - w_1)w_3 - (w_2 - w_3)w_1}{k(w_2 - w_1) - (w_2 - w_3)}$$每个步骤都会减小问题的大小，而不会引入额外的未知结构。 最终的表达式是直接求值。 

### 为什么它有效

 线性分数变换保留交叉比，因为它正是将圆和直线映射到圆和直线同时保持投影结构的变换组。 一旦三个点固定，任何第四个点与这三个点之间的交叉比在变换下是不变的。 这个不变量产生一个复杂的方程$w_0$，并且由于变换在扩展平面上是双射的，因此该方程具有唯一解。 推导崩溃了所有的自由度$a,b,c,d$转化为单一理性恒等式，保证计算值的正确性。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    T = int(input())
    out = []

    for _ in range(T):
        p1, q1, r1, s1 = map(int, input().split())
        p2, q2, r2, s2 = map(int, input().split())
        p3, q3, r3, s3 = map(int, input().split())
        p0, q0 = map(int, input().split())

        z1 = complex(p1, q1)
        z2 = complex(p2, q2)
        z3 = complex(p3, q3)
        z0 = complex(p0, q0)

        w1 = complex(r1, s1)
        w2 = complex(r2, s2)
        w3 = complex(r3, s3)

        num_k = (z0 - z1) * (z2 - z3)
        den_k = (z0 - z3) * (z2 - z1)
        k = num_k / den_k

        num = k * (w2 - w1) * w3 - (w2 - w3) * w1
        den = k * (w2 - w1) - (w2 - w3)
        w0 = num / den

        out.append(f"{w0.real:.10f} {w0.imag:.10f}")

    print("\n".join(out))

if __name__ == "__main__":
    solve()
```实现直接遵循导出的封闭形式。 每个测试用例从整数输入构建​​复数，计算不变量$k$，然后计算最终的有理式。 主要的微妙之处是确保运算分组与代数推导相匹配，从而最大限度地减少浮点消除。 使用 Python 内置的复杂算术可以使代码足够短且稳定，以满足给定的精度要求。

 ## 边缘情况

 一种重要的边缘情况是当点被排列时中间差异变得非常小，例如当$z_0$非常接近$z_1$或者$z_3$。 在这种情况下，系数的简单扩展会放大浮点误差。 交比形式通过保留分子和分母的对称结构来避免这种情况。 

另一种情况是变换的行为类似于简单的反转。 如果$w_1 = 1/z_1$,$w_2 = 1/z_2$,$w_3 = 1/z_3$，那么公式应该精确地减少到$w_0 = 1/z_0$。 插入算法后，分子和分母都简化为一致的倒数，无需特殊的大小写即可产生正确的值。 

最后一种情况是当$z_0, z_1, z_2, z_3$在复平面中几乎共线，这可以使交比值变得很大。 代数公式仍然可以正确处理这个问题，因为所有放大在分子和分母中对称出现，并在最终除法之前取消。
