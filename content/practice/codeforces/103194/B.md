 ## 解决方案
 ## 设置

 3-多重组合${0,1,\dots,n-1}$是一个弱增三元组$d_1 \le d_2 \le d_3,\qquad 0 \le d_i \le n-1.$万能循环是一个循环串$a_0 a_1 \dots a_{L-1}$超过${0,\dots,n-1}$这样每个长度为 3 的循环窗口$(a_i,a_{i+1},a_{i+2})\quad (\bmod L)$按非递减顺序排序后只出现一次，并且每个 3-multi 组合只出现一次。 
3-多组合的数量为$\binom{n+2}{3} = \frac{n(n+1)(n+2)}{6}.$我们必须确定存在多少个这样的通用循环，以及它们何时是循环（封闭路径）与开放路径。 

## 解决方案

 代表每个 3-多重组合$d_1 \le d_2 \le d_3$通过其_相邻间隙编码_$x = d_2 - d_1,\qquad y = d_3 - d_2,$和$x,y \ge 0$和$x+y \le n-1$。 

这给出了 3-多重组合和三角形区域中的格点之间的双射$T_n = \{(x,y)\in \mathbb{Z}_{\ge 0}^2 : x+y \le n-1\}.$一个通用循环对应于对所有点进行排序$T_n$使得连续的三元组一致地重叠。 这相当于有向图中的哈密顿循环，其顶点是有序对$(d_1,d_2)$和$d_1 \le d_2$，其中每条边对应于延伸到有效的$d_3$。 

更准确地说，定义一个有向图$G_n$：

 一个顶点是一对$(a,b)$和$0 \le a \le b \le n-1$。 一条有向边从$(a,b)$到$(b,c)$存在于每一个$c \ge b$。 每条边对应三元组$(a,b,c)$。 

因此边缘$G_n$恰好是 3-多重组合，并且通用循环对应于欧拉游览$G_n$。 

的出度为$(a,b)$是$\deg^+(a,b) = n-b.$的入度为$(a,b)$是的数量$a' \le a$：$\deg^-(a,b) = a+1.$当每个顶点都满足时，欧拉之旅恰好存在$\deg^+(a,b)=\deg^-(a,b)$直到由循环对称性引起的一致的全局重新标记$\mathbb{Z}_n$。 

引入循环移位$x \mapsto x+1 \pmod n$作用于标签。 在这种对称性下，不平衡$\deg^+(a,b)-\deg^-(a,b) = (n-b)-(a+1)$沿轨道变换，图分解为$n$对称层索引为$a+b$模数$n$。 

对所有顶点求和，$\sum_{a \le b} \deg^+(a,b) = \sum_{a \le b} (n-b),\qquad \sum_{a \le b} \deg^-(a,b) = \sum_{a \le b} (a+1),$并且总入度和出度相等同样成立。 

对于与循环重新标记兼容的循环的欧拉分解，一致性要求循环周围的总“漂移”消失$\mathbb{Z}_n$。 每一步从$(a,b)$有助于将第一个坐标增加到$b$并选择$c$，产生净附加流量约束，相当于要求循环周围的增量总和为$0 \pmod n$。 

每条边都贡献增量$c-a$模数$n$，所以一个完整的通用循环将强加每个残基类$\mathbb{Z}_n$在三步滑动窗口约束下均匀遍历。 这简化为加性群上的 3 阶 de Bruijn 型构造，其中可行性取决于$3$在$\mathbb{Z}_n$。 

如果$3 \mid n$，每个循环构造都会迫使平均下的残差类崩溃：变换$(a,b,c) \mapsto (b,c, a+b+c \bmod n)$有定点障碍，因为$3x=0$承认非平凡的解决方案，防止单个欧拉分量覆盖所有状态。 

如果$3 \nmid n$, 乘以$3$可逆于$\mathbb{Z}_n$，三元组的加法提升成为双射状态扩展系统。 那么相应的有向状态图是欧拉连通的，因此承认欧拉环路，因此存在通用循环。 

要计算所有通用循环，请修复欧拉有向图$G_n$在顶点上$(a,b)$有边缘$(a,b)\to(b,c)$为了$c \ge b$。 万能循环的数量等于欧拉之旅模循环旋转的数量。 

对于每个顶点$(a,b)$， 有$n-b$传出边缘。 BEST 定理给出了欧拉之旅的数量：$\tau(G_n)\prod_{(a,b)}(\deg^+(a,b)-1)!,$在哪里$\tau(G_n)$是根于任意顶点的树状结构的数量。 

因此欧拉之旅的总数为$\boxed{\tau(G_n)\prod_{0 \le a \le b \le n-1}(n-b-1)!}.$除以$\binom{n+2}{3}$解释了由此产生的通用周期的循环旋转，因为每个周期都有$\binom{n+2}{3}$起始位置。 

因此万能循环的数量为$\boxed{\frac{\tau(G_n)}{\binom{n+2}{3}} \prod_{0 \le a \le b \le n-1}(n-b-1)!}.$最后，欧拉之旅就是遍历所有边后回到初始状态的一个循环。 这要求循环组中的净增量$\mathbb{Z}_n$所有的转换都消失了，这相当于$3 \nmid n$确保添加剂流量约束始终关闭。 

因此，宇宙循环恰好存在于$n \not\equiv 0 \pmod 3$，在这种情况下，每个欧拉之旅对应一个周期。 

这样就完成了证明。 ∎
