 ## 解决方案
 ## 问题理解

 我们得到三个购买金额，表示为$t_1, t_2, t_3$。 还有一个函数叫做`discount(x)`这告诉我们，如果我们购买名义总价值的东西，我们实际支付的金额是多少$x$。 折扣函数可以以某种方式降低成本，并且假设已经以某种形式在输入中定义或提供。 

目标是在应用折扣之前决定如何对这三个值进行分组，因为分组会更改支付总额。 在应用折扣之前，我们可以单独支付商品或将它们合并成更大的捆绑包。 每种不同的分组策略都会导致不同的最终成本，我们必须选择尽可能小的成本。 

问题的结构很小并且形状固定：只有三个项目，因此有意义的分组模式的数量是恒定的。 即使折扣函数本身是任意的，我们应用它的组合结构也是有限的。 

从约束的角度来看，这立即排除了对任意大小的子集的排列或动态编程的任何搜索，因为输入大小没有增长。 就决策而言，问题的大小是恒定的，因此解决方案必须在每个测试用例的恒定时间内运行一次`discount`可以进行评估。 

这里经常出现的一个天真的误解是试图通过重复重新计算或更深层次的递归来考虑三个值上所有可能的二叉树结构。 这在逻辑上是正确的，但没有必要，而且很容易变得过于复杂。 

一个微妙的边缘情况是折扣函数是非线性的甚至是对抗性的。 例如，它的行为可能类似于：

 -`discount(x) = x`（无折扣）
 - 或`discount(x) = 1`对于任何正 x（统一定价）

 在这两种情况下，分组行为都会极大地改变最优策略，因此我们必须完全按照给定的方式评估所有五个结构化选项，而不是假设单调性或凸性。 

## 方法

 蛮力的想法是从字面上解释问题：我们尝试一切可能的方法将三个元素分组，计算每个分组的成本，并取最小值。 对于三个元素，分组意味着在应用折扣函数之前决定是将相邻项目合并还是将所有项目合并在一起。 

如果我们在没有结构的情况下扩展所有可能性，我们可能会想象将集合递归地分割成子集，但这很快就会推广到指数子集 DP。 对于更大的输入，将是$O(2^n)$，这是不可行的。 

关键的观察结果是，问题并不是真正要求任意分区。 该语句已经显式枚举了每个有效的分组配置，并且总共有五个：

 我们要么将折扣分别应用于所有三个，要么合并一对，或合并所有三个。 

这将问题从组合搜索分解为五个表达式的直接求值。 折扣函数的结构除了可调用之外并不重要。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 对所有分区进行暴力破解 |$O(2^n)$（广义）|$O(n)$| 太慢了|
 | 评估 5 个固定公式 |$O(1)$|$O(1)$| 已接受 |

 ## 算法演练

 我们将问题简化为计算五个候选值并取最小值。 

1.读取三个值$t_1, t_2, t_3$。 它们是固定且独立的，因此我们可以在整个计算过程中将它们视为常量。 
2. 计算$s_1 = discount(t_1) + discount(t_2) + discount(t_3)$。 这对应于从不合并项目，这意味着每个项目都通过折扣函数独立处理。 
3. 计算$s_2 = t_1 + discount(t_2 + t_3)$。 在这里，我们在应用折扣之前合并第二个和第三个项目。 第一项保持不变。 
4. 计算$s_3 = discount(t_1 + t_2) + t_3$。 这将合并前两项，而将第三项分开。 
5. 计算$s_4 = discount(t_1 + t_3) + t_2$。 这合并了第一项和第三项。 尽管这可能看起来不连续，但该问题允许任意配对，因此必须考虑此配置。 
6. 计算$s_5 = discount(t_1 + t_2 + t_3)$。 这会将所有内容合并为一个捆绑包并应用一次折扣。 
7. 返回五个计算值中的最小值。 

### 为什么它有效

 应用折扣的每种有效方式都对应于选择将三个元素划分为组，其中每个组的大小为 1、大小 2 或大小 3，并且通过折扣函数独立评估每个组。 对于三个元素，在给定结构下重要的不同分区的数量恰好是五个。 该算法枚举了所有这些值，没有重叠或遗漏，因此任何最佳解决方案都必须与这些计算值之一匹配。

 ## 边缘情况

 当所有值都相等时，问题通常隐藏对称性。 对于像这样的输入`t1 = t2 = t3 = x`，每个分组都会产生结构相似的表达式。 该算法仍然独立评估所有五种情况，因此不会遗漏任何情况，并且即使多个表达式并列，也能正确找到最小值。 

当一个值为零时，根据折扣函数，分组可能会出现意外行为。 例如，如果`discount(0) = 0`，那么与零合并可能是中性的或有益的，具体取决于如何`discount`表现在较大的金额上。 对所有组合的显式评估可确保在五个候选组合中的至少一个中捕获任何此类交互。
