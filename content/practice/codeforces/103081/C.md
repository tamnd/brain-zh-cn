 ## 解决方案
 ## 问题理解

 我们在平面上有一个矩形房间，锚定在原点并延伸到该点$(X, Y)$。 在这个矩形内有几个代表其他人的固定点。 爱丽丝开始于$(0, 0)$并且必须达到$(X, Y)$同时始终留在矩形内。 

关键约束不是寻找最短路径或任何特定路线，而是保持间隙：爱丽丝想要一条路径，使得她与其他人的距离始终至少为某个值$d$，我们被要求最大化这个$d$。 由于爱丽丝可以在任何方向上连续移动，因此问题是几何问题，并且取决于平面中的连续距离，而不是网格结构。 

输出是一个实数，表示从 Alice 的路径到任何给定点的最大可能的最小距离。 

和$N \le 1000$，我们大致可以负担得起$O(N^2)$或者$O(N^2 \log N)$推理，但是任何涉及平面或路径枚举的密集离散化的事情都是不可能的。 任何显式模拟路径或将平面离散化为精细网格的尝试都会立即失败，因为坐标连续到$10^6$规模。 

当多人靠得很近时，就会出现一个微妙的问题。 例如，如果两个人非常接近，即使每个人单独看来是可以避免的，他们之间的安全区域也可能会消失。 仅考虑与每个点独立的距离而不考虑安全空间的连通性的幼稚方法将会失败。 

当最佳路径必须恰好在两个人之间通过时，就会出现另一种边缘情况，其中限制距离是由他们的影响区域之间最窄的“走廊”定义的。 独立处理障碍物（例如沿直线取到任意点的最小距离）会低估或歪曲可行的运动，因为爱丽丝可以绕过障碍物。 

## 方法

 暴力解释是考虑候选距离$d$并询问 Alice 是否可以从$(0,0)$到$(X,Y)$同时保持至少距离$d$来自每个人。 从几何上来说，这相当于将每个人扩大成半径为1的圆$d$，并询问矩形内部是否存在一条避开所有圆的连续路径。 

对于固定的$d$，我们可以尝试对矩形的精细离散化进行连续的 BFS 或基于网格的洪水填充，标记至少$d$远离一切圈子。 如果存在一条路径，$d$是可行的。 使用二分搜索重复此操作$d$给出了一个解决方案。 然而，将平面离散化得足够精细以尊重$10^{-5}$精度需要巨大的网格，很容易超过$10^9$细胞，这是不可能的。 

关键的观察是，一旦我们解决问题，问题的结构就会发生变化$d$。 我们可以考虑禁区之间的连通性，而不是考虑连续空间。 每个人定义一个半径为的圆盘$d$，这些磁盘对空间进行分区。 爱丽丝想知道是否$(0,0)$和$(X,Y)$位于自由空间的同一连通分量中。 

这是一个经典的几何连通性问题，可以简化为障碍物和边界墙之间的图形连通性。 我们构建一个图，其中节点代表障碍物，并且还包括矩形四个边界的虚拟节点。 如果两个障碍物的展开圆盘重叠，则两个障碍物相连。 如果圆盘接触或穿过边界，则它连接到边界。 关键思想是，如果存在将起始侧与结束侧分开的连接的圆盘链，有效地形成跨越矩形的屏障，则爱丽丝将被阻挡。 

这将可行性检查转换为大小图上的并查找或 BFS 连接性检查$O(N)$，这在二分搜索中足够有效$d$。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 强力网格连接每$d$|$O(N \cdot G^2)$|$O(G^2)$| 太慢了 |
 | 二分查找+几何连通图 |$O(N^2 \log R)$|$O(N^2)$| 已接受 |

 这里$R$是坐标尺度。 

## 算法演练

 我们通过二分查找答案来解决问题$d$，并为每个候选人检查是否存在安全路径。 

1.我们设置一个搜索范围$d$，从$0$达到足够大的值，例如$\max(X, Y)$。 这个上限是安全的，因为任何大于房间尺寸的圆盘半径都会立即完全阻止移动。 
2. 对于固定候选人$d$，我们对待每个人$(x_i, y_i)$作为半径为阻塞的圆盘$d$。 
3. 我们构建一个图，其节点是这些圆盘加上代表矩形左壁、右壁、底壁和顶壁的四个附加边界节点。 
4. 如果两个人节点之间的欧几里得距离最大，则连接它们$2d$。 该模型模拟了重叠的禁区，这意味着它们形成了一个连接的障碍。 
5. 如果人员节点到边界的距离最多为，我们将其连接到边界节点$d$。 当禁止区域接触或与墙壁相交时，会捕获此信息。 
6. 然后，我们检查是否存在阻塞区域的连通分量，将左边界连接到右边界，或者将下边界连接到上边界，以分离的方式$(0,0)$从$(X,Y)$。 如果存在这样的障碍，则当前$d$是不可行的。 
7. 使用此可行性检查，我们对最大的进行二分搜索$d$这仍然是可行的。 

关键的几何解释是，当且仅当禁止圆盘的并集不会创建将矩形切割成不连续区域的连续屏障时，爱丽丝才能从头到尾通过。 

### 为什么它有效

 对于固定的$d$，禁区是封闭圆盘。 如果这些磁盘形成连接矩形相对侧的连接链，它们会将可用空间划分为断开的组件，从而防止从开始到结束出现任何连续路径。 相反，如果不存在这样的分离链，则圆盘的补集在两点之间保持连接。 并查结构正确地捕获了磁盘和边界之间的连通性，因此可行性测试相当于检查是否存在阻塞屏障。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

import math

def can(d, pts, X, Y):
    n = len(pts)
    # nodes: 0..n-1 persons, n=left, n+1=right, n+2=bottom, n+3=top
    L, R, B, T = n, n+1, n+2, n+3

    parent = list(range(n + 4))

    def find(x):
        while parent[x] != x:
            parent[x] = parent[parent[x]]
            x = parent[x]
        return x

    def union(a, b):
        ra, rb = find(a), find(b)
        if ra != rb:
            parent[rb] = ra

    def dist2(a, b):
        dx = pts[a][0] - pts[b][0]
        dy = pts[a][1] - pts[b][1]
        return dx*dx + dy*dy

    # connect person-person
    for i in range(n):
        for j in range(i+1, n):
            dx = pts[i][0] - pts[j][0]
            dy = pts[i][1] - pts[j][1]
            if dx*dx + dy*dy <= 4*d*d:
                union(i, j)

    # connect to boundaries
    for i, (x, y) in enumerate(pts):
        if x <= d:
            union(i, L)
        if X - x <= d:
            union(i, R)
        if y <= d:
            union(i, B)
        if Y - y <= d:
            union(i, T)

    # check blocking conditions:
    # left connects to right OR bottom connects to top
    return find(L) != find(R) and find(B) != find(T)

def solve():
    X, Y = map(float, input().split())
    n = int(input())
    pts = [tuple(map(float, input().split())) for _ in range(n)]

    lo, hi = 0.0, max(X, Y)
    for _ in range(60):
        mid = (lo + hi) / 2
        if can(mid, pts, X, Y):
            lo = mid
        else:
            hi = mid

    print(f"{lo:.10f}")

if __name__ == "__main__":
    solve()
```该解决方案依赖并查找结构来合并重叠的禁止磁盘及其与边界的相互作用。 关键的实现细节是使用平方距离来避免比较中的浮点不稳定。 二分搜索运行固定次数的迭代，以保证精度在所需的误差范围内。 

一个微妙的点是可行性条件。 我们不直接检查 Alice 的路径。 相反，我们检查禁区是否在相对两侧之间形成连续的障碍。 如果他们这样做了，爱丽丝就会被困住； 否则，存在一条穿过自由空间的路径。

 ## 边缘情况

 一个重要的边缘情况是当一个人正好位于或非常接近边界时。 例如，如果一个点位于$(0, y)$，那么即使是一个非常小的$d$立即将该磁盘连接到左墙上。 联盟的步骤`if x <= d`正确地捕获了这一点，因为圆盘已经触及半径处的边界$d$。 

另一种情况是当两点正好相距时$2d$。 在浮点运算中，这可能会在连接和断开之间振荡。 使用平方距离可以避免精度损失，因为我们比较$dx^2 + dy^2 \le 4d^2$始终如一。 

最后一个微妙的情况是无人存在的情况。 并查找图只有边界节点，并且不会发生并集，因此左侧永远不会连接到右侧或从下到上。 该算法正确返回完全自由度，允许$d = \max(X, Y)$作为限制二分搜索界限。
