 ## 解决方案
 ## 问题理解

 给出三堆糖果，每堆糖果可以独立地保持不变或加倍一次。 选择这些操作后，所有三堆糖果的总数是固定的。 问题是是否有可能使这个最终总和可以被三个人整除，这意味着总和必须能被 3 整除。 

因此，实际的决定不是如何分割桩，而是我们是否可以选择桩的子集加倍，以使所得的总和满足整除条件。 

每堆只有两种状态，原始状态或加倍状态，因此最终的总和可以由一小组组合形成。 目标是检查至少一种组合的总和是否能被 3 整除。 

约束很小，每个值最多 1000，因此枚举所有组合的任何推理实际上都是常数时间。 在任何时间限制下，即使对所有八种可能性进行简单的检查也是微不足道的。 关键是要认识到该结构是对三个独立元素的二元选择。 

一个常见的错误是专注于最后均匀地分配糖果，而不是意识到只有当总和能被 3 整除时才可能进行分配。另一个错误是尝试独立地对每一堆糖果进行局部推理，但整除性仅取决于全局总和。 

一个微妙的边缘情况是当原始总和已经可以被 3 整除时。在这种情况下，答案立即为“是”，无需任何操作。 另一种边缘情况是只有一个特定的加倍组合可以修复整除性，例如小输入，奇偶校验变化很重要，即使幅度很小。 

## 方法

 暴力方法枚举了应用加倍运算的所有方法。 由于三个桩中的每一桩都有两种状态，因此我们检查所有 2³ = 8 配置。 对于每个配置，我们计算结果总和并测试它是否能被 3 整除。这是有效的，因为状态空间是常数，所以正确性很简单。 

这种蛮力在这个问题上已经是最优的，但理解它为什么仍然有效是很有用的。 每堆独立贡献，除了最终总和之外，操作不会交互。 这种独立性保证了枚举所有组合可以捕获所有可能的结果。 

由于搜索空间的大小是固定的，因此没有有意义的渐近改进。 即使约束更大，相同的结构仍然会减少检查恒定数量的组合或模算术条件。 

关键的见解是，这个问题相当于检查是否存在一个值的子集，其加倍后会将总和模 3 变为零。

 ## 算法演练

 我们想要确定加倍选择的任何分配是否会导致总和被 3 整除。 

1. 读取代表初始桩的三个整数。 
2. 枚举要加倍的桩的所有子集。 每个子集代表一个从 0 到 7 的二进制掩码。 
3. 对于每个子集，根据是否选择了该堆，通过添加原始值或两倍来计算总和。 
4. 检查这个和是否能被 3 整除。 
5. 如果有任何配置满足条件，则立即返回YES。 
6. 如果没有，则返回 NO。 

这个过程足够的原因是每个有效的操作序列完全对应于桩的一个子集。 由于不允许进行其他操作，因此枚举已完成。 

### 为什么它有效

每堆贡献 x 或 2x。 这意味着每个可实现的最终总和的形式为 x + y + z 加上再次添加的 x、y、z 的某个子集。 该算法隐式检查所有此类线性组合。 由于能被 3 整除仅取决于最终的总和，并且对每个可能的最终总和进行了测试，因此不会错过任何有效的解决方案。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

x = int(input())
y = int(input())
z = int(input())

vals = [x, y, z]

for mask in range(8):
    total = 0
    for i in range(3):
        if mask & (1 << i):
            total += 2 * vals[i]
        else:
            total += vals[i]
    if total % 3 == 0:
        print("YES")
        sys.exit()

print("NO")
```该实现直接使用位掩码对子集枚举进行编码。 每个位指示一堆是否加倍。 0 到 7 的循环涵盖了所有可能性，不会遗漏任何配置。 

提前退出很重要，因为一旦找到有效的配置，就无需继续检查其他子集。 即使常数因子已经很小，这也可以保持解决方案的干净并避免不必要的计算。

 ## 边缘情况

 像 1 1 1 这样的最小情况总是产生 3 的和，它是可整除的，因此算法在第一个掩码处立即返回 YES。 即使所有后面的掩码都是不必要的，枚举也确认了正确性。 

像 1 1 2 这样的情况就表明失败。 原始总和为 4，任何子集加倍都会产生总和 4、5、6、7、6、7、8、9。只有 6 和 9 可以被 3 整除，但仔细检查表明，根据操作的解释，枚举仍然捕获是否存在任何有效配置。 该算法正确识别至少一个面罩是否有效。 

较大的相等值（例如 1000 1000 1000）不会改变行为，因为所有计算都保持在较小的常量范围内。 可除性检查不受大小的影响，仅受模块化结构的影响，该结构由枚举保留。
