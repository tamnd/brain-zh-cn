 ## 解决方案
 ## 问题理解

 我们得到一个数字$d$，我们想要决定是否可以将其拆分为两个非负实数$a$和$b$这样两个条件同时成立：它们的总和等于$d$，他们的乘积也等于$d$。 

因此，从几何上讲，我们正在寻找非负实数线上的两个点，其和与积与固定目标值一致。 每个测试用例都会询问这样的一对是否存在，如果存在，我们必须构造任何有效的对，不一定是唯一的。 

约束条件$d \le 10^3$和$t \le 10^3$意味着我们只需要每个测试用例进行恒定时间计算。 任何涉及以细粒度迭代候选值的方法仍然是可以接受的，但是任何指数或依赖于没有结构的精度搜索的方法都是不必要的。 方程的结构表明直接代数约简而不是数值暴力。 

主要的微妙之处在于，该问题允许实数的精度高达$10^{-6}$。 这排除了纯粹的整数推理，并迫使我们将条件解释为实值约束。 另一个微妙之处是浮点根靠近边界情况，尤其是周围$d = 0$和$d = 1$，在朴素公式中可能表现不同，因此需要显式处理。 

当试图假设任何二次方程总是产生实数非负解时，就会出现故障模式。 例如，$d = 1$导致判别式崩溃到零，从而使系统在所需的等式约束下不一致，即使代数操作可能会误导性地提出边界解决方案。 另一个边缘情况是$d = 0$，其中两个变量必须同时崩溃到零。 

## 方法

 蛮力观点会尝试选择一个候选人$a$并计算$b = d - a$，然后检查是否$a \cdot b \approx d$。 自从$a$是真实的，我们将离散化范围$[0, d]$分成许多小步骤。 原则上这是正确的，因为每个有效解都必须精确满足总和约束，但由于解空间是连续的并且需要极精细的采样才能满足精度要求，因此在计算上毫无意义。 即使每个测试用例有 10^6 个步骤，10^3 个测试的最坏情况输入也已经变得太慢了。 

关键的结构观察是，一旦我们替换，系统就完全确定了$b = d - a$。 这将乘积条件转换为单个二次方程$a$。 然而，我们不是每次都直接求解它，而是可以观察到可行性仅取决于所得二次方程是否在非负域中具有实根，并且可以显式构造这些根。 约束结构将问题简化为求解带有参数的二次方程$d$，并选择任何有效的根对。 

代数导致：$$a(d - a) = d \Rightarrow a^2 - da + d = 0$$这给出了判别条件：$$\Delta = d^2 - 4d = d(d - 4)$$所以只有当$d \ge 4$或者$d = 0$。 特殊情况$d = 4$给出重根，并且$d = 0$简单地崩溃到$(0, 0)$。 为了$d \in (0, 4)$，判别式为负，因此不存在真正的解。 

当解存在时，我们计算：$$a = \frac{d - \sqrt{d^2 - 4d}}{2}, \quad b = d - a$$| 方法| 时间复杂度| 空间复杂度| 判决 |

 ## 算法演练

 1.读取整数$d$。 目标是确定从和积条件导出的二次约束系统的可行性。 
2.如果$d = 0$,立即输出$a = 0$,$b = 0$。 这是和与积同时为零而不引入矛盾的唯一方法。 
3.如果$d < 4$，输出“N”。 这是从判别式得出的$d(d - 4)$，它变成负值并消除实数解。 
4. 否则计算判别式$\Delta = d^2 - 4d$。 该值确定从约束导出的二次方程是否有实根。 
5. 取平方根$\Delta$并计算较小的根：$$a = \frac{d - \sqrt{\Delta}}{2}$$选择较小的根可确保两个值保持非负且数值稳定。 
6. 计算$b = d - a$。 这精确地强制了总和约束，防止浮点漂移违反$a + b = d$。 
7. 输出“Y”后跟$a$和$b$。 

### 为什么它有效

 每个有效对必须同时满足两个约束，并且替换一个变量将系统简化为单个二次方程，其根精确地表征所有可行解。 判别式完全确定实解的存在性，并且构造的根通过代数等价自动满足和条件和乘条件。 这些根之外不能存在其他对，因为变换在约束上是双射的$b = d - a$。 

## Python 解决方案```python
import sys
import math

input = sys.stdin.readline

t = int(input())
for _ in range(t):
    d = float(input().strip())

    if d == 0:
        print("Y 0.0 0.0")
        continue

    if d < 4:
        print("N")
        continue

    disc = d * d - 4 * d
    root = math.sqrt(disc)

    a = (d - root) / 2
    b = d - a

    print("Y", a, b)
```代码直接实现了二次约简。 早期退出处理退化和不可行的范围，避免不必要的浮点运算。 判别计算是稳定的，因为$d \le 10^3$， 所以$d^2$保持在浮点精度限制之内。 计算$b$作为$d - a$准确地保留总和约束并避免复合舍入误差。

 ## 边缘情况

 对于$d = 0$，算法立即输出$(0, 0)$。 代入得到和 0 和积 0，完全匹配。 

为了$d = 3$，算法拒绝这种情况，因为$d < 4$。 判别式$9 - 12 = -3$确认没有实根，因此不存在有效的对。 

为了$d = 4$，判别式变为零，给出单个重复根$a = b = 2$。 该算法计算$\sqrt{0} = 0$，然后返回$a = 2$,$b = 2$，它完全满足这两个约束。
