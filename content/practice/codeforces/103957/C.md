 ## 解决方案
 ## 设置

 顺序真值表$n$是一个长度为的二进制字符串$2^n$。 一颗珠子就是一个真值表$\beta$这不是形式$\alpha\alpha$。 如果通过固定变量的任何前缀获得的每个子表都是珠子，则布尔函数是甜蜜的。 

让$S(n)$表示甜蜜布尔函数的数量$n$变量。 目标是计算$S(n)$为了$n\le 7$。 

级别的子函数$k$对应固定$x_1,\dots,x_k$，产生顺序真值表$n-k$。 甜味要求每个这样的子表都是一颗珠子。 

## 解决方案

 真值表$\tau$秩序$m$是一个珠子当且仅当它的两个顺序子表$m-1$不同。 因此，顺序的子函数$m$是一个珠子当且仅当其 LO 和 HI 限制为$x_1$是不同的。 

布尔函数是甜蜜的当且仅当对于每个子函数$\tau$由任何前缀分配产生，诱导对$(\tau_0,\tau_1)$满足$\tau_0\ne \tau_1$。 

这个条件相当于要求函数的完整二元决策树中的每个节点都具有不同的 LO 和 HI 子函数。 由于子函数完全由其真值表决定，因此节点的相等性与子表的相等性完全对应。 

因此，由 sweet 函数导出的结构是深度的完整二元决策树$n$其中每个节点都由不同的子函数标记，并且每个节点的顺序$m>0$由有序的不同子函数对决定$m-1$。 

让$T(m)$表示出现在级别的不同子函数的集合$m$在一个甜蜜的函数中$n$变量。 处于水平$0$只有一个函数，即原始函数。 在每个级别的细化步骤$m-1$达到水平$m$,各子函数$\tau$秩序$n-m+1$被替换为有序对$(\tau_0,\tau_1)$秩序的不同子函数$n-m$。 

因此，单个子函数的可接受的细化次数为$S(n-m) (S(n-m)-1)$， 自从$\tau_0$可以任意选择$S(n-m)$和$\tau_1$从剩余的$S(n-m)-1$的可能性。 

处于水平$m-1$有$2^{m-1}$子函数，并且它们的细化是独立的，因为不同的节点对应于由第一个子函数的不同分配确定的不同子问题$m-1$变量。 因此，从深度扩展甜蜜功能的方法总数$m-1$深入$m$是$$\bigl(S(n-m)(S(n-m)-1)\bigr)^{2^{m-1}}.$$环境$m=n$给出一个递归式$S(n)$按照$S(n-1)$：$$S(n) = \bigl(S(n-1)(S(n-1)-1)\bigr)^{2^{n-1}}.$$初始条件是$S(0)=2$，因为零变量的唯一布尔函数是常量$0$和$1$，两颗珠子。 

现在依次计算。 

为了$n=1$,$$S(1) = (S(0)(S(0)-1))^{1} = (2\cdot 1)^1 = 2.$$为了$n=2$,$$S(2) = (S(1)(S(1)-1))^{2} = (2\cdot 1)^2 = 4.$$为了$n=3$,$$S(3) = (4\cdot 3)^4 = 12^4 = 20736.$$为了$n=4$,$$S(4) = (20736\cdot 20735)^8.$$计算乘积：$$20736\cdot 20735 = 20736(20736-1) = 20736^2 - 20736.$$自从$20736^2 = 429981696$，减法给出$$20736\cdot 20735 = 429981696 - 20736 = 429960960.$$因此$$S(4) = (429960960)^8.$$为了$n=5$,$$S(5) = \bigl((429960960)^8((429960960)^8-1)\bigr)^{16}.$$为了$n=6$,$$S(6) = \Bigl(S(5)(S(5)-1)\Bigr)^{32}.$$为了$n=7$,$$S(7) = \Bigl(S(6)(S(6)-1)\Bigr)^{64}.$$因此，这些值完全由递归确定。 

因此，对于$n\le 7$,$$\boxed{
\begin{aligned}
S(0)&=2,\\
S(1)&=2,\\
S(2)&=4,\\
S(3)&=20736,\\
S(4)&=(429960960)^8,\\
S(5)&=\bigl((429960960)^8((429960960)^8-1)\bigr)^{16},\\
S(6)&=\Bigl(S(5)(S(5)-1)\Bigr)^{32},\\
S(7)&=\Bigl(S(6)(S(6)-1)\Bigr)^{64}.
\end{aligned}}$$这样就完成了解决方案。 ∎

 ## 验证

 递归仅取决于珠子条件，这需要每个节点处的 LO 和 HI 子表不相等。 这与第 7.1.4 节中珠子的定义相匹配。 

在每个细化步骤中，每个节点都对应于由唯一前缀分配确定的不同子函数，因此跨节点的独立性$2^{n-1}$通过将真值表分解为不相交的子问题来证明节点的合理性。 

的计算$20736^2$被验证为$$(20000+736)^2 = 400000000 + 2\cdot 20000\cdot 736 + 736^2
= 400000000 + 29440000 + 541696
= 429981696.$$减法$20736$产量$429960960$，确认修正后的产品。 

所有后续表达式仅依赖于这个更正后的基数。 

这样就完成了验证。 ∎
