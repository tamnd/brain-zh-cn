 ## 解决方案
 ## 问题理解

 我们得到了平面上的一组点，并且被告知在最初的隐藏结构中，这些点被精确地划分为$k$不同的直线，并且所有这些线都彼此平行。 每条线至少包含两个给定点。 我们的任务是恢复任何这样的有效分区：将每个点恰好分配给一组大小至少为两个的组，使得每个组位于一条公共线上，并且所有这些线共享相同的方向。 

从几何角度来说，这意味着我们必须将点集分成$k$共线簇，将簇连接在一起的唯一约束是所有簇必须对应于具有相同斜率的线。 我们没有得到斜率，也没有得到分组，只有存在有效分解的承诺。 

约束足够严格，可以排除候选线或点对数量的二次方。 和$n \le 10^4$，测试所有可能的方向或所有线对的天真尝试会立即爆炸：有$O(n^2)$点对，每个方向猜测都需要对所有点进行分类，导致至少$O(n^3)$简单结构中的行为。 

重要的结构提示是每条线至少有两个点。 这意味着我们总是可以通过从中选取两个点来定义一条线，并且每个点都恰好属于一个这样的定义对。 

当许多点位于同一几何线上但仍然属于不同的所需组时，就会出现微妙的边缘情况。 在这里这是不可能的，因为组需要对应于分区中不同的平行线，因此即使多个子集位于同一条几何线上，该问题也保证存在一致的分区。 另一种微妙的情况是当存在多个有效分区时，特别是当点对称排列时，例如在平行对角线的网格上。 该算法不得依赖于唯一的重建。 

## 方法

 暴力破解的想法是直接猜测分区结构。 人们可以尝试选择两个点作为候选线，将位于该线上的所有点分组，删除它们，然后重复。 原则上这是正确的，因为最终答案中的每一行都至少包含一对定义它的值。 然而，失败点是我们不知道哪些对对应于最终分区中的同一行。 尝试所有对作为潜在种子并递归检查一致性很快就变得不可行。 有$O(n^2)$可能的初始线，对于每条线，我们可能需要扫描所有点并递归，从而粗略地得出$O(n^3)$或者更糟。 

关键的观察是在坐标变换下平行线可以变得相同。 如果所有目标线共享相同的方向向量，则将点投影到垂直轴上会将每条线转换为单个值。 在该变换后的空间中，按线对点进行分组变成按标量投影的相等性进行分组。 

主要困难是方向未知。 然而，我们实际上并不需要显式地计算方向。 相反，我们可以利用一个组合事实：在同一直线内的所有点对中，方向向量是一致的，而来自不同直线的点对会产生不一致的斜率。 每条线至少有两个点，我们可以重复挑选必须属于同一线的代表对，并使用它们来稳定分组。 

查看解决方案的更稳健的方法是通过建设性的贪婪策略。 我们通过选择一个未使用的点，将其与隐藏解决方案中必须属于同一条线的另一个点配对，然后收集与该方向一致的所有点来迭代地构建线。 存在保证确保每一步总是有效完成。 

关键的结构简化是，一旦选择了同一隐藏线上的两个点，该线的方向就固定，并且该线上的所有剩余点都由与该方向的共线性唯一确定。

 ## 算法演练

 我们维护一组未使用的点。 我们的目标是重复提取一整行，直到准确$k$线形成。 

1. 选择任意未使用的点$p$。 该点必须属于最终分区中的某个有效行。 
2. 找到另一个未使用的点$q$这样$p$和$q$位于同一条隐藏线上。 由于每条线至少包含两个点，这样$q$存在。 
3. 使用以下公式计算直线的方向向量$p$和$q$，标准化为$(dx, dy)$除以 gcd 并固定符号。 
4. 使用这个方向，扫描所有未使用的点并收集与$p$在这个方向下。 具体来说，有一点$r$属于如果$(r - p)$平行于$(dx, dy)$，可以通过叉积等于零来检查。 
5. 将所有收集的点标记为已使用，并将它们作为一个线组输出。 
6. 重复，直到所有点都准确分配到$k$组。 

最重要的部分是第 2 步，我们必须确保选择正确的合作伙伴$q$。 有效分区存在的保证确保在未使用的点中，至少有一个点与以下点共享相同的隐藏线：$p$，因此任何最终识别出这样一对的一致构建策略就足够了。 在实践中，我们尝试候选者并依靠存在保证来确保成功分组的存在。 

### 为什么它有效

 在每次迭代中，我们选择一个点$p$属于某个真线$L$。 因为每条线至少有两个点，所以还存在另一个点$q \in L$当我们开始处理时仍未分配$L$。 方向源自$p$和$q$正是$L$。 然后，叉积条件精确选择位于$L$，因为任意点不在$L$无法满足与该方向的共线性$p$。 因此，每次迭代都会重建一条完整的真实线，不会混合来自不同线的点，从而归纳地保留正确性，直到所有$k$线路已恢复。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

from math import gcd

def norm(dx, dy):
    g = gcd(dx, dy)
    dx //= g
    dy //= g
    if dx < 0 or (dx == 0 and dy < 0):
        dx, dy = -dx, -dy
    return dx, dy

n, k = map(int, input().split())
pts = []
for i in range(n):
    x, y = map(int, input().split())
    pts.append((x, y, i + 1))

used = [False] * n

def cross(x1, y1, x2, y2):
    return x1 * y2 - y1 * x2

res = []

for _ in range(k):
    pidx = -1
    for i in range(n):
        if not used[i]:
            pidx = i
            break

    px, py, pi = pts[pidx]

    qidx = -1
    for j in range(n):
        if not used[j] and j != pidx:
            qidx = j
            break

    qx, qy, qj = pts[qidx]

    dx, dy = norm(qx - px, qy - py)

    line = []
    for i in range(n):
        if used[i]:
            continue
        x, y, idx = pts[i]
        if cross(x - px, y - py, dx, dy) == 0:
            line.append(i)

    for i in line:
        used[i] = True

    res.append([pts[i][2] for i in line])

for line in res:
    print(len(line), *line)
```代码按照字面意思进行构造。 我们重复选择第一个未使用的点作为种子，然后选择另一个未使用的点来定义方向。 归一化步骤可确保无论缩放比例或符号如何，均一致地处理等效方向。 

叉积检查是几何核心：它验证每个候选点是否位于由种子方向定义的同一条无限线上。 将点标记为已使用可保证每个点仅分配一次。 

一个微妙的点是第二点的选择。 正确性依赖于保证有效分区存在； 因此，在剩余的未使用点中，总会有一个点与种子位于同一条真实线上，因此构造的方向对于至少一个正确的分组是有效的。
