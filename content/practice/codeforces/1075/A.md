 ## 解决方案
 ## 问题理解

 我们被赋予了一个$n \times n$网格。 一位国王从左下角开始$(1,1)$另一个从右上角开始$(n,n)$。 一枚硬币被放置在$(x,y)$，两位国王都试图尽快到达目的地。 它们像国际象棋国王一样移动，因此只需一步，它们就可以到达八个相邻单元格中的任何一个，包括对角线。 

这个问题的关键数量是国王到达目标单元所需的最少移动次数。 由于允许对角线移动，因此国王在可能的情况下有效地同时减少了行和列的差异。 这使得两个单元格之间的距离等于切比雪夫距离：$$\text{dist}((a,b),(c,d)) = \max(|a-c|, |b-d|)$$白王先走，然后是黑王，交替轮流走。 对于移动的国王来说，每一步的距离正好是一步。 谁达到谁就是胜利者$(x,y)$第一的。 

约束条件允许$n$最多$10^{18}$，因此任何一步一步的运动模拟都是不可能的。 我们必须在常数时间内计算距离。 

一个微妙的边缘情况是当国王开始硬币时。 如果$x,y = 1,1$，白棋在黑棋走之前立即获胜。 如果$x,y = n,n$，黑棋立即获胜。 任何解决方案都必须正确处理这些即时胜利。 

另一个边缘情况是平价：即使黑棋的绝对距离更近，如果黑棋需要再走一步但先下棋，白棋仍然可能获胜。 

## 方法

 暴力模拟将一步步移动两个国王，总是选择向硬币移动的最短路径。 每个国王都需要$O(n)$在最坏情况下移动，并且每次移动都是常数时间，导致$O(n)$每个国王。 自从$n$可以是$10^{18}$，这是完全不可行的。 

关键的观察是，最佳游戏是确定性的：两个国王总是沿着最短路径移动，因此他们的到达时间完全取决于他们的初始距离$(x,y)$。 唯一剩下的微妙之处是回合顺序。 

我们计算：

 -$d_w = \max(|x-1|, |y-1|)$-$d_b = \max(|n-x|, |n-y|)$白衣准时到达$2 \cdot d_w$或者更准确地说$2 \cdot d_w$游戏时间线的轮次，但由于白棋先走，他的有效到达轮次是$2d_w - 1$。 

黑方轮到$2d_b$。 

因此比较简化为：$$2d_w - 1 < 2d_b$$如果为真，则白方获胜，否则黑方获胜。 

这将整个问题简化为计算两个切比雪夫距离并将它们与一个小的奇偶校验移位进行比较。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | --- | --- | --- | --- |
 | 暴力模拟|$O(n)$|$O(1)$| 太慢了 |
 | 距离+奇偶校验比较|$O(1)$|$O(1)$| 已接受 |

 ## 算法演练

 1. 使用切比雪夫度量计算白色国王到硬币的距离。 这代表白棋发挥最佳效果时所需的最少步数。 
2. 以同样的方式计算黑王到硬币的距离，从$(n,n)$。 
3. 将两个距离转换为“到达硬币时的圈数”。 白棋轮流到达$1,3,5,\dots$，所以他的到达轮次是$2d_w - 1$。 黑棋转弯时到达$2,4,6,\dots$，所以他的就是$2d_b$。 
4. 比较这两个值。 如果白棋的到达回合严格小于白棋，则白棋获胜。 否则黑方获胜。 

步骤 3 有效的原因是，双方玩家每次移动都会将距离减少 1 个单位，并且任何移动都不能减少超过 1 个单位的切比雪夫距离。 

### 为什么它有效

 关键的不变量是，在最佳玩法下，每个国王到目标的距离每一步都会减少一分，并且没有其他策略可以更快地减少它。 由于走棋顺序是固定的（白棋总是先手），唯一的不对称性是白棋的一步优势。 将距离转换为转弯指数可以准确地捕捉到这一优势，因此比较$2d_w - 1$和$2d_b$完全决定胜负。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    n = int(input())
    x, y = map(int, input().split())

    dw = max(abs(x - 1), abs(y - 1))
    db = max(abs(n - x), abs(n - y))

    white_turn = 2 * dw - 1
    black_turn = 2 * db

    if white_turn < black_turn:
        print("White")
    else:
        print("Black")

if __name__ == "__main__":
    solve()
```白色和黑色距离是使用切比雪夫度量计算的，因为国王可以沿对角线移动，这会同时减少两个坐标。 转向回合指数的转换至关重要：白方的第一步棋使他在交替序列中获得了一步优势，这是通过在他的最终到达时间中减去一来捕获的。 

比较是严格的，因为如果两者在同一回合到达，则黑色因第二次移动而被认为获胜。
