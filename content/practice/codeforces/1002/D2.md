 ## 解决方案
 ## 问题理解

 我们有一个小型量子电路设置，其中输入由$N$-量子位寄存器$x$，单个目标量子位$y$，和一个经典的二元向量$b$长度$N$。 每个量子位都可以处于任意量子态，因此该操作必须作为可逆量子变换而不是经典函数评估来实现。 

任务是实现一个通过翻转输出量子位来转换联合状态的预言机$y$取决于输入位的布尔函数$x_1, x_2, \dots, x_N$。 该函数定义为每个索引贡献的模 2 之和，其中每个索引$i$贡献要么$x_i$或其否定$1 - x_i$，取决于是否$b_i$分别为1或0。 最终的函数值是所有这些贡献的异或，并且预言机必须应用$y \mapsto y \oplus f(x)$。 

关键的限制是$N \le 8$，这意味着我们不会被要求对大输入进行渐近优化，而是要仔细推理量子可逆性以及如何使用受控门实现异或奇偶校验计算。 任何直接计算状态的经典副本上的函数的朴素解释都是无效的，因为测量会破坏叠加性。 

一个微妙的边缘情况来自否定项$1 - x_i$。 如果处理不当，人们可能会尝试暂时“反转”量子位而不恢复它们，从而破坏可逆性。 例如，如果$b = [0]$和$x_1 = 1$，那么贡献就是$1 - x_1 = 0$。 一个不小心的实施会翻转$x_1$并直接使用它而不恢复它会破坏输入状态。 

另一个陷阱是忘记对否定位进行异或会引入全局奇偶校验移位，而不仅仅是独立否定。 必须在不引入意外纠缠的情况下处理这种全球转变。 

## 方法

 蛮力经典思维方式会尝试评估该函数$f(x)$通过读取所有位，计算每一项$x_i$或者$1-x_i$，然后翻转$y$因此。 虽然这在传统上是简单的，但在量子环境中是无效的，因为读取量子位会破坏它们的状态。 即使我们想象模拟所有$2^N$基础指出，成本随着量子比特数量呈指数级增长，这使得它在概念上和实践上都不合适。 

正确的方法来自于认识到该函数在 XOR 上是线性的。 每个术语是$x_i$或者$1 \oplus x_i$，这些项的 XOR 可以重新排列为所有项的总和$x_i$项加上恒定的奇偶校验移位，具体取决于存在的否定数量。 这使我们能够仅使用 CNOT 门以及可能在输出量子位上使用单个 X 门来实现预言机。 

具体来说，对所有$x_i$值转化为$y$可以通过一系列 CNOT 操作来完成。 否定项贡献了一个附加常数，该常数等于中零条目数量的奇偶校验$b$。 该常数可以通过翻转来应用$y$如果需要的话一次。 

这减少了从推理量子态的复杂布尔表达式到构建奇偶校验电路的问题。

 ## 算法演练

 我们使用可逆异或累积来构建预言机。 

1. 计算满足多少个指标$b_i = 0$。 设这个数为$k$。 我们只关心$k \bmod 2$，因为异或运算甚至会忽略重复。 
2. 对于每个索引$i$，应用 CNOT 门$x_i$到$y$。 这会累积所有的异或$x_i$值到目标量子位中。 
3.如果$k \bmod 2 = 1$，应用 Pauli-X 门$y$。 这说明了这样一个事实：每个$b_i = 0$贡献一个否定位$1 \oplus x_i$，为每个此类项引入全局 XOR 1。 
4. 不做任何其他事情。 输入寄存器$x$保持不变，保持可逆性。 

### 为什么它有效

 每个受控非执行转换$y \mapsto y \oplus x_i$，并将它们链接起来产生$y \mapsto y \oplus \bigoplus_i x_i$。 对于索引，其中$b_i = 0$, 替换$x_i$经过$1 \oplus x_i$每个此类索引添加额外的 XOR 1。 对常数进行异或运算$k$时间的贡献正是$k \bmod 2$。 因此最终的变换正是$y \mapsto y \oplus f(x)$，同时留下所有$x_i$不变。

## 工作示例

 ### 示例 1

 让$N = 3$,$b = [1, 0, 1]$。 

我们计算贡献：

 -$i=1$:$x_1$-$i=2$:$1 \oplus x_2$-$i=3$:$x_3$所以$f(x) = x_1 \oplus x_2 \oplus x_3 \oplus 1$。 

该电路首先对所有的进行异或$x_i$进入$y$，然后翻转$y$一次，因为其中正好有一个零$b$。 

| 步骤| 行动| 影响$y$|
 | ---| ---| ---|
 | 1 | 中国诺特$x_1 \to y$|$y \oplus x_1$|
 | 2 | 中国诺特$x_2 \to y$|$y \oplus x_1 \oplus x_2$|
 | 3 | 中国诺特$x_3 \to y$|$y \oplus x_1 \oplus x_2 \oplus x_3$|
 | 4 | 翻动$y$|$y \oplus x_1 \oplus x_2 \oplus x_3 \oplus 1$|

 这确认了转换与函数定义完全匹配。 

### 示例 2

 让$N = 2$,$b = [1, 1]$。 
没有谈判发生，所以$k = 0$。 

| 步骤| 行动| 影响$y$|
 | ---| ---| ---|
 | 1 | 中国诺特$x_1 \to y$|$y \oplus x_1$|
 | 2 | 中国诺特$x_2 \to y$|$y \oplus x_1 \oplus x_2$|

 不应用最终翻转。 结果纯粹是输入的异或。 

这表明，当所有系数均为 1 时，预言机会简化为标准奇偶校验计算。

 ## 边缘情况

 当所有条目$b$为零，每一项都变成$1 \oplus x_i$，所以该函数是所有位加上一个常数的异或$N \bmod 2$。 该算法可以正确处理此问题，因为它计算零条目和翻转$y$仅当它们的奇偶校验为奇数时，同时仍对所有内容进行异或$x_i$进入目标。 

为了$b$由于多个 0 中只有一个 0，因此仅引入了一次全局翻转。 CNOT 结构保持不变，最终的 X 门正确地解释了孤立的否定，而无需修改任何输入量子位。 

什么时候$N = 1$，该函数简化为单个位的同一或否定，并且电路正确地变为单个 CNOT 或 CNOT 后跟 X$y$，匹配两种可能的布尔结果。
