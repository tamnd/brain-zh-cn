 ## 解决方案
 ## 问题理解

 我们从最多八个量子位的寄存器开始，所有量子位都在全零计算基础状态下初始化。 除此之外，我们还得到了另一个基本状态的经典描述，即布尔数组，其中每个条目指示相应的量子位在该基本配置中是否应该处于状态 1 还是 0。 该数组的第一个条目保证为 true，这意味着目标基础状态的第一个量子位始终等于 1。 

任务不是“计算”经典意义上的任何东西。 相反，我们必须应用一系列量子操作，以便最终的量子态成为两个基本状态的相等叠加：全零状态和输入数组描述的基本状态。 

形式上，最终状态应该是 (|00…0⟩ + |b⟩) / √2，其中 |b⟩ 是由布尔数组编码的计算基础状态。 

由于 N 至多为 8，因此任何对每个量子位应用恒定数量的量子门的方法就足够了。 不用担心 N 中线性的渐近缩放，因为量子位的总数很小。 真正的限制是概念性的：转换必须使用有效的量子运算来完成，无需测量，也无需直接“写入”振幅。 

一个微妙的边缘情况是由于两个基本状态在潜在的许多量子位中不同的要求而产生的。 一个天真的想法可能是“准备|b⟩，然后以某种方式添加|0…0⟩”，但量子力学不允许直接添加状态。 另一种诱人但不正确的方法是将哈达玛门应用于所有量子位并尝试选择性地取消振幅，这将在所有状态上产生均匀叠加，而不是所需的两态叠加。 

关键的困难在于，我们需要一种从 |0…0⟩ 开始的可逆结构，并将一个额外的计算基础状态引入叠加，而不引入任何其他状态。 

## 方法

 蛮力思维会尝试通过推理幅度然后将分解应用于门来显式构造目标状态向量。 虽然这在理论上是可能的，但它很快就会变得不必要的复杂性：将任意两种状态叠加分解为没有结构的基本门将涉及处理所有 2^N 基本状态的干扰模式，这远远超出了 N ≤ 8 所需的范围。 

目标状态的结构非常特殊。 它恰好包含两个基本状态，其中之一是初始状态。 这表明我们在量子电路中只需要一个“分支点”：一个将振幅分成两个计算路径的操作，然后用一种确定性的方法将其中一个路径转换为 ​​|b⟩。 

关键的观察是，我们可以使用单个量子位作为控制寄存器来决定是保持 |0…0⟩ 还是转变为 |b⟩。 如果我们使用哈达玛门将第一个量子位置于叠加状态，它可以充当两个分支之间的开关。 然后，来自该量子位的受控操作可以有条件地翻转其他量子位，以便只有一个分支演变成所需的基础状态。 

这将问题简化为仅使用受控的 NOT 运算从 |0…0⟩ 构建 |b⟩，但仅在叠加的“1”分支上激活。

 ## 算法演练

 我们直接使用量子位寄存器和描述目标基本状态的布尔数组。

1. 将 Hadamard 门应用于第一个量子位。 这会创建两个计算分支的叠加：一个第一个量子位为 0，另一个为 1。此时，两个分支中所有其他量子位都保持为 0。 
2. 将第一个量子位解释为控制位，决定我们是保持在全零状态还是向目标基础状态移动。 第一个量子位为 0 的分支必须保持不变，因此进一步的操作不应影响它。 
3. 对于每个大于 0 且位[i]为真的索引 i，应用具有控制量子位 0 和目标量子位 i 的 CNOT 门。 这确保了只有在控制量子位为1的分支中，相应的目标量子位才从0翻转到1。 
4. 对位[0]不执行任何操作，因为第一个量子位已经是控制并且已经表示目标基础状态中所需的值。 

在这些步骤之后，状态演变如下：第一个分支保持 |00…0⟩，而第二个分支恰好变为 |b⟩，因为每个所需的量子位翻转都已被有条件地应用。 

### 为什么它有效

 该结构在整个计算过程中保持严格的两分支结构。 在哈达玛门之后，状态恰好是两个基本配置的线性组合，仅在第一个量子位上有所不同。 随后的每个 CNOT 均由第一个量子位控制，因此它对 |0⟩ 分支中的所有幅度具有相同的作用，并且仅修改 |1⟩ 分支。 

由于每个 CNOT 完全对应于在需要时将目标基础状态的一位设置为 1，因此第二个分支确定性地转换为 |b⟩，而不影响第一个分支。 没有引入额外的基础状态，因为每个操作都是以控制量子位为条件的计算基础向量的排列。

```该实现反映了这样一个事实：我们不是模拟量子态，而是发出门操作。 第一个量子位上的哈达玛是创建叠加的唯一操作。 每个后续步骤都是由同一量子位驱动的条件位翻转。 

唯一微妙的一点是在 CNOT 循环中跳过索引 0。 由于bits[0]保证为真，因此已经通过选择第一个量子位作为控制来考虑它； 将一个量子位的 CNOT 应用到其自身是没有意义的，应该避免。

 ## 边缘情况

 对于 N = 1 且位 = [true] 的最小情况，系统从 |0⟩ 开始。 将哈达玛应用于唯一的量子位会产生 (|0⟩ + |1⟩)/√2。 没有可应用的 CNOT 操作，因此状态立即保持正确。 

对于除第一个位之外的所有位都为假的情况，例如位 = [true, false, false]，Hadamard 仍然创建 (|00…0⟩ + |10…0⟩)/√2，并且不应用额外的门。 第二个分支已经与目标基础状态完全匹配。 

对于最大尺寸情况 N = 8，bits[i] 为 true 的每个量子位恰好接收一次受控翻转。 由于所有操作均由量子位 0 控制，因此第一个分支永远不会受到干扰，而第二个分支确定性地累积完整的位串，而不会创建任何中间不需要的状态。
