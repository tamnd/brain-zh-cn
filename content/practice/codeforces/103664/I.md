 ## 问题理解

 我们被要求构造一个序列$n$不同的正整数，使得从序列中选择的任何三个数字都可以形成非退化三角形的边长。 用几何术语来说，对于每个三元组$a \le b \le c$从集合来看，三角不等式必须成立，这意味着$a + b > c$。 所有选择的值也必须是不同的，并且每个值不得超过$10^9$。 

输入只是整数$n$，输出是任何有效的集合$n$满足条件的数。 

约束条件$n \le 10^5$立即排除任何试图显式验证三元组的构造。 直接检查所有三元组需要$O(n^3)$的操作，远远超出了可行的限度。 即使根据候选第三个元素检查所有对也会导致$O(n^2)$，在这种规模下也太慢了。 

天真的思维的一个微妙的失败案例是选择任意数字或没有结构的随机递增序列。 例如，选择两个的幂，例如$1, 2, 4, 8$立即中断，因为$1 + 2 \le 4$，因此即使很小的子集也会违反三角形条件。 这表明增长率至关重要：快速增长的序列会失败，因为最大的元素主导了两个较小元素的总和。 

因此，挑战在于设计一个保持足够密集的序列，以便即使任何三元组中最小的两个元素仍然可以超过最大的元素。 

## 方法

 暴力尝试将生成候选集并检查每个三元组的三角形条件。 对于固定集，验证有效性需要检查所有$\binom{n}{3}$三元组，大约是$n^3 / 6$检查。 和$n = 10^5$，这是一个天文数字，无法在任何时间限制内运行。 

关键的观察结果是，三角不等式最难满足三元组中最大元素的最小可能和。 如果我们对序列进行排序，则条件会减少为仅检查索引方面的连续三元组，因为如果它适用于任何子集中的最小两个元素和最大元素，那么它也适用于所有其他元素。 

这导致结构简化：我们想要一个序列，其中两个最小元素的总和始终大于任何三元组中最大可能元素。 保证这一点的一个简单方法是使序列从足够大的基数开始连续的整数。 连续的数字可以最大限度地减少间隙，从而使两个最小元素相对于最大元素的总和最大化。 

如果我们选择顺序$n-1, n, n+1, \dots, 2n-2$，那么最坏情况的三元组是$n-1, n, 2n-2$。 即使在这种极端情况下，我们也有：$$(n-1) + n = 2n - 1 > 2n - 2$$所以三角不等式始终成立。 任何其他三元组仅改善左侧或减少右侧。

 ## 算法演练

 该构造的工作原理是输出向上移动的连续整数序列，以便即使两个元素的最小可能和仍然支配任何三元组中的最大元素。 

1. 选择起始值为$n-1$。 这确保了最小可能的对和相对于我们将使用的最大元素来说已经很大。 
2. 通过列出来自的所有整数来构造序列$n-1$最多$2n-2$包容性。 这正好产生$n$不同的数字。 
3. 按升序输出这些数字。 之后不需要排序，因为构建已经保证了顺序。 

选择这个特定区间的原因是它尽可能紧密地压缩范围，同时仍然保证极端配置的三角不等式，其中最小和最大元素之间的差距最大化。 

### 为什么它有效

 排序序列中的任何三元组至少有其最小的两个元素$n-1$和$n$，而它的最大元素至多是$2n-2$。 最坏情况的三角不等式检查减少到三重$(n-1, n, 2n-2)$。 自从$2n-1 > 2n-2$，在最严格的情况下，不等式严格成立。 所有其他三元组仅增加两个较小元素的总和或减少最大值，因此不会发生违规。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    n = int(input().strip())
    res = [str(i) for i in range(n - 1, 2 * n - 1)]
    print(" ".join(res))

if __name__ == "__main__":
    solve()
```实施直接遵循构建。 范围开始于$n-1$并结束于$2n-2$，精确地产生$n$价值观。 一次转换为字符串并连接可以避免重复的 I/O 开销，这对于$n = 10^5$。 

一个常见的错误是范围端点相差一。 正确的端点是包含在内的$2n-2$，在 Python 中对应于`range(n - 1, 2 * n - 1)`因为上限是排他的。

 ## 边缘情况

 对于最小的输入$n = 3$, 施工产出$2, 3, 4$。 唯一可能的三元组是全套本身，并且它满足$2 + 3 > 4$。 这证实了该构造在不存在灵活性的下边界处是有效的。 

对于大型$n$， 例如$n = 10^5$，序列跨度为$99999$到$199998$。 三角形条件仍然简化为最坏情况$99999 + 100000 > 199998$，这严格成立。 即使在最大规模下，线性增长也能确保最小总和与最大元素之间的差距保持在正确的方向上。
