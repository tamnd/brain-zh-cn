 ## 设置

 让$n=s+t$。 练习 64 考虑所有长度字符串的集合$n$确切地包含$t$星号和$s$数字，其中每个数字是$0$或者$1$。 因此，顶点集是选择一个的笛卡尔积$t$-星号位置的子集并将二进制标签分配给其余的$s$位置，因此顶点集有大小$N = 2^s \binom{n}{t}.$如果两个顶点在允许的变换之一上不同，则它们是相邻的$\ast 0 \leftrightarrow 0 \ast$,$\ast 1 \leftrightarrow 1 \ast$， 或者$0 \leftrightarrow 1$。 genlex 格雷路径是该图中的哈密顿路径，遵循练习 64 的 genlex 构造，并且 genlex 格雷循环是哈密顿循环。 

任务是枚举所有 genlex Gray 路径并确定其中有多少是循环。 

## 解决方案

 每个顶点由两个独立的组件组成。 其中一个组成部分是放置$t$星号，这是一个$(s,t)$-组合。 另一个组成部分是分配$0$和$1$到剩下的$s$职位。 

转变$\ast 0 \leftrightarrow 0 \ast$和$\ast 1 \leftrightarrow 1 \ast$仅修改单个星号的位置，同时保留该位置携带的数字。 这在约翰逊图上产生了邻接$(s,t)$- 组合，其中每次移动对应于交换$\ast$与相邻的数字位置。 

转变$0 \leftrightarrow 1$仅修改一个数字，同时保留$\ast$位置固定。 这会导致邻接$s$维超立方体$Q_s$。 

因此完整的图是笛卡尔积$G = J(n,t) \,\square\, Q_s,$在哪里$J(n,t)$是约翰逊图$(s,t)$- 组合和$Q_s$二元超立方体是$s$位。 

练习 64 通过同步哈密顿循环来构建 genlex Gray 循环$J(n,t)$哈密​​顿循环$Q_s$以便每个坐标在其因子周期前进时准确地变化。 一旦初始顶点固定，构造就是确定性的，因为在每一步，genlex 规则都会选择唯一的可接受的变换，以保持两个组件的字典顺序一致性。 

修复起始顶点。 genlex 规则确定每个顶点的唯一出边，因为允许的变换中字典顺序第一个适用的变换是由练习 64 的构造唯一定义的。这迫使遍历成为覆盖所有顶点的单个有向循环。$N$顶点。 反转所有选择会产生反向遍历，这又是一个有效的 genlex 灰色路径。 

没有其他自由了。 单个顶点的任何偏差都将违反构造中使用的字典条件，因为组合更新规则和位更新规则都是由第 7.2.1.3 节中的算法 L 及其练习 64 中的灰色循环细化所依据的“最右变化”原理唯一确定的。因此，每个 genlex 灰色路径都与构造的循环或其反转一致。 

这给出了完整顶点集上的两个有向哈密顿路径，对应于同一循环的两个方向。 

哈密​​顿循环与起点无关，因此两条有向路径表示相同的无向循环。 因此，恰好存在一个 genlex 格雷循环。 

## 验证

 的每个顶点$G$有学位$s+t$，因为在允许的变换下，每个数字位置和每个星号位置都可以进行一次移动。 genlex 构造在每一步选择一个输出边，生成一个 1-正则生成子图，因此是有向循环的不相交并集。 

因为构造在返回之前恰好访问每个顶点一次，所以生成子图由单个循环组成$N$顶点。 任何替代的 genlex 路径都需要在某个顶点有不同的出边，这与字典顺序确定的移动的唯一性相矛盾。 因此，全球范围内不存在分支选择。 

反转循环保留了邻接性和允许的变换，因此恰好存在两条有向哈密顿路径，并且它们仅在方向上有所不同。 因此无向循环的数量为$1$。
