 ## 解决方案
 ## 问题理解

 有`n`电梯内的人，包括您自己。 每个人都选择了目的地楼层。 在所有选定的目的地中，`m`出现不同的楼层。 

您可以自由选择自己的目的地楼层，您的目标是最大限度地增加包括您自己在内的人数在您所在楼层一起离开电梯的数量。 正是因为`m`必须代表不同的楼层，并将目的地分配给其他楼层`n - 1`人员可以通过满足此要求的任何有效方式进行分配。 

对于每个测试用例，我们仅给出总人数和不同目标楼层的数量。 我们必须计算可以离开与我们同一楼层的最大可能群体。 

最大输入值仅约为`10^4`，每个测试用例只包含两个整数。 即使有`10^4`测试用例，一个`O(1)`每个案例的解决方案总体上只执行一些算术运算。 任何更复杂的算法都是不必要的，因为答案仅取决于简单的计数参数。 

一个容易犯的错误是忘记每个不同的楼层必须至少有一名乘客。 

例如，```
n = 5
m = 5
```正确答案是`1`。 每个乘客必须选择不同的楼层，因此没有人可以分享您的目的地。 

另一个常见的错误是假设所有剩余的乘客始终可以选择您的楼层。 

例如，```
n = 6
m = 3
```正确答案是`4`， 不是`6`。 除了你自己的楼层之外，还必须有另外两个不同的楼层。 每个楼层必须至少分配一名乘客，这样您所在的楼层就只剩下四个人。 

最小的输入也值得关注。```
n = 1
m = 1
```答案是`1`。 你是唯一的乘客，所以大家一起离开。 

## 方法

 暴力的观点是想象分配所有`n`乘客正好`m`不同的楼层并检查每个有效的分布。 对于每个分布，我们记录分配给单个楼层的最大组，并保留所有可能性中的最大值。 

这种方法是正确的，因为它检查了每一个合法的分配。 不幸的是，此类分布的数量呈指数增长`n`，即使对于中等大小的输入也完全不切实际。 

关键的观察是我们永远不需要知道确切的分布。 为了最大限度地增加与我们一起离开的人数，每位没有被迫到其他楼层的乘客都应该选择我们的楼层。 

因为一定有确切的`m`不同的楼层，其中一层是我们的，其余的`m - 1`每个楼层至少需要一名乘客。 为其他每个楼层分配一名乘客。 这使用`m - 1`人们，离开```
n - (m - 1) = n - m + 1
```我们这一层的人，包括我们自己。 

不可能有更大的答案，因为每增加一个人搬到我们的楼层就会使所需的不同楼层之一空出。

 ## 算法演练

 1. 读取数值`n`和`m`。 
2. 为每人预订一名乘客`m - 1`不同的楼层。 这是仍然保留所有所需楼层的最小可能分配。 
3. 将所有剩余乘客安置在您的楼层上。 由于不存在其他约束，这会产生尽可能多的群体一起离开。 
4. 将答案计算为`n - m + 1`。 
5. 输出当前测试用例的结果。 

### 为什么它有效

 每项有效的分配必须在每个航班上至少包含一名乘客`m - 1`楼层和你的不一样。 这些乘客也不能与您一起离开。 这意味着至少`m - 1`人们被排除在您的小组之外，因此您的小组规模最多为`n - (m - 1)`。 该算法通过仅将一名乘客分配到每隔一个楼层并将其他所有乘客都安排在您的楼层上来准确地达到此值。 由于达到了上限，因此结果是最优的。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    t = int(input())
    out = []

    for _ in range(t):
        n, m = map(int, input().split())
        out.append(str(n - m + 1))

    sys.stdout.write("\n".join(out))

if __name__ == "__main__":
    solve()
```程序首先读取测试用例的数量。 每个测试用例都是独立处理的，因此用例之间不需要存储任何信息。 

整个解就是算术表达式`n - m + 1`。 这直接来自算法中开发的计数参数。 不需要在乘客或楼层上形成环路。 

使用`sys.stdin.readline`即使测试用例数量达到最大，也能保持输入快速。 答案会累积在一个列表中并在最后一起打印，从而避免了许多单独的输出操作。 

不存在溢出问题，因为 Python 整数具有任意精度，并且问题约束已经很小。 唯一的边界条件是当`m = n`，在这种情况下，公式正确计算为`1`。

 ## 边缘情况

 考虑尽可能最小的输入。```
1
1 1
```该算法计算`1 - 1 + 1 = 1`。 没有其他需要的楼层，因此单个乘客可以单独离开。 输出正确`1`。 

现在考虑每个乘客必须选择不同楼层的情况。```
1
5 5
```该算法计算`5 - 5 + 1 = 1`。 由于除了我们的楼层外还需要其他四个不同的楼层，因此其他每位乘客都必须选择其中一个楼层。 没有人可以分享我们的目的地，所以答案是正确的。 

最后，考虑只有一层的情况。```
1
8 1
```该算法计算`8 - 1 + 1 = 8`。 不需要额外楼层，因此每位乘客都可以选择我们的楼层。 所有八名乘客一起离开，这显然是最佳选择。
