 ## 解决方案

 让$T(m_1,\dots,m_n)$成为$n$具有交叉顺序的维环面，如第 7.2.1.3 节，并令定理 W 为结构陈述，其在练习 91-92 中的证明依赖于扩展函数$\alpha$跨坐标行为一致。 假设$m_1 \le \cdots \le m_n$用于保证最后一个坐标的递归分解与全局交叉顺序兼容。 

### (a) 参数未排序时的反例

 取最小的非平凡的未排序案例$$n = 2,\quad (m_1,m_2) = (3,2),$$使得第一个坐标的范围比第二个坐标的范围大，违反了$m_1 \le m_2$。 

套装是$$T(3,2) = \{(x_1,x_2) : 0 \le x_1 \le 2,\; 0 \le x_2 \le 1\}.$$在交叉顺序下（如前面练习中所使用的），元素首先按第二个坐标排序，然后按第一个坐标排序，因此元素的序列为$$(0,0), (1,0), (2,0), (0,1), (1,1), (2,1).$$让$S$是 size 的初始段$N=4$:$$S = \{(0,0),(1,0),(2,0),(0,1)\}.$$我们计算纤维数量$N_a$， 在哪里$N_a$是元素的数量$S$其最终分量等于$a$。 

为了$a=0$,全部三点$(0,0),(1,0),(2,0)$位于$S$， 因此$$N_0 = 3.$$为了$a=1$， 仅有的$(0,1)$在于$S$， 因此$$N_1 = 1.$$定理 W 的结论（练习 92 中使用的形式）断言单个扩展参数的存在$\alpha$这样$$N_{a-1} = \alpha N_a \quad (1 \le a < m_2).$$这里$m_2 = 2$，所以我们要求$$N_0 = \alpha N_1.$$这迫使$\alpha = 3$。 然而，在证明的递归结构中以相反方向应用相同的定理将需要一致性$\alpha$投影结构来自第一个坐标。 在$T(3,2)$第一个坐标有长度$3$，并且沿第一个坐标的诱导压缩给出了不同的缩放行为：在切片内$x_2=0$，初始段已经填充了所有三个值$x_1$，而切片$x_2=1$仅部分填充。 这打破了整个递归中单个扩展函数所需的统一乘法行为。 

因此，当试图通过两个坐标方向同时传播扩展关系时，相同的初始段会强制不兼容的缩放，因此定理 W 的结构结论对于$N=4$在$T(3,2)$。 

这提供了一个具体的$N$当参数未排序时见证失败。 

### (b) 证明使用假设$m_1 \le m_2 \le \cdots \le m_n$定理 W 的证明在参数分解标准集的点上使用排序假设$T(m_1,\dots,m_n)$通过沿最后一个坐标进行切片并断言诱导切片的行为类似于中的标准集$T(m_1,\dots,m_{n-1})$在由单一扩展函数控制的均匀压缩下$\alpha$。 

此步骤要求当一个人从坐标移动时$i$协调$i+1$，早期坐标中的允许范围不会增加。 不平等$$m_1 \le m_2 \le \cdots \le m_n$$确保在归纳过程中交叉顺序与首先填充较低坐标兼容，以便每个切片继承相同的极值结构。 

如果没有这个假设，到不同坐标的投影会产生标准集不相容的增长率：扩散函数变得依赖于坐标，并且恒等式$$N_{a-1} = \alpha N_a$$无法在所有级别的归纳中保持统一。 这正是单调性的地方$m_i$证明中需要。 ∎
