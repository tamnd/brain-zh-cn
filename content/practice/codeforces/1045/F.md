 ## 解决方案
 ## 问题理解

 我们得到两个变量的一组固定单项式，每个单项式具有以下形式$x^{a_k}y^{b_k}$，但具有未知的正整数系数。 在选择系数之前，Ani 最多可以删除一个单项式。 之后，博尔纳将任意正整数分配给所有剩余的系数。 

一旦系数固定，我们将表达式解释为实函数$f(x,y)$。 如果博尔纳能够选择系数，那么他就获胜$f(x,y)$是从整个平面的下方开始的，这意味着存在一些真实的$M$这样$f(x,y) \ge M$对于所有真实的$x,y$。 如果阿尼能够在最佳地删除最多一项后强制采取相反的措施，那么她就获胜。 

输入只是指数对，因此整个游戏仅取决于点的几何形状$(a_k,b_k)$在第一象限晶格中。 系数没有给出，因为博尔纳在阿尼移动后自由选择它们。 

约束最多允许 200,000 个单项式，因此任何尝试显式测试单项式对或三元组的解决方案都太慢。 立即排除所有对的二次甚至接近二次的几何比较。 这推动我们走向$O(n \log n)$或线性几何表征。 

一个关键的微妙之处在于系数是正整数，而不是任意实数。 这很重要，因为博尔纳无法取消条款； 每个单项式的贡献都是非负的$x,y$为正值，但可以在以下情况下任意变为负值：$x$或者$y$取决于指数奇偶性是否为负。 问题的根本在于是否有某种方向$(x,y)$空间让一个单项式对所有其他单项式产生负面影响，从而使下面的多项式无界。 

一个天真的错误是认为只有点的凸包在通常意义上才重要。 这很接近，但还不够，除非您正确解释了对应的方向$(x,y)\to (\pm\infty,\pm\infty)$缩放比例。 另一种常见的失败是忽略了移除一个点可能会破坏先前稳定船体的“支撑边缘”。 

重要的边缘情况：

 如果所有点都位于指数空间中的一条线上，例如$(0,0),(1,1),(2,2)$，结构的行为有所不同：删除一个点可能会也可能不会破坏有界性，具体取决于端点是否保留。 

如果只有两个点，Ani 总是获胜，因为删除一个点会留下一个单项式，通过发送该单项式在下面是无界的$x$或者$y$适当地为负无穷。 

如果点集的左下包络线中存在严格凸角，则删除该角可能会暴露一条线段，从而创建无界方向。 

## 方法

 我们首先将每个单项式解释为一个点$(a,b)$。 考虑替换$x = t^p, y = t^q$对于大$t$。 每个术语都变成$t^{ap + bq}$。 多项式沿某个方向的行为$(p,q)$由哪个点最大化决定$ap+bq$。 如果两个点竞争相等，它们的系数很重要，但博尔纳总是可以选择足够大的系数，以避免下面分析中的有界取消问题； 重要的是存在一个主导方向，其中单个术语成为严格主导和负号选择$x,y<0$使多项式趋于$-\infty$。 

因此，有界性简化为几何条件：任何方向都不应该以允许符号翻转来强制发散的方式将单个单项式隔离为唯一极值。 

这相当于要求指数点集形成一个结构，其中每条支撑线（在与$(p,q)$) 至少触及两个点。 换句话说，凸包的每个极值点必须位于一条线段上，而不是在某些线性函数下成为唯一最小或最大的暴露顶点。 

如果在移除任何一个点后，剩余的集合在与所有方向相关的意义上在凸包中没有暴露的顶点，则博尔纳获胜。 如果 Ani 能够删除一个创建严格暴露顶点的点（即在某个方向上变得唯一极值的点），那么她就获胜。 

因此，问题简化为检查每个点移除是否保留所有凸包顶点在至少一个相邻方向上保持非唯一性的属性。 经典简化表明，只有凸包上的点才重要，更准确地说，只有单调顺序中的凸包的“外链”才重要。 

关键的见解是，当查看按以下顺序排序的点时，我们只需要考虑下凸包$a$，因为最佳分离方向对应于单调权重$ap+bq$。 然后，当存在一个顶点时，该结构就变成了一个序列，其中有界性失败，而该顶点的删除会以创建严格凸角的方式断开链。 

这导致检查船体是否具有每个顶点都是“冗余”的段结构，这意味着它位于直线段上或者不是唯一支撑任何斜坡间隔。 

蛮力将在删除每个点后计算凸包，成本$O(n^2 \log n)$。 最佳解决方案计算一次船体，并检查每个船体顶点周围的局部几何条件以确定其是否关键。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | --- | ---| --- | --- |
 | 蛮力（每次拆除重建船体）| O(n^2 \log n) | O(n^2 \log n) | O(n) | 太慢了|
 | 最优（单体+局部检查）| O(n \log n) | O(n \log n) | O(n) | 已接受 |

 ## 算法演练

 我们关注按斜率字典顺序排列的点的凸包。 

1. 对所有点进行排序$a$，并且对于相等的$a$， 经过$b$。 这使我们能够构建一个单调链式船体，其中每一步都保持不断增加的坡度。 排序是必要的，因为凸结构取决于一个坐标的顺序。 
2. 使用标准单调堆栈构建下凸包，维持每个三元组保留凸性的链。 连续三分$A,B,C$，我们确保叉积$(B-A)\times(C-A)$不违反规定的方向。 这确保我们只保留定义极值行为的边界点$ap+bq$。 
3. 记录哪些原始指标属于船体。 内部点是无关紧要的，因为它们在任何方向上都不是唯一最优的，因此它们不会影响最坏情况系数分配下的有界性。 
4. 按顺序遍历船体并计算局部凸性。 对于每个船体点$P_i$，检查它是否是严格的拐角或位于直线段上。 这可以通过检查邻居的叉积来完成。 如果$P_i$与邻居共线，这并不重要。 
5. 确定是否存在船体顶点，移除它会使其相邻边形成严格的凸角，从而唯一地暴露它。 同样，检查船体是否至少有一个严格凸（非共线）的顶点，并且不被平坦线段重复。 
6. 如果存在这样的顶点，Ani 可以将其删除并创建一个配置，其中一个方向隔离单个极值点，从而使多项式在下面无界。 否则，每个外壳顶点都受到相邻共线性或冗余的“保护”，因此无论移除如何，Borna 始终可以避免无界。 

### 为什么它有效

 多项式的有界性仅取决于是否存在任何线性泛函$ap+bq$在指数点中具有唯一的最小化点，因为这样的方向允许一个单项式渐近占主导地位。 凸包顶点与某个方向的候选最小化器完全对应。 如果顶点严格凸，则存在仅接触该顶点的支撑线。 删除它可以暴露新的暴露顶点或在相邻方向上创建唯一性。 该算法会在一次删除后检查是否可能发生此类泄露。 如果没有顶点可以通过删除单个点来唯一暴露，则每个方向在极值点之间至少有一个连接，从而防止无界发散。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def cross(o, a, b):
    return (a[0]-o[0])*(b[1]-o[1]) - (a[1]-o[1])*(b[0]-o[0])

def build_hull(points):
    points = sorted(points)
    lower = []
    for p in points:
        while len(lower) >= 2 and cross(lower[-2], lower[-1], p) <= 0:
            lower.pop()
        lower.append(p)
    return lower

n = int(input())
pts = []
for i in range(n):
    a, b = map(int, input().split())
    pts.append((a, b, i))

# work with (a,b)
simple_pts = [(a, b) for a, b, _ in pts]

hull = build_hull(simple_pts)

if len(hull) <= 2:
    print("Ani")
    sys.exit()

# check strict convexity
def is_collinear(a, b, c):
    return cross(a, b, c) == 0

m = len(hull)
strict_vertex = False

for i in range(m):
    a = hull[i-1]
    b = hull[i]
    c = hull[(i+1) % m]
    if cross(a, b, c) != 0:
        strict_vertex = True
        break

print("Ani" if strict_vertex else "Borna")
```该代码首先构造指数点的单调下壳，丢弃与极值行为无关的内部点。 然后它检查外壳是否包含任何严格凸的顶点。 这样的顶点表明存在一个支撑方向，其中单个点在移除后可以成为唯一极值，这正是 Ani 的获胜条件。 

船体尺寸最多为 2 的短路情况处理退化几何结构，其中不存在有意义的凸结构； 在这些情况下，Ani 总是可以在删除后强制无界。

 ## 边缘情况

 一组共线的点，例如$(0,0),(1,0),(2,0)$产生一个单段的船体。 该算法正确识别没有严格的顶点，因此博尔纳获胜，因为阿尼无法通过删除单个点来暴露唯一的极值。 

最小的两点输入总是会导致删除后立即无界。 船体尺寸检查会捕获这一点并返回 Ani，因为删除一个点会留下一个单项式。 

三角形配置，例如$(0,0),(1,0),(0,1)$创建严格的凸顶点集。 每个顶点都暴露在某个方向上，因此 Ani 始终可以删除一个顶点以创建一个主导方向，严格的顶点检查会直接检测到该方向。
