 ## 解决方案
 ## 问题理解

 该情况描述了两名宇航员和黑洞系统引起的相对论时间差。 一名宇航员利奥 (Leo) 留在地球上，而另一名宇航员艾玛 (Ema) 则前往一颗遥远的星球，那里的时间过得更慢。 关键细节是，地球上的时间并不与地球时间一对一线性匹配，而是按固定因子拉伸。 

我们得到三个整数。 第一个是艾玛离开地球时狮子座的初始年龄。 第二个是时间膨胀因子，表明与地球相比，地球上的时间流逝得慢多少。 第三个是根据星球本身的时钟，艾玛在星球上度过了多少年。 

任务是确定艾玛返回地球时利奥的年龄，这完全取决于艾玛旅行期间地球时间过去了多少时间。 

关键的解释步骤是将在地球上度过的时间转换为地球时间。 如果地球上的一年对应于地球上的 X 年，则 Y 个行星年的持续时间对应于 X × Y 个地球年。 狮子座在整个地球期间正常衰老。 

输入限制非常小，所有值都以 100 为界。这立即排除了任何优化或高级数据结构的需要。 即使直接算术计算在恒定时间内也足够了。 

当乘法步骤处理不当时，就会出现微妙的失败情况。 例如，如果 X 和 Y 被视为浮点，或者在较大约束下可能出现整数溢出，则结果可能不正确。 然而，这里的值足够小，标准整数算术是安全的。 

另一个潜在的错误是误解时间膨胀方向。 短语“时间慢了X倍”可能会被错误地理解为除法而不是乘法。 如果有人计算 Y / X 而不是 X × Y，即使输入像 A = 20、X = 3、Y = 4 这样的简单输入，他们也会得到更小且不正确的结果，其中正确的地球时间贡献是 12，而不是 1。 

## 方法

 对这个过程的天真的解释是逐年模拟时间。 人们可以想象将艾玛的停留时间一次提前一个行星年，并通过乘以 X 将每个时间转换为地球时间，然后相应地增加狮子座的年龄。 这在逻辑上是可行的，但会在 Y 步骤上引入不必要的迭代。 每个步骤都执行恒定的工作，因此总复杂度为 O(Y)，这在这里仍然微不足道，但在概念上变得低效。 

关键的观察结果是从行星时间到地球时间的转换是线性且均匀的。 地球上的每一个时间单位都会扩展到地球上的 X 个单位。 由于这个固定的比例，整个时长可以通过一次乘法来转换，而不是重复累加。 

这将问题简化为直接的算术表达式：Leo 的最终年龄等于初始年龄加上 Ema 任务期间经过的总地球时间，即 X × Y。 

中间状态之间没有依赖性，因此不需要模拟或状态跟踪。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | --- | --- | --- | --- |
 | 逐步模拟| O(Y) | O(1) | O(1) | 已接受但没有必要 |
 | 直接计算 | O(1) | O(1) | O(1) | O(1) | 已接受 |

 ## 算法演练

 1. 读取初始年龄 A、时间膨胀因子 X 和行星持续时间 Y。这些是描述系统状态的独立标量值。 
2. 通过 X 和 Y 相乘，计算 Ema 任务期间经过的总地球时间。这种转换反映了两个时间尺度之间的物理关系。 
3. 将这个地球时间添加到狮子座的初始年龄 A. 狮子座的衰老速度与地球时间相同，因此每个地球年他的年龄都会增加一岁。 
4. 输出结果值作为Ema返回时刻Leo的年龄。 

## 为什么它有效

正确性依赖于时间膨胀在整个间隔内恒定的事实。 每个行星年恰好贡献 X 个地球年，因此从行星持续时间到地球持续时间的映射是线性缩放。 由于 Leo 的老化仅取决于地球时间，而不取决于 Ema 当地时间，因此结果完全由 A + X × Y 决定。没有中间交互或状态变化影响结果。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    a = int(input().strip())
    x = int(input().strip())
    y = int(input().strip())
    print(a + x * y)

if __name__ == "__main__":
    solve()
```该解决方案按照指定从不同的行读取三个整数。 乘法`x * y`直接执行，无需任何循环或转换。 结果被添加到`a`，已经代表了狮子座的初始年龄。 

一个常见的实现错误是尝试从一行读取所有值，但输入格式显式地将每个值放在自己的行上，因此每次调用`input()`必须单独处理。 

另一个细节是确保乘法发生在加法之前，尽管运算符优先级使表达式`a + x * y`自然正确。

 ## 边缘情况

 直接的极端情况是时间膨胀因子为零。 在这种情况下，无论艾玛在地球上停留多久，地球时间都不会过去。 用于输入`A=10, X=0, Y=100`，计算得出`10 + 0 = 10`，这意味着 Leo 在任务间隔期间不会衰老。 

另一种情况是艾玛没有在地球上度过任何时间。 为了`A=10, X=5, Y=0`，地球时间贡献变为零，狮子座年龄保持10岁不变。 

最终的边界情况是当所有值在约束内都是最大值时，例如`A=100, X=100, Y=100`。 相乘得到 10000 个地球年，最终年龄为 10100，这仍然很容易符合整数限制，并表明在此问题设置中不需要溢出预防措施。
