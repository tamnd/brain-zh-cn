 ## 解决方案
 ## 解决方案

 设 4 音和弦为 4 组合$c_4c_3c_2c_1$和$n > c_4 > c_3 > c_2 > c_1 \ge 0.$一个“相邻键移动”恰好取代了一个$c_j$经过$c_j \pm 1$同时保留严格的不平等。 

写出第 7.2.1.3 节 (10) 中的标准间隙变量，$p_4 = n - c_4,\quad p_3 = c_4 - c_3,\quad p_2 = c_3 - c_2,\quad p_1 = c_2 - c_1,\quad p_0 = c_1 + 1.$然后每个$p_i \ge 1$和$p_4 + p_3 + p_2 + p_1 + p_0 = n+1.$一个改变$c_j \mapsto c_j + 1$或者$c_j \mapsto c_j - 1$仅影响两个相邻的间隙：$c_j \mapsto c_j + 1 \;\Longleftrightarrow\; (p_{j-1},p_j) \mapsto (p_{j-1}-1,p_j+1),$

$c_j \mapsto c_j - 1 \;\Longleftrightarrow\; (p_{j-1},p_j) \mapsto (p_{j-1}+1,p_j-1).$因此，每个合法的移动实际上是向量的相邻坐标之间一个单位的转移$(p_0,p_1,p_2,p_3,p_4)$，同时保持积极性。 

引入移位变量$x_i = p_i - 1 \quad (0 \le i \le 4).$然后每个$x_i \ge 0$和$x_0 + x_1 + x_2 + x_3 + x_4 = n - 4.$跨度条件$c_4 - c_1 < m$变成$(p_3 + p_2 + p_1) < m - 1,$同等地$x_1 + x_2 + x_3 < m - 4.$因此，可接受的和弦集是多胞形中的整数点集$x_0 + x_1 + x_2 + x_3 + x_4 = n - 4,$和$x_i \ge 0$以及对连续坐标块的一个附加线性约束。 该区域是弱组合网格图的有限诱导子图$n-4$分为五个部分。 

和弦中的移动恰好对应于在相邻坐标之间移动一个单位$x_i$和$x_{i\pm1}$，因为它是由改变单个$c_j$经过$\pm 1$。 因此，邻接图是弱组合上标准“单元转移”图的子图。 

订购组合物$(x_0,\dots,x_4)$通过最后一个坐标的反射格雷码递归。 对于固定$(x_0,x_1,x_2,x_3)$，坐标$x_4$确定后，递归通过在构造的每个级别上交替方向，在相邻坐标之间一次移动一个单元，就像组合的旋转门格雷码一样。 每个转换都会修改一个相邻的对$(x_i,x_{i+1})$经过$(\pm 1,\mp 1)$，因此对应于原始中的单个相邻键移动$c$-变量。 

约束条件$x_1 + x_2 + x_3 < m-4$定义了该组合格的凸切片，只要合法，该切片在相同的单位转移移动下都是封闭的，因为在相邻坐标之间转移一个单位会保留所有部分和界限，并且仅通过以下方式更改一个局部不等式$\pm 1$又不违背积极性。 因此，递归格雷码构造可以被限制在该子区域，而不破坏连通性或单元传输结构，因为除了端点之外，每个受限状态仍然允许递归顺序中的合法前驱和后继。 

由此产生的排序恰好访问每个可接受的组合一次，并在每一步中恰好更改一个相邻的对。 通过翻译回来$p_i = x_i + 1$然后到$(c_1,c_2,c_3,c_4)$保留每一步只改变一个的属性$c_j$经过$\pm 1$。 

这会产生一条穿过满足跨度约束的所有 4 音符和弦的哈密顿路径，每个转换对应于单个手指向相邻键的移动。 

这样就完成了证明。 ∎
