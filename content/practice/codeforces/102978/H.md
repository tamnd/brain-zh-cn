 ## 解决方案
 ## 解决方案

 一片混乱${0,1,\dots,n}$正是布尔格中的反链：一个族$\mathcal C$的子集，使得没有两个不同的成员满足$\alpha \subset \beta$。 大小向量$(M_0,M_1,\dots,M_n)$记录有多少成员$\mathcal C$位于每个级别$\binom{[n]}{t}$。 

### (a) 部分：特征描述

 让$\mathcal C_t \subseteq \binom{[n]}{t}$是以下的亚科$t$-设置在混乱中，与$|\mathcal C_t| = M_t$。 杂波的定义条件是对于所有$s<t$, 没有设置$\mathcal C_s$包含在任意集合中$\mathcal C_t$。 

等价地，对于每对$s<t$，家人满意$$\mathcal C_s \cap \nabla(\mathcal C_t) = \varnothing,$$在哪里$\nabla(\mathcal C_t)$表示向上闭合$\mathcal C_t$。 

因此，当且仅当存在不相交族时，大小向量才是可行的$$\mathcal C_t \subseteq \binom{[n]}{t}, \quad |\mathcal C_t| = M_t,$$使得没有成员$\mathcal C_s$是任何成员的子集$\mathcal C_t$为了$s<t$。 

这个条件是精确的：任何这样的选择都会产生混乱，而每个混乱都会产生这样的分解。 

这样就完成了特征描述。 

∎

 ### (b) 部分：枚举$n=4$写出以下级别$\binom{[4]}{t}$作为$$|\binom{[4]}{0}|=1,\quad |\binom{[4]}{1}|=4,\quad |\binom{[4]}{2}|=6,\quad |\binom{[4]}{3}|=4,\quad |\binom{[4]}{4}|=1.$$可行的大小向量对应于反链，因此任何选定的集合都不能包含另一个选定的集合。 

＃＃＃＃ 等级$0$空集包含在每个非空集中，因此如果$M_0=1$然后所有其他$M_t=0$。 这产生向量$$(1,0,0,0,0).$$从现在开始假设$M_0=0$。 

#### 纯级反链

 任何单个级别本身就是一个反链，因此以下是可行的：$$(0,1,0,0,0),\ (0,2,0,0,0),\ (0,3,0,0,0),\ (0,4,0,0,0),$$

$$(0,0,1,0,0),\dots,(0,0,6,0,0),$$

$$(0,0,0,1,0),\dots,(0,0,0,4,0),$$

$$(0,0,0,0,1).$$这些对应于单个级别内的任意子集，因为在固定基数内不会发生包含。 

#### 混合级别$1$和$2$一个$2$-set 恰好包含两个$1$-套。 如果一个家庭包含$M_2$清楚的$2$-集合，它们的并集至少有大小$M_2+1$（通过星形配置实现）最多$2M_2$（如果不相交达到大小限制$4$）。 

该联合中的每个元素都禁止包含相应的单例。 

因此，单例的最大可能数量最多为$$M_1 \le 4 - (M_2+1) = 3 - M_2.$$这个界限对于所有可行的情况都是可以实现的$M_2$通过采取一颗星$2$-套。 因此，级别之间的混合级别约束$1$和$2$正是$$0 \le M_2 \le 3,\quad 0 \le M_1 \le 3 - M_2.$$#### 混合级别$2$和$3$通过互补下的对称性$[4]$, 一个$3$-set 恰好包含一个互补项$1$-设置并对应于先前的情况。 一个$3$-套装包含三个$2$-子集，因此对称参数给出$$0 \le M_3 \le 3,\quad 0 \le M_2 \le 3 - M_3.$$#### 混合级别$1$和$3$如果一个$3$-set被选择，它包含三个单例。 如果$M_3>0$，至少有 3 个顶点被占用，因此最多有一个单例可以保持不相交。 前面的不平等已经暗示了这一点。 

＃＃＃＃ 等级$4$如果$M_4=1$，那么其他所有级别都必须为零，因为$[4]$包含所有子集。 

如果$M_4=0$，不再产生进一步的限制。 

#### 最终枚举

 所有可行的大小向量都满足以下条件：$$M_0 \in \{0,1\}, \quad \text{and if } M_0=1 \text{ then } M_1=M_2=M_3=M_4=0,$$否则$M_0=0$和$$0 \le M_2 \le 3,\quad 0 \le M_3 \le 3,$$

$$0 \le M_1 \le 3 - M_2,\quad 0 \le M_3 \le 3 - M_2,$$连同上面已经包含的对称约束，以及$M_4 \in {0,1}$。 

具体来说，非平凡的可行向量（其中$M_0=0$) 是通过选择任何可接受的对获得的所有整数解$(M_2,M_3)$和$0\le M_2,M_3\le 3$和$M_2+M_3\le 3$，然后选择$$0 \le M_1 \le 3 - M_2,\quad 0 \le M_3 \le 3 - M_2,$$没有设定尺寸的一致性$3$可以在其支持内与单例共存。 

与纯级别案例和单例一起$(0,0,0,0,1)$，这些耗尽了杂波的所有大小向量$4$元素。 

这样就完成了解决方案。 ∎
