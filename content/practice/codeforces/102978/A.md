 ## 解决方案
 ## 问题理解

 我们被要求计算有多少个大小相同的矩阵$N \times M$可以用整数填充$1$到$K$这样当向右或向下移动时，值永远不会减少。 换句话说，每行从左到右非递减，每列从上到下非递减，因此矩阵在两个方向上都是单调的。 

一个特定的细胞$(R, C)$预先固定才具有价值$V$，并且所有有效矩阵也必须遵守此约束。 任务是计算此类单调矩阵的模数$998244353$。 

限制条件$N, M \le 200$和$K \le 100$立即排除任何矩阵的直接枚举，甚至排除所有配置上的行动态规划。 单行已经有$K^M$没有单调性约束的可能性，因此暴力破解远远不可行。 

一个更微妙的约束是固定单元格。 没有它，这个问题已经是一个经典的二维单调填充计数问题。 固定值引入了全局约束，将结构分成两个相互作用的区域。 

查看边缘情况的一个有用方法是考虑极端放置$(R, C)$。 

如果$R = C = 1$，左上角是固定的。 这迫使所有条目至少$V$，因为单调性会向下和向右传播值。 一种简单的方法，在局部处理约束$(R, C)$但忽略传播仍会错误地对其他地​​方出现较小值的矩阵进行计数。 

如果$R = N$和$C = M$，右下角是固定的。 那么上面和左边的所有内容最多都必须是$V$。 同样，任何不通过单调性全局传播约束的方法都会错误计数。 

另一个微妙的失败案例出现在$V = 1$或者$V = K$。 为了$V = 1$，西北地区的一切都被逼得紧紧的，而对于$V = K$，东南方的一切都被逼的紧紧的。 任何假设围绕任意值对称而不考虑边界的方法都可能会重复计算或错过这些边界饱和度。 

## 方法

 蛮力解释是生成所有$N \times M$网格中的值$[1, K]$，然后按行和列检查单调性并强制执行固定单元格。 即使进行了修剪，每个前缀仍然分支为最多$K$每个单元的选择，因此总状态空间增长如下$K^{NM}$，即使对于$N = M = 10$。 该结构过于刚性，局部回溯无法切割足够的分支。 

关键的观察是单调矩阵相当于一堆嵌套的“阈值层”。 对于每个值$t$，考虑至少具有值的单元格集合$t$。 因为矩阵在两个方向上都是不减的，所以这些集合形成了类似杨图的形状，嵌套为$t$增加。 每层都是单调点阵路径结构，层与层之间不交叉。 

固定单元约束转化为有多少层穿过或位于该位置上方的条件。 我们不再考虑单个单元格的值，而是将矩阵重新解释为$K$相互作用的单调边界分隔价值水平。 

这将问题转化为计算非相交晶格路径，并在固定点附加约束。 此类结构的标准工具是 Lindström-Gessel-Viennot (LGV) 框架，其中非相交路径族的计数表示为行列式。 

如果没有固定的约束，我们将构建$K-1$表示值之间边界的路径$1,2,\dots,K$。 固定细胞力精确地$V-1$这些边界处于特定的位置关系$(R, C)$。 该条件可以通过引入标记变量并提取系数以代数方式强制执行，从而将约束转变为行列式内的受控权重分配。 

因此，最优解将组合对象简化为多项式值行列式评估问题。 自从$K \le 100$，我们可以计算多个点的行列式值并插入与所需层数相对应的系数。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 暴力枚举|$O(K^{NM})$|$O(NM)$| 太慢了 |
 | LGV + 多项式评估 |$O(K^2(N+M) + K^4)$|$O(K^2)$| 已接受 |

 ## 算法演练

 1. 用水平集替换原来的矩阵解释。 不要直接跟踪值，而是考虑每个阈值$t$至少具有值的单元格区域$t$。 这将矩阵转换为$K-1$嵌套的单调形状。 
2. 将连续级别之间的每个边界建模为从网格左下边界到右上角边界的单调网格路径。 矩阵的单调性保证这些路径永远不会相交。 
3. 使用平移坐标系将非相交条件转换为独立路径系统。 每条路径都是偏移的，以便相交约束成为 DAG 上的标准不相交约束。 
4. 使用 LGV 行列式对有效路径族的数量进行编码，其中每个条目对网格图中相应起点和终点之间的路径进行计数。 
5. 纳入约束$a_{R,C} = V$将其解释为对相对于该点的边界路径数量的限制$(R,C)$。 这是通过分配符号权重来强制执行的$x$跨越某个区域的边缘，导致“低于”该点。 
6. 将结果矩阵的行列式计算为多项式：$x$。 系数为$x^{V-1}$与正确数量的层穿过相关区域的配置完全对应。 
7. 在几个不同的值处评估这个多项式行列式$x$，通过插值重建它，并提取所需的系数模$998244353$。 

### 为什么它有效

 正确性取决于每个有效的单调矩阵唯一对应于以下配置的事实：$K-1$不相交的晶格路径，反之亦然。 LGV 行列式精确地计算了这些配置，因为它扩展了路径端点的排列，其中非相交族是唯一贡献非零权重的族。 

固定细胞条件不会破坏这种双射； 它仅通过对相对于固定几何分隔符的路径数量施加全局条件来限制允许哪些族。 通过权重变量对此限制进行编码，将全局组合约束转换为行列式内的系数提取问题，在保持精确计数的同时保持代数上的易处理性。

 ## 边缘情况

 当固定单元处于$(1,1)$，单调性迫使整个矩阵中的所有条目至少$V$。 在路径解释中，这意味着每个边界必须严格位于原点上方，仅留下所有层向上移动的配置。 LGV 公式仍然正确计数，因为所有起点路径均统一移动，保留了非交叉结构。 

当固定单元处于$(N,M)$，所有条目最多必须是$V$。 在路径图中，这迫使所有边界曲线位于终端角下方，从而有效地折叠上层。 该位置的多项式权重变得微不足道，因此只有与零交叉相对应的系数幸存下来，与正确的计数相匹配。 

什么时候$V = 1$，系数提取目标$x^0$，这意味着没有路径贡献标记区域的权重。 这将多项式行列式折叠为未加权的 LGV 计数，并且算法无需修改即可简化为基本情况。 

什么时候$V = K$，提取的目标是最高功率，这意味着所有边界都被迫进入加权区域。 这将固定点的解释从下限约束翻转为上限饱和条件，但多项式表示仍然捕获它，因为所有有效配置一致地贡献最大指数权重。
