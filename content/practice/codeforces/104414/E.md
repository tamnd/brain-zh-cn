 ## 解决方案
 ## 问题理解

 我们正在使用星形图$n$节点，其中一个节点作为中心，其余节点$n-1$节点是叶子。 每片叶子都与固定的“原始油漆颜色”相关联，并且有确切的$n-1$颜色不同，每片叶子一个。 

最初，每个节点都是未着色的。 我们被允许进行两种类型的操作。 首先，我们可以激活一片叶子并为其分配自己的预定义颜色。 其次，一旦一个节点已经着色，我们可以将其颜色复制到相邻节点。 这意味着颜色可以沿着边缘流动，但只能从已经着色的顶点向外流动。 

如果每个节点在这些操作的某些序列之后最终都被着色，则着色被认为是有效的。 如果至少一个节点具有不同的颜色，则两种最终颜色被认为是不同的。 

任务是计算在这些规则下可以产生多少种不同的最终完整着色，模数$10^9 + 7$，对于多个值$n$， 在哪里$n$可以大到$10^7$测试用例的数量可以大到$10^5$。 

该约束立即告诉我们，任何依赖于迭代节点或模拟操作的解决方案都是不可能的。 甚至$O(n)$每个测试用例都会太慢。 答案必须简化为封闭式表达式，可以在每个查询的恒定时间内进行计算，可能涉及模幂或直接组合公式。 

这个过程中的一个微妙之处是颜色在一开始就不能自由分配； 它们仅源自叶子。 然而，一旦叶子被激活，它的颜色就会扩散到中心并可能覆盖其他叶子。 这种交互使得推理约束变得非常重要，因为颜色可以传播并随后再次被覆盖。 

暴露混乱的一种边缘情况是$n=3$。 有两片具有固定颜色的叶子。 即使在这种最小的非平凡情况下，中心也可以暂时采用一片叶子的颜色，然后切换到另一片叶子，从而根据时间产生不同的最终配置。 这表明最终的结构不是简单的单源传播，而是允许多个独立的覆盖。 

## 方法

 强力解释将模拟所有可能的操作序列。 在每一步中，我们要么激活一片叶子，要么沿着边缘传播颜色。 这构建了一个巨大的状态空间，其中每个节点可以处于多个中间状态，并且转换取决于先前的选择。 即使对于小$n$，序列的数量呈指数增长，因为我们正在有效地探索排序颜色传播和重新着色的所有方法。 

关键的结构观察是该图是一个星形，因此叶子之间的每个相互作用都必须经过中心。 这意味着中心是唯一的颜色混合点。 一旦叶子颜色到达中心，它就可以重新分配到任何其他叶子。 相反，任何叶子稍后都可以重新引入其原来的颜色。 

这产生了解耦效果：每个节点的最终颜色不受固定传播树的约束，因为颜色可以通过中心任意多次重新引入。 因此，每个节点的行为几乎独立于其最终的颜色选择，只要该颜色是其中之一$n-1$叶子的颜色。 

这将全局组合结构简化为一个简单的计数问题：每个$n$节点可以独立地结束于任何$n-1$可用颜色。 除了由叶子颜色着色之外，该中心没有特殊限制，并且叶子不限于保留其原始颜色。 

因此，有效着色的总数变成了纯粹的分配计数。

 ## 算法演练

 我们现在将结构转化为直接计算。 

1. 观察是否有$n-1$提供不同的颜色，每种颜色都来自一片叶子。 
2. 认识到通过重复使用该中心作为传输枢纽，任何节点最终都可以获得其中任何一个$n-1$颜色。 
3. 得出结论，最终状态不取决于操作顺序，仅取决于节点颜色的最终分配。 
4. 对于每一个$n$节点，独立选择其中之一$n-1$可用颜色。 
5. 在所有节点上乘以选择，产生$(n-1)^n$。 
6. 返回该值的模数$10^9 + 7$。 

基本的推理步骤是中心不限制全局一致性。 在传播过程中创建的任何临时结构都可以在以后被覆盖，这意味着叶子之间不存在持久的依赖关系图。 

### 为什么它有效

 关键的不变量是每种颜色都源自一片叶子，并且可以随时重新引入系统。 由于中心可以任意多次重新着色，然后用于对任何叶子重新着色，因此任何节点都不会永久受到早期传播选择的限制。 这消除了最终节点颜色之间的任何依赖性，因此最终配置空间正是从节点到$n-1$可用颜色。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

MOD = 10**9 + 7

def modpow(a, e):
    res = 1
    while e:
        if e & 1:
            res = res * a % MOD
        a = a * a % MOD
        e >>= 1
    return res

t = int(input())
for _ in range(t):
    n = int(input())
    print(modpow(n - 1, n))
```该实现将整个问题简化为快速模幂运算。 每个测试用例计算$(n-1)^n \bmod (10^9+7)$。 二进制求幂函数确保即使对于$n$最多$10^7$，计算仍然有效。 

A common pitfall here is mixing base and exponent, since both depend on$n$。 另一个微妙的问题是确保仅针对中间乘法在模下完成求幂，而不是针对指数本身。

 ## 边缘情况

 对于$n=2$，只有一片叶子和一个中心。 公式给出$1^2 = 1$，意味着只有一种可能的颜色。 这符合两个节点最终都必须具有唯一可用颜色的事实。 

对于较大的$n$， 例如$n=10^7$，计算结构不变。 求幂以对数时间运行，并且没有中间状态依赖于超出算术参数的图形大小，因此即使在极端输入限制下，算法也保持稳定。 

独立性假设在这些极端情况下也成立，因为星形结构确保所有相互作用仍然通过中心漏斗，而不会在叶子之间引入额外的约束。
