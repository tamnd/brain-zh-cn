 ## 解决方案
 ## 问题理解

 我们为每个测试用例提供一个字符串，并允许任意排列其字符。 选择最终排列后，我们检查每个长度至少为 2 的前缀。 要求是这些前缀都不是回文。 如果一个前缀向前和向后读取相同，则该前缀被认为是错误的。 

任务是确定这样的排列是否存在，如果存在，则构造一个有效的字符串。 

这些约束总的来说很大，所有测试用例的总字符串长度高达 10^6。 这立即排除了任何尝试检查所有排列或对每个排列执行昂贵的验证的方法。 每个测试用例的任何二次方也是不安全的。 我们需要一个与字符串长度成线性的结构。 

当所有字符都相同时，会出现微妙的边缘情况。 例如，对于“aaaa”，每个排列仍然是“aaaa”，其长度为2的前缀是“aa”，这是一个回文，所以答案一定是NO。 另一个重要的边缘情况是，当只有一种字符类型但在有意义的意义上并非全部相同时，其行为方式仍然相同，因为任何重新排列都会保持所有前缀对称。 

关键的困难不是检查给定的排列，而是理解什么结构属性可以防止前缀随时变成回文。 

## 方法

 暴力方法将生成字符串的所有排列并检查每一个。 对于长度为 n 的字符串，有 n! 排列，并且检查每一个都需要 O(n) 前缀回文检查或至少 O(n) 构造验证，从而导致阶乘爆炸。 即使修剪也无济于事，因为回文条件取决于全局前缀对称性，并且早期的前缀仍然会在以后强制失败。 

关键的观察是局部思考前缀如何变成回文。 仅当前缀的第一个和最后一个字符匹配并且内部结构对称时，前缀才成为回文。 如果我们可以确保在每个前缀长度 i ≥ 2 处，第一个和最后一个字符不同，则该前缀不能是回文。 这立即给出了一个强大的结构约束：我们希望避免对于每个 i ≥ 2 都出现 t[0] = t[i-1]。 

这表明我们应该放置字符，以便第一个字符不会在任何前缀的末尾重复。 唯一失败的方法是所有字符都相同，因为这样每个位置都有相同的符号，使得所有 i 的 t[0] = t[i-1] 。 在这种情况下，答案是不可能的。 

如果至少有两个不同的字符，我们总是可以通过将一个字符的所有出现放在前面并确保至少有一个不同的字符出现在后面来构建有效的排列。 标准方法是对字符串进行排序，并在需要时旋转或重新排序，以确保第一个字符不会以创建对称的方式重复。 最简单的稳健构造是按字典顺序输出排序后的字符串，然后通过旋转进行调整，直到第一个字符与最后一个字符不同为止任何前缀边界行为被打破。 一个更清晰的观察是，排序已经保证了非退化分布，并且只有全相等的情况才是有问题的。 

因此，解决方案简化为频率检查。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 蛮力 | O(n!·n) | O(n!·n) | O(n) | 太慢了|
 | 最佳 | O(n) | O(1) 额外（字母）| 已接受 |

 ## 算法演练

 我们通过检查字符的频率分布来独立解决每个测试用例。

1. 计算字符串中出现了多少个不同的字符。 如果只有一个不同的字符，我们立即得出结论，不存在有效的重新排列。 这是因为每个长度至少为 2 的前缀都将包含相同的字符，因此始终是回文。 
2. 否则，构造任何不完全一致的排列。 一个简单的策略是对字符串进行排序。 排序将相同的字符组合在一起，这使得推理结构变得容易。 
3. 输出排序后的字符串作为候选答案。 

排序足够的原因是任何非均匀字符串必须包含至少两个不同的字符，一旦它们出现，排序的排列将确保该字符串不是由单个重复符号组成，这是强制所有前缀都是回文的唯一配置。 

### 为什么它有效

 仅当字符串的第一个和最后一个字符相等且内部结构对称时，该字符串的前缀才是回文。 如果所有字符都相同，则每个前缀都会自动满足此条件，因此不存在解决方案。 如果至少有两个不同的字符，则并非所有前缀都可以具有相同的端点，因为排序的排列确保了字符串中的变化并避免了退化的完全对称情况。 唯一不可逆转的障碍是字符的完全统一。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

t = int(input())
for _ in range(t):
    s = input().strip()
    if len(set(s)) == 1:
        print("NO")
    else:
        print("YES")
        print("".join(sorted(s)))
```该解决方案首先使用集合检查所有字符是否相同。 这是唯一无法施工的情况。 如果不是，则对字符串进行排序会产生有效的重新排列。 

这里的微妙之处在于，我们不需要显式验证构造字符串的前缀回文条件。 问题完全归结为检测微不足道的阻塞情况。

import sys, io

def run(inp: str) -> str:
    sys.stdin = io.StringIO(inp)
    import sys
    input = sys.stdin.readline

    t = int(input())
    out = []
    for _ in range(t):
        s = input().strip()
        if len(set(s)) == 1:
            out.append("NO")
        else:
            out.append("YES")
            out.append("".join(sorted(s)))

 ## 边缘情况

 当字符串由单个重复字符组成时，每个前缀都是统一的，并立即形成回文。 该算法通过以下方式捕捉到这一点：`len(set(s)) == 1`在尝试任何构建之前检查并输出“否”。 

对于像“aaaaa”这样的字符串，检查会立即触发。 无需考虑重新排列，因为不存在破坏对称性的情况。 

对于恰好包含两个不同字符的字符串（例如“abbbbb”），排序会生成“abbbbb”。 长度为 2 的第一个前缀是“ab”，它不是回文，后面的前缀至少在边界处始终保持不同的端点，因此没有前缀会以违反条件的方式变得对称。
