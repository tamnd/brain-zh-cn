 ## 解决方案
 ## 设置

 让$G_{s,t}$表示顶点都是长度子立方体的图$s+t$拥有$s$中的数字${0,1}$和$t$星号，其边缘由变换给出$\ast 0 \leftrightarrow 0\ast$,$\ast 1 \leftrightarrow 1\ast$， 和$0 \leftrightarrow 1$。 

genlex 灰色路径是哈密顿路径$G_{s,t}$它源自练习 64 中的 genlex 构造，这意味着在每一步，后继者都是由归纳表示的字典顺序确定的（交换最左边可用的相邻对，或者在不可能交换时翻转数字），与第 7.2.1.3 节中算法 L 的底层构造完全相同。 

genlex 格雷循环是由该构造的循环版本生成的同一图中的哈密顿循环。 

任务是确定 genlex 灰色路径的总数$G_{s,t}$并确定其中有多少是循环。 

## 解决方案

 的每个顶点$G_{s,t}$唯一地分解为两个独立的组件。 第一个组成部分是删除星号后得到的二进制字，是${0,1}^s$。 第二个组成部分是星号的位置模式，它对应于$(s,t)$- 组合并由一个单词表示${0,1}^{s+t}$恰好与$t$，相当于第 7.2.1.3 节中的网格路径表示。 

在这种分解下，变换$\ast 0 \leftrightarrow 0\ast$和$\ast 1 \leftrightarrow 1\ast$通过交换相应格路径表示中的相邻步骤，仅更改组合分量，而变换$0 \leftrightarrow 1$仅更改二进制数字部分。 没有任何举动可以混合这两种效果。 由此可见$G_{s,t}$与笛卡尔积同构$$G_{s,t} \cong Q_s \,\square\, C_{s,t},$$在哪里$Q_s$是$s$长度二进制串上的维超立方体$s$， 和$C_{s,t}$是的图$(s,t)$- 相邻交换操作下的组合，相当于第 7.2.1.3 节中算法 L 的底层图。 

笛卡尔积中的哈密顿路径$G \square H$由哈密顿路径的交错确定$G$和哈密顿路径$H$，因为产品中的每一步都精确地更改一个坐标，同时保留另一个坐标。 如果哈密顿路径为$G$用途$|V(G)|-1$边和哈密顿路径$H$用途$|V(H)|-1$边，然后每个哈密顿路径$G \square H$对应于这两个边缘序列的洗牌，保留每个因子内的内部顺序。 

在本案中，$$|V(Q_s)| = 2^s, \qquad |V(C_{s,t})| = \binom{s+t}{t}.$$哈密​​顿路径$Q_s$正好有$2^s-1$边，以及哈密顿路径$C_{s,t}$正好有$\binom{s+t}{t}-1$边缘。 因此任何哈密顿路径$G_{s,t}$对应于两个固定长度序列的洗牌$2^s-1$和$\binom{s+t}{t}-1$，产生长度序列$$(2^s-1) + \left(\binom{s+t}{t}-1\right) = 2^s\binom{s+t}{t} - 1.$$练习 64 中的 genlex 规则在每个顶点通过确定性优先级固定下一步移动：在允许的交换和翻转之间按字典顺序排列的第一个可接受的变换。 该规则产生了一个独特的后继函数$G_{s,t}$，因为在每个顶点，选择是由交换的第一个位置唯一确定的$\ast 0 \to 0\ast$或者$\ast 1 \to 1\ast$可以按字典顺序排列； 否则唯一的剩余可接受的移动是$0 \leftrightarrow 1$。 由于后继在各处都是唯一定义的，因此由该规则导出的有向图具有出度$1$在每个顶点。 

每个顶点都有出度的有限有向图$1$分解为有根树的定向循环。 genlex 构造产生哈密顿遍历，因此没有顶点可以位于非平凡的内树中； 每个顶点必须位于从初始配置开始的唯一定向轨道上。 因此，后继规则根据最后是否重新访问起始顶点来定义单个有向循环或单个有向路径。 

genlex 构造中的初始配置是固定的（字典顺序最小的子立方体），因此确定性后继函数生成一次覆盖所有顶点的单个轨道。 因此 上恰好存在一条 genlex 灰色路径$G_{s,t}$。 

对于循环性，相同的确定性后继规则要么关闭轨道，要么只留下一个没有返回初始状态边缘的顶点。 练习 64 中的构造明确地产生了所有的循环$s,t$，因此后继函数与周期性边界闭合一致。 由于后继函数是唯一的，任何 genlex 循环都必须与该规则产生的循环一致。 因此，恰好存在一个 genlex 格雷循环。 

## 验证

 分解为$Q_s \square C_{s,t}$是合理的，因为每个允许的移动要么只改变数字位置，要么只改变星号位置，并且没有移动同时改变这两个部分。 这种分离在笛卡尔积结构中产生独立的坐标更新。 

每个因子中顶点的计数与第 7.2.1.3 节一致，因为长度的二进制字符串$s$是$2^s$， 和$(s,t)$- 组合是$\binom{s+t}{t}$。 

笛卡尔乘积中哈密顿路径的洗牌解释源于这样一个事实：乘积中的每条边都投影到一个因子，因此任何哈密顿遍历都会产生两个单调边序列，它们的交错唯一地重建路径。 

genlex 规则的确定性遵循可接受局部变换的词典优先顺序，其结构与算法 L 相同，算法 L 在每一步分配唯一的下一步。 这消除了后继关系中的分支，从而强制全局遍历的唯一性。 

练习 64 中循环的存在确保了确定性后继轨道是封闭的，并且后继规则的唯一性可防止出现额外的循环。 

## 结果

 子立方体上的 genlex 灰色路径总数为$$\boxed{1},$$这些循环路径的数量是$$\boxed{1}.$$
