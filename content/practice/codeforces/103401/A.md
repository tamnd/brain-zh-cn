 ## 解决方案
 ## 设置

 让$a_{s+t-1}\dots a_1a_0$是一个的二进制表示$(s,t)$- 组合，所以每个$a_i \in {0,1}$和$\sum a_i = t$。 

长度前缀的旋转$j+1$是转变$$a_j a_{j-1}\dots a_0 \;\leftarrow\; a_{j-1}\dots a_0 a_j,$$所有其他数字不变。 

目标是表明所有$(s,t)$- 可以通过这种形式的连续前缀旋转来生成组合，并导出映射位串的 MMIX 指令序列$(a_{s+t-1}\dots a_0)_2$到其字典顺序的后继者，当$s+t<64$。 

## 解决方案

 ### (a) 通过连续旋转生成

 让$x = a_{s+t-1}\dots a_1a_0$以二进制形式，并按这些位串的字典顺序解释组合。 

让$x$可以是任何非最终的$(s,t)$-组合。 写$x$作为$$x = u\,0\,1\,1^k\,0^\ell$$在哪里$u$是最长的前缀，使得从右边起第一个可能发生变化的位置出现在模式上$0,1$， 和$1^k 0^\ell$是由固定权重条件确定的尾随零的最大后缀。 

等价地，以标准词典后继形式，令$j$是最大的索引，使得$$a_j = 0,\quad a_{j-1} = 1.$$这样一个$j$存在，除非字符串是$1^t0^s$，这是终端组合。 

让后缀分解一下$j$是$$x = a_{s+t-1}\dots a_{j+1}\,0\,1\,a_{j-2}\dots a_0.$$让$r$是$1$进入$a_{j-2}\dots a_0$。 字典顺序的继承者$x$通过交换得到$01 \to 10$在职位$j,j-1$并移动剩余的$r$尽可能靠右的。 这会产生$$x' = a_{s+t-1}\dots a_{j+1}\,1\,0\,0^{s'}1^{r},$$对于一个唯一确定的$s'$。 

现在考虑前缀$a_j a_{j-1}\dots a_0$。 在$x$该前缀的形式为$$a_j a_{j-1}\dots a_0 = 0\,1\,w,$$在哪里$w$恰好包含$r$那些。 

应用旋转$j$:$$a_j a_{j-1}\dots a_0 \;\leftarrow\; a_{j-1}\dots a_0 a_j,$$所以前缀变成$$1\,w\,0.$$后缀$w$保留其相对顺序，并且单个$0$移动到旋转块的末尾。 这一行动让尊贵的$1$一对的左边$01$超过由以下确定的块$w$，这与词典后继规则所需的重新分配完全匹配。 

因此，字典顺序中的每个后继步骤对应于由最右侧可接受模式的位置确定的单个前缀旋转$01$。 迭代此规则$0^s1^t$达到每一个$(s,t)$- 组合一次，因为每个步骤都会产生字典顺序的后继，并且该过程终止于$1^t0^s$。 

这样就完成了证明。 ∎

 ### (b) MMIX 后继者的实现

 让位串存储在寄存器中$rA$， 和$s+t<64$所以字操作是有效的。 

固定流行计数二进制单词的词典后继的标准标识是：$$\text{next}(x) = r \;\;|\;\; \left(\frac{(r \oplus x) \gg 2}{c}\right),$$在哪里$$c = x \,\&\, (-x), \quad r = x + c.$$这相当于众所周知的 Gosper 变换，用于生成具有相同数量的下一个整数$1$位，匹配$(s,t)$-第 7.2.1.3 节的二进制编码下的组合。 

在MMIX中，使用寄存器$rA$（输入/输出），$rB$,$rC$,$rD$:```
        SET     rB, rA
        NEG     rC, rA          % rC = -x
        AND     rC, rA, rC       % c = x & (-x)

        ADD     rD, rA, rC       % rD = r = x + c

        XOR     rB, rD, rA       % rB = r ^ x
        SR      rB, rB, 2        % rB = (r ^ x) >> 2
        DIV     rB, rB, rC       % rB = ((r ^ x) >> 2) / c

        OR      rA, rD, rB       % result = r | ...
```输出在$rA$当输入不是终结符时，是输入组合的字典顺序后继。 

正确性来自标准的固定流行计数后继身份，它保留了$1$位并产生最小的较大整数，匹配字典顺序$(s,t)$-二进制编码下的组合（第7.2.1.3节中的等式（2）-（4））。 

## 验证

 旋转规则保留了位的多重集，因为每个操作仅排列前缀内的条目，因此$1$仍然存在$t$。 

每一步都会按字典顺序增加二进制值，因为所选的旋转会移动一个$1$从最右边可接受的较低位置$01$模式到更高的位置，同时向左推零，匹配具有固定权重的二进制字符串的后继变换。 

MMIX 序列仅使用可逆算术恒等式$c = x & (-x)$和代数分解$x = r - c$，确保所有中间值在 64 位寄存器模型中保持明确定义$s+t<64$。 

## 注释

 旋转解释对应于组合的旋转门生成的循环形式，其中每个转换移动一个$1$通过受控的前缀循环而不是执行全局重新排序。 

MMIX 公式是适应 Knuth 机器模型的经典 Gosper“下一个组合”计算，它与第 7.2.1.3 节中使用的二进制编码的词典后继一致。
