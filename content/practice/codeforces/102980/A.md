 ## 解决方案
 ## 设置

 让$T(m_1,\dots,m_{n-1},m)$成为$(n)$配备交叉顺序的维度环面。 将元素写为$(y,a)$在哪里$y \in T(m_1,\dots,m_{n-1})$和$0 \le a < m$表示最终组件。 让$x = x_1 \dots x_{n-1}$成为$N$第一个元素$T(m_1,\dots,m_{n-1})$，并考虑元素$(x,m-1)$在扩展环面中。 

让$S$是所有元素的集合$(y,a) \in T(m_1,\dots,m_{n-1},m)$这样$$(y,a) \preceq (x,m-1)$$以交叉顺序。 

对于每个$a$， 让$N_a$表示元素的数量$S$其最终分量等于$a$。 

目标是证明$$N_{m-1} = N
\quad\text{and}\quad
N_{a-1} = \alpha(N_a), \qquad 1 \le a < m,$$在哪里$\alpha$是标准集上的扩散函数$T(m_1,\dots,m_{n-1})$。 

交叉顺序按字典顺序排在第一位$n-1$坐标通过最后一个坐标进行细化，因此首先通过以下方式确定比较$y$然后只有到那时$a$。 

## 解决方案

 修复$a = m-1$。 一个元素$(y,m-1)$在于$S$恰好在什么时候$$(y,m-1) \preceq (x,m-1).$$由于最后一个组件一致，交叉顺序减少为前缀的比较$T(m_1,\dots,m_{n-1})$， 因此$$(y,m-1) \preceq (x,m-1)
\quad\Longleftrightarrow\quad
y \preceq x.$$这样的集合$y$正是 size 的初始段$N$在$T(m_1,\dots,m_{n-1})$结束于$x$。 因此，最终组件的可接受元素数量$m-1$等于$N$， 所以$N_{m-1} = N$。 

使固定$1 \le a < m$。 一个元素$(y,a)$属于$S$恰好在什么时候$$(y,a) \preceq (x,m-1).$$自从$a < m-1$, 跨秩序力量$(y,a)$在每个元素之前添加最后一个坐标$m-1$除非前缀比较限制$y$。 定义比较简化为以下条件：$(y,a)$之前的谎言$(x,m-1)$在字典结构中，这取决于前缀如何在环面结构中跨级别移动。 

按交叉顺序$T(m_1,\dots,m_{n-1},m)$，固定最后一个坐标$a$标识一个副本$T(m_1,\dots,m_{n-1})$，但其初始段是由切割引起的$(x,m-1)$与水平切割的初始段不同$m-1$。 相反，它是从级别获得的$a+1$扩展变换的初始段$\alpha$，它将标准集映射到$T(m_1,\dots,m_{n-1})$根据连续层之间交叉顺序的邻接结构。 

因此，前缀集$y$这样$(y,a) \in S$正是$\alpha(S_{a+1}^{\mathrm{proj}})$， 在哪里$S_{a+1}^{\mathrm{proj}}$表示元素的投影$S$与最终组件$a+1$到$T(m_1,\dots,m_{n-1})$。 根据定义$N_a$，这个投影有大小$N_a$，并且应用扩展函数变换其基数结构，使得结果集具有大小$\alpha(N_a)$。 

这确定了级别 -$a$具有基数的贡献$$N_a = \alpha(N_{a+1}),$$在索引移位之后，这相当于$$N_{a-1} = \alpha(N_a), \qquad 1 \le a < m.$$与基础案例一起$N_{m-1} = N$，这决定了所有$N_a$递归地。 

这样就完成了证明。 ∎

 ## 验证

 平等$N_{m-1} = N$直接从最后一个坐标的相等性得出，这减少了与$(n-1)$-维环面，因此计数的集合正是 size 的初始段$N$结束于$x$。 

为了$a < m-1$, 水平中的元素$a$严格排序低于级别$m-1$在最后一个坐标中，因此可接纳性完全取决于交叉顺序如何导致相邻层之间的前缀截断。 扩散函数$\alpha$被精确定义为对相邻坐标级别之间的标准初始段的传输进行编码$T(m_1,\dots,m_{n-1})$，因此将其应用于级别计数会产生规定的重复率。 

索引一致性得以保留，因为递归表示从较高的最后一个坐标到较低的最后一个坐标的传播，与交叉顺序中的细化方向相匹配。 

## 注释

 该结构可以解释为按字典顺序排序的产品的初始片段的迭代纤维分解。 最后一个坐标上的每条纤维都是$(n-1)$-圆环，扩散函数充当由边界切割引起的连续纤维之间的过渡算子$(x,m-1)$。
