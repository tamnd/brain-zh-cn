 ## 解决方案
 ## 问题理解

 我们得到一个小写字符串，并被要求决定它的字母是否可以重新排列以形成回文。 任务不是构造回文，只是确定这种重新排列是否存在。 

回文具有字符围绕其中心镜像的特性。 这立即限制了每个字符可以出现的次数。 字符必须对称配对，长度为奇数时可能有一个中间字符除外。 

输入长度最多可达 100000，这排除了任何二次或更差的方法。 任何试图模拟排列或构建候选字符串的东西都会发生组合爆炸。 即使基于排序的构造也可以，但是涉及重新排列所有排列的任何事情都是不可能的。 

当只关注字符串本身是否是回文而不是它是否可以排列成回文时，就会出现一个幼稚的错误。 例如，像这样的字符串`cacao`不是书面的回文，但可以重新排列成`acoca`，这是有效的。 

另一个微妙的边缘情况是多个字符具有奇数频率的字符串。 例如，`aabbcc`无法形成任何回文，因为需要多个字符占据中心位置。 仅检查总长度奇偶校验的粗心实现将错误地接受此类情况。 

## 方法

 暴力策略将尝试字符串的所有排列并测试它们中是否有一个是回文。 这在概念上是正确的，因为它直接匹配 dralinpome 的定义，但它是不可行的。 排列的数量是 n 的阶乘，并且每个回文检查都是线性的，即使对于中等的 n（如 20）也会导致天文数字的运行时间。 

关键的观察是我们不关心排列，只关心频率计数。 在回文中，字符必须以镜像对的形式出现在中心周围。 这意味着每个字符计数必须为偶数，但如果字符串长度为奇数，则可能有一个字符的计数为奇数。 

因此，整个问题归结为计算有多少个字符出现了奇数次。 如果该计数最多为 1，则存在回文重排。 这避免了任何结构构造，并将问题压缩到弦上的单频传递中。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | --- | --- | --- | --- |
 | 蛮力 | O(n!·n) | O(n!·n) | O(n) | 太慢了|
 | 频率计数 | O(n) | O(1) | O(1) | 已接受 |

 ## 算法演练

 我们通过计算每个字符的出现次数来处理字符串，然后评估其中有多少个字符是奇数。 

1. 统计字符串中每个字符出现的频率。 这给出了每个字母出现次数的完整摘要，与顺序无关。 
2.遍历频率表，统计有多少个字符出现频率为奇数。 每个奇数频率表示一个未配对的字符，需要占据回文中的镜像位置。 
3. 如果奇数频率字符的数量大于 1，则立即断定不可能进行回文重排。 
4. 否则，得出存在有效重排的结论。 

### 为什么它有效

 回文字符围绕其中心对称配对。 左边的每个位置 i 必须与右边的相应位置匹配。 这强制要求每个字符的出现都必须可整除为对，但当长度为奇数时，可能有一个剩余字符可能位于中间。 如果多个字符未配对出现，则无法在不破坏对称性的情况下放置它们，因此任何排列都不能形成回文。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def main():
    s = input().strip()
    
    freq = [0] * 26
    for ch in s:
        freq[ord(ch) - ord('a')] += 1
    
    odd_count = 0
    for f in freq:
        if f % 2 == 1:
            odd_count += 1
    
    if odd_count <= 1:
        print("yes")
    else:
        print("no")

if __name__ == "__main__":
    main()
```该解决方案首先读取字符串并为小写字母构建固定大小的频率数组。 这避免了字典开销并保持每个字符的操作不变。 

然后我们扫描 26 个计数并累加奇数个数。 决策逻辑是回文重排条件的直接翻译。 

一个常见的实现陷阱是使用字符串长度奇偶校验而不是频率奇偶校验。 这是不正确的，因为偶数长度的字符串仍然是不可能的（例如`abcabcx`)，奇数长度的字符串仍然有效。

 ## 边缘情况

 一个最小的字符串，例如`a`包含一个频率为奇数的字符，算法正确地计算出一个奇数条目并接受它。 这对应于一个平凡的回文。 

像这样的字符串`ab`产生两个奇数频率，每个字符一个。 对频率数组的扫描将奇数计数器增加两次，最终条件拒绝它，这与在对称结构中配对两个字符的可能性相匹配。 

更大的结构化案例，例如`abcabcx`表明顺序无关紧要。 频率是`{a:2, b:2, c:2, x:1}`，因此仅存在一个奇数。 该算法接受它，并正确反映`abcxcba`是一个有效的重排。
