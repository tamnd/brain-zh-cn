 ## 解决方案
 ## 问题理解

 我们得到一个被分成单词序列的句子。 每个单词仅由大写英文字母组成。 任务是确定句子是否满足涉及其第一个单词的特定结构属性。 

如果第一个单词可以通过按顺序取出句子中每个单词的第一个字母来重建，则该规则将句子定义为“可爱”。 换句话说，如果我们将句子写成单词列表，提取每个单词的首字符并将它们连接起来，则生成的字符串必须与第一个单词完全匹配，没有多余或丢失的字母。 

输入的大小非常小，最多100个单词，每个单词最多100个字符。 这立即意味着任何在所有字符上以线性时间运行的解决方案都足够快。 即使是多次重复扫描单词的解决方案在限制下仍然是安全的。 

主要风险不是性能而是正确性。 一些微妙的情况很重要：

 一种边缘情况是第一个单词的长度与单词数量不同。 例如，如果句子有 3 个单词，但第一个单词的长度为 5，则它们不可能匹配，因为首字母缩略词的长度只能等于单词数。 

另一个边缘情况是单词是单个字母时。 这实际上是正常情况，必须一致处理，因为首字母缩略词构造仅依赖于每个单词的第一个字符，而不管单词长度如何。 

第三种情况是第一个字母匹配但顺序破坏了对齐。 例如，即使单词之间存在所有字母，序列也必须按顺序完全匹配，而不是作为集合比较。 

## 方法

 一种直接的方法是通过迭代所有单词、获取每个单词的第一个字符并将它们连接成一个新字符串来显式构造首字母缩略词字符串。 然后我们将这个构造的字符串与第一个单词进行比较。 

这是正确的，因为“可爱句子”的定义纯粹是结构性的，除了提取和比较之外不需要任何转换。 成本主要是扫描每个单词一次，因此总工作量与单词长度之和成正比。 

暴力变体可能会尝试通过重复扫描单词或使用嵌套循环检查匹配位置来验证第一个单词的每个字符。 虽然在限制下仍然可行，但它不必要地复杂，并且可能会引入索引错误。 在最坏的情况下，这种方法可以执行 O(N^2) 字符检查，这在这里仍然很小，但对于较大的输入来说扩展性很差。 

关键的观察是，首字母缩略词是由每个单词的第一个字符唯一确定的，因此我们不需要考虑任何其他结构。 这将问题简化为单遍构造和比较。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 蛮力 | O(N²) | O(1) | O(1) | 已接受但没有必要 |
 | 最佳 | O(总字符数) | O(N) | 已接受 |

 ## 算法演练

 1. 读取单词数和单词列表。 我们保持它们有序，因为位置结构至关重要。 
2. 提取每个单词的第一个字符并构建一个新字符串。 该字符串表示句子定义所隐含的首字母缩略词。 
3. 将构建的首字母缩略词与句子中的第一个单词进行比较。 
4. 如果相同，则输出“Yes”，否则输出“No”。 

每个步骤都直接反映定义，因此除了简单迭代之外不需要辅助结构或预处理。 

### 为什么它有效

第一个单词需要完全等于所有单词的第一个字符的串联。 由于每个单词恰好为首字母缩略词贡献了一个字符，并且保留了顺序，因此构造的字符串是唯一可能的候选者。 因此，构造的字符串和第一个单词之间的相等对于正确性来说既是必要的也是充分的。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    n = int(input().strip())
    words = input().split()

    acronym = ''.join(w[0] for w in words)

    if acronym == words[0]:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    solve()
```解决方案首先读取整数计数，然后读取句子。 使用`split()`确保我们正确地分隔单词，无论间距是否不规则。 

核心操作是生成器表达式`w[0] for w in words`，它安全地提取每个单词的第一个字符。 由于每个单词都保证非空，因此不存在索引错误的风险。 

最后，我们将构建的首字母缩略词与`words[0]`，它代表所需的目标词。

 ## 边缘情况

 一种关键的边缘情况是只有一个单词时。 对于输入：

 1

 一个

 该算法通过获取唯一单词的第一个字符来构造首字母缩略词，生成“A”。 然后它将其与第一个单词（也是“A”）进行比较，因此输出为“Yes”。 这正确地处理了句子由单个单词组成的退化情况。 

另一种情况是第一个单词长于单词数。 例如：

 3

 ABCDAB

 首字母缩略词变为“AAB”，而第一个单词是“ABCD”。 由于字符串立即不同，算法无需任何特殊处理即可正确输出“否”。 

最后，当所有单词都是相同的单个字符时，例如：

 4

 X X X X

 首字母缩略词为“XXXX”，与第一个单词完全匹配，因此输出为“Yes”。 这证实了重复的字符和统一的输入在相同的逻辑下行为正确。
