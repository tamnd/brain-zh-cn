 ## 解决方案
 ## 问题理解

 我们得到一个由小写拉丁字母序列书写的单词。 标记从第一个字符开始，并根据确定性规则移动，该确定性规则完全取决于当前字符在单词中出现的次数。 

如果标记下的字符在字符串中出现多次，则允许标记跳转到同一字符的任何其他出现处。 如果该字符恰好出现一次，则令牌将被迫向右移动一步。 如果它已经在最后一个位置并且该字符是唯一的，则该过程结束。 

问题是这个过程是否总是终止，或者是否可以通过选择相等字符之间的跳转来使其永远继续下去。 

约束 n 高达 100000 意味着任何逐步探索转换的模拟如果可以重复循环状态，都是不安全的。 该过程可以多次重新访问位置，因为重复的字母允许任意跳转，因此状态空间实际上是一个具有多达 100000 个节点和由相同字符引起的许多边的图。 在最坏的情况下，由于反复重新访问，运行直到终止或循环检测的简单模拟可能会退化为二次行为。 

当所有字符都相同时，会出现微妙的边缘情况。 例如，在像“aaa”这样的字符串中，令牌永远不会确定性地前进，并且总是可以来回跳跃，从而产生无限循环。 另一方面，在像“abc”这样的字符串中，每个字符都是唯一的，因此令牌总是向右移动并立即终止。 

关键的困难是识别跳跃结构何时创建循环，而不是实际模拟该过程。 

## 方法

 强力解释将每个位置视为一个状态。 从位置 i 开始，如果当前字符出现多次，我们考虑转换到具有相同字符的所有其他索引。 否则，我们移动到 i + 1。我们继续这个过程，直到我们要么掉下绳子，要么重新访问之前在保证重复的配置中看到的位置。 

从概念上讲，这是可行的，因为一旦我们确定了跳转的选择，该过程就是确定性的，但问题允许在相同的字母之间进行任意选择，这意味着该过程不是单一路径，而是一个分支系统。 直接模拟需要探索所有可能的选择或在隐式状态图上维护访问集。 在最坏的情况下，每个位置都可能与具有相同特征的许多其他位置相连，并且重复访问可能会造成指数或二次爆炸。 

关键的观察结果是，取得进展的唯一方法是通过在整个过程的后缀行为中只出现一次的字符。 如果在任何时刻标记位于多次出现的角色上，我们总是可以选择跳到较早的出现位置并重新创建以前见过的情况。 这种返回到先前状态的能力意味着系统是循环的，除非强制进展占主导地位。 

因此，问题简化为检查该过程是否可以达到这样一个点：我们登陆的每个角色在剩余的可到达结构中都是唯一的，从而强制确定性向右移动直到终止。 这归结为频率上的结构条件：如果存在频率至少为 2 的任何字符，则始终可以通过在该字符的出现之间弹跳来构造循环，因为规则中没有任何内容可以阻止无限期地返回到较早的索引。 

因此，当且仅当字符串中的所有字符都不同时，才会发生终止。

 ## 算法演练

 该解决方案基于计算每个字符的出现次数并检查是否有任何字符出现多次。 

1. 统计字符串中每个字符出现的频率。 这是通过输入的单次传递完成的，由于只使用小写拉丁字母，所以维护了一个大小为 26 的固定数组。 此步骤捕获过程中的所有结构模糊性。 
2. 扫描频率表，检查是否有任何字符的频率大于1。存在这样的字符意味着至少存在一个位置，使得令牌总是可以选择跳回到另一个相同的字符。 
3. 如果任意频率超过1，则断定该过程可以无限并输出NO。 否则，如果所有频率都恰好为 1，则令牌始终向右移动，直到退出字符串，因此输出 YES。 

### 为什么它有效

 仅当字符出现多次时，该过程才变得不确定。 在这样的字符下，令牌可以自由地跳转到另一个相同的位置，这始终保留重新访问先前配置的能力。 由于位置是有限的，在这种自由度下的反复重访就形成了一个循环。 如果没有字符重复，则每次移动都强制到下一个索引，并且状态严格进行直到终止。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    n = int(input().strip())
    s = input().strip()

    freq = [0] * 26
    for ch in s:
        freq[ord(ch) - ord('a')] += 1

    for c in freq:
        if c > 1:
            print("NO")
            return

    print("YES")

if __name__ == "__main__":
    solve()
```解决方案首先读取长度和字符串。 频率数组跟踪每个字符在恒定空间中的出现次数。 扫描重复项是决定性的一步：它并不尝试模拟运动，因为唯一重要的属性是分支是否存在于任何位置。 

支票`c > 1`就足够了，因为即使是单个重复字符也会引入可逆转换机制，允许令牌通过交替出现来无限循环。

 ## 边缘情况

 像“a”这样的最小单字符字符串以唯一字符开头，因此令牌立即移出界限并终止。 频率检查发现没有重复并返回 YES。 

像“aa”这样的字符串演示了立即非终止：两个位置共享相同的字符，因此即使令牌可以移动，它总是有一个替代的事件可以跳转到，从而允许两个索引之间的无限循环。 该算法正确标记频率 2 并返回“否”。 

像“abcde”这样的字符串不包含重复项，因此每个步骤都强制向右，没有分支，保证在字符串末尾终止。
