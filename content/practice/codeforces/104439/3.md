 ## 解决方案
 ## 问题理解

 我们得到一段连续的整数$A$到$B$，包含在内。 两个人用不同的规则在这个段内计数：一个人计算有多少个数字可以被$K$，另一个计算有多少个数字可以被整除$M$。 任务不是输出两个计数，而只是输出它们的差值：有多少倍数$K$减去多少倍数$M$，限制在区间内$[A, B]$。 

重要的一点是我们没有明确列举数字。 范围可以非常大，最多可达$2 \cdot 10^9$，因此直接扫描每个整数是不可能的。 任何循环的解决方案$A$到$B$会需要$O(B - A)$，在最坏的情况下远远超出了可行的限度。 

相反，这个问题从根本上讲是关于计算一个范围内的倍数，这表明了一种封闭形式的算术方法。 

出现微妙的边缘情况时$K = M$。 在这种情况下，每个间隔的两个计数都是相同的，因此答案必须始终为零。 另一种边缘情况是当区间边界与倍数完全对齐时，例如当$A$或者$B$可以整除$K$或者$M$，这可能会暴露简单公式中的差一错误。 

## 方法

 暴力解决方案将迭代每个整数$A$到$B$，检查整除性$K$，计算匹配项，然后重复$M$。 这是正确的，因为它直接遵循问题的定义。 然而，它的运行时间取决于间隔的大小。 如果$A = 1$和$B = 2 \cdot 10^9$，这将需要数十亿次操作，这在典型的时间限制下是不可行的。 

关键的观察是计算前缀中的倍数$[1, X]$很简单：能被整除的整数个数$K$最多$X$是$\lfloor X / K \rfloor$。 由此，算入$[A, B]$变为两个前缀计数之差。 这将问题从迭代一个范围转变为计算恒定数量的算术表达式。 

同样的推理独立地适用于两者$K$和$M$，然后我们减去结果。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | --- | --- | --- | --- |
 | 蛮力 |$O(B - A + 1)$|$O(1)$| 太慢了 |
 | 最佳 |$O(1)$|$O(1)$| 已接受 |

 ## 算法演练

 ## 算法演练

 1. 读取整数$K, M, A, B$。 它们定义了同一时间间隔内的两个独立的计数过程。 
2. 定义辅助函数$f(x, d)$计算中有多少个整数$[1, x]$可以被整除$d$。 这简直就是$x // d$。 这样做的原因是每个块$d$连续的数字恰好贡献一倍。 
3. 计算有多少个倍数$K$位于$[A, B]$作为：$f(B, K) - f(A - 1, K)$。 这减去了之前的所有内容$A$，只保留有效范围。 
4. 计算相同的数量$M$:$f(B, M) - f(A - 1, M)$。 
5. 将第一个计数减去第二个计数并输出结果。 

关键思想是前缀计数避免了任何迭代，并将范围约束转换为两个端点上的算术。 

### 为什么它有效

 对于任意固定除数$d$，倍数序列具有周期性$d$。 完整周期数$[1, x]$正是$\lfloor x / d \rfloor$，并且每个时期恰好贡献一个有效数字。 减去前缀计数可以保留任何间隔的精确包含-排除，因此不会重复计算或遗漏任何数字。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def count(x, d):
    if x <= 0:
        return 0
    return x // d

K = int(input())
M = int(input())
A = int(input())
B = int(input())

k_cnt = count(B, K) - count(A - 1, K)
m_cnt = count(B, M) - count(A - 1, M)

print(k_cnt - m_cnt)
```辅助函数干净地处理前缀逻辑。 显式检查$x \le 0$避免负除法边缘情况$A = 1$， 自从$A - 1$变为零或负数。 在这些情况下，前缀中没有有效数字，因此计数为零。 

主要计算应用包含-排除思想两次，每个除数一次，然后计算所需的差值。

 ## 边缘情况

 一个关键的边缘情况是当$A = 1$，这使得$A - 1 = 0$。 例如，与$K = 4$,$A = 1$,$B = 10$，前缀减法变为$f(10, 4) - f(0, 4)$。 助手返回零$f(0, 4)$，所以结果很简单$\lfloor 10/4 \rfloor = 2$，对应数字4和8。 

另一个边缘情况是当$K = M$。 在这种情况下，每个间隔的两个计数都是相同的。 用于输入$K = M = 7, A = 3, B = 50$，两个前缀表达式的计算结果相同，因此减法始终产生零，这与预期的取消行为相匹配。 

最终边界情况发生在$A = B$。 如果单个数可以被$K$但不是$M$，结果是$1$。 如果能被两者整除，则为$0$。 前缀公式仍然可以正确处理这一问题，因为它通过减法精确地隔离了一个端点。
