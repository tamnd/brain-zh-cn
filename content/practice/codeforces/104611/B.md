 ## 解决方案
 ## 问题理解

 我们得到了几堆独立的石头。 每堆开始时都有一定数量的石子，两名玩家交替移动。 在一轮中，玩家精确地选择一堆石子，并根据规则从其中移除石子，该规则取决于适合该堆大小的最大正方形。 

如果一堆有$m$石头，我们计算$s = \lfloor \sqrt{m} \rfloor$，这是最大的边长$s \times s$适合的正方形$m$。 规则描述了我们在概念上将这个正方形结构与剩余的石头分开，然后允许玩家删除该正方形的一定数量的整行。 移动后，正方形部分中的堆变空，并且根据移除的行数，可能仅保留较小的正方形结构。 

当所有堆上的所有石子都被移走时，游戏结束，移走最后一块石子的玩家获胜。 任务是在假设两名玩家都发挥最佳的情况下确定获胜者。 

从约束中得到的重要观察结果是，每个桩尺寸在选择方面都是独立的，并且桩之间唯一的耦合是通过轮流发生的。 这强烈表明对不相交组件的 Sprague-Grundy 公式。 自从$n$最多可达约$10^5$和每个$a_i$可以大到$10^9$，任何显式模拟移动或构建每堆游戏图的方法都是不可能的。 即使枚举每堆的移动也已经太大了，因为一堆的大小$m$可以表示最多$O(\sqrt{m})$结构选择。 

天真的思维的一个微妙的失败案例是根据语句中的公式将移动解释为任意减法，并尝试模拟每个状态的所有可到达状态$m$。 例如，与$m = 10^9$，即使是单个堆也会生成巨大的转换图。 另一个常见的错误方向是将游戏视为标准 Nim 堆，其中每次移动都会删除任意正整数； 这忽略了平方分解所施加的强结构，导致错误的 Grundy 值。 

## 方法

 直接的暴力方法会将每一堆视为游戏图中的一个状态，并尝试通过枚举所有有效的移动来计算其 Grundy 值。 对于一堆大小$m$, 可以计算出$s = \lfloor \sqrt{m} \rfloor$，然后考虑要删除的行数的所有可能选择，并递归计算结果状态。 这在理论上是正确的，因为斯普拉格-格伦迪理论适用，但它很快就变得不可行。 每个州都可能分为$O(s)$接下来的状态，从那时起$s$是$O(\sqrt{m})$，即使输入适度，所有桩的总工作也会爆炸。 

关键的简化来自于注意到最大正方形之外的堆的详细剩余部分不会影响未来的决策。 一旦我们分离出大小的平方$s \times s$，每次移动仅通过删除整行来改变有效方块大小。 这意味着一堆的游戏状态仅取决于$s = \lfloor \sqrt{m} \rfloor$，而不是精确值$m$。 

一旦进行了这种减少，每个堆实际上相当于一个状态为整数的堆$s$，并且移动会改变$s$成任何更小的$t < s$通过删除行。 这将游戏折叠成一个非常简单的拆除结构，其中来自状态$s$, 所有州$0, 1, \dots, s-1$是可达的。 

这将问题转化为计算经典递减选择游戏的格伦迪数，从而产生直接封闭形式。 

| 方法| 时间复杂度| 空间复杂度| 判决 |
 | ---| ---| ---| ---|
 | 蛮力（每堆状态图）| 指数为$m$| 大型递归/备忘录表| 太慢了 |
 | 减少 Grundy 的平方大小 |$O(n)$|$O(1)$| 已接受 |

 ## 算法演练

 我们现在将每一堆压缩成一个整数$s = \lfloor \sqrt{a_i} \rfloor$，并计算每个这样的 Grundy 值$s$。 

1. 对于每一堆，计算$s = \lfloor \sqrt{a_i} \rfloor$。 这捕获了影响未来移动的堆的唯一部分，因为所有操作都取决于存在多少个完整的方形行。 
2. 定义概念状态$G(s)$代表一个桩，其有效平方边长为$s$。 从这个状态开始，移动允许选择任何数字$k$行，减少方边$s$到$s-k$， 在哪里$k \ge 1$。 
3. 观察每个状态$t$和$0 \le t < s$通过选择只需一步即可到达$k = s - t$。 这使得过渡集从$s$等于所有较小的整数。 
4. 从以下位置开始自下而上计算 Grundy 值$G(0) = 0$。 对于每个$s \ge 1$，Grundy 值的可达集合恰好是$\{G(0), G(1), \dots, G(s-1)\}$。 
5. 集合的混合体$\{0, 1, \dots, s-1\}$是$s$， 所以$G(s) = s$。 
6. 因此，每堆的 Grundy 值就是$s = \lfloor \sqrt{a_i} \rfloor$。 
7. 计算所有的异或$s$价值观。 如果结果非零，则第一个玩家有获胜策略； 否则第二个玩家获胜。 

核心不变量是，归约后，每个堆都表现为一个独立的 Nim 堆，其大小等于其平方根。 由于每次移动都会严格减小该值并允许达到所有较小的值，因此格伦迪序列变得严格$G(s)=s$，这保证了跨堆的异或组合完全决定了结果。 

## Python 解决方案```python
import sys
input = sys.stdin.readline
import math

def solve():
    n = int(input())
    a = list(map(int, input().split()))
    
    x = 0
    for v in a:
        s = math.isqrt(v)
        x ^= s
    
    if x:
        print("First")
    else:
        print("Second")

if __name__ == "__main__":
    solve()
```该实现直接应用派生的归约。 唯一重要的步骤是使用整数平方根，这可以避免浮点精度问题。 每堆都在恒定时间内转换成其 Grundy 贡献，最终的 XOR 根据 Sprague-Grundy 理论确定获胜者。 

## 工作示例

 考虑带有桩的输入$[3, 8, 15]$。 

对于我们计算的每个值$s = \lfloor \sqrt{a_i} \rfloor$, 给予$[1, 2, 3]$。 XOR 是逐步计算的。 

| 桩| 价值$a_i$|$s = \lfloor \sqrt{a_i} \rfloor$| 异或到目前为止 |
 | ---| ---| ---| ---|
 | 1 | 3 | 1 | 1 |
 | 2 | 8 | 2 | 3 |
 | 3 | 15 | 15 3 | 0 |

 最终的 XOR 为零，因此第二个玩家获胜。 这符合在最佳玩法下桩完美平衡的直觉。 

现在考虑$[1, 4, 9]$。 平方根是$[1, 2, 3]$再次，但移除结构表明每一堆都已经是一个完美的正方形，因此每一堆都做出了充分的贡献。 

| 桩| 价值$a_i$|$s$| 异或到目前为止 |
 | ---| ---| ---| ---|
 | 1 | 1 | 1 | 1 |
 | 2 | 4 | 2 | 3 |
 | 3 | 9 | 3 | 0 |

 结果再次是第一个玩家输了，这表明 Grundy 值的对称性决定了结果，而不是原始堆大小。

 ## 边缘情况

 一个关键的边缘情况是当一堆已经是一个完美的正方形时。 例如，$m = 16$给出$s = 4$。 在这种情况下，游戏状态的行为与类似 Nim 递减选择形式的大小为 4 的标准堆完全相同。 该算法仍然为其分配 Grundy 值 4，因此它的处理方式一致，无需特殊分支。 

另一个边缘情况是$m = 1$。 这里$s = 1$，唯一的一步会导致 0，如果被孤立的话，这对于下一个玩家来说是一个失败的位置。 计算产生 Grundy 值 1，它正确反映了获胜位置。 

大值，例如$m = 10^9$也是安全的，因为该算法仅计算整数平方根。 该结构确保不会发生溢出或深度递归，并且每个堆在恒定时间内独立处理。
