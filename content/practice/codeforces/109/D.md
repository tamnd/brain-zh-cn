 ## 解决方案
 ## 问题理解

 给定一个正整数数组，我们希望按非降序对其进行排序。 不同之处在于，只有当涉及的两个数字中至少有一个是幸运的时，我们才能交换元素。 幸运数字被定义为仅包含数字 4 和 7 的数字。目标是找到完成排序的交换序列或确定这是不可能的。 交换序列的长度不得超过 2*n。 

这里的关键约束是数组的大小，最多 10^5 个元素。 这排除了主要排序逻辑时间复杂度低于 O(n log n) 的任何算法。 重复检查所有可能的交换对的简单暴力尝试是立即不可行的，因为迭代所有对将需要 O(n^2) 操作，在最坏的情况下大约为 10^10。 

当数组根本没有幸运数字时，就会出现一个重要的边缘情况。 在这种情况下，由于交换至少需要一个幸运数字，因此不可能移动任何元素，并且输出必须为-1。 另一个微妙的情况是当数组已经排序时； 该算法应该认识到这一点并产生零交换。 如果数组中有幸运数字，我们需要确保交换策略始终有效地使用它们作为“枢轴”将其他元素移动到位。 

例如，如果输入是`5 3 2 1`如果没有幸运数字，输出是`-1`。 如果输入是`4 2 7 1`只要有一个幸运数字 7，就可以使用 7 来安排交换来对数组进行排序。 

## 方法

 暴力方法很简单：重复扫描数组中的无序对，其中至少有一个数字是幸运的，执行交换，然后重复，直到数组排序完毕。 这在原则上是正确的，但效率很低，因为它可能需要最多 O(n^2) 次操作。 当 n 达到 10^5 时，这将导致时间复杂度达到 10^10，这对于 3 秒的时间限制来说远远超出了可行的范围。 

关键的观察是幸运数字充当可移动的枢轴。 如果至少有一个幸运数字，我们总是可以用它作为临时保存点对数组进行排序。 该过程是识别任何幸运数字的索引，然后使用该主元模拟选择排序。 当某个数字不合适时，我们将其与幸运枢轴交换，然后在必要时将枢轴移回原处。 这将问题简化为可管理的 O(n) 交换序列，因为每个元素最多需要与主元进行恒定数量的交换。 

如果没有幸运数字并且数组尚未排序，则排序是不可能的。 否则，使用一个幸运数字作为灵活的中介可以保证我们可以将所有元素放置在正确的位置，交换次数不超过 2*n，从而满足问题的约束。

 ## 算法演练

 1. 扫描数组，找出所有幸运数字。 如果不存在且数组未排序，则立即返回 -1。 否则，选择任何幸运数字作为主元。 
2. 创建数组的排序副本。 这代表每个数字的最终目标位置。 
3. 维护一个当前位置数组，以快速定位每个数字在排序数组中应该出现的位置。 
4. 从左到右迭代数组。 对于每个元素，如果它已经位于正确的位置，则移动到下一个元素。 如果不是，如果元素本身不幸运，则将其与幸运的主元元素交换。 记录每次交换。 
5.如果当前元素幸运的话，可以直接与任何放错位置的元素进行交换。 否则，首先将当前元素与枢轴交换，然后将枢轴放置在其最终位置。 
6. 重复此操作，直到所有元素都按排序顺序排列。 确保交换总数不超过2*n。 

为什么有效：不变的是主元幸运数字可以在数组中的任何位置移动并返回，从而提供了重新定位任何元素的途径。 每个未就位的元素要么是幸运的，可以直接移动，要么可以使用枢轴作为临时固定点来移动。 该算法保证每个元素最多 2 次交换后，它达到排序配置。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def is_lucky(x):
    s = str(x)
    return all(c in '47' for c in s)

def main():
    n = int(input())
    a = list(map(int, input().split()))
    
    lucky_indices = [i for i, val in enumerate(a) if is_lucky(val)]
    
    if not lucky_indices:
        if a == sorted(a):
            print(0)
        else:
            print(-1)
        return
    
    pivot = lucky_indices[0]
    sorted_a = sorted(a)
    pos = {v: i for i, v in enumerate(a)}
    swaps = []
    
    for i in range(n):
        correct_val = sorted_a[i]
        while a[i] != correct_val:
            j = pos[correct_val]
            if i != pivot:
                swaps.append((pivot + 1, j + 1))
                a[pivot], a[j] = a[j], a[pivot]
                pos[a[pivot]] = pivot
                pos[a[j]] = j
            swaps.append((i + 1, pivot + 1))
            a[i], a[pivot] = a[pivot], a[i]
            pos[a[i]] = i
            pos[a[pivot]] = pivot
    
    print(len(swaps))
    for x, y in swaps:
        print(x, y)

if __name__ == "__main__":
    main()
```该代码首先识别幸运数字。 如果不存在且数组未排序，则输出 -1。 否则，它会使用一个幸运数字作为枢轴来重复交换放错位置的元素。 字典跟踪当前位置以进行高效查找。 交换序列通过确保每个元素最多移动两次来遵守 2*n 限制。
