 ## 解决方案
 ## 问题理解

 我们有一个标准的六面骰子，但它的面不是通常的值 1 到 6，而是包含整数 2、3、4、5、6 和 7，全部不同。 每掷一次都会产生这些数字中的一个，一系列掷骰的分数是可见面的总和。 

对于每个查询值$x$，我们被要求确定任意数量的卷$k$这样就可以准确地获得总和$x$准确地使用$k$卷。 我们不需要构建掷骰子的序列，只需输出有效的$k$。 

关键的解释是一旦我们修复$k$，问题就变成了我们能否代表$x$作为总和$k$整数，每个都选自$\{2,3,4,5,6,7\}$。 

限制很小：$t \le 100$和$x \le 100$。 这立即排除了任何繁重的组合或搜索的需要。 任何在每个查询的恒定时间内计算每个答案的解决方案就足够了。 

一个微妙的点是，许多不同的值$k$可以为同样的工作$x$。 该问题不要求最小值或最大值，只要求任何有效的值。 这种灵活性使得构造变得简单。 

不存在涉及不可能的真正边缘情况，因为该语句保证每个查询都存在一个解决方案。 

## 方法

 暴力解释是尝试所有可能的次数$k$，并且对于每个$k$，检查是否$x$可以使用形成$k$2 到 7 之间的数字。对于固定的$k$，这相当于询问是否$x$位于范围内$[2k, 7k]$，因为 2 是掷骰的最小贡献，而 7 是最大。 

如果我们用蛮力解决所有问题$k$直到，说，$x$，并且对于每个$k$执行检查，在给定约束的情况下，复杂性仍然很小，但这是不必要的。 问题的结构表明了直接构造。 

关键的观察是对于任何选择的$k$，可达到的总和形成一个连续区间$2k$到$7k$。 这个间隔没有间隙，因为我们可以以 1 为步长调整各个卷，同时保持在允许的面值范围内。 因此，我们不需要搜索序列，而只需要找到任意整数$k$这样$2k \le x \le 7k$。 

重新排列，我们想要：$$\frac{x}{7} \le k \le \frac{x}{2}$$自从$k$必须是整数，此范围内的任何整数都可以。 一个简单的建设性选择是取最小的有效值$k$仍然允许到达$x$，即：$$k = \left\lceil \frac{x}{7} \right\rceil$$一旦我们解决这个问题$k$, 剩余差值$x - 2k$总是可以通过将一些卷从 2 增加到 7 来分配。

 ## 算法演练

 1.读取查询数量$t$。 每个查询给出一个目标总和$x$。 
2. 对于每个$x$，计算最小整数$k$这样$7k \ge x$。 这确保了即使所有的掷骰数都是 7，我们也可以达到或超过$x$。 
3.输出这个$k$作为查询的答案。 

我们选择天花板的原因$x/7$的优点是它最大限度地减少了卷筒数量，同时保证了足够的总容量以达到$x$。 任何更小的$k$将限制以下最大可实现金额$x$，使其不可能。 

### 为什么它有效

 对于任何固定数量的卷$k$，可达到的总和的集合正是区间$[2k, 7k]$。 这是因为每个卷独立地贡献 2 到 7 之间的任何值，并且将一个卷调整 ±1 会使总和恰好改变 1，同时保持有效。 因此该范围内的所有整数都是可达的。 

选择$k = \lceil x/7 \rceil$保证$x \le 7k$。 同时，自从$k \ge 1$，我们总是有足够的结构来表示$x$从所有 2 开始并增加一些骰子。 因此$x \ge 2k$也自动适用于问题约束中的此构造范围，并且可以通过将选定的卷增加到 7 来填补任何剩余的间隙。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    t = int(input())
    for _ in range(t):
        x = int(input())
        k = (x + 6) // 7
        print(k)

if __name__ == "__main__":
    solve()
```该代码独立处理每个查询。 表达式`(x + 6) // 7`是标准整数上限除法$x/7$，确保我们总是选择最小的$k$这样$7k \ge x$。 

该实现避免了任何掷骰子的模拟。 该逻辑直接计算最小可行的滚动次数，这已经足够了，因为该问题允许任何有效的配置。

 ## 边缘情况

 一个重要的界限是何时$x$能被 7 整除。例如，$x = 14$。 该算法给出$k = 2$。 对于两卷，最大为$14$，通过实现$7 + 7$，确认上边界的正确性。 

另一种情况是当$x$刚好高于 7 的倍数，例如$x = 8$。 这里$k = 2$。 范围是$[4, 14]$，因此 8 是可以实现的，例如$2 + 6$。 这表明，即使一卷纸不够用，添加第二卷纸也能产生足够的灵活性来填补空白。 

最后，对于非常小的值，例如$x = 2$，公式给出$k = 1$。 唯一可能的总和恰好是 1 卷，必须是 2，完全符合要求。
