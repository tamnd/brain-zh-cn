 ## 解决方案
 ## 问题理解

 我们正在与一系列$n$顶点从左到右排列，每对相邻的顶点相连，形成一条简单路径。 配置是一个长度为二进制的字符串$n$，其中 1 表示该顶点存在闪烁，0 表示该顶点为空。 

时间以离散的步骤演变。 每一秒，每个闪烁都会同时分裂成两部分。 一个部分向左移动一个顶点，另一部分向右移动一个顶点。 直观上讲，每个被占用的顶点以单位速度向两个方向向外发送质量。 

有两种重要的破坏性相互作用。 如果两个闪烁体同时到达同一个顶点，它们就会湮灭。 此外，还有一种特殊的“边缘碰撞”情况，当相反的运动穿过边缘或在顶点周围相遇时，根据位置的奇偶性，也可以取消。 所有这些规则的效果是，当对称流碰撞时，系统的行为就像粒子左右移动并相互抵消。 

我们每秒观察占用顶点的配置，产生一个序列$C_0, C_1, \dots, C_{2n}$。 我们需要选择一个满足两个条件的初始配置。 首先，在每个时间步，配置都不为空，这意味着至少一个顶点始终包含一个幸存的闪烁。 二、在第一个内$2n+1$状态，序列必须重复，所以存在$i < j$和$C_i = C_j$。 

输出是满足这些约束的任何有效初始配置，如果不存在则为“Unlucky”。 

约束条件$n \le 123$足够小，二次甚至三次推理都是合理的，但配置的状态空间是指数的。 对所有初始状态进行简单模拟是不可能的，因为有$2^n$可能性，并且每个模拟运行$O(n)$步骤，给出不可行的$O(n2^n)$规模。 

一个关键的边缘情况是全零配置。 它立即违反了第一个条件。 另一个微妙的例子是单个 Twinkle。 它很快就传播到境界并消失，所以不能保持永恒的非空性。 这排除了稀疏配置在不维持碰撞的情况下向外塌陷的情况。 

真正的困难在于，进化可以被视为构型的可逆或循环变换，我们必须构建一个避免灭绝的状态，同时保证有限时间内的循环。 

## 方法

 暴力尝试将尝试所有初始二进制字符串并模拟最多的演化$2n$步骤，检查非空和重复。 每个模拟步骤都需要更新所有顶点的贡献，因此每个步骤都是$O(n)$，并且对于$2^n$表明这变成$O(n^2 2^n)$，即使对于小规模来说，这也远远超出了可行的限制$n$。 

关键的观察结果是，所描述的动力学行为就像路径上的线性传播系统，其中贡献独立地左右移动，并在它们相遇时抵消。 这在结构上相当于跟踪重叠影响区间的奇偶性。 我们没有从物理闪烁的角度思考，而是将系统重新解释为具有强对称性的比特串的确定性变换。 

关键的见解是，由于系统是有限的和确定性的，因此每个轨道如果从未达到全零状态，则最终必须循环。 因此，问题归结为构造任何其轨迹避免吸收零状态的起始状态。 一旦我们确保不灭绝，有界前缀内的重复就会自动遵循鸽巢原理$2n+1$州。 

因此，真正的任务变成构建一个在该传播规则下永远不会完全抵消的配置。 该过程的结构意味着对称图案能够自我维持，而交替图案会产生持久的干扰图案，从而防止彻底湮灭。 人们可以验证一种简单的周期性模式（例如交替位）是否会生成稳定的行波，从而使至少一个活动顶点始终保持活动状态。 

该解决方案简化为构建一个周期性的二进制串，该二进制串会引起整个链上的波的永久运动。

 ## 算法演练

 该结构来自于强制重复的空间模式，保证持久传播而不是湮灭。 一个自然的候选者是跨顶点的交替结构，它确保每个传播步骤总是在相邻位置生成新的活动，而不是同步到完全取消。 

1. 我们将初始配置构建为长度为 2 的重复模式，通常在整条线上重复“10”。 这可确保占用的顶点均匀分布。 
2.我们直接输出这个二进制串作为初始状态。 

这种特定选择很重要的原因是，任何孤立的集群都倾向于向外扩散并在边界消亡，但交替占据可以保证从 1 开始的每次向外移动都会遇到由 0 和 1 组成的结构化环境，从而产生持续的重新激活而不是完全取消。 

## 为什么它有效

 演化可以被视为有限状态空间上的确定性变换。 任何不崩溃为全零配置的状态最终都必须重复。 交替模式可以防止完全抵消，因为它不断地产生不对称的传播前沿。 这些前沿以一种确保至少一个顶点始终保持活动状态的方式进行干扰，从而防止吸收。 由于系统保持在有限的非零轨道上，因此保证在有限的时间内重复。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

n = int(input().strip())

# Construct alternating pattern
ans = []
for i in range(n):
    if i % 2 == 0:
        ans.append('1')
    else:
        ans.append('0')

print("".join(ans))
```代码如下：$n$并构建一个简单的交替二进制字符串。 偶数索引设置为 1，奇数索引设置为 0。这直接实现了算法中讨论的周期结构。 由于结构是静态的，因此不需要模拟。 

关键的实现细节是索引：使用从零开始的索引可确保模式以 1 开头，这对于避免第一步立即出现空配置非常重要。 这种模式的任何转变都可以，但从 1 开始可以保证立即有效。

 ## 边缘情况

 最小输入$n = 2$产生`10`。 开始于`01`也将是有效的，但是`10`确保左边界立即出现闪烁，而不会出现早期灭绝对称性的风险。 

为了$n = 3$，输出`101`保持两端的活动，防止快速崩溃为空状态。 如果我们开始于`010`，系统将表现对称，并且由于同时向外抵消而可能更快崩溃。 

最大时$n = 123$，交替模式保持一致并且不会引入边界漏洞，因为每隔一个顶点出现一个 1 确保没有传播波可以完全消除单个方向上的所有源。
