 ## 问题理解

 我们在三维空间中得到两个向量，每个向量由三个整数坐标描述。 任务是构造同时垂直于两个给定向量的任何非零整数向量。 从几何角度来说，这意味着我们需要一个与两个输入向量所跨越的平面正交的方向。 

输入保证两个向量都不是零向量并且它们不相同。 语句中隐藏的关键几何假设是两个向量不共线，否则它们不会定义唯一的平面方向，尽管即使在这种退化情况下，叉积公式仍然会产生有效的正交向量。 

输出必须是一个整数向量，其坐标位于有界范围内并且不是零向量。 由于存在多个有效答案，因此任何正确的垂直方向都是可接受的。 

约束非常小，坐标范围为 0 到 10。这立即排除了对浮点算术或复杂数值方法的任何需要。 任何生成有界整数的正确构造就足够了，并且在 Python 中溢出不是问题。 

一个微妙的边缘情况是两个向量平行。 例如，如果两个向量都是`(1, 2, 3)`和`(2, 4, 6)`，天真的几何直觉可能表明问题定义不明确，但叉积仍然产生`(0, 0, 0)`在这种情况下，这是无效的。 然而，该语句保证向量不相等，但没有明确表明它们不共线，因此我们必须处理直接叉积变为零的可能性。 

另一种边缘情况是当组件很小但结构使得取消导致一个或多个坐标为零时。 例如，`(1, 0, 0)`和`(2, 0, 0)`产生零交叉积，必须通过选择后备正交结构来避免这种情况。 

## 方法

 蛮力的想法是搜索一个小的整数向量`(x, y, z)`在允许的输出范围内并测试它是否垂直于两个输入向量。 我们将迭代所有值`[-200, 200]^3`，跳过零向量，并检查点积。 每个候选者需要两个点积，因此总复杂度大致为`(401^3) * 6`算术运算，大约有4亿次检查。 这是临界点，但从概念上讲，在 Python 中，这对于 1 秒来说太慢了，而且考虑到几何结构，这也是不必要的。 

关键的观察是，在三个维度中，可以使用叉积直接构造与两个给定向量正交的向量。 叉积精确地编码垂直于两个输入向量的方向，并且它始终满足代数恒等式的点积约束。 这消除了任何搜索并将问题简化为常数时间计算。 

然而，当输入向量共线时，叉积可以变成零向量。 在这种情况下，任何垂直于共享方向的矢量都是可接受的。 我们可以安全地切换到确定性回退，例如通过交换坐标并取反一个分量来构造与第一个向量正交的向量，确保我们避免零输出。

 ## 算法演练

 1.读取两个输入向量`a = (x1, y1, z1)`和`b = (x2, y2, z2)`。 它们定义平面方向，除非它们共线。 
2. 计算叉积`v = a × b`，使用标准公式`v = (y1*z2 - z1*y2, z1*x2 - x1*z2, x1*y2 - y1*x2)`。 

选择这种结构是因为每个坐标的形成都是为了抵消与两个输入向量的点积。 
3. 检查是否`v`是零向量。 如果不为零，则直接输出它，因为它保证垂直于两个输入。 
4. 如果叉积为零，则向量共线，这意味着任何向量垂直于`a`也垂直于`b`。 构造一个简单的垂直向量，例如`(y1, -x1, 0)`除非`(x1, y1)`都为零。 
5. 如果`(x1, y1)`为零，进一步回落至`(0, z1, -y1)`或任何确保至少一个非零分量的循环排列。 这保证了所需范围内的有效有界整数解。 

### 为什么它有效

 叉积被定义为使得其与每个输入向量的点积通过代数抵消而完全为零。 如果它不为零，则直接提供有效答案。 如果为零，则输入向量是线性相关的，这意味着它们仅定义空间中的单个方向。 在这种情况下，正交补集是一个完整的平面，因此构造与该方向正交的任何向量总是可能的，并且简单的坐标交换构造保证了边界内的非零结果。 

## Python 解决方案```python
import sys
input = sys.stdin.readline

def solve():
    x1, y1, z1 = map(int, input().split())
    x2, y2, z2 = map(int, input().split())

    # cross product
    x = y1 * z2 - z1 * y2
    y = z1 * x2 - x1 * z2
    z = x1 * y2 - y1 * x2

    if x != 0 or y != 0 or z != 0:
        print(x, y, z)
        return

    # fallback: vectors are collinear
    if x1 != 0 or y1 != 0:
        print(y1, -x1, 0)
    else:
        print(0, z1, -y1)

if __name__ == "__main__":
    solve()
```核心计算是叉积，直接用整数运算实现。 这保证了恒定时间内的正确性。 后备分支处理退化情况，即所有叉积分量由于共线性而相互抵消。 

后备建设`(y1, -x1, 0)`选择它是因为它的点积为`(x1, y1, z1)`简化为`y1*x1 - x1*y1 = 0`，确保正交性，无论`z1`。 如果前两个坐标均为零，则向量位于 z 轴上，因此 xy 平面中的任何向量都有效，并且`(0, z1, -y1)`仍然确保非零正交方向。

 ## 边缘情况

 当两个向量平行时，叉积为零，因此初级构造失败。 例如，输入`(1, 2, 3)`和`(2, 4, 6)`产量`(0, 0, 0)`。 该算法检测到这一点并切换到`(y1, -x1, 0) = (2, -1, 0)`。 直接检查确认正交性，因为`2*1 + (-1)*2 + 0*3 = 0`。 

当向量完全位于一个轴上时，例如`(1, 0, 0)`和`(2, 0, 0)`，同样的后备产生`(0, 1, 0)`，保证非零且垂直。 点积立即减为零，因为输入向量中只有一个坐标处于活动状态。 

当向量位于z轴上时，例如`(0, 0, 5)`和`(0, 0, 7)`，后备分支`(0, z1, -y1)`产生`(0, 5, 0)`，它是正交的，因为它的点积仅取决于 z 分量，而 z 分量在构造的向量中为零。
