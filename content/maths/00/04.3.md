## 4.3 同构同性

 同构是结构相同的数学概念。 当两个对象之间存在保留相关结构的可逆映射时，它们是同构的。 

平等通常过于严格。 出于理论目的，两个对象可以具有不同的表示，但仍然表现相同。 

例如，整数的加法群和偶数的加法群作为集合不相等：$$
\mathbb{Z} \neq 2\mathbb{Z}
$$但它们作为群体是同构的。 地图$$
f : \mathbb{Z} \to 2\mathbb{Z}, \quad f(n) = 2n
$$是双射且保留加法：$$
f(n+m) = 2(n+m) = 2n + 2m = f(n) + f(m).
$$所以这两个组具有相同的组结构。 

### 定义

 同构是具有逆态射的态射。 

如果$A$和$B$是结构化对象，同构于$A$到$B$是一个态射$$
f : A \to B
$$使得存在态射$$
g : B \to A
$$和$$
g \circ f = \operatorname{id}_A
$$和$$
f \circ g = \operatorname{id}_B.
$$当这样的映射存在时，我们写：$$
A \cong B
$$这意味着$A$和$B$结构上是相同的。 

### 同构取决于结构

 同构的含义取决于所研究对象的类别。 

| 结构| 同构 |
 | ----------------- | ------------------------------------------------------ |
 | 设置| 双射 |
 | 集团| 双射群同态|
 | 戒指| 具有兼容运算的双射环同态 |
 | 向量空间| 可逆线性映射|
 | 拓扑空间| 同胚 |
 | 度量空间| 等距|
 | 图| 逆保邻双射 |

 相同的两个底层集合在一种结构中可能是同构的，而在另一种结构中则不同。 

例如，两个空间可能是同构的，但不是等距的。 它们具有相同的拓扑结构但不同的度量结构。 

这说明同构并不是绝对的。 这与我们选择保留的结构有关。 

### 相等与同构

 平等说两个对象在底层形式系统中实际上是同一个对象。 

同构说两个对象具有相同的结构。 

| 关系 | 意义| 实力|
 | ----------- | ----------------------- | -------- |
 |$A = B$| 字面上的平等| 更强|
 |$A \cong B$| 结构相同| 较弱|
 |$A \sim B$| 一些选择的等价物| 最弱|

 在普通的数学实践中，一旦同构成立，同构对象通常被视为相同的对象。 这是有效的，但应该谨慎执行。 

例如：$$
V \cong k^n
$$对于每一个$n$维向量空间$V$超过$k$。 但$V$不等于$k^n$除非它是这样定义的。 

选择一个基给出同构：$$
V \to k^n.
$$不同的碱基给出不同的同构。 因此，这种识别通常是不规范的。 

### 规范同构

 当一个同构是由给定结构自然决定的，没有任意选择时，它就是规范的。 

例如，有限维向量空间$V$自然同构于其双对偶：$$
V \cong V^{**}
$$地图发送每个向量$v \in V$到评估函数$V^*$:$$
\phi \mapsto \phi(v).
$$这种构造不需要选择基础。 

相比之下，同构$$
V \cong k^n
$$需要选择一个基础。 因此，它总体上是非规范的。 

| 声明| 需要选择吗？ | 典范？ |
 | -------------------------------------------------------------------------- | ----------------------: | ---------: |
 |$V \cong k^n$| 是的，选择一个基础| 没有 |
 |$V \cong V^{**}$对于有限维$V$| 无需基础 | 是的 |
 | 有限集$n$元素是双射的${1,\dots,n}$| 是的，选择订购| 没有 |

 规范同构用作标识更安全，因为它们不依赖于隐藏的选择。 

### 同构下的不变量

 不变量是同构所保留的属性。 

如果$A \cong B$，那么任何结构不变量$A$等于相应的不变量$B$。 

示例：

 | 结构| 不变|
 | ----------------------- | -------------------------------------------------------------------- |
 | 有限集 | 基数 |
 | 向量空间| 尺寸|
 | 相似度下的矩阵 | 迹、行列式、特征值 |
 | 集团| 阶、阿贝尔性质、子群结构 |
 | 拓扑空间| 紧凑性、连通性 |
 | 图| 顶点数、度数序列、连通分量 |

 不变量有助于证明两个对象不是同构的。 如果一个不变量不同，则不存在同构。 

例如，同一域上不同维度的向量空间不是同构的：$$
k^m \cong k^n \quad \text{if and only if} \quad m = n.
$$对于固定域上的有限维向量空间，维数是完全不变量。 

### 完全和不完全不变量

 如果一个不变量完全将对象分类到同构，那么它就是完整的。 

维数是固定域上有限维向量空间的完全不变量。 

对于有限群来说，群阶并不是完全不变量。 可以存在具有相同元素数量的非同构群。 

| 领域| 不变| 完全的？ |
 | ---------------------------------------------------- | ---------------------------- | --------- |
 | 有限维向量空间$k$| 尺寸| 是的 |
 | 有限集| 基数 | 是的 |
 | 有限群| 订单| 没有 |
 | 图表| 顶点和边的数量 | 没有 |
 | 拓扑空间| 连通性 | 没有 |

 分类问题需要完整的不变量或规范代表。 

### 同构类

 我们经常研究同构类，而不是研究单个对象。 

对象的同构类$A$是同构于的所有对象的集合$A$:$$
[A] = {B : B \cong A}.
$$分类定理描述了这些类别。 

例如：

 每个有限维向量空间$k$恰好属于一个同构类，由其维数决定。 

这意味着分类是：$$
0, 1, 2, 3, \dots
$$在哪里$n$对应于$k^n$。 

### 同构和传输

 同构允许结构和定理的传输。 

如果$f : A \to B$是同构，那么关于的任何结构事实$A$有一个相应的事实$B$。 

例如，如果$G \cong H$和$G$是阿贝尔的，那么$H$是阿贝尔的。 

证明草图：

 让$h_1,h_2 \in H$。 自从$f$是满射的，选择$g_1,g_2 \in G$和$$
f(g_1)=h_1,\quad f(g_2)=h_2.
$$然后$$
h_1h_2 = f(g_1)f(g_2)=f(g_1g_2)=f(g_2g_1)=f(g_2)f(g_1)=h_2h_1.
$$所以$H$是阿贝尔的。 

由于保留了组操作，因此保留了该属性。 

### 自同构

 自同构是对象与其自身的同构。$$
f : A \to A
$$自同构描述了内部对称性。 

| 对象| 自同构 |
 | ----------------- | -------------------------------------- |
 | 设置| 排列|
 | 集团| 保结构双射 |
 | 向量空间| 可逆线性映射 |
 | 图| 顶点重新标记保留边 |
 | 拓扑空间| 从空间到自身的同胚|

 对象的自同构在组合下形成群：$$
\operatorname{Aut}(A).
$$该组测量对称性$A$。 

### 结构实践

 在结构数学中，陈述通常在同构下应该是不变的。 如果当对象被同构副本替换时属性发生变化，则该属性取决于表示而不是结构。 

例如，短语“第一个坐标$v$为零”取决于所选的基础。它不是向量的结构属性$v \in V$除非坐标系是固定的。 

相比之下，声明“$v = 0$”是结构性的。

 | 声明| 结构性的？ |
 | -------------------------------------------------- | --------------------------: |
 |$v = 0$| 是的 |
 | 第一个坐标为$v$为零 | 不可以，除非基础是固定的 |
 |$T$是单射 | 是的 |
 | 矩阵为$T$左上角的条目中有一个零 | 不可以，除非底座是固定的 |
 |$\dim V = n$| 是的 |

 这种区别有助于防止依赖于表示的争论。 

### 实用规则

 当两个物体看起来相似时，问：

 | 问题 | 目的|
 | -------------------------------------------------- | ----------------------------------------------------------- |
 | 什么样的结构被保留？ | 确定同构的含义 |
 | 地图可以反转吗？ | 区分同构和一般态射 |
 | 同构是规范的吗？ | 检测隐藏的选择 |
 | 保留了哪些不变量？ | 转让财产|
 | 我们使用的是相等还是同构？ | 防止夸大|

 同构形式化了这样的想法：对象可以在表示上不同但在结构上相同。 它是使抽象变得精确的主要工具之一。
