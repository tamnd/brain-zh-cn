## 1.1 抽象对象和结构

 数学通过剔除不相关的细节来研究物体。 数字、函数、图形、空间和群并不是作为物理事物来研究的。 通过它们满足的规则、它们支持的关系以及维护它们的转换来研究它们。 

抽象对象是通过其数学属性而不是其物质形式来考虑的对象。 整数`3`可以数苹果、点数、美元或步数。 在每种情况下，物理对象都不同。 数学对象保持不变，因为`3`由它在自然数系中的位置决定。 

结构是对象以及关系、操作或规则的集合。 例如，自然数形成加法、乘法、阶数和归纳的结构。 图形成具有顶点和边的结构。 向量空间形成一个由向量​​、标量乘法、向量加法以及连接这些运算的公理组成的结构。 

相同的底层集可以支持不同的结构。 实数集可以被视为有序集、域、度量空间、拓扑空间或有理数上的向量空间。 每个观点都会暴露不同的信息。 

| 实数视图 | 主要结构| 典型问题 |
 | -------------------- | -------------------------------------- | -------------------------------- |
 | 订购套装 |`<`| 哪些元素更大？ |
 | 领域 |`+`,`*`,`0`,`1`| 方程如何表现？ |
 | 度量空间| 距离 | 收敛是什么意思？ |
 | 拓扑空间| 开集| 哪些性质是连续的？ |
 | 向量空间`Q`| 加法和有理缩放 | 存在什么线性关系？ |

 抽象思维使数学能够重用思想。 一旦我们证明了关于所有群的定理，它就适用于加法下的整数、乘法下的非零有理数、平方的对称性以及许多其他系统。 该定理取决于群结构，而不取决于元素的具体性质。 

这是抽象的主要经济。 被证明处于正确水平的结果可以多次应用。 

结构通常具有三层。 

| 层| 意义| 示例|
 | ----------------------- | ------------------------------------------ | --------------------------------- |
 | 承运人 | 底层对象 | 集合的元素 |
 | 操作或关系 | 物体相互作用的方式| 加法、顺序、邻接 |
 | 定律或公理| 交互满足的规则 | 结合性、恒等性、对称性 |

 对于一个群体来说，载体就是一个集合。 该操作将两个元素组合成另一个元素。 这些定律需要结合性、单位元和逆元。 这些要求定义了结构。```text
Group = set + binary operation + group axioms
```元素本身可以是数字、矩阵、排列、函数或几何对称。 组概念忽略了它们的具体起源，只保留与操作相关的行为。 

物体和结构之间的这种分离是核心。 数学通常不太关心对象“是什么”，而更关心它在系统内的行为方式。 

当两个结构之间存在结构保持对应关系时，它们可以被视为相同。 这个想法导致了同构。 例如，偶数的加法群的行为类似于所有整数的加法群：每个整数`n`对应偶数`2n`，并且保留加法。```text
n + m  corresponds to  2n + 2m
```这些对象作为数字子集而有所不同，但它们的组结构是相同的。 

这个原则随处可见。 在线性代数中，域上的有限维向量空间按其维数分类为同构。 在图论中，如果两个图的顶点可以在保留边的情况下被重命名，那么它们在结构上是相同的。 在拓扑学中，空间通过连续映射和同胚进行比较。 

抽象对象还取决于用于描述它们的语言。 三角形可以通过欧几里得几何、坐标、向量、变换或度量关系来研究。 每种语言强调不同的结构。 没有一个是自动最终的。 有用的语言取决于问题。 

定义选择重要的结构。 

例如，在定义度量空间时，我们只保留一个集合和一个满足三个基本定律的距离函数。 我们会丢弃角度、坐标和平滑度，除非稍后添加它们。 这使得这个概念足够广泛，可以包括欧几里得空间、具有路径距离的图、函数空间和许多其他示例。 

好的抽象是有代价的。 它可以隐藏具体的直觉。 一个高度普遍的定理可能很强大，但更难理解。 一个具体的例子可能更容易计算，但更难转移。 数学实践在这些层次之间移动。 

| 水平| 实力| 风险|
 | ------------------ | -------------------------------------- | ----------------------- |
 | 具体例子| 易于计算和可视化 | 不可一概而论|
 | 抽象结构| 可跨域重用 | 可能隐藏直觉|
 | 正式制度| 精确可查| 可能会变得难以阅读 |

 该技能是为任务选择正确的抽象级别。 

在学习新的数学分支时，问四个问题：

 | 问题 | 目的|
 | --------------------------------- | -------------------------------------- |
 | 对象是什么？ | 识别承运人 |
 | 它们具有什么结构？ | 识别操作和关系 |
 | 哪些地图保留了结构？ | 识别自然变换 |
 | 哪些属性保持不变？ | 确定理论研究的内容 |

 这些问题将一个主题变成一个可用的系统。 在群论中，映射是同态，不变量包括阶数、子群和商结构。 在拓扑中，映射是连续函数，不变量包括连通性、紧致性和空洞。 在线性代数中，映射是线性变换，不变量包括维数、秩、特征值和规范形式。 

当通过结构观察对象时，数学变得连贯。 抽象并不是消除意义。 对于一类问题来说，重要的是意义的选择。
