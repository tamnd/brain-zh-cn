## 2.5 跨领域示例

 真理、可证明性和独立性之间的区别并不局限于数理逻辑。 每当一个陈述依赖于公理、解释或所选结构时，它就会出现在整个数学中。 

在算术中，预期的结构通常是自然数。```text
N = {0, 1, 2, 3, ...}
```某个陈述对于标准自然数可能是正确的，但在选定的形式系统中却无法证明。 这是不完整的核心教训。 算术足够简单，足以说明基本事实，但也足够具有表达能力，可以对证明、程序和自引用进行编码。 

在几何中，真理取决于空间公理。 欧几里得几何、球面几何和双曲几何使用不同的假设。 关于平行线的陈述可能在一种几何中成立，而在另一种几何中则失败。 

| 领域 | 声明| 取决于 |
 | ---------- | ----------------------------------- | -------------------------- |
 | 算术| 一些真实的陈述是无法证明的 | 正式制度|
 | 几何| 平行线行为 | 几何公理|
 | 集合论| 连续统假说| 集合论公理|
 | 代数 | 基地的存在| 选择原则|
 | 分析| 病理功能| 集合论假设 |
 | 拓扑 | 紧性定理 | 选择和覆盖公理|

 集合论提供了一些最直接的例子。 连续统假设与 ZFC 无关，假设 ZFC 是一致的。 这意味着集合论的通常公理并不能严格决定自然数和实数之间是否存在基数。 

代数常常隐藏基本假设。 “每个向量空间都有一个基”这一说法是经典数学中的标准，但它取决于完全一般性的选择公理。 对于有限维向量空间，不会出现这样的问题。 基础可以通过有限过程构建。 

分析包含许多存在性定理，其证明使用非构造性原理。 一些定理断言对象存在而没有给出可计算的构造。 这在经典分析中是可以接受的，但它改变了构造性或可计算分析中的解释。 

拓扑给出了另一种常见模式。 紧致性陈述通常将无限数据转换为有限数据。 有些形式是建设性的，而另一些则需要类似选择的原则。 确切的状态取决于空间和覆盖概念。 

一个实用的规则是在解释定理之前先确定框架。 

| 问题 | 为什么这很重要 |
 | -------------------------------------------------------- | ------------------------------------------------------ |
 | 公理是什么？ | 确定可以证明什么 |
 | 预期的模型是什么？ | 确定语义真实性 |
 | 证明有建设性吗？ | 确定是否有证人 |
 | 是否使用了选择原则？ | 决定基础力量 |
 | 该命题在等价条件下是否不变？ | 确定结构意义 |

 在各个领域，同样的教训都在重复。 定理绝不只是一个句子。 它属于一种语言、一种结构、一种证明系统。 

当这些是明确的时，数学陈述就变得可移植。 当它们被隐藏时，同一句话可能会从一个领域改变到另一个领域。
