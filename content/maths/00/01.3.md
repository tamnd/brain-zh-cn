## 1.3 平等、同一和等价

 数学依赖于精确的相同性概念。 符号`=`看似简单，但根据上下文的不同，它具有不同的含义。 本节分为三个层次：平等、同一和等价。 

平等是最严格的观念。 如果两个对象是底层系统中的同一个对象，则它们相等。```text
a = b
```在集合论中，相等是可扩展的。 当两个集合具有完全相同的元素时，它们相等。```text
A = B  if and only if  ∀x (x ∈ A ↔ x ∈ B)
```该定义忽略了集合的描述方式。 套装`{1,2,3}`和`{3,2,1}`是平等的，因为成员资格匹配。 

平等的表现是可预测的：

 | 物业 | 意义|
 | ------------ | ------------------------------------------------------ |
 | 反身性|`a = a`|
 | 对称|`a = b`意味着`b = a`|
 | 及物|`a = b`和`b = c`意味着`a = c`|
 | 替代| 将任意表达式中的 equals 替换为 equals |

 替补至关重要。 如果`a = b`，那么任何关于`a`当`a`被替换为`b`。 

身份更加微妙。 它询问结构中的两个对象是否应被视为相同。 身份可能取决于对象的构造或标记方式。 

例如，考虑有序对：```text
(1,2) ≠ (2,1)
```元素作为一个集合进行匹配`{1,2}`，但顺序很重要。 身份包括超越成员资格的结构。 

在代数中，恒等通常指运算的中性元素：```text
a + 0 = a
a * 1 = a
```这里，`0`和`1`是加法和乘法的单位元。 这是“身份”一词的不同用法。 上下文决定意义。 

在类型理论环境中，身份可能需要证据或证人。 两个物体的价值可能相同，但结构不同。 这种区别出现在形式验证和证明助手中。 

等价是对平等的有控制的削弱。 如果两个对象共享感兴趣的属性，则它们是等效的，即使它们不相等。 

等价关系是一种关系`~`满足：

 | 物业 | 意义|
 | ---------- | --------------------------------- |
 | 反身性|`a ~ a`|
 | 对称|`a ~ b`意味着`b ~ a`|
 | 及物 |`a ~ b`和`b ~ c`意味着`a ~ c`|

 等价将集合划分为类。 每个类都包含在所选关系下被视为相同的对象。 

例子出现在数学中。 

| 域名 | 等价关系| 解读|
 | -------------- | -------------------- | -------------------------- |
 | 整数|`a ≡ b (mod n)`| 余数相同 |
 | 几何| 一致性| 形状和尺寸相同|
 | 线性代数 | 行等价| 相同的解决方案空间|
 | 拓扑 | 同胚 | 相同的拓扑结构|
 | 代数 | 同构 | 相同的代数结构 |

 等价性降低了复杂性。 我们不是单独研究所有对象，而是研究在给定标准下表现相同的对象类别。 

同构是一个中心例子。 当存在保留结构的双射映射时，两个结构是同构的。```text
f(a ⋆ b) = f(a) ⋆ f(b)
```该方程表示操作的保存`⋆`。 确切的形式取决于结构。 在小组中，`⋆`是团体运作。 在向量空间中，它包括加法和标量乘法。 在图中，它保留邻接性。 

出于结构目的，同构对象被视为相同。 这个想法允许对结果进行分类。 例如，域上的所有有限维向量空间都由其维数确定为同构。 

平等、同一和对等形成了一个等级制度。 

| 水平| 严格| 使用 |
 | ----------- | ----------------- | ------------------- |
 | 平等| 最强| 精确替换 |
 | 身份 | 上下文相关 | 结构相同|
 | 等价| 最弱| 分类|

 当这些级别混合时就会产生混乱。 例如，写`A = B`当仅存在同构时隐藏了重要的结构。 仔细写下来，我们可以区分：```text
A = B        (exact equality)
A ≅ B        (isomorphism)
A ~ B        (equivalence)
```每个符号代表不同的主张。 

在实践中，一旦上下文变得清晰，数学家常常会模糊这些区别。 在证明两个结构同构后，它们可以被视为可互换。 这简化了推理，但假设结构保留图已经建立。 

精确的方法遵循以下模式：

 | 步骤| 行动|
 | ---- | ------------------------------------------------ |
 | 1 | 定义对象和结构 |
 | 2 | 定义等价概念 |
 | 3 | 证明性质在等价条件下不变 |
 | 4 | 将对象分类至等效 |

 这种模式出现在很多领域。 在群论中，群被分类为同构。 在拓扑学中，空间被分类为同胚。 在线性代数中，矩阵根据相似度进行分类。 

选择正确的相同性概念是一个设计决策。 太严格了，理论就会变得僵化。 太弱了，区别就消失了。 适当的级别取决于问题。 

数学的进步是通过在可能的情况下用结构化的等价代替原始的等价来实现的。 这种转变揭示了更深层次的不变量并降低了复杂性。
