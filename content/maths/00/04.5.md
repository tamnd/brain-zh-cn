## 4.5 示例：群、空间、图

 通过示例，结构思维变得更加清晰。 本节展示相同的原则如何出现在不同的领域。 

我们关注三个领域：群、拓扑空间和图。 每个都有自己的对象、态射、不变量和分类问题。 
### 1. 团体

 一个组就是一个集合$G$具有满足结合性、恒等性和逆性的二元运算。 

#### 对象和地图

 * 对象：组$(G, \ast)$* 态射：群同态$$
  f(a \ast b) = f(a) \cdot f(b)
  $$* 同构：双射同态

 #### 示例实例

 | 集团| 描述 |
 | ---------------------------- | -------------------------------------- |
 |$(\mathbb{Z}, +)$| 加法下的整数 |
 |$(\mathbb{Q}^\times, \cdot)$| 乘法下的非零有理数 |
 |$S_n$| 的排列$n$元素|
 |$D_4$| 正方形的对称性|

 这些是同一结构类型的不同表示。 

#### 不变量

 | 不变| 意义| | |
 | ------------------ | -------------------- | - | -------------------------------- |
 | 命令$            | G                     | $| 元素数量（有限情况）|
 | 阿贝尔性质 | 无论$ab = ba$| | |
 | 子组结构 | 内部组织| | |
 | 元素顺序 | 循环行为 | | |

 #### 分类示例

 * 每组素数阶$p$同构于$\mathbb{Z}/p\mathbb{Z}$* 有限阿贝尔群可以通过分解成循环因子来分类

 这显示了通过不变量的结构分类。 
### 2.拓扑空间

 拓扑空间是一个集合$X$以及一系列开放集。 

#### 对象和地图

 * 对象：空间$(X, \mathcal{T})$* 态射：连续映射
 * 同构：同构

 一张地图$f : X \to Y$是连续的，如果$$
f^{-1}(U) \text{ is open in } X \quad \text{for every open } U \subseteq Y.
$$#### 示例实例

 | 空间| 描述 |
 | -------------- | ---------------------------------------------------- |
 |$\mathbb{R}$| 具有通常拓扑的实线 |
 |$[0,1]$| 闭区间|
 | 圆圈$S^1$| 单位距离内的点$\mathbb{R}^2$|
 | 离散空间| 每个子集都是开放的 |

 #### 不变量

 | 不变| 意义|
 | -------------------- | --------------------------------------- |
 | 连通性 | 不能分成不相交的开集 |
 | 紧凑性| 有限子覆盖性质|
 | 组件数量| 分解成碎片|
 | 基础组| 循环结构（高级）|

 #### 示例比较

 *$[0,1]$和$(0,1)$不是同胚的
 *$\mathbb{R}$和$(0,1)$是同胚的

 这表明拓扑忽略距离但保留开集结构。 
### 3.图表

 图由顶点和边组成。 

#### 对象和地图

 * 对象：图表$G = (V, E)$* 态射：图同态
 * 同构：双射保持邻接

 #### 示例实例

 | 图| 描述 |
 | -------------------- | ----------------------- |
 | 路径图| 顶点链|
 | 循环图| 闭环|
 | 完整图$K_n$| 所有对均已连接 |
 | 树| 连通无环图 |

 #### 不变量

 | 不变| 意义|
 | -------------------- | ---------------------------------- |
 | 顶点数 | 尺寸|
 | 边数 | 连接水平|
 | 学位序列 | 顶点度数 |
 | 连接组件| 连接结构|
 | 循环的存在 | 循环结构|

 #### 示例比较

 具有不同度序列的两个图不能同构。 
然而，具有相同度序列的图仍然可能是非同构的。 

这表明不变量可能是不完整的。 
### 4.跨域比较

 相同的结构模式出现在所有三个域中。 

| 概念 | 团体 | 拓扑 | 图表|
 | -------------- | ------------------- | -------------------------- | ------------------- |
 | 对象| 团体 | 空间 | 图表|
 | 态射 | 同态| 连续地图 | 图同态|
 | 同构 | 群同构| 同胚 | 图同构|
 | 不变量 | 阿贝尔秩序 | 紧凑性、连通性| 度、成分|
 | 分类| 部分或完整| 经常偏袒| 常常辛苦|

 该表显示结构思维在各个领域是一致的。 
### 5. 表示与结构

 每个域都有多种表示形式。 

| 对象| 代表| 结构|
 | ------ | -------------------- | ------------------ |
 | 集团| 乘法口诀表| 抽象操作|
 | 空间| 坐标| 公开组|
 | 图| 邻接表 | 连接性|

 不同的表示可以描述相同的结构。 

例如，图表可以存储为：

 * 邻接表
 * 邻接矩阵
 * 边缘设置

 它们的表示方式不同，但编码相同的结构。 
### 6. 重要经验教训

 * 结构决定了什么是重要的
 * 态射定义允许的变换
 * 同构定义相同性
 * 不变量检测差异
 * 分类组织对象

 这些想法在数学中不断重复。 

一个领域中的问题通常在另一个领域中也有类似问题。 结构思维允许结果转移。 

例如：

 * 群中的分解与向量空间中的分解并行
 * 图形中的连通性与拓扑中的连通性并行
 * 几何中的对称性与代数中的自同构相似
 ### 7. 实际工作流程

 当面对新的结构时：

 | 步骤| 行动|
 | ---- | ---------------------- |
 | 1 | 识别物体 |
 | 2 | 识别态射 |
 | 3 | 确定不变量 |
 | 4 | 通过不变量进行比较 |
 | 5 | 尝试分类|

 此工作流程适用于跨域。 
当你在不同的环境中看到相同的模式时，结构性思维就会变得有效。 组、空间和图形表面上看起来不同，但它们遵循相同的底层设计。
