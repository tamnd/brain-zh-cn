## 4.4 不变量和分类

 不变量是在选定的变换类别下保持不变的属性、数量或结构。 

不变量是结构思维的核心。 它们使我们能够在不依赖于表示的情况下比较对象。 

如果两个对象是同构的，则它们的结构不变量一致。 因此，如果不变量不同，则对象不能同构。 

例如，具有不同基数的两个有限集不能是双射的。 同一域上不同维数的两个向量空间不能同构。 具有不同数量连通分量的两个图不能同构。 

| 结构| 转型| 不变|
 | ----------------- | ------------------ | ---------------------------------------------------- |
 | 有限集| 双射 | 基数 |
 | 向量空间| 线性同构| 尺寸|
 | 矩阵| 基础变更 | 秩、迹、行列式|
 | 图| 图同构 | 度数序列，连通分量 |
 | 拓扑空间| 同胚 | 紧凑性、连通性 |
 | 集团| 群同构| 序、阿贝尔性质、子群格 |

 不变量很有用，因为它在表示形式发生变化后仍然存在。 

仅在选择碱基后，矩阵才表示线性映射。 更改基数可能会更改矩阵条目，但不会更改诸如等级之类的结构量。 

如果$A$和$B$表示不同基下的同一个线性算子，那么它们是相似的：$$
B = P^{-1}AP.
$$在相似性下，迹和行列式是不变的：$$
\operatorname{tr}(B) = \operatorname{tr}(A)
$$和$$
\det(B) = \det(A).
$$这意味着迹和行列式属于运算符，而不仅仅是选定的矩阵表示。 

### 不变量区分对象

 不变量的第一次使用是否定的：它证明两个对象不等价。 

认为$V$和$W$是同一域上的有限维向量空间$k$。 如果$$
\dim V \neq \dim W,
$$然后$$
V \not\cong W.
$$道理很简单。 线性同构保留维度。 如果维度不同，则不存在线性同构。 

相同的模式出现在各个领域。 

| 观察| 结论 |
 | ------------------------------------------------------ | ---------------------------------------------------------- |
 | 有限集有不同的大小 | 无双射 |
 | 团体有不同的顺序| 没有群同构|
 | 图有不同的度数序列 | 没有图同构 |
 | 空间具有不同的紧凑性行为 | 没有同胚 |
 | 矩阵有不同的秩 | 行和列操作不等价 |

 这通常是分离对象的最快方法。 

### 不变量可能不完整

 即使对象不是同构的，不变量也可能无法区分对象。 

例如，两个有限群可以具有相同的阶数但不同的结构。 有序循环群$4$和克莱因四群都有四个元素，但它们不是同构的。$$
\mathbb{Z}/4\mathbb{Z} \not\cong \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}.
$$顺序相同，但元素顺序不同。 在$\mathbb{Z}/4\mathbb{Z}$，有一个顺序元素$4$。 在克莱因四群中，每个非同一性元素都有序$2$。 

因此，对于有限群来说，单独的阶数并不是完全不变量。 

| 不变| 可能会失败，因为 |
 | ---------------------------------- | -------------------------------------------------------------------------- |
 | 集合的基数 | 有限集的完整 |
 | 向量空间的维数 | 固定域上有限维向量空间的完备性 |
 | 团体顺序| 不同的团体结构可以共享订单 |
 | 图的度数序列 | 非同构图可以共享度数序列 |
 | 空间的连通性| 许多非同胚空间是相连的 |

 不完全不变量仍然有价值。 它可以排除等价性、指导分类并提出更强的不变量。 

### 完整的不变量

 完全不变量将对象完全分类到选定的等价性。 

对于有限集，基数是完整的。$$
A \cong B \quad \text{if and only if} \quad |A| = |B|.
$$对于固定域上的有限维向量空间$k$，维度已完成。$$
V \cong W \quad \text{if and only if} \quad \dim V = \dim W.
$$完全不变量给出两个方向：

 | 方向 | 意义|
 | ------------------------------------------- | ---------------------- |
 | 如果对象是等价的，则不变式同意 | 保留不变式 |
 | 如果不变量一致，则对象是等价的 | 不变量已完成 |

 第二个方向通常更难。 

例如，证明同构向量空间具有相同的维数就是保值。 证明任意两个具有相同有限维数的向量空间同构需要构造同构。 

### 分类

 分类是描述一个类中的所有对象直至相等的过程。 

分类定理通常具有以下形式：```text
Objects of type X are classified up to equivalence by invariant I.
```示例：

 | 对象| 分类不变式|
 | --------------------------------------------------- | ------------------------ |
 | 有限集| 基数 |
 | 有限维向量空间$k$| 尺寸|
 | 有限生成阿贝尔群 | 排名和扭转系数|
 | 正交相似下的实对称矩阵| 特征值 |
 | 紧凑的连接表面 | 属性和定向性|

 分类将大量对象减少为更简单的参数列表。 

对于有限维向量空间，分类尤其清晰：

 每个有限维向量空间$k$同构于$k^n$为了一个独特的$n$。 

这里$n = \dim V$。 

### 规范形式

 规范形式是每个等价类的选定代表。 

对于有限维向量空间，$k^n$是尺寸的标准代表$n$。 

对于矩阵，行简化梯形形式是行等价下的规范形式。 

对于某些类型的矩阵，乔丹范式给出了代数闭域上的规范形式（有条件）。 

| 设置| 规范形式|
 | ------------------------------------------------ | ------------------------ |
 | 有限集|${1,\dots,n}$|
 | 向量空间 |$k^n$|
 | 行等价下的矩阵 | 简化行梯形|
 | 某些线性算子 | 乔丹范式 |
 | 合适域上的二次形式 | 对角形式|

 规范形式很有用，因为它们将等价问题转化为代表之间的平等问题。 

而不是问是否$A$和$B$是等价的，计算它们的规范形式并进行比较。 

### 强不变量和弱不变量

 不变量的强度不同。 

弱不变量很容易计算，但区分的对象较少。 强不变量可以区分更多对象，但可能更难计算。 

| 不变| 实力| 成本|
 | -------------------- | -------------------------------- | -------------- |
 | 基数 | 从低到高取决于具体情况 | 通常较低 |
 | 尺寸| 向量空间的强项 | 低|
 | 学位序列 | 图表弱 | 低|
 | 矩阵的谱 | 中度至强| 中等|
 | 基础组| 强拓扑不变量 | 高|
 | 同源群 | 强大但不完整| 中到高|

 数学实践经常使用一系列不变量。 从廉价的不变量开始。 如果它们不能将物体分开，则转向更坚固的物体。 

### 地图下的不变性

 不变量是相对于一类映射来定义的。 

属性在一种变换下可能是不变的，但在另一种变换下则不然。 

例如，距离在等距下是不变的，但在同胚下则不然。 连通性在同胚下是不变的，但不能确定度量距离。 

| 物业 | 保存者 |
 | ------------- | ------------------------------------------------ |
 | 距离 | 等距|
 | 角度| 等形地图，在适当的设置下 |
 | 连通性 | 同胚 |
 | 尺寸| 线性同构|
 | 排名| 矩阵等价|
 | 团体订购 | 群同构|

 这意味着不变量必须始终与其等价关系一起读取。 

问题不仅仅是“保存什么？” 但“在什么转变下得以保存？”

 ### 分类工作流程

 典型的分类问题遵循以下模式：

 | 步骤| 行动|
 | ---- | ------------------------------------------------ |
 | 1 | 定义对象 |
 | 2 | 定义等价关系 |
 | 3 | 识别不变量 |
 | 4 | 证明不变量被保留 |
 | 5 | 测试不变量是否完整 |
 | 6 | 寻找规范代表|
 | 7 | 证明代表的唯一性 |

 此工作流程将问题分成可管理的部分。 

例如，对有限维向量空间进行分类：

 | 步骤| 结果 |
 | ------------------------ | ---------------------------------------------------- |
 | 对象| 有限维向量空间$k$|
 | 等价| 线性同构|
 | 不变| 尺寸|
 | 保存| 同构保留碱基 |
 | 完整性| 相同的维度给出了同构 |
 | 规范代表|$k^n$|
 | 独特性|$n$是独一无二的|

 结果是一个完整的分类。 

### 分类的限制

 并非每个分类问题都有简单的答案。 有些类太大，太狂野，或者对微小的变化太敏感。 

有限维向量空间很容易分类。 有限群要困难得多。 图表一般来说是困难的。 拓扑空间极其广阔。 

在困难的环境中，分类可能是不完整的。 人们可以对特殊情况进行分类，定义不变量，或者证明在选定的约束下不存在合理的完整分类。 

| 情况| 结果|
 | -------------------------- | -------------------------------------- |
 | 简单课堂| 分类齐全|
 | 中等班级| 具有几个不变量的分类 |
 | 狂野类| 仅部分分类 |
 | 重计算类| 算法分类|
 | 基础敏感类| 分类取决于公理 |

 在这种情况下，即使没有完全分类，不变量仍然有用。 

### 实用规则

 当比较数学对象时，问：

 | 问题 | 目的|
 | ---------------------------------------------------- | ------------------------------------------- |
 | 使用什么等价关系？ | 定义相同性 |
 | 保留了哪些属性？ | 给出不变量|
 | 不变量是否完整？ | 确定分类能力 |
 | 我们可以计算它们吗？ | 决定实际用途|
 | 有规范代表吗？ | 简化比较 |

 不变量是结构的可测量残基。 当代表发生变化时，它们仍然存在。 分类根据这些残基来组织物体，并询问它们是否讲述了整个故事。
