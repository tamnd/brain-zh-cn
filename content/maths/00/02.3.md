## 2.3 一致性和完整性

 判断一个正式系统的部分标准是它能证明什么，部分是它避免证明什么。 两个核心属性是一致性和完整性。 

一致性意味着系统不会证明矛盾。 如果没有声明，系统是一致的`P`使得两者`P`和`¬P`是可证明的。```text
not both:  ⊢ P  and  ⊢ ¬P
```不一致的系统无法用于普通数学，因为矛盾会破坏辨别力。 在经典逻辑中，人们可以从矛盾中推导出任何陈述。 这个原理称为爆炸。```text
P, ¬P ⊢ Q
```如果一个系统证明了一个矛盾，那么每个陈述都变得可证明。 错误的算术、错误的几何和武断的主张都随之而来。 证明不再区分有效陈述和无效陈述。 

完整性有多种含义，区别很重要。 

如果每个逻辑上有效的陈述都是可证明的，那么证明系统在语义上是完整的。```text
If every model satisfies φ, then ⊢ φ.
```从这个意义上来说，一阶逻辑是完整的。 如果一阶句子在每种解释下的每种结构中都是正确的，那么就有了它的形式证明。 

如果对于每个句子，一个理论在句法上是完整的`φ`用它的语言，要么`φ`或者`¬φ`是可证明的。```text
For every φ:  ⊢ φ  or  ⊢ ¬φ
```这意味着理论决定了用其语言表达的每一个陈述。 

完整性的这两种用途不应混淆。 

| 物业 | 适用于 | 意义|
 | ---------------------- | ------------------ | ---------------------------------------------------- |
 | 语义完整性 | 证明系统| 每个逻辑上有效的公式都是可证明的 |
 | 句法完整性 | 一个理论| 每一句话都已决定|
 | 一致性| 系统或理论| 无法证明任何矛盾 |

 一个理论可以是一致的但不完整的。 这很常见。 这意味着该理论避免了矛盾，但仍有一些未定论。 

算术给出了中心例子。 任何充分表达、有效公理化、一致的算术理论都不能决定每一个算术陈述。 有些陈述独立于理论。```text
T ⊬ φ
T ⊬ ¬φ
```声明`φ`然后是不可判定的`T`。 

这并不意味着`φ`没有意义。 这意味着所选择的公理不足以解决它。 更强的公理可能会决定它。 

一致性通常是最低要求。 完整性更强，但往往不可用。 

| 系统| 持续的？ | 完全的？ |
 | ------------------- | ------------------------ | ------------------------ |
 | 命题逻辑| 是的，符合标准规则 | 语义完整 |
 | 一阶逻辑 | 是的，符合标准规则 | 语义完整 |
 | 皮亚诺算术 | 预期一致| 语法不完整 |
 | ZFC 集合论 | 预期一致 | 语法不完整 |

 对于强理论，一致性通常不能从同一理论内部得到证明。 一个足够强的一致理论无法证明其自身的一致性，假设它以所需的形式方式表达。 这是形式化的主要限制之一。 

为了证明一致性，人们常常采用一种更强的元理论。 例如，人们可能会证明一种理论有一个模型。 如果模型存在，则理论是一致的，因为该模型中的所有公理都是正确的，并且有效的推理规则保留了真理。```text
Model exists  ⇒  theory is consistent
```这将一致性与语义联系起来。 

完整性将证据与决策联系起来。 完整的理论给出了其语言中的每个句子的答案。 不完整的理论会留下空白。 这些差距并不一定是缺陷。 他们可能表明该语言足够丰富，可以表达需要附加原则才能回答的问题。 

几何提供了一个有用的例子。 欧几里得几何包括平行公设。 如果删除该假设，欧几里德几何和非欧几里德几何都成为可能。 其余公理不决定并行公设。 

| 公理系统| 平行假设状态 |
 | ------------------- | -------------------------------------- |
 | 欧几里得几何| 假设为真 |
 | 双曲几何| 被不同的公理取代|
 | 中性几何| 独立|

 独立性表明可以存在多个一致的扩展。```text
T + φ      consistent
T + ¬φ     consistent
```当两个扩展一致时，原始理论不能确定该陈述。 

完整性还与类别性相关。 如果一个理论的所有模型在同构方面本质上都是相同的，那么它就是绝对的。 如果一个理论有许多非同构模型，那么它的公理不会唯一地孤立一个结构。 

这在算术和集合论中很重要。 一阶理论通常有非标准模型。 即使预期的模型非正式地很清楚，正式公理也可能承认其他模型。 

实用的阅读策略是：

 | 问题 | 检查什么 |
 | -------------------------------------------------- | ---------------------- |
 | 该系统能证明矛盾吗？ | 一致性|
 | 证明系统是否捕获了所有有效的推理？ | 语义完整性 |
 | 理论决定每句话吗？ | 句法完整性 |
 | 有多种型号吗？ | 类别性 |
 | 声明未定吗？ | 独立|

 数学基础平衡强度和安全性。 更强的公理证明了更多的定理，但可能带有更大的一致性假设。 较弱的公理更安全，但决定的问题更少。 

一致性可以防止推理崩溃。 完整性衡量系统可以决定的程度。 它们之间的紧张关系解释了为什么形式数学既有力量又有局限性。
