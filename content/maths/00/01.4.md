## 1.4 有限对象与无限对象

 数学对有限对象和无限对象进行了严格区分。 许多定义在这两种环境中看起来很相似，但它们的行为在本质上有所不同。 理解这个边界对于推理、证明设计和算法思维是必要的。 

如果一个集合的元素可以完整列出并可以用自然数来计数，则该集合是有限的。```text id="m7kq2p"
|A| = n   for some n ∈ N
```如果不存在这样的有限计数，则集合是无限的。 更准确地说，如果一个集合可以与其自身的真子集一一对应，那么它就是无限的。```text id="w8p1sx"
A is infinite  if  ∃B ⊂ A such that A ≅ B
```对于有限集，不会出现此性质。 它抓住了无穷大的核心特征：约简下的自相似性。 

示例阐明了区别。 

| 设置| 尺寸| 观察|
 | ------------------- | -------- | -------------------------------------- |
 |`{1,2,3}`| 有限| 可以进行完整的枚举 |
 | 自然数`N`| 无限|`N ≅ N \ {0}`|
 | 偶数| 无限| 尺寸相同`N`|
 | 实数 | 无限| 严格大于`N`|

 有限集支持穷举推理。 每个元素都可以直接检查。 证明通常通过列举、案例分析或归纳来进行。 

无限集需要不同的方法。 直接枚举是不可能的。 证明依赖于结构、限制、紧凑性或不变量。 

这种差异体现在基本属性上。 

| 物业 | 有限情况| 无限案例|
 | ---------------------------- | -------------------------------------- | ----------------------- |
 | 子集大小 | 除非相等，否则严格小于 | 可以匹配整体尺寸|
 | 注入意味着喷射 | 真实| 假 |
 | Surjection 意味着注入 | 真实| 假 |
 | 有界序列 | 永远有限| 可能是无限|
 | 终止| 保证 | 需要证明 |

 对于有限集，当域和余域大小相等时，单射函数自动成为满射函数。 对于无限集，这会失败。 例如地图：```text id="h3v8td"
f(n) = n + 1
```在自然数上是单射的，但不是满射的。 

基数衡量集合的大小。 对于有限集，基数是自然数。 对于无限集，基数使用双射来比较大小。 

如果两个集合之间存在双射，则它们具有相同的基数。```text id="b9q4rn"
A ≅ B  ⇔  ∃ bijection f : A → B
```这导致了令人惊讶的结果。 自然数集和偶数集具有相同的基数。 映射：```text id="z2l6yx"
f(n) = 2n
```是双射。 

无限集的大小仍然可以不同。 实数严格大于自然数。 这是从对角论证得出的，表明它们之间不存在双射。 

| 设置| 基数类型 |
 | ---------------- | ---------------- |
 | 自然数| 可数 |
 | 整数 | 可数|
 | 有理数 | 可数 |
 | 实数 | 不可数|

 可数性意味着元素可以按序列列出，即使序列永远不会结束。 不可数意味着不存在这样的列表。 

有限与无限的区别也影响代数和几何。 

在线性代数中，每个有限维向量空间都有一个具有固定数量元素的基。 无限维空间可能具有无法显式构造的基并且需要额外的公理。 

在图论中，有限图可以直接存储和处理。 无限图需要局部规则和结构属性。 

在分析中，有限和的表现与无限级数不同。 融合成为一个中心问题。 重新排列有限和中的项不会改变结果。 重新排列无穷级数可能会改变值或破坏收敛性。```text id="p6n2cf"
∑_{i=1}^{n} a_i     (finite sum)
∑_{i=1}^{∞} a_i     (infinite series)
```极限形式化了关于无限过程的推理。 如果序列的项接近固定值，则序列收敛。```text id="r4x8ko"
lim_{n→∞} a_n = L
```这用近似代替了直接计算。 

紧致性是区分有限推理和无限推理的另一个原则。 在有限的设置中，覆盖参数是微不足道的。 在无限设置中，紧致性确保某些无限集合的行为类似于有限集合。 例如，在拓扑中，紧凑的空间允许每个开放的覆盖层减少到有限的子覆盖层。 

计算突出了这种区别。 算法始终对有限数据进行操作，即使在对无限对象进行建模时也是如此。 实数由有限近似表示。 函数由代码表示。 通过有限的描述可以访问无限的结构。 

这导致了两种推理模式。 

| 模式| 描述 |
 | -------- | --------------------------------- |
 | 有限| 直接计算、枚举 |
 | 无限| 近似值、极限、不变量 |

 弥合这些模式是一项中心任务。 许多定理将有限行为与无限结构联系起来。 例如，多项式由有限多个系数确定，但在无限域上定义函数。 

构造性数学在对待无穷大时附加了额外的约束。 无限对象必须由产生其元素的规则给出。 经典数学使用诸如选择公理之类的原理，允许存在而无需明确的构造。 

这种差异会影响有效证明的定义。 

| 方法| 无限远景|
 | ------------ | ---------------------------------- |
 | 建设性的| 必须提供明确的构造 |
 | 古典| 可以间接断言存在|

 有限对象在计算下是稳定的。 无限的物体需要解释。 关于无限结构的证明通常会将其简化为有限近似或有限见证。 

实用指南：

 | 问题 | 有限答案 | 无限答案 |
 | -------------------------------- | ------------- | ---------------- |
 | 我可以列出所有元素吗？ | 是的 | 没有 |
 | 我可以检查所有案例吗？ | 是的 | 没有 |
 | 我需要限制或收敛吗？ | 没有 | 是的 |
 | 注入是否意味着双射？ | 是的 | 没有 |

 许多错误都是由于将有限的直觉应用于无限的设置而产生的。 仔细的声明区分适用哪种制度。 

通过使用受控工具将有限推理扩展到无限领域，数学得以进步。 这包括限制、基数、紧凑性和抽象。
