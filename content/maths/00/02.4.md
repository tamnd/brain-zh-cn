## 2.4 独立现象

 当系统既不能证明该命题也不能证明其否定时，该命题就独立于公理系统。 独立性是不可判定性的一种精确形式。```text
T ⊬ φ
T ⊬ ¬φ
```这里`T`是一个理论并且`φ`是该理论语言中的一个句子。 该符号表示`φ`不能衍生自`T`， 和`¬φ`不能衍生自`T`任何一个。 

独立并不意味着该声明含糊不清。 这意味着所选公理不包含足够的信息来决定它。 

证明独立性的常用方法是模型构建。 为了表明`φ`独立于`T`，建立两个模型：```text
M ⊨ T + φ
N ⊨ T + ¬φ
```第一个模型满足公理`φ`。 第二个模型满足公理`¬φ`。 如果两个模型都存在，那么`T`独自无法决定`φ`，假设背景推理是合理的。 

| 目标| 方法|
 | ----------------- | -------------------------- |
 | 展示`T ⊬ φ`| 建立一个模型`T + ¬φ`|
 | 展示`T ⊬ ¬φ`| 建立一个模型`T + φ`|
 | 展现独立性 | 构建这两种模型 |

 平行公设是经典的几何例子。 欧几里得几何接受它。 双曲几何拒绝它并用平行线的不同行为代替它。 其余的公理不强制任何选择。 

| 几何| 并行行为 |
 | ---------------- | --------------------------------------------------- |
 | 欧几里得| 恰好通过线外一点的一条平行线 |
 | 双曲线 | 通过这一点的平行线不止一条 |
 | 中性几何| 不决定问题|

 这表明，由于熟悉的模型，语句可能在几何上看起来很明显，同时仍然独立于较小的公理基础。 

集合论给出了更深入的例子。 连续统假设涉及无限集的大小。 它询问是否存在一个基数严格位于自然数和实数之间的集合。 在标准 ZFC 公理中，假设 ZFC 是一致的，则连续统假设既不能被证明也不能被反驳。 

这有一个重要的后果。 两种扩展都可以作为形式理论：```text
ZFC + CH
ZFC + ¬CH
```相对于一致性假设，每一个都可以作为一个合法的数学宇宙来研究。 

独立性也出现在算术中。 哥德尔式的陈述实际上是为了表明它们在给定系统中是不可证明的。 如果系统是一致的并且足够强大来表达算术，那么这样的陈述在预期的自然数中可能是正确的，但在系统内部是无法证明的。 

这造成了以下之间的区别：

 | 状态 | 意义|
 | ------------------ | ----------------------------------------------------------- |
 | 假| 在预期的解释中反驳|
 | 未经证实| 目前尚无已知证据|
 | 无法证明在`T`| 不存在来自公理的证明`T`|
 | 独立于`T`| 无论是该陈述还是其否定都无法证明`T`|

 “未经证明”和“无法证明”之间的区别尤其重要。 一个猜想今天可能未被证明，但明天就可以被证明。 如果不改变公理或形式设置，就无法解决独立的陈述。 

独立性总是与理论相关的。 独立于一种理论的陈述可能由更强大的理论决定。```text
T ⊬ φ
S ⊢ φ
```在哪里`S`延伸`T`。 

例如，添加一个公理可以决定一个先前独立的陈述。 代价是更强的理论取决于新公理的可接受性和一致性强度。 

| 行动| 效果|
 | ------------------------------------------ | ---------------------------------------------------- |
 | 添加`φ`作为公理| 部队`φ`新理论是正确的|
 | 添加`¬φ`作为公理| 部队`φ`新理论是错误的|
 | 添加更强的结构公理 | 可以决定`φ`间接|
 | 改变逻辑或基础 | 可能会改变什么才是证据|

 独立现象表明公理不仅仅是起点。 它们塑造了正在研究的数学宇宙。 

在实践中，独立有多种用途。 

首先，它澄清了理论的局限性。 如果一个陈述是独立的，那么在同一公理系统内进一步的工作将无法解决它。 

其次，它将定理证明与公理选择分开。 一旦独立性被知晓，问题就从“我们能证明这一点吗？”转变为“我们能证明这一点吗？” 到“哪个扩展为预期目的提供了正确的理论？”

 第三，它暴露了隐藏的假设。 证明可能看起来只使用基本推理，但实际上依赖于更强有力的原理。 

选择公理提供了一个有用的例子。 代数、拓扑和分析中的许多结果都使用了它，有时甚至是隐式的。 有些定理相当于弱或强选择形式。 当省略选择时，这些定理可能会失败或变得独立。 

| 声明或原则 | 与选择的关系|
 | ------------------------------------------ | --------------------------------------- |
 | 每个向量空间都有一个基 | 用途选择|
 | 吉洪诺夫定理| 相当于常见设置中的选择 |
 | 佐恩引理 | 相当于选择|
 | 良序定理| 相当于选择|

 因此，独立性会影响数学风格。 建设性的证明可以避免更强的公理。 经典证明可以自由地使用它们。 基本的仔细证明说明了需要哪些原则。 

独立性也会影响分类。 具有独立陈述的理论允许多个非等价的扩展。 可能存在一系列一致的数学世界，而不是一个最终的宇宙。 

这在集合论中很常见，其中强制和内部模型方法构建具有不同属性的模型。 相同的基本公理可以支持有关基数、可定义性和组合原理的问题的不同答案。 

解决独立性问题的一个有用的工作流程是：

 | 步骤| 问题 |
 | ---- | ------------------------------------------------- |
 | 1 | 什么理论`T`正在被使用？ |
 | 2 | 具体是什么说法`φ`正在测试？ |
 | 3 | 能`T + φ`被建模？ |
 | 4 | 能`T + ¬φ`被建模？ |
 | 5 | 哪些附加公理决定了该陈述？ |
 | 6 | 这些公理引入了哪些数学成本？ |

 独立性并不会削弱数学。 它使公理的作用变得明确。 这说明有些问题不是单靠逻辑就能解决的。 他们需要选择框架。 

成熟的理论通过命名其假设、比较扩展以及跟踪哪些结果取决于哪些公理来处理独立性。
