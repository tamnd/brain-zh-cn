## 2.1 真理与可证明性

 数学陈述有两种不同的状态：它们可以是真实的，并且它们可以是可证明的。 这些概念相关但不相同。 

当一个陈述符合给定的解释或结构时，它就是**真实**。 这是一个语义概念。 这取决于意义。 

当一个陈述可以使用推理规则从公理推导出来时，它就是**可证明的**。 这是一个句法概念。 这取决于符号的形式操作。 

| 概念| 域名 | 定义|
 | ----------- | ---------| -------------------- |
 | 真相| 语义 | 保持在结构中 |
 | 可证明性| 语法 | 由公理推导|

 为了使这一点更加精确，我们修复了语言和结构。 

让`L`成为一种正式语言。 让`M`是解释该语言的结构。 对于一个声明`φ`：```text
M ⊨ φ      (φ is true in M)
⊢ φ        (φ is provable in the system)
```这些符号将意义与推导分开。 

一个基本的例子来自算术。 考虑以下声明：```text
∀x (x + 0 = x)
```在自然数的标准结构中，这种说法是正确的。 这也可以从通常的算术公理中证明。 在这种情况下，真理和可证明性是一致的。 

这种对齐方式并不总是成立。 

如果每个可证明的陈述在其预期解释中都是正确的，则系统是**健全的**。```text
⊢ φ  ⇒  M ⊨ φ
```健全性确保证明不会产生虚假陈述。 

如果每个真实的陈述都是可证明的，那么系统就是**完整的**。```text
M ⊨ φ  ⇒  ⊢ φ
```完整性确保所有真理都可以通过证明获得。 

这些属性取决于系统和域。 

| 系统| 声音| 完成 |
 | ---------------------------------- | -----| -------- |
 | 命题逻辑 | 是的 | 是的 |
 | 一阶逻辑 | 是的 | 是的 |
 | 算术（丰富的系统）| 是的 | 没有 |

 对于命题逻辑和一阶逻辑，完整性在精确的技术意义上成立。 对于算术等具有足够表达能力的系统，完整性是失败的。 存在系统内无法证明的真实陈述。 

这种限制源于与库尔特·哥德尔相关的不完整性现象等结果。 这些结果表明，任何能够表达基本算术的一致系统都具有真实但在该系统内无法证明的陈述。 

真理与可证明性之间的差距可以通过独立性来说明。 

声明`φ`如果两者都不是，则**独立于系统**`φ`它的否定也不能被证明：```text
⊬ φ   and   ⊬ ¬φ
```在这种情况下，系统不会决定该语句。 系统的不同扩展可能会分配不同的真值。 

例子出现在集合论中。 某些命题不能仅使用标准公理来解决。 额外的公理导致不同的一致理论。 

真理和可证明性之间的区别也取决于解释。 一个陈述可能在一种结构中为真，而在另一种结构中为假。 

| 声明| 结构| 状态 |
 | ------------------ | ------------------- | ------ |
 |`x^2 + 1 = 0`| 实数 | 假|
 |`x^2 + 1 = 0`| 复数 | 真实|
 | 平行假设| 欧几里得几何| 真实|
 | 平行假设| 双曲几何| 假|

 相比之下，可证明性仅取决于所选的公理和规则，而不取决于特定的结构。 

这导致了两种互补的观点。 

| 查看 | 焦点 |
 | ---------| ------------------ |
 | 语义| 含义和型号|
 | 句法| 形式推导 |

 证明在语法上起作用。 它根据规则转换语句。 真理是从语义上评估的。 它检查语句是否包含在结构中。 

精心设计的系统会尽可能地对齐这些视图。 健全性确保证明保持真理。 完整性确保可以通过证明得出真理。 当完整性失败时，必须仔细区分什么是真实的，什么是可以证明的。 

在实践中，数学家经常在标准解释内工作。 对于算术来说，这是自然数。 为了分析，这是实数。 在这些情况下，真理具有预期的含义，即使正式系统无法捕获全部含义。 

工作指导方针：

 | 问题 | 解读|
 | -------------------------- | ------------------------ | |
 | 该陈述是否可证明？ | 取决于公理和规则 |
 | 该说法属实吗？ | 取决于结构|
 | 系统可以决定吗？ | 取决于完整性 |

 当阅读定理时，识别这两个方面是很有用的。 定理通常断言可证明性。 它的意义往往在于它所表达的关于结构的真相。 

数学的进步是通过完善双方来实现的。 它开发出更强大的系统来证明更多的陈述，并开发更丰富的结构来解释它们。 真理与可证明性之间的界限标志着形式推理的局限性。
