## 4.2 态射和映射

 态射是保留结构的映射。 对于给定的数学设置，这是正确的变换概念。 

普通函数将一个集合的元素发送到另一集合的元素。$$
f : X \to Y
$$态射的作用更多。 它尊重定义结构的操作、关系或法律$X$和$Y$。 

对于群来说，态射是群同态。 一张地图$f : G \to H$是群同态如果$$
f(a \ast b) = f(a) \cdot f(b)
$$为所有人$a,b \in G$。 操作$\ast$属于$G$，以及操作$\cdot$属于$H$。 

对于向量空间，态射是线性映射。 一张地图$T : V \to W$是线性的，如果$$
T(v+w) = T(v) + T(w)
$$和$$
T(\lambda v) = \lambda T(v)
$$为所有人$v,w \in V$和所有标量$\lambda$。 

对于拓扑空间，态射是连续映射。 一个功能$f : X \to Y$如果每个开集的逆像是连续的$Y$开放于$X$。 

| 结构| 态射| 保存的数据|
 | ----------------- | ------------------------------------------------------ | ---------------------------------- |
 | 设置| 功能| 仅元素 |
 | 集团| 同态 | 运营|
 | 戒指| 环同态| 加法和乘法|
 | 向量空间| 线性图| 加法和标量乘法 |
 | 拓扑空间| 连续图 | 开集结构 |
 | 度量空间| 等轴测图、Lipschitz 贴图或连续贴图 | 距离相关结构 |
 | 图| 图同态| 邻接 |

 态射的选择定义了所进行的数学类型。 如果态射改变，理论就会改变。 

例如，可以使用等距图、Lipschitz 图或连续图来研究度量空间。 每个选择都会保留不同数量的结构。 

| 地图类型| 它保留了什么 |
 | -------------- | -------------------------------------------------- |
 | 等距| 精确距离 |
 | 利普希茨地图| 达到受控失真的距离|
 | 连续图| 从拓扑意义上讲，附近的点仍然是附近的 |

 因此，根据允许的映射，相同的对象可以支持不同的理论。 

### 态射作为比较

 态射比较结构。 地图来自$A$到$B$告诉我们信息如何$A$可以在里面表示$B$。 

单射态射通常将一个结构嵌入到另一个结构中。 满射态射常常将结构折叠成商。 同构给出了可逆比较。 

| 类型 | 意义|
 | ------------------- | -------------------------------------- |
 | 内射态射 | 保留元素之间的区别 |
 | 射射态射 | 覆盖目标|
 | 同构 | 可逆地保留结构 |
 | 自同态 | 从物体到自身的态射 |
 | 自同构| 从对象到自身的同构 |

 对于群同态$f : G \to H$，内核测量崩溃的内容：$$
\ker(f) = {g \in G : f(g) = e_H}.
$$图像测量的是哪一部分$H$达到：$$
\operatorname{Im}(f) = {f(g) : g \in G}.
$$这些构造将态射转变为结构数据。 

### 成分

 态射组成。 如果$f : A \to B$和$g : B \to C$是同类态射，那么它们的复合也是态射：$$
g \circ f : A \to C.
$$成分定义为$$
(g \circ f)(a) = g(f(a)).
$$这个简单的规则是态射成为核心的主要原因之一。 它允许从短的转换构建长的转换。 

组合是关联的：$$
h \circ (g \circ f) = (h \circ g) \circ f.
$$每个对象也有一个恒等态射：$$
\operatorname{id}_A : A \to A.
$$恒等态射什么也不做：$$
\operatorname{id}_A(a) = a.
$$对象、态射、恒等映射和组合共同构成了范畴论的基本模式。 

### 保存和反思

 态射可以保留属性、反映属性或两者兼而有之。 

如果源中的属性暗示目标中的相应属性，则映射会保留该属性。 

如果目标中的属性暗示源中的相应属性，则映射会反映该属性。 

例如，连续函数在许多熟悉的空间中保留序列的收敛性：$$
x_n \to x \quad \Rightarrow \quad f(x_n) \to f(x).
$$同构通常保留并反映所有可用理论语言表达的结构。 

这就是为什么同构对象可以被视为结构相同的原因。 

### 通过态射进行传输

 态射允许信息移动。 

如果$f : A \to B$是同构，任意结构$A$可以运送到$B$。 传输结构被定义为$f$变得结构保持。 

对于二元运算$\ast$在$A$，定义一个操作$\star$在$B$经过$$
b_1 \star b_2 = f(f^{-1}(b_1) \ast f^{-1}(b_2)).
$$这使得$A$和$B$相当于结构化对象。 

传输在线性代数中很常见。 选择一个基给出同构$$
V \cong k^n.
$$这将抽象向量传输到坐标元组中。 然后可以进行计算$k^n$并解释回$V$。 

### 健忘的地图和添加的结构

 有些地图忘记了结构。 群可以被视为一个集合，忽略它的操作。 拓扑空间可以被视为一个集合，忽略它的开集。 

这种操作在分类语言中通常被称为健忘函子。 

| 结构化对象| 被遗忘的景色|
 | ----------------- | -------------------------------- |
 | 集团| 底层集合 |
 | 戒指| 基础附加组或集合|
 | 向量空间| 底层集合|
 | 拓扑空间| 底层集合|
 | 度量空间| 底层拓扑空间|

 忘记结构可以提供更多函数，但适用的定理却更少。 

基础集合之间的函数不必是更丰富结构的态射。 向量空间之间的映射可以是不是线性的集合函数。 

### 自同构和对称性

 自同构是对象与其自身的同构。$$
f : X \to X
$$自同构描述了对称性。 它们是结构保留转换，使对象保持相同的结构类型。 

示例：

 | 对象| 自同构 |
 | ----------------- | ---------------------------------------------------- |
 | 设置| 排列|
 | 集团| 群自同构|
 | 向量空间| 可逆线性映射 |
 | 图| 保留邻接性的顶点排列 |
 | 拓扑空间| 从空间到自身的同胚|

 对象的自同构通常形成一个群。 该组测量物体的对称性。 

例如，正方形的对称性在组合下形成群。 每个对称性都保留了正方形的几何结构。 

### 态射决定理论

 选择对象而不选择态射给出了一个不完整的理论。 

例如，假设对象是度量空间。 有几种可能的选择：

 | 态射 | 结果强调|
 | ---------------------------------- | ----------------------- |
 | 等距 | 精确的公制几何形状 |
 | 利普希茨地图| 定量失真|
 | 连续地图 | 拓扑 |
 | 均匀连续映射 | 结构均匀 |

 每个选择都会产生不同的同一性概念。 

对于等距，只有当距离完全保持不变时，两个空间才等效。 对于同胚，只有开集结构才重要。 线段和曲线在度量上可能不同，但在拓扑上是相同的。 

### 实际工作流程

 当研究结构时，总是问：

 | 问题 | 目的|
 | ------------------------ | | ---------------------------------- |
 | 对象是什么？ | 确定研究领域 |
 | 什么是态射？ | 识别允许的转换 |
 | 态射保留了什么？ | 识别结构|
 | 有哪些同构？ | 识别相同性 |
 | 什么是自同构？ | 识别对称性 |

 此工作流程可以防止一个常见错误：当理论需要保留结构时，将任何函数视为可接受。 

态射是映射的严格形式。 它将物体从一种环境转移到另一种环境，同时尊重使这些物体数学化的结构。
