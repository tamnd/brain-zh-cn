有效性和蕴涵是语义概念，用于比较所有结构中的公式，而不是在一个固定结构内。 满意度告诉我们一个公式在特定分配下的特定结构中是否为真，而有效性和蕴涵则询问是否在语言的每种可能的解释中都强制存在真理。 

### 有效性

 如果一个句子在其语言的每个结构中都成立，则该句子是有效的。 

### 定义 2.41（有效性）

 让$A$是一个句子。 我们这么说$A$如果满足以下条件则有效：$$
\mathcal{M} \models A
$$对于每个结构$\mathcal{M}$对于语言$A$。 

在这种情况下我们写：$$
\models A
$$有效性意味着事实的真实性$A$仅取决于其逻辑形式，而不取决于特定结构的任何特殊属性。 

### 示例 2.42

 句子：$$
\forall x \, (x = x)
$$是有效的，因为相等在每个结构中都是自反的。 

句子：$$
\forall x \, P(x)
$$是无效的，因为我们可以选择一个结构，其中谓词$P$并不适用于域的每个元素。 

例如，令域为：$$
M = \{0,1\}
$$并让：$$
P^{\mathcal{M}} = \{0\}
$$然后：$$
\mathcal{M} \not\models \forall x \, P(x)
$$因为$1$不在$P^{\mathcal{M}}$。 

### 反模型

 为了证明一个句子无效，给出一个该句子是错误的结构就足够了。 这种结构称为反模型。 

### 定义 2.43（反模型）

 让$A$是一个句子。 一个反模型$A$是一个结构体$\mathcal{M}$这样：$$
\mathcal{M} \not\models A
$$反模型很重要，因为它们提供了具体的证据，证明句子不能仅从逻辑得出。 

### 语义蕴涵

 蕴涵通过允许假设来概括有效性。 

### 定义 2.44（语义蕴涵）

 让$\Gamma$是一组句子，并且让$A$是同一种语言的句子。 

我们这么说$\Gamma$语义上蕴涵$A$如果每个模型的$\Gamma$也是一个模型$A$。 

在这种情况下我们写：$$
\Gamma \models A
$$等效地：$$
\Gamma \models A
$$意味着对于每个结构$\mathcal{M}$， 如果：$$
\mathcal{M} \models B
$$对于每一个$B \in \Gamma$， 然后：$$
\mathcal{M} \models A
$$因此$A$是一个语义结果$\Gamma$恰好在什么时候$A$在所有假设都成立的每个结构中都是正确的$\Gamma$是真的。 

### 示例 2.45

 让$\Gamma$包含两个句子：$$
\forall x \, \forall y \, (x < y \to ¬(y < x))
$$和：$$
\forall x \, \forall y \, \forall z \, ((x < y \land y < z) \to x < z)
$$第一句话说的是$<$是不对称的，第二句话说$<$是传递性的。 

然后：$$
\Gamma \models \forall x \, ¬(x < x)
$$为了看到这一点，让$\mathcal{M}$可以是任何模型$\Gamma$，并假设存在一个元素来矛盾$a$这样：$$
a < a
$$根据不对称性，由：$$
a < a
$$我们得到：$$
¬(a < a)
$$这是不可能的，所以没有这样的$a$存在，因此：$$
\mathcal{M} \models \forall x \, ¬(x < x)
$$自从$\mathcal{M}$是任意的，蕴涵成立。 

### 无假设的有效性

 有效性是蕴涵的一个特例。 

对于一句话$A$:$$
\models A
$$意思与以下内容相同：$$
\varnothing \models A
$$这表示满足空集中所有句子的每个结构也满足$A$。 Since every structure satisfies the empty set of assumptions, this is exactly the statement that$A$is true in every structure.

 ### 逻辑等价

 如果两个句子相互蕴含，那么它们在逻辑上是等价的。 

### Definition 2.46 (Logical Equivalence)

 让$A$和$B$是句子。 我们这么说$A$和$B$在逻辑上是等价的，如果：$$
A \models B
$$和：$$
B \models A
$$等效地：$$
\models A \leftrightarrow B
$$这意味着$A$和$B$在每个结构中都具有相同的真值。 

### 示例 2.47

 句子：$$
¬\forall x \, P(x)
$$和：$$
\exists x \, ¬P(x)
$$逻辑上是等价的。 

事实上，结构满足：$$
¬\forall x \, P(x)
$$正是当并非每个元素都满足时$P$，并且当某些元素无法满足时，这恰好成立$P$。 

因此：$$
\models ¬\forall x \, P(x) \leftrightarrow \exists x \, ¬P(x)
$$### 满意度

 如果一组句子至少有一个模型，则它是可满足的。 

### Definition 2.48 (Satisfiability)

 让$\Gamma$是一组句子。 我们这么说$\Gamma$如果存在结构则可满足$\mathcal{M}$这样：$$
\mathcal{M} \models \Gamma
$$如果不存在这样的结构，那么$\Gamma$是无法满足的。 

可满足性询问假设是否可以在某种结构中同时实现。 

### 示例 2.49

 套装：$$
\Gamma = \{\exists x \, P(x), \forall x \, ¬P(x)\}
$$是无法满足的。 

第一句话说至少有一个对象满足$P$，而第二句话说没有对象满足$P$。 

没有任何结构可以使两个句子同时为真。 

### 引理 2.50（蕴含性和不可满足性）

 让$\Gamma$是一组句子，并且让$A$是一个句子。 然后：$$
\Gamma \models A
$$当且仅当：$$
\Gamma \cup \{¬A\}
$$是无法满足的。 

证明

 首先假设：$$
\Gamma \models A
$$然后满足所有句子的每个结构$\Gamma$也满足$A$。 

因此没有一个结构能够满足所有句子$\Gamma$并且还满足$¬A$，因为满足$¬A$意味着$A$是假的。 

因此：$$
\Gamma \cup \{¬A\}
$$是无法满足的。 

相反，假设：$$
\Gamma \cup \{¬A\}
$$是无法满足的。 

让$\mathcal{M}$可以是这样的任何结构：$$
\mathcal{M} \models \Gamma
$$如果$\mathcal{M} \not\models A$， 然后：$$
\mathcal{M} \models ¬A
$$因此：$$
\mathcal{M} \models \Gamma \cup \{¬A\}
$$这与以下假设相矛盾$\Gamma \cup \{¬A\}$是无法满足的。 

因此每个型号的$\Gamma$满足$A$， 所以：$$
\Gamma \models A
$$### 示例 2.51

 我们展示：$$
\{p \to q, p\} \models q
$$使用不可满足性。 

足以证明：$$
\{p \to q, p, ¬q\}
$$是无法满足的。 

假设估值满足所有三个公式。 然后：$$
p
$$是真的，并且：$$
q
$$是假的。 

但如果$p$是真的并且$q$为假，则：$$
p \to q
$$是假的。 

这与以下假设相矛盾：$$
p \to q
$$是真的。 

因此，没有任何估值能够满足所有三个公式，因此：$$
\{p \to q, p\} \models q
$$### 自由变量蕴涵

 对于具有自由变量的公式，蕴涵是使用结构和赋值来定义的。 

让$\Gamma$是一组公式并让$A$是一个公式。 我们写：$$
\Gamma \models A
$$如果对于每个结构$\mathcal{M}$以及每一个作业$s$，每当：$$
\mathcal{M}, s \models B
$$对于每一个$B \in \Gamma$， 然后：$$
\mathcal{M}, s \models A
$$这个定义扩展了先前的句子定义，因为句子不依赖于赋值。 

### 示例 2.52

 蕴涵：$$
x < y, \quad y < z \models x < z
$$并不在所有具有任意二元关系的结构中成立$<$。 

如果包含传递性作为假设，则它变得有效：$$
\forall x \, \forall y \, \forall z \, ((x < y \land y < z) \to x < z), \quad x < y, \quad y < z \models x < z
$$这个例子表明，蕴涵取决于已包含的假设，并且关系符号没有固定的含义，除非公理限制它们的行为。
