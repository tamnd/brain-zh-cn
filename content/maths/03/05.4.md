当一阶理论承认满足其所有公理但与预期或标准解释不同的模型时，就会出现非标准模型，并且它们的存在自然地遵循紧性定理和 Lowenheim Skolem 定理。 

### 定义 5.22（标准模型）

 标准模型是一种旨在代表理论的通常解释的结构。 

例如，算术的标准模型是：$$
\mathbb{N} = \{0,1,2,\dots\}
$$与通常的操作和关系。 

非标准模型是同一理论的任何模型，但与标准模型不同构。 

### 通过紧凑性进行构建

 我们说明了非标准算术模型的构造。 

让$\mathcal{L}$成为算术语言，并通过添加新的常量符号来扩展它$c$。 

考虑一组句子：$$
\Gamma = \text{Th}(\mathbb{N}) \cup \{c > 0, c > 1, c > 2, \dots\}
$$这里$\text{Th}(\mathbb{N})$表示标准模型中所有真实的算术语句。 

每个有限子集$\Gamma$仅包含有限多个不等式：$$
c > 0, c > 1, \dots, c > n
$$这样的有限集可以满足$\mathbb{N}$通过解释$c$作为一个大于$n$。 

因此每个有限子集$\Gamma$是可以满足的。 

通过紧凑性，$\Gamma$有一个模型$\mathcal{M}$。 

在$\mathcal{M}$, 元素解释$c$大于每个标准自然数，因此$\mathcal{M}$不能同构于$\mathbb{N}$。 

这给出了一个非标准的算术模型。 

### 定义 5.23（非标准元素）

 在非标准算术模型中，如果某个元素大于与标准自然数对应的每个元素，则该元素称为非标准。 

从外部的角度来看，这些元素的行为就像无限多个。 

### 非标准模型的结构

 非标准算术模型具有丰富的内部结构。 

它们包含与标准自然数同构的初始段：$$
0,1,2,\dots
$$除了这个初始段之外，还有一些非标准元素可以无限期地扩展顺序。 

标准和非标准元素之间没有间隙，并且排序的行为类似于具有附加无限部分的离散线性顺序。 

### 示例 5.24（无限整数）

 在非标准模型中，可以考虑一个元素$c$这样：$$
c > n
$$对于每个标准自然数$n$。 

该元素的行为就像一个无限整数。 

还可以形成如下表达式：$$
c + 1, \quad c + 2, \quad 2c, \quad c^2
$$所有这些也是非标准元素，它们满足与普通自然数相同的代数定律。 

### 重新审视斯科勒姆悖论

 非标准模型有助于阐明斯科伦悖论。 

一个理论可以证明不可数集合的存在，但仍然有一个可数模型。 

在模型内部，某些集合和自然数之间不存在双射，因此这些集合在模型内被认为是不可数的。 

然而，从外部来看，整个模型是可数的。 

因此，可数性的概念取决于观察结构的角度。 

### 一阶属性的转移

 非标准模型的一个关键特征是，只要理论完整，它们就满足与标准模型完全相同的一阶句子。 

因此，任何在标准模型中为真的一阶陈述在每个非标准模型中也都为真，反之亦然。 

这意味着不能仅通过一阶属性将非标准模型与标准模型区分开来。 

### 示例 5.25（超出标准部分的感应失败）

 尽管归纳法在非标准模型中适用于所有一阶可定义属性，但存在不可定义的域子集，并且归纳法可能会失败。 

这表明一阶逻辑中的归纳仅适用于可定义的属性，不适用于任意子集。 

### 推论 5.26（非标准模型的存在）

 如果一阶理论有一个无限模型，那么它就有一个非标准模型。 

这是根据 Lowenheim Skolem 定理和紧致性得出的，因为人们可以构建不同大小的模型或使用标准模型中无法实现的附加元素来扩展理论。 

### 概念意义

 非标准模型表明，一阶逻辑不能唯一地表征许多数学结构，因为总是存在满足相同公理但在重要方面有所不同的替代模型。 

它们还提供了有用的数学工具，例如非标准分析，其中引入无穷小和无限元素以新的方式研究极限和连续性。 

因此，非标准模型的存在既是一种限制，也是新数学思想的源泉，这表明逻辑系统通常允许比最初预期更丰富的解释。
