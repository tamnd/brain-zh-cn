紧性定理是一阶逻辑的核心结构结果之一，它表达了理论的有限行为和无限行为之间的精确关系，表明全局可满足性完全由有限片段控制。 

### 定义 5.1（结构和满意度）

 让$\mathcal{L}$是一阶语言，并且让$\mathcal{M}$成为一个$\mathcal{L}$-结构。 

对于一句话$A$在$\mathcal{L}$，我们写：$$
\mathcal{M} \models A
$$意思是$A$是真的$\mathcal{M}$。 

如果$\Gamma$是一组句子，我们写：$$
\mathcal{M} \models \Gamma
$$意思是：$$
\mathcal{M} \models A
$$对于每一个$A \in \Gamma$。 

因此$\mathcal{M}$是一个模型$\Gamma$恰好当它满足中的每个句子时$\Gamma$。 

### 定义 5.2（可满足性和一致性）

 一组句子$\Gamma$是可满足的，如果存在$\mathcal{L}$-结构$\mathcal{M}$这样：$$
\mathcal{M} \models \Gamma
$$一组句子$\Gamma$如果不存在这样的结构，则不可满足。 

一组句子$\Gamma$如果满足以下条件，则语法一致：$$
\Gamma \nvdash \bot
$$在一阶逻辑中，完整性定理通过说一组句子在语法上一致时是可满足的来连接这两个概念。 

### 定义 5.3（有限可满足性）

 一组句子$\Gamma$如果每个有限子集是有限可满足的：$$
\Gamma_0 \subseteq \Gamma
$$是可以满足的。 

这意味着每当一个人只从其中选择有限多个句子时$\Gamma$，存在一种结构，其中所有选定的句子同时为真。 

### 定理 5.4（紧性定理）

 让$\Gamma$是一组一阶句子。 

如果$\Gamma$是有限可满足的，那么$\Gamma$是可以满足的。 

等价地，如果每个有限子集$\Gamma$有一个模型，那么就有一个单一的结构，它是所有句子的模型$\Gamma$。 

该定理表明，只有当这些需求的某些有限部分已经无法建立模型时，无限组一阶需求才可能无法建立模型。 

假设$\Gamma$是有限可满足的，并假设矛盾$\Gamma$是不能满足的。 

然后：$$
\Gamma \models \bot
$$根据一阶逻辑的完备性定理：$$
\Gamma \vdash \bot
$$形式证明是有限的，因此矛盾的推导只能使用有限的多个假设$\Gamma$。 因此存在一个有限子集：$$
\Gamma_0 \subseteq \Gamma
$$这样：$$
\Gamma_0 \vdash \bot
$$通过健全性，或者等效地通过相反方向应用的完整性，这个有限子集是不可满足的：$$
\Gamma_0 \models \bot
$$这与以下假设相矛盾：$\Gamma$是可以满足的。 所以$\Gamma$一定是可以满足的。 

### 定理 5.5（结果形式的紧致性）

 让$\Gamma$是一组句子，并且让$A$是一个句子。 

如果：$$
\Gamma \models A
$$则存在有限子集：$$
\Gamma_0 \subseteq \Gamma
$$这样：$$
\Gamma_0 \models A
$$认为：$$
\Gamma \models A
$$那么就没有模型了：$$
\Gamma \cup \{¬A\}
$$因此：$$
\Gamma \cup \{¬A\}
$$是无法满足的。 

根据紧致性，某些有限子集已经不​​可满足。 因此存在有限集：$$
\Gamma_0 \subseteq \Gamma
$$这样：$$
\Gamma_0 \cup \{¬A\}
$$是无法满足的。 

这意味着每个型号$\Gamma_0$必须是一个模型$A$，因此：$$
\Gamma_0 \models A
$$### 示例 5.6（有限条件下的无限模型）

 让语言包含一个一元谓词符号$P$。 

对于每个自然数$n \geq 1$， 让$A_n$是句子：$$
\exists x_1 \cdots \exists x_n
\left(
\bigwedge_{1 \leq i < j \leq n} x_i \neq x_j
\land
\bigwedge_{1 \leq i \leq n} P(x_i)
\right)
$$这句话$A_n$说至少有$n$不同的元素满足$P$。 

让：$$
\Gamma = \{A_n : n \geq 1\}
$$每个有限子集$\Gamma$仅包含有限多个句子：$$
A_{n_1}, A_{n_2}, \dots, A_{n_k}
$$让：$$
N = \max(n_1,n_2,\dots,n_k)
$$一个结构与$N$元素齐全，都很满意$P$，是所有这些有限多个句子的模型。 

因此每个有限子集$\Gamma$是可以满足的。 

通过紧凑性，$\Gamma$本身是可以满足的。 在任何模型中$\Gamma$，对于每个自然数$n$至少有$n$不同的元素满足$P$。 因此，元素集合满足$P$是无限的。 

### 例 5.7（非标准算术模型）

 让$\mathcal{L}$成为常用的算术语言，并通过添加新的常量符号来扩展它$c$。 

让$\Gamma$由算术公理和句子组成：$$
c > 0,\quad c > 1,\quad c > 2,\quad \dots
$$每个有限子集$\Gamma$只提到有限多个不等式：$$
c > 0,\quad c > 1,\quad \dots,\quad c > n
$$这个有限集可以通过解释在标准自然数中得到满足$c$作为一个大于$n$。 

因此，每个有限子集$\Gamma$是可以满足的。 

通过紧凑性，整套$\Gamma$有一个模型。 在该模型中，元素表示为$c$大于每个标准自然数。 

这样的模型不能成为标准模型$\mathbb{N}$，因为没有一个普通自然数大于所有普通自然数。 因此，一阶算术具有非标准模型。 

### 推论 5.8（有限性不是一阶公理化的）

 不存在其模型完全是有限结构的一阶句子集合。 

假设，为了矛盾，$\Gamma$是一组一阶句子，其结构满足$\Gamma$恰好在它是有限的时候。 

对于每个$n \geq 1$， 让$B_n$是表示至少有$n$不同的元素：$$
\exists x_1 \cdots \exists x_n
\bigwedge_{1 \leq i < j \leq n} x_i \neq x_j
$$考虑：$$
\Gamma \cup \{B_n : n \geq 1\}
$$该理论的每个有限子集只需要有限多个不同的元素，因此它有一个满足的有限模型$\Gamma$。 

通过紧致性，整个理论有了一个模型。 

该模型满足每个$B_n$，所以它是无限的。 也满足$\Gamma$，因此通过假设的属性$\Gamma$它必须是有限的。 

这是不可能的。 因此，有限性不能用任何一阶理论来表征。 

### 推论 5.9（任意大的有限模型给出无限模型）

 让$\Gamma$是一阶理论。 

如果$\Gamma$具有任意大尺寸的有限模型，则$\Gamma$有一个无限的模型。 

对于每个$n \geq 1$， 让$B_n$是表示至少有$n$不同的元素：$$
\exists x_1 \cdots \exists x_n
\bigwedge_{1 \leq i < j \leq n} x_i \neq x_j
$$考虑：$$
\Gamma \cup \{B_n : n \geq 1\}
$$该理论的每个有限子集仅需要模型大小的下限。 自从$\Gamma$具有任意大尺寸的有限模型，每个这样的有限子集都是可满足的。 

根据紧性，整个理论是可满足的。 

整个理论的任何模型至少有$n$每个的独特元素$n$，因此它是无限的。 

### 概念意义

 紧致性表示一阶可满足性由有限信息控制。 

这是一个强大的模型构建原理。 要构建具有无限多个要求的模型，只需证明这些要求的每个有限选择都可以得到满足。 

同时，紧致性揭示了一阶逻辑的局限性。 需要在有限和无限行为之间进行真正全局区分的属性通常不能用一阶句子来表达。
