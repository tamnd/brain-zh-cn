量词是允许一阶逻辑同时表达关于许多对象的陈述的符号，它们是将一阶逻辑与命题逻辑分开的主要特征。 

### 通用量词和存在量词

 有两个基本量词：$$
\forall
$$和：$$
\exists
$$全称量词$\forall$读作“for all”，它用于说明一个公式适用于所考虑的域中的每个对象。 

存在量词$\exists$读作“存在”，用于说明某个公式适用于所考虑的域中的至少一个对象。 

例如：$$
\forall x \, P(x)
$$说每个对象都有属性$P$， 尽管：$$
\exists x \, P(x)
$$表示至少一个对象具有属性$P$。 

### 定义 2.9（范围）

 量词的范围是量词所应用的公式的一部分。 

式中：$$
\forall x \, (P(x) \to Q(x))
$$的范围$\forall x$是：$$
P(x) \to Q(x)
$$因此，两次出现$x$该公式中的 是由量词控制的。 

相反，在公式中：$$
(\forall x \, P(x)) \to Q(x)
$$的范围$\forall x$只是：$$
P(x)
$$的发生$x$在$Q(x)$超出了量词的范围，因此是自由的。 

### 定义 2.10（自由和绑定出现）

 如果公式中变量的出现位于同一变量的量词范围内，则该变量的出现是绑定的。 

如果变量未绑定，则它的出现是自由的。 

例如，在：$$
\forall x \, P(x, y)
$$的发生$x$是有界的，而发生$y$是免费的。 

在：$$
\exists y \, (P(x, y) \land Q(y))
$$的出现$y$是绑定的，而发生$x$是免费的。 

这种区别很重要，因为具有自由变量的公式的真实性取决于对这些变量的赋值。 

### 示例 2.11

 考虑：$$
\forall x \, (x < y)
$$变量$x$是有约束力的，因为它发生在$\forall x$。 

变量$y$是自由的，因为没有量词约束它。 

因此，该公式表示每个对象都小于分配给的对象$y$，其真实性取决于哪个对象$y$表示。 

### 定义 2.12（句子）

 句子是一个没有自由变量的公式。 

例如：$$
\forall x \, (x = x)
$$是一个句子。 

公式：$$
x = x
$$不是一个句子，因为变量$x$自由发生。 

句子是可以在结构中分配真值的公式，而不需要对自由变量进行额外的分配。 

### 量词的顺序

 量词的顺序很重要。 

公式：$$
\forall x \, \exists y \, R(x, y)
$$说对于每个对象$x$，有一些物体$y$相关于$x$。 

公式：$$
\exists y \, \forall x \, R(x, y)
$$说有一个对象$y$与每个对象相关$x$。 

这两个公式通常具有不同的含义。 

例如，如果$R(x, y)$方法：$$
x < y
$$然后：$$
\forall x \, \exists y \, (x < y)
$$说每个物体都有一个更大的物体。 

但：$$
\exists y \, \forall x \, (x < y)
$$说有一个物体比所有物体都大。 

在自然数中，第一个陈述为真，而第二个陈述为假。 

### 定义 2.13（限制量词）

 受限制的全称量词：$$
\forall x \in S \, A(x)
$$是以下缩写：$$
\forall x \, (x \in S \to A(x))
$$受限存在量词：$$
\exists x \in S \, A(x)
$$是以下缩写：$$
\exists x \, (x \in S \land A(x))
$$这些翻译中蕴涵和连词之间的区别很重要。 

对于通用语句，外部对象$S$应该不重要，所以使用暗示。 

对于存在性陈述，对象必须都属于$S$并满足$A(x)$，因此使用连词。 

### 示例 2.14

 声明：$$
\forall x \in \mathbb{N} \, (x + 0 = x)
$$方法：$$
\forall x \, (x \in \mathbb{N} \to x + 0 = x)
$$声明：$$
\exists x \in \mathbb{N} \, (x^2 = 4)
$$方法：$$
\exists x \, (x \in \mathbb{N} \land x^2 = 4)
$$这些不是一阶逻辑的新原始形式，而是方便的缩写。 

### 引理 2.15（否定量词）

 对于每个公式$A(x)$，以下等价成立：$$
¬\forall x \, A(x) \equiv \exists x \, ¬A(x)
$$和：$$
¬\exists x \, A(x) \equiv \forall x \, ¬A(x)
$$证明

 公式：$$
¬\forall x \, A(x)
$$说事实并非如此$A(x)$对于每个对象都成立$x$。 

这意味着至少有一个对象$x$为此$A(x)$失败。 

因此：$$
¬\forall x \, A(x) \equiv \exists x \, ¬A(x)
$$同样，公式：$$
¬\exists x \, A(x)
$$说没有对象$x$为此$A(x)$成立。 

这意味着每个对象$x$未能满足$A(x)$。 

因此：$$
¬\exists x \, A(x) \equiv \forall x \, ¬A(x)
$$### 示例 2.16

 否定：$$
\forall x \, P(x)
$$是：$$
\exists x \, ¬P(x)
$$这说明并非每个对象都具有属性$P$恰好当至少一个对象不具有属性时$P$。 

否定：$$
\exists x \, P(x)
$$是：$$
\forall x \, ¬P(x)
$$这表示不存在具有属性的对象$P$正是在每个对象都无法拥有属性的时候$P$。 

### 定义 2.17（变量捕获）

 当替换导致自由变量受到量词约束时，就会发生变量捕获。 

例如，考虑：$$
\forall x \, P(x, y)
$$变量$y$是免费的。 

如果我们更换$y$经过$x$在不首先更改绑定变量的情况下，我们得到：$$
\forall x \, P(x, x)
$$新插入的$x$现在属于范围内$\forall x$，因此它已成为绑定。 

这会改变变量的语法角色，并可能改变公式的含义。 

### 引理 2.18（重命名绑定变量）

 只要没有捕获自由变量，绑定变量就可以被重命名。 

例如：$$
\forall x \, P(x, y)
$$可以重命名为：$$
\forall z \, P(z, y)
$$前提是$z$尚未以会造成混乱或捕获的方式发生。 

证明

 绑定变量仅充当域中对象的占位符。 

更改绑定变量的名称不会更改量化的结构，只要更改在整个量词范围内一致进行并且不捕获任何自由变量即可。 

因此公式：$$
\forall x \, P(x, y)
$$和：$$
\forall z \, P(z, y)
$$具有相同的逻辑内容。 

关于捕获的限制是必要的，因为替换$x$通过公式中已经自由的变量可以改变哪些出现被绑定以及哪些出现保持自由。
